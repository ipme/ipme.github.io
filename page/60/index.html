读书破万卷，下笔如有神；读万卷书，行万里路

发表于 2018-06-14 | 分类于 读书名言

1、我一生的嗜好，除了革命之外，就是读书。我一天不读书，就不能够生活。读书破万卷，下笔如有神。读万卷书，行万里路。读书之法无他，惟是笃志虚心，反复详玩，为有功耳。
2、读书无嗜好，就能尽其多。不先泛览群书，则会无所适从或失之偏好，广然后深，博然后专。读书之法，在循序渐进，熟读而精思。
3、读书务在循序渐进；一书已熟，方读一书，勿得卤莽躐等，虽多无益。读书是学习，摘抄是整理，写作是创造。
4、看书不能信仰而无思考，要大胆地提出问题，勤于摘录资料，分析资料，找出其中的相互关系，是做学问的一种方法。
5、书犹药也，善读之可以医愚。读书破万卷，胸中无适主，便如暴富儿，颇为用钱苦。知古不知今，谓之落沉。知今不知古，谓之盲瞽。

6、读书名言，举一纲而万目张，解一卷而众篇明。知识，主要是靠主动“抓”出来的，不是靠“教”出来的。学而不厌，诲人不倦。
7、欲速是读书第一大病，功夫中在绵密不间断，不在不速也。不积跬步，无以至千里；不积小流，无以成江海。学而时习之，不亦悦乎！
8、倘能生存，我当然仍要学习。热爱书吧，这是知识的泉源！我读书越多，书籍就使我和世界越接近，生活对我也变得越加光明和有意义。
9、书籍是在时代的波涛中航行的思想之船，它小心翼翼地把珍贵的货物运送给一代又一代。读书对于智慧，也象体操对于身体一样。
10、当我们第一遍读一本好书的时候，我们仿佛觉得找到了一个朋友；当我们再一次读这本书的时候，仿佛又和老朋友重逢。理想的书籍，是智慧的钥匙。

你可听懂风传递的情话，爱你永远！

发表于 2018-06-14 | 分类于 表白的话

1、你是我今生不变的思念，你的美铭刻在我的心尖，难忘你醉人如花的笑靥，眼眸闪烁着爱的诗篇，让我痴迷而不悔的眷恋。你可听懂风传递的情话，爱你永远！
2、借一缕阳光为笔，蓝天为宣，写下爱的诗篇，与你共享温馨浪漫，牵着时光的手，织一段幸福美满人生路，与你共度一世美好无限。此生爱你不变！
3、静守温馨小窝，凝视美好未来，把纷纷扰扰关在窗外，在甜蜜缱绻中悠然徘徊，用深情关爱烹煮生活的欢快，用携手同心演绎人生的精彩。此生爱你不变！

4、表白的话，潮水滚滚，不停地向上疯涨，那是我对你的思念，缓缓溢出心底，源源不断地向你流去，潮涨潮落，都是我对你最深的情意。
5、借问随风叶，飘落欲为谁？空留昨日恨，青翠付流水。世间真情难觅，往事依依难寻。思念满腔藏不住，祝福一声只为君；惜君幸福莫相问，做得一生逍遥人！
6、你的微笑，似春风一缕，挑逗着我的孤寂；想你的甜蜜，飘飘移移散步风雨里；梦中你的身影时时刻刻萦绕在心底；好像大喊告诉你：我真的很爱你！

不变的是思念，爱你今生到永远

发表于 2018-06-14 | 分类于 表白的话

1、思念总是在心底鼓噪，后悔年轻的心沉迷骄傲，把分手当成一种玩笑，现在的我已经知道，只有真爱值得炫耀，亲爱的你对我非常重要，快快回来我身边报到！
2、茫茫人海，凭你的名字导航；凄凄寒夜，拥你的名字取暖；漫漫人生，携你的名字同游。想你，寻不到你；追你，赶不上你；原来，你在我心里。
3、玉树临风踏古城，携手相拥游乘兴，亭台凭栏寻旧梦，古今天地尽英雄。想你，就在你我闲庭信步欣赏惬意的风景，如每一次快门的按动存储了你永恒的温情。

4、谁弹奏着思念的琴弦，把爱恋藏在我心里面；谁刻画了美好的明天，把遇见变成最美好的情缘。看沧海桑田，不变的是思念，爱你今生到永远。
5、表白的话，孤枕夜晚难成眠，读书三更才入睡；心中牵挂远方妻，思念伴随到天明；早醒想起近日事，自编短信记心情。
6、情深无言以表，爱浓有心拥抱。遇一个人不容易，牵一双手不简单。是缘分让我们相遇，是真情让我们相牵，爱你，只盼与你共走这一路，简单幸福就好！

1、这一生，不管我往哪个方向去，都只为与你相遇；不管我在哪个季节留，都只为与你相守；不管我用哪一只手牵，都只为与你相伴。爱你，情不绝！
2、再香的花，也浓不过爱的芬芳；再大的风，也吹不散爱的藤蔓；再远的路，也阻不断爱的思念；再美的景，也比不了爱的画卷。爱你不变，从现在，到未来！
3、我没有拖延症，快递一拿到我就会拆，闹钟一响我就会按，零食一开我就会吃，一想到你我就会笑，你一离开我就会闹，你一高兴我就手舞足蹈，爱你就要这样到老！
4、我要缠你一辈子，千难险阻不逃退。不管你说的对不对，一切就当你全对。不管现实无情的社会，牵你双手向前走，愿为你心碎为你累，为你憔悴为你醉，为你守候一生不后悔！

5、爱你，是心灵的呼喊；爱你，是直觉的感应；爱你，是上天的安排；爱你，是幸福的拥抱；爱你，是无悔的相守；爱你，是最美的誓言：亲爱的，我爱你！
6、表白的话，风儿轻轻吻着蓝天，蝶儿双双舞动缠绵。朝朝暮暮的思念，两颗心紧紧相连，不管天地相隔远，感受痴情的眷恋。爱你的情不断，爱你一万年！
7、我熬呀熬，终于熬到周末来。我盼呀盼，终于可以和你再相见。先亲亲，再抱抱，花前月下情谊深。爱也深，情也真，一分一秒不分离。周末有你就够了。
