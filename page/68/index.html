你要懂得珍惜，才能够懂得爱情

发表于 2018-06-15 | 分类于 爱情说说

1、由爱故生忧，由爱故生怖，若离于爱者，无忧亦无怖。爱情和做事一样没有失败只有放弃。既不回头，何必不忘。既然无缘，何须誓言。
2、这一切都是一种心境。心若无物就可以一花一世界，一草一天堂。参透这些，一花一草便是整个世界，而整个世界也便空如花草！
3、爱情说说，承诺只怕锦书难托，一念执着，换三生迷离烟火；一生多情愁，来回多紧锁，燃烧的福祸，忘记你我。
4、不要为了寂寞去恋爱，时间是个魔鬼，天长日久，如果你是个多情的人，即使不爱对方，到时候也会产生感情，到最后你怎么办？

5、你来过一下子，我却要想念一辈子。不要认为他爱你便不会离开你。在你的一再伤害下，他也是会离开，那因为你不懂得珍惜，不值得他去爱。你要懂得珍惜，才能够懂得爱情。
6、有时候，爱情需要在对的时间和对的地方才能遇见对的人，有相对的开始，与相对的人在一起，一辈子才有最大的幸福。

和有情人，做快乐事，别问是劫是缘

发表于 2018-06-15 | 分类于 爱情说说

1、如果我们能看清“苦”的真相，看出苦由何处而生，由何处消逝，那么便容易做到不令苦的形成，也就能够控制苦。
2、爱情说说，和一个生活习惯有很多差异的人恋爱不要紧，结婚要慎重，想想你是否可以长久忍受彼此的不同。
3、爱本是恨的来处。胡汉不归路，一个输，一个哭，宁愿你恨的糊涂，中了爱的迷毒，一面满足一面残酷。
4、问佛：如果遇到了可以爱的人，却又怕不能把握怎么办？佛曰∶留人间多少爱，迎浮世千重变；和有情人，做快乐事，别问是劫是缘。

5、感情的事基本上没有谁对谁错，他（她）要离开你，总是你有什么地方不能令他满足，回头想想过去在一起的日子，总是美好的。当然，卑劣的感情骗子也有，他们的花言巧语完全是为了骗取对方和自己上床，这样的人还是极少数。
6、彼岸花，开一千年，落一千年，花叶永不相见。情不为因果，缘注定生死。佛曰：人生有七苦，生，老，病，死怨憎恨，爱别离，求不得。

爱情不是做买卖，失意了就要倾家荡产

发表于 2018-06-15 | 分类于 爱情说说

1、恋爱不是慈善事业，不能随便施舍的。感情是没有公式，没有原则，没有道理可循的。佛曰：缘为冰，我将冰拥在怀中；冰化了，我才发现缘没了。
2、缘来天注定，缘去人自夺。种如是因，收如是果，一切唯心造。笑着面对，不去埋怨。悠然，随心，随性，随缘。注定让一生改变的，只在百年后，那一朵花开的时间。
3、你来过。一下子，我却要想念一辈子。不要认为他爱你便不会离开你。在你的一再伤害下，他也是会离开，那因为你不懂得珍惜，不值得他去爱。你要懂得珍惜，才能够懂得爱情。

4、爱情不是做买卖，失意了就要倾家荡产；爱情也不是下赌注，输掉了就变成穷光蛋。爱情是两颗心的碰橦和相许，如同和谐的乐谱的音符，谱写人间的圣洁，相随一生。
5、爱情说说，就在那一夜，我忘却了所有，抛却了信仰，舍弃了轮回，只为，那曾在佛前哭泣的玫瑰，早已失去旧日的光泽。
6、凡夫误用经典的章句，只是把它拿来说说，而无身体力行。他只是得到章句的表皮，没有掌握到真谛。缘为冰。我将冰拥在怀中，冰化了，我才发现缘没了。

菩提树下的轮回，是几世遗留的姻缘

发表于 2018-06-15 | 分类于 爱情说说

1、霓虹灯下的幻影，是因为思念的脚步轻轻踏出红尘万丈；千丝万缕的情意，直到遇见了你，才划过一道忧郁的泪痕。
2、在找到智慧之前得先找到快乐，找到快乐之前得先找到自己。人要寻求最高的觉悟，得先明白自己和自己与时空的关系。
3、短短今生一面遇，前世多少香火缘。牵手是有一种缘，回眸是一种缘；生命是有一种缘，爱情是一种缘；假如还有来生，来生也是一种缘。组成人生的，就是那无数个缘；让你去结，去解。惜缘，随缘，才能以佛心看缘起缘落。

4、佛陀说：“净化自己完全在于自己，别人无法代替。”但是心要怎么净化呢？诸恶莫作，众善奉行，自净其意，是诸佛教。
5、爱情是一种遇见…遇见是一种缘分…佛又说五百年的修行才换来今生的擦肩而过…人生在世，如身处荆棘之中！心不动，人不妄动，不动则不伤。
6、爱情说说，菩提树下的轮回，是几世遗留的姻缘，等你来度我；石桥下等你来过，岁月的蹉跎让我犯了错，回首前尘，只愿涅盘成佛。
