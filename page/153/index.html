人人都有歇不下的累，人人都有道不尽的苦

发表于 2018-06-14 | 分类于 心情不好

人活着，哪有容易的。人人都有歇不下的累；人人都有道不尽的苦；人人都有流不出的泪。再累再苦又怎样，还不得坚强面对；再难再烦又如何，日子也得照样继续。
人活着，挣钱不易。天亮始天黑归，一天的工作也不敢耽搁；风里来雨里去，一年到头也不敢多休息。不是有多稀罕钱，少了钱，缺了钱，拿什么孝敬父母，拿什么养育儿女！
人活着，相处不易。谁虚伪，谁真诚，假假真真太难分；谁重情，谁重利，辛辛苦苦太难寻。交对了人，一生受益；交错了人，一生受损。处好了友，让你暖心；处坏了友，让你寒心。
人活着，做人不易。男人有男人的难，女人有女人的酸，各人都有各人的烦。有钱了，说你炫；没钱了，说你烂。老实了，说你傻；精明了，说你算。人活着，哪有容易的。

自己吃的苦，自己知道；自己受的累，自己明白；自己走的路，自己清楚。人活着，哪有容易的。酸甜苦辣咸，是一辈子的滋味；喜怒哀乐愁，是一辈子的体会！
在人之上，要把别人当人；在人之下，要把自己当人。人再厉害，也不会风光永远，人再窝囊，也有灵光一闪。
不要看轻自己，你的特长，别人没有；也不要盲目的去仰慕别人，人人有所短，人人有所长。
人生无非是笑笑人家，再被人家笑笑而已。无论遇到了什么，请坚持对自己笑一笑，你会发现：生活，原来没什么大不了！
命运给你洗了一次牌，但玩牌的还是你自己。牌好牌差，都要好好打；生活顺不顺，都要好好过！
心情不好，要随意指责别人的生活，多一份平和，多一点温暖，生活才有阳光，才会快乐。
当你伸出手指责别人时，你另外四个指头正指着自己。你把别人想得太复杂，是因为你也不简单。心简单，世界就简单，幸福才会生长。

忍一忍风平浪静，退一步海阔天空

发表于 2018-06-14 | 分类于 心情不好

人活在世上不可能事事尽如人意，遇到困难和烦心的事情，听听别人的奉劝，也有好处。別担心，一切都会好的上天对每个人都是公平的，它在关上一扇门的同时，必定会打开一扇窗；无论多么糟糕的东西，世界都为其预留了位置。
相信雨点不会仅仅落在你一个人的屋顶之上，相信你自己，大千世界总有属于你的角落。拥有积极乐观的态度，是解决和战胜任何困难的第一步。
別害怕，天是不会塌的！害怕时，把心思放在必须做的事情上，如果曾经彻底准备，便不会害怕；去做你害怕的事，害怕自然就会消失。
心情不好，人生就像“碰钉子”，碰一回钉子，长一分见识，增一分阅历；天塌下来，还有大汉撑着，体验一次吧！人的整个生命其实就是一场体验。

別后悔，谁都会做错事。世界上没有永远不犯错误的人，做错事，千万别后悔。后悔是一种耗费精神的情绪，后悔是比损失更大的损失，比错误更大的错误。
別失望，机会还会有的。人之所以能，是相信能。一个人最大的破产是绝望，最大的资产是希望。生活很多时候是会有很多不公平的，不要抱怨，人生充满了尝试与错误，一次失败不代表你就出局了。
別放弃，坚持就有希望。人生最遗憾的，莫过于轻易地放弃了不该放弃的。你可以不拥有任何东西，除了对生活的激情和对未来的希望。人生只有一条路不能选择，那就是放弃的路；只有一条路不能拒绝，那就是成长的路。
別生气，学会宽宏大量。生活中总会有伤害你的人，你千万别生气。生气，是拿别人的错误来惩罚自己，忍一忍风平浪静，退一步海阔天空。好脾气是一个人在社交中所能穿着的最佳服饰，宽容是人与人相互理解和信任的桥梁，学着换位思考。

人活着，多装些阳光在心里，路途才会少些阴霾

发表于 2018-06-14 | 分类于 心情不好

简单是福，人生之苦，苦于繁忙。忙财富、忙名利、忙着争抢，忙于计较得失荣辱。争来抢去终是空。简单的人，勤劳节俭，一切随缘而安。
我们曾渴望命运的波澜，到最后才发现，人生最曼妙的风景，竟是内心的淡定与从容；我们曾期盼外界的认可，到最后才知道，世界是自己的，与他人毫无关系。
心情不好，我们曾计较付出的回报，到最后才懂得，一切得到终将失去的，只能空留一抹浮名。
凡事太计较，全都要搞清，岂不是活得太累了。有些话，听完就随风飘走，既是垃圾，何必去脏自己的耳。

有些人，微微一笑不去计较，既是无品，何必脏自己的心，人活着，多装些阳光在心里，路途才会少些阴霾。
越来越明白：活得简单，才能活得自由，心简单，活着就简单；心自由，活着才自由。用一颗平常心，平凡的活着。从容达观一些，轻松自在一些，豁达随意一些。
心净才能无染，无染才能舒心。淡淡的，守住做人本色。淡泊宁静无需高仰，生活还需清凉一品。与这尘世，甘苦担当，再包含一份拈花微笑的默契。

自然界没有风风雨雨，大地就不会春华秋实

发表于 2018-06-14 | 分类于 座右铭

1、压力在事业成功的道路上，你是无知者颓丧的前奏，更是有志者奋进的序曲。彩云飘在空中，自然得意洋洋，但最多只能换取几声赞美；唯有化作细雨，才能给世界创造芳菲。
2、马行软地易失蹄，人贪安逸易失志。种子牢记着雨滴献身的叮嘱，增强了冒尖的勇气。岸边的奇花异草，是拘留不住奔腾向前的江水的。
3、座右铭，克服困难，勇敢者自有千方百计，怯懦者只感到万般无奈。外在压力增加时，就应增强内在的动力。经受了火的洗礼泥巴也会有坚强的体魄。
4、失败缘于忽视细处，成功始于重视小事。没有一种不通过蔑视忍受和奋斗就可以征服的命运。千万人的失败，都有是失败在做事不彻底，往往做到离成功尚差一步就终止不做了。
5、痛苦是性格的催化剂，它使强者更强，弱者更弱，仁者更仁，暴者更暴，智者更智，愚者更愚。如果我们做与不做都会有人笑，如果做不好与做得好还会有人笑，那么我们索性就做得更好，来给人笑吧！

6、这个世界并不是掌握在那些嘲笑者的手中，而恰恰掌握在能够经受得住嘲笑与批评不断往前走的人手中。成功需要成本，时间也是一种成本，对时间的珍惜就是对成本的节约。
7、行动是治愈恐惧的良药，而犹豫拖延将不断滋养恐惧。投资知识是明智的，投资网络中的知识就更加明智。忍别人所不能忍的痛，吃别人所不能吃的苦，方能收获别人所得不到的收获。
8、生命之灯因热情而点燃，生命之舟因拼搏而前行。拥有梦想只是一种智力，实现梦想才是一种能力。种子牢记着雨滴献身的叮嘱，增强了冒尖的气。
9、自然界没有风风雨雨，大地就不会春华秋实。只会幻想而不行动的人，永远也体会不到收获果实时的喜悦。希望是生命的源泉，失去它生命就会枯萎。
10、只有品味了痛苦，才能珍视曾经忽略的快乐；只有领略了平凡，才会收藏当初丢弃的幸福。每个人的一生都有许多梦想，但如果其中一个不断搅扰着你，剩下的就仅仅是行动了。
