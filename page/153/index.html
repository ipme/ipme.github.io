母爱似针，父爱似线，一针一线把亲情线连

发表于 2018-06-15 | 分类于 短句子

“感恩的心，感谢有你，伴我一生，让我有勇气做我自己……。”每当听到这首歌时，我就会想起自己的父母。
正如诗歌中所说“父爱如山，无论有多大困难总是依靠的屏障：父爱如伞为你遮风挡雨：父爱如路伴你走完人生”。
我的爸爸是一个不善言辞的人，当我刚上学的时候，由于各种原因，我们那里的学校被合并了，我们那里的孩子都需要去镇上的学校住宿或寄宿亲戚家中。
爸爸为了让我有更好的学习环境，决定让妈妈陪我到大雁来上学，照顾我的生活和学习，自己却留在家里上班。
爸爸的工作特殊他是农场的一名工人，每天都是早出晚归，爸爸一个人在默默承受许许多多的困难，可是爸爸从没有怨言。

当他看到我取得的一张纸奖状时，他总是语重心长地说：“只要你好，爸爸就开心，”朴实的话语中包含着爸爸多少无私的爱呀。
母爱是船载着我们日益走向成熟，小时候妈妈买回荔枝，妈妈只吃一点尝尝，总是说她不爱吃，留给我吃。
父母的爱，我很纳闷，这么好吃的荔枝妈妈还不喜欢，我索性就吃个精光，我索性就吃个精光。
后来我才发现我爱吃的东西妈妈都说她不爱吃，长大了我才明白，那是妈妈不舍得吃想留给我多吃点。
我生病时，父母对我悉心照料：我犯错时，父母对我的尊尊教诲：我失落时，父母对我的鼓励……
从小到大，父母为我做的事数也数不清，点点滴滴都透露着父母对我的爱，父母的爱像一杯浓茶，需要我们细细品味，“母爱似针，父爱似线，一针针一线线把亲情线连”。
是啊，无论我们长多大无论我们走多远，父母的养育之恩我们都要牢牢记在心间。

父母的爱是汪洋中的航标，指明了新生的希望

发表于 2018-06-15 | 分类于 短句子

父母的爱是冬日里的一道阳光，驱散了寒冷的寒霜；父母的爱是久旱后的一场甘露，湿润了龟裂的心田；父母的爱是汪洋中的航标，指明了新生的希望。
脑海中，父母对我的爱数不胜数，但却又这么一件小事，让我印刻在心……
我还记得曾有那么一回，我生病了。那一天早上，我躺在床上，病魔使我无法振作起来去上学，于是我有气无力地对爸爸妈妈说：“爸爸妈妈，我头好晕啊！”。
爸爸妈妈见到此状，都焦急了许多。后面爸爸便凑过来关心地说：“那你先躺着，今天不用去学校了。”说完，妈妈走过来用手触摸着我的额头，之后便闪电般地缩了回来。
“你的头好烫啊！看来烧的不轻，我们上医院去。”我那时头昏脑涨，四肢发软，根本没力气站起来。在朦胧之中感觉一双强劲有力的手把我抱起，放在那冰冷的车座上，汽车的噪声使我心烦意乱。

到了医院，护士给我量了体温，39度。护士叫我躺在病床上吊盐水。我躺在床上，看着四周安安静静，便生起了一丝睡意，不一会儿我闭眼进入了梦乡。
光阴似箭，时间如梭。转眼间已经到了中午，我起来一看，我的爸爸正在趴在床上呢！爸爸好像有感应似的，一下就起床了。
后面妈妈端着盒饭进来了，妈妈神神秘秘地凑过来对我说：“你爸爸一直守在你身旁呢！”我听到这我感动了。
这时我以饥肠辘辘，眼睛一亮，看见了妈妈做的盒饭，我像一匹饿狼向饭扑过去，津津有味的吃起来。
后面我感到不妙，看了看爸爸妈妈，才发现他们也没有吃，我把饭推给了爸爸妈妈。可是，他们却苦笑地对我说：“你刚刚吊完，先吃，我们吃过了。”
那时您们简简单单一句话让我感动了，虽然只是一句话。我爱您们！
“谁言寸草心，报得三春晖”，是的，父母像风，我像帆，父母助我扬帆起航，我要为父母劈风斩浪，飞溅起一片精彩的浪花。

父母的爱，是一股清泉，让心灵愈加纯洁

发表于 2018-06-15 | 分类于 短句子

父母的爱，是一股清泉，让心灵愈加纯洁；父母的爱，是一盏明灯，指引正确的航向；父母的爱，是一缕凉风，消去内心的燥热。
父母的爱，不只在温柔体贴中展现，那么直接。有时，需要我们用心去细细体会。
那一天，我写的一篇文章得到了老师的称赞，很是兴奋。一路哼着小曲回到家，我一见到妈妈，就眉飞色舞地说：“妈妈，今天有个好消息告诉您！”
“什么呀？”妈妈饶有兴致地问到。“今天老师表扬我作文写得好，不信您看看！”说着，我把作文本递过去，满怀期待地看着妈妈。
妈妈接过本子，细细地看了一会，飘过来一句：“嗯，还行，继续加油。”我仿佛被浇了盆冷水，一下子纳闷了，不该是热情如火的表扬吗？
我像是争辩似的跟妈妈说：“可是……”“可是什么呀，还不去做作业？”我委屈地朝自己的房间走去，闷闷不乐：为什么？表现好都得不到表扬？难道，妈妈不爱我了吗？”

我终于控制不住自己的眼泪了，任她像断线的珠子般流淌。天，也在这时下起了蒙蒙细雨，给我的心平添了几分忧伤……
晚上，我继续胡思乱想，翻来覆去睡不着。在起来上厕所的时候，爸爸妈妈的房间还亮着灯，里面有小声的话语传出，我便俯过身去侧耳倾听。
“孩子这么喜欢写作文，照这样下去，以后绝对能青出于蓝，你这语文老师压力大啊！”这是妈妈的声音。
“什么压力，我倒是希望她早些超过我，孩子超过父母，天经地义，是为人父母最大的心愿。”
爸爸的声音里透出些自豪，“我知道你也在为女儿骄傲，但刚才为什么要那么轻描淡写？”
妈妈的声音有些不好意思：“我这不是怕表扬过头会导致她骄傲吗？失去了写作动力怎么办？你们五年级语文课文有一篇《‘精彩极了’和‘糟糕透了’》，里面怎么写的来着？”
爸爸似乎是在点头：“嗯嗯，有道理，其实我早看出你这点小心思了”……
听到这些，我的眼角湿润了，泪水又一次不争气地流出了眼眶。我想，我已经明白了爸爸妈妈对我的良苦用心。
爸爸妈妈把对我的肯定深藏在心里，我也会把对他们的感谢珍藏在心里，让他们看到我每天的进步和成长，来回报他们！
清晨起床，枕头上略微有些潮湿，但外面已是晴空万里，艳阳高照。

父母的爱是温暖的外套，时时刻刻温暖着我

发表于 2018-06-15 | 分类于 短句子

父爱如山，母爱如海。相信大家大家都听过这句话吧！意思是说，父亲的爱就像一座山一样“险峻”。母亲的爱就像无边无际的大海。
记得有一次，我们一家人去大良溜冰。我们一家人坐着车子，高高兴兴地到了大良的顺峰山公园。
到了之后，看见许多人在溜冰，看着她们那矫健的步伐，我不禁想去尝试一下。我兴高采烈地换好了溜冰鞋。
一开始我以为很容易，所以，我就立马站了起来，差点就摔了个屁股跤。等我第二次重新站起来的时候，我加紧了“防备”，以免会被别人撞倒。
我扶着那张石凳，慢慢地站起身来。先把右脚慢慢地抬起来向前迈过去。第一步，我成功了。当我想去告诉爸爸妈妈听的时候，妈妈就说：“当心着点，小心啊，别摔了。”
我就说：“不回的，你放心吧”当我真的摔了的时候，爸爸却冷冷地说：“你在哪里摔倒就要从哪里爬起来。”爸爸又对妈妈说：“不要去扶她。”

当我听到这句话的时候，突然觉得眼眶都湿透了。我生气地从地上爬起来，换好自己的鞋子，狠狠地把溜冰鞋摔在地上，不满地走开了。
当我回到家以后，把自己关在房间里不肯出来，妈妈在外面叫：“女儿啊，你以后就会懂得爸爸妈妈的这片苦心的。你也不要怪爸爸妈妈。”听了这句话只后，我还是不明白爸爸今天说的那句话是什么意思。
直至有一天，我打开妈妈的电脑，本想去网上查找资料的，但是，我看见了电脑的桌面上有一个文档，题目是可怜天下父母心。我好奇地打开了那个文档，上面写着
今天，我们一家人出去外面溜冰。本是一个开心的晚上，但是，我的女儿，因为太骄傲，导致了摔跤。最后受到了她爸爸的严厉教导，可是，她还是不明白爸爸妈妈为什么这么做的原因。
其实我们这么做只是想让她自己独立，成长，迟早我们都要离开她的。这个方法是我和她爸爸花了很长时间才想出来的最佳教育方法。
读了妈妈写的这封信，我终究明白了，天下的父母都是为自己的儿女好的，并不是因为一个严厉的批评而不爱自己的儿女。我现在真想上前去抱抱爸爸妈妈，告诉他们，我长大了。
我学会独立了。那时因为我的一时任性，导致了你们为我操碎了心。我想对你们说“爸爸妈妈，我爱你们。”我也想向你们说一句“对不起”这所谓，父爱如山，母爱如海。
父母的爱是温暖的外套，时时刻刻温暖着我，我现在真想真诚地对爸爸妈妈说一句：我爱你们。
