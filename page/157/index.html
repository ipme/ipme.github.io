时间象弹簧，可以缩短，也可以拉长

发表于 2018-06-15 | 分类于 励志语录

1、你目前拥有的，都将随着你的死亡而成为他人的。那为何不现在就布施给真正需要的人呢？恨别人，痛苦的却是自己。每天醒来，敲醒自己的不是钟声，而是梦想。
2、命好不如习惯好。养成好习惯，一辈子受用不尽。比别人多一点执着，你就会创造奇迹。一时的忍耐是为了更广阔的自由，一时的纪律约束是为了更大的成功。
3、我们只有一条命，要卖给识货的人。人生舞台的大幕随时都可能拉开，关键是你愿意表演，还是选择躲避。时间象弹簧，可以缩短，也可以拉长。

4、生命是那样美好，建议大家多做深呼吸，体会空气的清新，体味事物的美好。我喝水时总会想这也许是我喝过的最美味的水，时时要保持一种爱，学会欣赏美，惟有爱和美才是心灵的故乡。
5、若想改变人生，首先要改变我们自己。而改变我们自己，首先要正确的认识自己，敢于否定自己。谁虚度年华，青春就要裉色，生命就会抛弃他们。
6、青春励志，凡是想获得优异成果的人，都应该异常谨慎地珍惜和支配自己的时间。合理安排时间，就等于节约时间。目标越接近，困难越增加。

若想改变人生，首先要改变我们自己

发表于 2018-06-15 | 分类于 励志语录

1、我只惋惜一件事，日子太短，过得太快。一个人从来看不出作成什么，只能看出还应该作什么？只要朝着一个方向努力，一切都会变得得心应手。
2、世界上最快而又最慢，最长而又最短，最平凡而又最珍贵，最容易忽视而又最令人后悔的就是时间。逝者如斯夫，不舍昼夜。
3、面对悬崖峭壁，一百年也看不出一条缝来，但用斧凿，得进一寸进一寸，得进一尺进一尺，不断积累，飞跃必来，突破随之。
4、坚硬优质的钢条，是经过千锤百炼而成的；瑰丽美观的贝壳是经过水冲日曝而得的。我们的意志和毅力也必须在火热的斗争中接受严峻的考验，去接受长期的锻炼。只有这样才能使自己在困难面前，永远热情奋发，斗志昂扬。

5、许多人企求着生活的完美结局，殊不知美根本不在结局，而在于追求的过程。我们无法选择自己的出身，可是我们的未来是自己去改变的。
6、青春励志，人生就像一道漫长的阶梯，任何人也无法逆向而行，只能在急促而繁忙的进程中，偶尔转过头来，回望自己留下的蹒跚脚印。

人生太短，要干的事太多，我要争分夺秒

发表于 2018-06-15 | 分类于 励志语录

1、别想一下造出大海，必须先由小河川开始。不要让未来的你，讨厌现在的自己，困惑谁都有，但成功只配得上勇敢的行动派。
2、人生最大的喜悦是每个人都说你做不到，你却完成它了！如果你真的愿意为自己的梦想去努力，最差的结果，不过是大器晚成。
3、不忘初心，方得始终。一个最困苦最卑贱最为命运所屈辱的人，只要还抱有希望，便无所怨惧。一切伟大的行动和思想，都有一个微不足道的开始。

4、有些人，因为陪你走的时间长了，你便淡然了，其实是他们给你撑起了生命的天空；有些人，分开了，就忘了吧，残缺是一种大美。
5、不论你在什么时候开始，重要的是开始之后就不要轻言放弃。花开不是为了花落，而是为了开的更加灿烂。随随便便浪费的时间，再也不能赢回来。
6、青春励志，人生太短，要干的事太多，我要争分夺秒。一寸光阴一寸金，寸金难买寸光阴。光景不待人，须臾发成丝。

比别人多一点执着，你就会创造奇迹

发表于 2018-06-15 | 分类于 励志语录

1、谁若游戏人生，他就一世无成，谁不能主宰自己，永远是一个奴隶。不要刻意去猜测他人的想法，如果你没有智慧与经验的正确判断，平日都会有错误的。
2、人格的完善是本，财富的确立是末。只有在开水里，茶叶才能展开生命浓郁的香气。在比夜更深的地方，一定有比夜更黑的眼睛。
3、每个人都有潜在的能量，只是很容易：被习惯所掩盖，被时间所迷离，被惰性所消磨。从来不跌倒不算光彩，每次跌倒后能再站起来，才是最大的荣耀。

4、青春励志，这个世界到处充满着不公平，我们能做的不仅仅是接受，还要试着做一些反抗。你不能拼爹的时候，你就只能去拼命！
5、如果人生的旅程上没有障碍，人还有什么可做的呢。我们无法选择自己的出身，可是我们的未来是自己去改变的。励志名言：比别人多一点执着，你就会创造奇迹。
6、伟人之所以伟大，是因为他与别人共处逆境时，别人失去了信心，他却下决心实现自己的目标。所虑时光疾，常怀紧迫情，蹒跚行步慢，落后最宜鞭。
