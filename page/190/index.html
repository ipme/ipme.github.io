慢慢的才知道，有很多东西是不属于你的

发表于 2018-06-15 | 分类于 人生感悟

慢慢的才知道，没有风的日子，云是雨的守望；没有梦的日子，等待会荒废时光。这个世界上，谁也不是谁的永远，唱一曲风花雪月，吟一阕岁月静好，烟火、流年、红尘、沧桑，浅浅遇，淡淡忘。
慢慢的才知道，生命于我们，只是沧海一粟，然而，却承载了太多的情非得已。聚散离合，痛苦欢笑，呐喊寻觅……因为没有人知道那个最终的谜底，所以，我们唯一的选择，必须是：向前，笑看风，笑对雨……
慢慢才知道，一个人在乎你！再忙也会抽出一分钟的时间来给你打个电话或发个信息，因为他在想你；慢慢才知道，一个人在乎你，再忙也会抽出一分钟的时间来回你打个电话或回信息，因为他知道你在想他。

慢慢才知道，一个人在乎你，再忙也会抽出一分钟的时间来关注你，因为他在担心你；慢慢才知道，一个人在乎你，再忙也会抽出一分钟的时间来跟你报告，因为他知道你在担心他。
感悟人生，慢慢才知道，不要把自己想的有多高，没有绝对性的胜利，也没有绝对性的失败；慢慢才知道，给人留一线日后好相见的真正意义，没有永远的敌人只有永远的朋友，凡事不要做的太绝，事情的结局都是用嘴巴说出来的。

人之所以会烦恼，就是记性太好

发表于 2018-06-15 | 分类于 人生感悟

人之所以会烦恼，就是记性太好。该记的，不该记的都会留在记忆里。而我们又时常记住了应该忘掉的事情，忘掉了应该记住的事情。
有时要做可爱、可笑的傻瓜，忘记人们对他的嘲笑与冷漠，忘记人世间的恩恩怨怨，忘记世俗的功名利禄，忘记这个世界的一切，活在自己的世界里随心所欲地快乐着，傻傻地笑着。
人生感悟，电影里有一句台词，说的是：人最大的烦恼，就是记性太好，如果什么都可以忘了，以后的每一天，都会是新的开始。
记性不好，也就全然没有了负担，往事飘缥缈渺，好像过眼烟云，恩恩爱爱，已是明日黄花，早就不记得了，记住那些干什么呢？

记性太好，有时候也是一种负担。容易忘记往事的人，反而更容易过得没心没肺并快乐，这样的人，何尝又不是幸福的。
想要感情幸福，不值得的情你得忘，念念不忘，只会增添烦恼。想要心情愉悦，不开心的事你得忘，整天瞎想，只会情绪糟糕。想要相处融洽，处不来的人你得忘，好相处就处，不好相处就不处。

人活着，能忘就忘，不能忘，就放一放！

发表于 2018-06-15 | 分类于 人生感悟

人活着，记性不能太好。很多事情，想多了，心神不宁；记久了，作践心灵。不如把往事清零，还自己一分清净。能忘就忘，不能忘，就放一放！
忘记是是非非，不能因一次的矛盾，否定了别人的品行。忘记恩恩怨怨，记住别人的好，宽恕别人的错。有时候真的挺恨自己记性太好。活得太明白的人总过得太累。
吵架的时候，记不住你撂的所有狠话，记不住你对我毫不留情的责骂。好像这段感情里都是歌舞升平，只要看不到华丽衣袍下面的虱子，就好像不存在似的。
分手的时候，记不住你手指的温度，记不住你充满爱意的眼神。这些会让我胸膛响起玻璃碎裂的声音，让我意识到从此以后这些都不是为我，而是给了另一个人。

为人心胸宽阔，别人佩服，自己快乐！忘，不是失忆，而是不想记起！忘，不是无情，而是删除多余！忘，不是伤感，而是心态乐观！忘，不是糊涂，而是活得理智！
人生苦短，烦恼，忘了吧，痛苦，忘了吧，不满，忘了吧，委屈，忘了吧。所有让自己不开心的事，统统都忘了吧。
生命只有一次，你我只活一回，与其背着烦恼活，不如带着美好过！与其在纠结焦虑中渡过，不如在乐观洒脱中活着！
人生感悟，谁的人生，没有点困难；谁的感情，没有点遗憾；谁的心情，没有点烦恼；谁的生活，没有点辛酸。
人生苦短，与其在纠结焦虑中渡过，不如在乐观洒脱中活着！该放就放了吧，该弃就弃了吧，该舍就舍了吧，该忘就忘了吧！
