天下快意之事莫若友，快友之事莫若谈

发表于 2018-06-14 | 分类于 友情的短句子

1、于患难风豪杰。——魏禧
2、人生贵相知，何用金与钱。——李白
3、一贵一贱交情见。——骆宾王
4、相知在急难，独好亦何益。——李白
5、交心不交面，从此重相忆。——白居易

6、少年乐相知，衰暮思故友。——韩愈
7、乐莫乐兮新相知。——屈原
8、知音世所稀。——孟浩然
9、天下快意之事莫若友，快友之事莫若谈。——蒲松龄
10、交心不交面，从此重相忆。——白居易

人生结交在终始，莫为升沉中路分

发表于 2018-06-14 | 分类于 友情的短句子

1、人之相识，贵在相知，人之相知，贵在知心。——孟子
2、君子之交淡若水，小人之交甘若醴。君子淡以亲，小人甘以绝。——庄子
3、衣不如新，人不如故。——《汉乐府》
4、酒逢知己千杯少，话不投机半句多。
5、人生结交在终始，莫为升沉中路分。——贺兰进明

6、与朋友交，言而有信。——子夏
7、布衣之交不可忘。——李延寿
8、君子与君子以同道为朋，小人与小人以同利为朋。——欧阳修
9、人生乐在相知心。——王安石
10、冤家宜解不宜结，各自回头看后头。——冯梦龙

顺利时结交朋友，逆境中考验朋友

发表于 2018-06-14 | 分类于 友情的短句子

1、顺利时结交朋友，逆境中考验朋友。——拉丁谚语
2、朋友间必须是患难相济，那才能说得上是真正的友谊。——莎士比亚
3、相识满天下，知心能几人。冯梦龙桃花潭水深千尺，不及汪伦送我情。——李白
4、同心而共济，始终如一。——（宋）欧阳修
5、一死一生，乃知交情。一贫一富，乃知交态。一贵一贱，交情乃见。——《史记》
6、未言心相醉，不再接杯酒。——陶渊明

7、人生交契无老少，论交何必先同调。——杜甫
8、人生所贵在知已，四海相逢骨肉亲。——《雁门集》
9、相知在急难，独好亦何益。——李白
10、大丈夫处世处，当交四海英雄。——《三国志·蜀书·刘巴传》
