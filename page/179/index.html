父母的爱是温暖的外套，时时刻刻温暖着我

发表于 2018-06-14 | 分类于 短句子

父爱如山，母爱如海。相信大家大家都听过这句话吧！意思是说，父亲的爱就像一座山一样“险峻”。母亲的爱就像无边无际的大海。
记得有一次，我们一家人去大良溜冰。我们一家人坐着车子，高高兴兴地到了大良的顺峰山公园。
到了之后，看见许多人在溜冰，看着她们那矫健的步伐，我不禁想去尝试一下。我兴高采烈地换好了溜冰鞋。
一开始我以为很容易，所以，我就立马站了起来，差点就摔了个屁股跤。等我第二次重新站起来的时候，我加紧了“防备”，以免会被别人撞倒。
我扶着那张石凳，慢慢地站起身来。先把右脚慢慢地抬起来向前迈过去。第一步，我成功了。当我想去告诉爸爸妈妈听的时候，妈妈就说：“当心着点，小心啊，别摔了。”
我就说：“不回的，你放心吧”当我真的摔了的时候，爸爸却冷冷地说：“你在哪里摔倒就要从哪里爬起来。”爸爸又对妈妈说：“不要去扶她。”

当我听到这句话的时候，突然觉得眼眶都湿透了。我生气地从地上爬起来，换好自己的鞋子，狠狠地把溜冰鞋摔在地上，不满地走开了。
当我回到家以后，把自己关在房间里不肯出来，妈妈在外面叫：“女儿啊，你以后就会懂得爸爸妈妈的这片苦心的。你也不要怪爸爸妈妈。”听了这句话只后，我还是不明白爸爸今天说的那句话是什么意思。
直至有一天，我打开妈妈的电脑，本想去网上查找资料的，但是，我看见了电脑的桌面上有一个文档，题目是可怜天下父母心。我好奇地打开了那个文档，上面写着
今天，我们一家人出去外面溜冰。本是一个开心的晚上，但是，我的女儿，因为太骄傲，导致了摔跤。最后受到了她爸爸的严厉教导，可是，她还是不明白爸爸妈妈为什么这么做的原因。
其实我们这么做只是想让她自己独立，成长，迟早我们都要离开她的。这个方法是我和她爸爸花了很长时间才想出来的最佳教育方法。
读了妈妈写的这封信，我终究明白了，天下的父母都是为自己的儿女好的，并不是因为一个严厉的批评而不爱自己的儿女。我现在真想上前去抱抱爸爸妈妈，告诉他们，我长大了。
我学会独立了。那时因为我的一时任性，导致了你们为我操碎了心。我想对你们说“爸爸妈妈，我爱你们。”我也想向你们说一句“对不起”这所谓，父爱如山，母爱如海。
父母的爱是温暖的外套，时时刻刻温暖着我，我现在真想真诚地对爸爸妈妈说一句：我爱你们。

自己要先看得起自己，别人才会看得起你

发表于 2018-06-14 | 分类于 励志语录

1、再长的路，一步步也能走完，再短的路，不迈开双脚也无法到达。多一分心力去注意别人，就少一分心力反省自己。人生就像钟表，可以回到起点，却已不是昨天！
2、人生就像弈棋，一步失误，全盘皆输。在必要时候需要弯一弯，转一转，因为太坚强容易折断，我们需要更多的柔软，才能战胜挫折。
3、励志语录，人生就像一个动物园，当你以为你在看别人耍猴的时候，却不知自己也是猴子中的一员！从绝望中寻找希望，人生终将辉煌。
4、越是成熟的稻穗，越懂得弯腰。江海之所以能成为百谷之王，是因为懂得身处低下。每个人都有潜在的能量，只是很容易：被习惯所掩盖，被时间所迷离，被惰性所消磨。

5、生气是拿别人做错的事来惩罚自己。后悔是一种耗费精神的情绪。后悔是比损失更大的损失，比错误更大的错误，所以请不要后悔。
6、人生有几件绝对不能失去的东西：自制的力量，冷静的头脑，希望和信心。不是境况造就人，而是人造就境况。自己要先看得起自己，别人才会看得起你。
7、要铭记在心：每天都是一年中最美好的日子。爱的力量大到可以使人忘记一切，却又小到连一粒嫉妒的沙石也不能容纳。人是可以快乐地生活的，只是我们自己选择了复杂，选择了叹息！

不要抱怨，不要沮丧，一切都会过去的

发表于 2018-06-14 | 分类于 励志语录

1、现在你不玩命的学，以后命玩你。耐得住寂寞，守得住繁华！再苦再累也要坚强，只为那些期待眼神。当事情已经发生，不要抱怨，不要沮丧，笑一笑吧，一切都会过去的。
2、我不知道年少轻狂，我只知道胜者为王。第二名意味着你是头号输家。成功就是你坚持不住的时候，在坚持一下。成功不了就是你太弱了，哪来那么多原因。
3、还没死就别把自己当废物。如果你很聪明，为什么不富有呢？只有傻瓜才用双脚去试河水的深浅。炫耀是需要观众的，而炫耀恰恰让我们失去观众。
4、青春励志，理论是一码事，实际又是一码事。知识玩转财富。有人在光明中注视着阴影，有人在阴影中眺望着光明。人总是珍惜未得到的，而遗忘了所拥有的。

5、用最少的悔恨面对过去。用最少的浪费面对现在。用最多的梦面对未来。大多数人想要改造这个世界，但却罕有人想改造自己。得意时应善待他人，因为你失意时会需要他们。
6、一千个人就有一千种生存方式和生活道路，要想改变一些事情，首先得把自己给找回来。任何的限制，都是从自己的内心开始的。忘掉失败，不过要牢记失败中的教训。

一步一步走，不去用嘴说，而是用心做

发表于 2018-06-14 | 分类于 励志语录

1、在我们能掌控和拼搏的时间里，去提升我们生命的质量。态度决定高度。你什么都没有，所以你只能拼命！人生就像一杯茶，不会苦一辈子，但总会苦一阵子。
2、有些路很远，走下去会很累，可是不走，会后悔！有梦就去追啊，愣在原地做什么。规划我的路，一步一步走，不去用嘴说，而是用心做。
3、要想人前显贵，必先人后受罪。前行的路上，我必须孤身一人，因为没有人能跟的上我的脚步。当你不能成就伟业，请你把握住平凡的幸福；当你不能让自己辉煌灿烂，请保持恒久的微笑。
4、不经历苦痛与磨难，哪来澄明平静的心态！要想人前显贵，必须人后受罪！当你决定坚持一件事情，全世界都会为你让路。
5、不逼自己一把，怎么知道自己不行。今天所有的懒惰都会换来明天的卑微。你要让你的实力配得上自己的野心。我讨厌什么都不做，却有所期待的自己！

6、如果努力就可以做到的事情，为什么不去做。父母给的那叫背景，自己打的才叫江山。花开不是为了花落，而是为了开的更加灿烂。
7、往事不必遗憾，若是美好，叫做精彩。若是糟糕，叫做经历。没有耐心和耐力，就别怪现实太现实。你都没竭尽全力，哪有资格哭泣。
8、你永远都无法借别人的翅膀，飞上自己的天空。一个人的强大在于控制情绪，而不是外在型体。努力不一定成功，不努力一定不成功。
9、青春励志，现在的努力，是为了小时候吹过的牛逼！每一个不努力的日子就是对生命的辜负。让努力成为一种习惯。
10、要有最遥远的梦想和最朴素的生活，即使明天天寒地冻，路远马亡。无论明日，有多落魄，至少今天，没有蹉跎。生前何必久睡，死后自会长眠。
