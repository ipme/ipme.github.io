人一生下就会哭，笑是后来才学会的

发表于 2018-06-14 | 分类于 人生格言励志

1、出路出路，走出去了，总是会有路的。困难苦难，困在家里就是难。
2、学的到东西的事情是锻炼，学不到的是磨练。

3、过错是暂时的遗憾，而错过则是永远的遗憾！
4、环境不会改变，解决之道在于改变自己。
5、勇气是控制恐惧心理，而不是心里毫无恐惧。
6、还能冲动，表示你还对生活有激情，总是冲动，表示你还不懂生活。
7、在实现理想的路途中，必须排除一切干扰，特别是要看清那些美丽的诱惑。
8、人生格言：人一生下就会哭，笑是后来才学会的。所以忧伤是一种低级的本能，而快乐是一种更高级的能力。
9、两个人共尝一个痛苦只有半个痛苦，两个人共享一个欢乐却有两个欢乐。
10、放弃该放弃的是无奈，放弃不该放弃的是无能，不放弃该放弃的是无知，不放弃不该放弃的是执著！

人之所以痛苦，在于追求错误的东西

发表于 2018-06-14 | 分类于 人生格言励志

1、人生哲理的人生格言，行动是治愈恐惧的良药，而犹豫、拖延将不断滋养恐惧。
2、你把周围的人看作魔鬼，你就生活在地狱；你把周围的人看作天使，你就生活在天堂。

3、人生格言：人之所以痛苦，在于追求错误的东西。
4、与其说是别人让你痛苦，不如说自己的修养不够。
5、如果你不给自己烦恼，别人也永远不可能给你烦恼，烦恼都是自己内心制造的。
6、好好管教自己，不要管别人。
7、你硬要把单纯的事情看得很严重，那样子你会很痛苦。
8、一杯清水因滴入一滴污水而变污浊，一杯污水却不会因一滴清水的存在而变清澈。
9、运气就是机会碰巧撞到了你的努力。
10、得之坦然，失之淡然，顺其自然，争其必然。

只有不断找寻机会的人才会及时把握机会

发表于 2018-06-14 | 分类于 人生格言励志

1、时间是治疗心灵创伤的大师，但绝不是解决问题的高手。
2、天道酬勤。也许你付出了不一定得到回报，但不付出一定得不到回报。

3、逆境是成长必经的过程，能勇于接受逆境的人，生命就会日渐的茁壮。
4、人生格言：只有不断找寻机会的人才会及时把握机会。
5、做一个决定，并不难，难的是付诸行动，并且坚持到底。
6、如果你能像看别人缺点一样，如此准确的发现自己的缺点，那么你的生命将会不平凡。
7、无论你觉得自己多么的了不起，也永远有人比你更强；无论你觉得自己多么的不幸，永远有人比你更加不幸。
8、背负着过去的痛苦，夹杂着现实的烦恼，这对于人的心灵而言是无任何益处。
9、坚持最难，但成果也最大。
10、只有你学会把自己已有的成绩都归零，才能腾出空间去接纳更多的新东西，如此才能使自己不断的超越自己。

人生就像打电话，不是你先挂就是我先挂！

发表于 2018-06-14 | 分类于 人生格言励志

人生就像打电话，不是你先挂就是我先挂！
人生就像坐公交车，坐过站了就不可能倒退着开，只有自己下车原路返回。

人生就像环型跑道，当你奋力跑到终点的时候才发现你仍然在幸福就在身边，可很多人都把握不住，当他们经过很多弯路在回来的时候，幸福却已经不在。
人生如梦。生命从无到有，又从有走向无。
人生如月，残缺后终有圆满的到来。
人生如溪流，一路历尽坎坷崎岖的重重的惊险，最终汇入平静的湖泊。
人生如蜡烛，火的燃烧和吞噬使蜡烛在烛光消失的一瞬间得到永恒。
人生如瀑布，惊险一霎那，此后便是平坦的前途。
人生如茶，粗品是苦的，细品是香的。
人生如云，历经漂泊不定的旅行，终成甘露滋润世间万物。
