只要够坚强，就没有什么能把自己打垮

发表于 2018-06-14 | 分类于 经典台词

1、胜利贵在坚持，要取得胜利就要坚持不懈地努力，饱尝了许多次的失败之后才能成功。人在旅途，难免会遇到荆棘和坎坷，但风雨过后，一定会有美丽的彩虹。
2、每一个成功者都有一个开始。勇于开始，才能找到成功的路。有了坚持不一定成功；但没有坚持，就注定失败。对于成功，坚持的塑造必不可少！
3、莫言，如果你有想法就要赶紧付出行动，别担心失败或者困难重重，只有在不停地实践与追求中，你才能超越自我，创造属于自己的辉煌。
4、只要不放弃，就没有什么能让自己退缩；只要够坚强，就没有什么能把自己打垮！我们最大的悲哀，是迷茫地走在路上，看不到前面的希望；我们最坏的习惯，是苟安于当下生活，不知道明天的方向。
5、生命就是一场变幻莫测的颠簸，谁也无法左右，但是却是能左右自己的心。爱走了，还是有下一场的花开，真心付出就无怨无悔，曾经的美好也是买不到的回忆。

6、人生是需要奋斗的，只有你奋斗了，失败后才会问心无愧；人生是单行路，只有奋斗了，才会有光明的前途；人生中有许多的竞争对手，正因有这么多的竞争对手，所以我们更得奋斗！
7、时光是个宝，随时都在跑，任你去追逐，逝去又无踪，若要紧跟随，珍惜每一秒，发奋又图强，人生好荣光。因害怕失败而不敢放手一搏，永远不会成功。
8、我匆匆而过，你慢慢地走。我走了，你来了，我们不曾相见。岁月不饶人，我们一直在努力。天下没有廉价的成功。心中长存善解、包容、感恩、知足、惜福。
9、人生几个秋，弹指一挥间，岁月总无情，生命多悔恨，丝染无复白，鬓白无重黑，今发此短信，愿君多谨记，努力爱青春，一失不再得，时间犹为重，惜之精彩至！
10、总是在最初的青春挥霍着岁月的生命，总是在最后的时光忆起最初的懈怠，时光一去不返，生命耗费不再，愿你能够珍爱生命，珍惜时间！

生活不会放弃你，命运不会抛弃你

发表于 2018-06-14 | 分类于 经典台词

1、时光如沙漏般流逝，带走了我们的青春、我们的年华，留下了那一抹淡淡的忧伤，是谁在风中轻轻歌唱，挥着翅膀飞向远方，岁月，神偷？再回首已是耄耋之年。
2、躁动的心灵，不安的灵魂。在彷徨中挣扎，在迷失中探索。时光飞逝，努力不变。春花秋月何时了，时间知多少。一寸光阴一寸金，不要往事多劳心。
3、再见，也许永远不见，我内心固执地追求，只有我自己看得见，但我希望我没错。偶尔给自己一点压力，适时让自己绽放一次，你会发现其实自己很优秀，很超凡。
4、我们因为拥有青春而幸福快乐，不要给自己留下太多的遗憾，不要等到失去的时候才懂得珍惜。这条路，尽管洒满泪水，却不是一条废弃之路。
5、人生应该有所追求，但暂时得不到并不会阻碍日常生活的幸福，因此，拥有一颗平常心，是人生必不可少的润滑液剂。

6、斧头虽小，但经历多次劈砍，终能将一棵最坚硬的橡树砍刀。人在旅途，难免会遇到荆棘和坎坷，但风雨过后，一定会有美丽的彩虹。
7、在这样的时刻，我们需要耐心等待，满怀信心地去期待，相信，生活不会放弃你，命运不会抛弃你。如果你耐不住寂寞，就看不到繁华。
8、生命不在长而在于好，只要每一次尽力的演示，都值得鼓励与喝采。你的努力，也许有人会讥讽；你的执着，也许不会有人读懂。在别人眼里你也许是小丑，在自己心中你就是国王！
9、在成功的道路上，坚持是非常重要的，面对挫折时，要告诉自己：饱尝甜头。逆境总是有的，人生总要进击。愿你不要屈从于命运的安排，坚韧不拔，锲而不舍！做永远的生活强者！
10、莫言，功到自然成，成功之前难免有失败，然而只要能克服困难，坚持不懈地努力，那么，成功就在眼前。

你是我生命中，唯一值得等待的女生

发表于 2018-06-14 | 分类于 生日祝福语

1、朋友生日，愿我的心灵是一朵鲜花，盛开在你的天空下，为你的生日增添一点温馨的情调，为你的快乐增添一道美丽的色彩。
2、亲爱的，明天就是你的生日了，送上一条祝福短信，告诉你，我随时待命，等待你的传唤；想要对你说的是，你是我生命中，唯一值得等待的女生。
3、八月桂花香，母亲大寿时。古稀之年身体健，劳作不辍精气足。宽容仁爱邻里赞，善良节俭美德传。亲朋好友齐祝愿：健康快乐，寿比南山！

4、生日哈皮！许个心愿，让满天的星都为你祝福。荧荧的背景灯照耀你的脸颊，虽然微弱，却可以延伸很远，羡慕你的生日是这样浪漫，充满诗情画意，只希望你的每一天都快乐健康美丽。
5、由于你的降临，这一天成了一个锦绣的日子，从此世界便多了一抹诱人的色彩！生日的烛光中摇曳一季繁花，每一支都是我的祝愿：生日快乐！
6、留不住的是时间，带不走的是思念。忘不了的是情谊，斩不断的是祝福。每年都有一个特别的日子，每年都有我最真诚的祝福：生日快乐，我的朋友。

生日快乐，年年有今日，岁岁有今朝

发表于 2018-06-14 | 分类于 生日祝福语

1、一年中总有几个日子难忘，一生中总有几个人最值得珍惜，在这个最浪漫温情的日子，送去最真挚地问候，愿你想到我的每一天都快乐无比！生日快乐！
2、我亲爱的朋友，你还好吗？虽然现在我们相隔天涯，但是我对您的思念却没有因此而丝毫减分，祝您在今天这个特殊的日子里，生日快乐，年年有今日，岁岁有今朝。
3、久违地祝福，无限地期待。终于让我等到了今天，和你一起的日子，让我陶醉，让我痴狂。虽天各一方，然无限真情：祝你生日快乐！

4、朋友生日，摘一颗星当礼物，采一片云作锦盒，载入思念的信号寄给你，在这特别的日子里，衷心祝你生日快乐甜蜜，从此称心如意。
5、快点看过来：有一颗心在追随着你，有一双眼在凝望着你，有一份真情在陪伴着你，有一份守候在等待着你，还有一个朋友在远方祝福着你：生日快乐！
6、祝你生日快乐，笑口常开！虽然你不能改变天气，但你可以改变心情；你不能选择容貌，但你可以选择表情；你不能预支明天，但你必须用好今天！
