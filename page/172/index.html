要想改变命运，首先改变自己

发表于 2018-06-15 | 分类于 努力奋斗

1、努力奋斗：要想改变命运，首先改变自己。
2、今天的成功是因为昨天的积累，明天的成功则依赖于今天的努力。成功需要一个过程。
3、命运掌握在自己手里，命运的好坏由自己去创造。
4、如果我们投一辈子石块，即使闭着眼睛，也肯定有一次击中成功。
5、好好的管教你自己，不要管别人。

6、先把鱼网打开，鱼儿才能找到渔网的入口。
7、不幸往往来自比较，而幸福也是来自比较。
8、风暴再大，它终不能刮到你的内心去。
9、不宽恕众生，不原谅众生，是苦了你自己。
10、阻止你前行的，不是人生道路上的一百块石头，而是你鞋子里的那一颗石子。

懂得珍惜才配拥有，知道感恩才能长久！

发表于 2018-06-15 | 分类于 努力奋斗

真心是什么？可以说是世上最珍贵，也最不值钱的东西。如果真心能换来真心，世上就不会有那么多的放弃；如果不离能换取不弃，生命就不会有千疮百孔的心灵。
多少主动被视作自作多情，多少深情到最后痛不欲生。真心，有时候是不值钱，可没了真心，所有的感情全都不值钱。
一个人对你好，叫真心。一个人帮助你，也叫真心。一个人惦记你，心疼你，更叫真心。
真心有多珍贵，谁都买不起。如果你真的得到了，请拿出自心用一辈子的期限，去换，去还，去珍惜！

平坦的路，是人一步一步走出来的；真挚的情，是心一点一点换回来的。没有无缘无故的离开，只因没被善待；没有毫无缘由的放手，只因少了关怀。
缘分，不是一个人的挽留，而是两个人的坚守；真情，不是一颗心单方面的付出，而是两颗心共同的呵护。
人心不是一天走远的，感情不是一天变淡的。一个人说再多，若少了应答，心情总会慢慢低落；一颗心爱再深，若缺了回应，感情难免悄悄淡化。
懂得珍惜，才配拥有；知道感恩，才能长久！

不要轻言放弃，否则对不起自己！

发表于 2018-06-15 | 分类于 努力奋斗

1、掌握坚持的人是成功的，是永不言弃的。
2、胜利贵在坚持，要取得胜利就要坚持不懈地努力，饱尝了许多次的失败之后才能成功。
3、我相信所有坚韧不拔的努力，迟早会得好报酬。
4、功到自然成，成功之前难免有失败，然而只要能克服困难，坚持不懈地努力，那么，成功就在眼前。

5、朋友，人生之路并非只有坦途，也有不少崎岖与坎坷，甚至会有一时难以跨越的沟坎儿。在这样的紧要关头我们只有一种选择：再向前跨出一步！尽管可能非常艰难，但请相信：只要坚持下去，你的人生会无比绚丽！
6、逆境总是有的，人生总要进击。愿你不要屈从于命运的安排，坚韧不拔，锲而不舍！做永远的生活强者！
7、励志的话，骆驼走得慢，但终能走到目的地。
8、在成功的道路上，坚持是非常重要的，面对挫折时，要告诉自己：饱尝甜头。
9、失败是什么？没有什么，只是更走近成功一步，成功是什么？就是走过了所有通向失败的路，只剩下一条路，那就是成功的路。
10、不要轻言放弃，否则对不起自己！

你知道别人都在拼命，你怎么好意思在浪费时间

发表于 2018-06-15 | 分类于 努力奋斗

1、你知道别人都在拼命，你怎么好意思在浪费时间。你的脸是为了呈现上帝赐给人类最贵重的礼物是微笑，一定要成为你工作最大的资产。人生的烦恼，多在于知道的太多，而做的太少。
2、能把在面前行走的机会抓住的人，十有八九都会成功。征服畏惧建立自信的最快最确实的方法，就是去做你害怕的事，直到你获得成功的经验。
3、于千万人之中，遇见你所遇见的人；于千万之中，时间的无涯荒野里，没有早一步，也不晚一步，刚巧赶上了。人如果没有诚信，就算有聪明能力，永远只是个危险人物。
4、你的努力，终将成就无可替代的自己。努力从来不会白费，今日撒下种子，正在你看不见想不到的某处，悄悄地生根发芽。你必须非常的努力，才能够看起来毫不费力。

5、只要你足够努力，连命运都不忍心辜负你。为了对得起来未来的自己，你不能贪图安逸。努力不是为了现在得到别人的夸耀，而是为了将来活得有底气的尊严。
6、在自己的世界里努力，在别人的世界里发光。活出精彩的自己。一无所有就是拼博的理由。现在你不拼，老了你拿什么跟孙子说你曾经牛逼的故事。
7、累吗？累就对了，舒服是留给死人的。做的技艺来自做的过程。自己打败自己是最可悲的失败，自己战胜自己是最可贵的胜利。每一发奋努力的背后，必有加倍的赏赐。
8、知识决定竞争力。每一个成功者都有一个开始。勇于开始，才能找到成功的路。自弃者扶不起，自强者击不倒。昨晚多几分钟的准备，今天少几小时的麻烦。
9、努力奋斗，如果在胜利前却步，往往只会拥抱失败；如果在困难时坚持，常常会获得新的成功。过错是暂时的遗憾，而错过则是永远的遗憾！
10、凡过于把幸运之事归功于自己的聪明和智谋的人多半是结局很不幸的。我们最值得自豪的不在于从不跌倒，而在于每次跌倒之后都爬起来。
