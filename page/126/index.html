有些失去是注定的，爱一个人不一定会拥有

发表于 2018-06-15 | 分类于 心情不好

1、一切淡定，可怕的是时间太快。我们都在迷茫着，徘徊着，剧情也许是不变的，虽然已经泛黄的很模糊，但始终会尘封在眼眶，既然迷了情，受了伤，索性一路歌唱到尽头，爱情的帷幕，让我们一起走过。
2、当过去的真爱已不存在，情感的繁花已被冬雨打得残红飘零时，人们总是习惯于停栖在爱情的树枝上低吟浅唱，不是心里仍眷恋那份柔情，等待伤害自己的人回心转意，就是也决心以同样的方式实行报复。但这都是不明智、不潇洒、不可爱的，最恰当的方法就是微笑地向他道声珍重。
3、缠绵在红尘，每逢丝丝微风吹起，便是刺骨般疼痛。可是，你还是没有回头，哪怕是一瞥。记得那时你说，念旧的人就是喜欢用曾经来悲伤自己的现在，用过去感伤自己的未来，可我怎么能忘记你。
4、你说我是你的，你的女人，谁也不能抢，你忘了吗？其实；那天我想说的是，我们永远也不要分开了好不好。
5、你们在一起暧昧的画面，对于我一个前任女友来讲，很难堪。唇齿间旳诺言，都是扯淡。青春、一场盛大旳流离失所。

6、心情不好，有些失去是注定的，有些缘分是永远不会有结果的，爱一个人不一定会拥有，拥有一个人就一定要好好去爱她。
7、相识是缘，相知是份，缘分的天空下，我们共同经历过很多刻骨铭心的爱，真正的幸福不是最后的拥有，而是彼此的珍惜。
8、当最后一个音符响起，想念被逼画上休止符，再美好的句子也要写下句号。然而，这并不预示着结束，而是代表着另一个悲伤的延续。
9、我们的生活有太多无奈，我们无法改变，也无力去改变，更糟的是，我们失去了改变的想法。终于知道，有些人重要，却买不到回程车票。
10、新酒依旧用的老配方，就像遇到再多新人却看出故人模样。时间会证明那些用键盘敲出来的承诺不堪一击。向往天空的，都是寂寞的。

每个人都有一段悲伤，想隐藏却欲盖弥彰

发表于 2018-06-15 | 分类于 心情不好

1、没有正式的告别，我们匆匆分开，犹似两条仅有一个交点的直线，再没机会存在交集。每个人都有一段悲伤，想隐藏却欲盖弥彰。
2、世上最凄绝的距离是两个人本来距离很远，互不相识，忽然有一天，他们相识，相爱，距离变得很近。然后有一天，不再相爱了，本来很近的两个人，变得很远。
3、当我最想说些什么的时候，往往也是我最沉默的时候。如果可以，请把我放回还不认识你的路上。每次打完喷嚏，都会骗自己说你想我了！
4、不是我装清高不食人间烟火，而是你错过了当初认真的我。我好像从没重要过，只是偶尔被需要。
5、我不去多想是否能够成功，既然选择了那块绿洲，便不再瞻前顾后！朋友，再会！朋友，珍重！流水匆匆，岁月匆匆，唯有支情永存心中。

6、多年离别后，抑或再相逢，相逢何所语？泪流默无声。我们用双手紧紧地握别，让感觉在手中轻轻撩过，共享一份难忘的温馨……
7、你来我世界一场，给我一生兵荒马乱。很多东西不见了就真的没有了，比如错过的末班车，比如你。异地恋不容易，不能见面也不能照顾对方。
8、心情不好，别去打扰一个不愿意理你的人，难受的始终是自己。你做了我的逃兵，却成了她的盖世英雄。
9、尘缘尽，一怀柔情化作云烟随风飘走，泪洒尽，换回的只是若有若无的温宁。昨夜星辰今相似，东风依旧，却再没有往日的绚烂与温柔。芳华易逝，孤窗外挂着清冷的目光，夜风撕扯着叹息，在一堆流血的文字里找寻往事，心总是痛的无法呼吸。
10、笑脸的背后流着别人无法知道的眼泪，生活中我们笑得比谁都开心，可是当所有的人潮散去的时候，我们比谁都落寂，很多人，因为寂寞而错爱了一人，但更多的人，因为错爱一人，而寂寞一生。

心痛的让人想哭，再也说不出一个字

发表于 2018-06-15 | 分类于 心情不好

1、回忆的时候，有一瞬间是疼痛的，无奈的，绝望的。明明难过明明舍不得，还是一副你走了我无所谓的样子。时间久了就发现，身边能怪的人越来越少她们谁都有自己的理由。
2、我是坚信缘的，有缘的人，哪怕只是擦肩时的一瞥便可回眸终生，无份的人，即使四目相对亦能互不相识。每个人来到世上，都是匆匆过客，有些人与之邂逅，转身忘记；有些人与之擦肩，必然回首。许是因为一杯茶，一首歌，一处美景，抑或是一个眼神，而我们的相遇缘于一首诗。
3、爱到深处，总会说：“如果开始选择两不相识又如何？”然而我们都不是佛陀，又怎能熏染菩提之心，化爱恨为禅意，拈过往入莲心，行至水穷处，坐看云起？
4、分手的时候，他说：我会打电话给你。这个时候，你顾不了尊严，低声下气的问他：你什么时候会打电话给我？他没给你一个明确的日期，只是说：我会找你。他走了之后，没有再打电话来。

5、我还能够说些什么，我还能够做些什么？我好希望你会听见，因为爱你我让你走了……淋过雨的空气，疲倦了的伤心，我记忆里的童话已经慢慢的融化。
6、如果爱上，就不要轻易放过机会。莽撞，可能使你后悔一阵子；怯懦，却可能使你后悔一辈子。我终生的等待，换不。来你刹那的凝眸。
7、我想变成一棵树，长在你的必经之路，我不是偷窥你，我只是暗恋你。我们以为爱得很深、很深，岁月让我知道，它不过很浅、很浅。
8、往往你以为是自己最珍贵的东西，在你曾经的情人眼里还不如他新欢的一件旧衣裳，让人情何以堪。心痛的让人想哭，再也说不出一个字。
9、心情不好，习惯难受，习惯思念，习惯等你，可是却一直没有习惯看不到你。一个是华丽短暂的梦，一个是残酷漫长的现实。
10、脸上的快乐，别人看得到。心里的痛又有谁能感觉到。有时，爱也是种伤害。残忍的人，选择伤害别人，善良的人，选择伤害自己。

你在的地方，再也不是我眼睛的方向

发表于 2018-06-15 | 分类于 心情不好

1、其实你不用冷淡，我从未想过纠缠。哪有什么好聚好散，不过是想显出落落大方与不纠缠。总有人相濡以沫十几年，却输给天真或无邪一张脸。
2、兜兜转转，相互猜疑争吵，来来回回，不愿面对的，只是内心对失去的恐惧，有勇气去直面风雨，才会真正明白：却已经失去不再是自己的。
3、也许你并不知道在你放开风筝的那一刻，它并没有飞远而是从高空中摔到了深渊，我看着你的眼睛有些疲惫似乎也有眷恋，我知道那双眼是我永生再也遇不到的海。
4、相爱却不能相恋，相恋却不相爱。既相爱又相恋的我们却不能相守一生。我不想喜欢你了，你把你的冷漠蛮横，忽冷忽热都收回去吧，它们威胁不到我了。
5、心情不好，你是酒馆却早早打烊，满心欢喜的我冻死在长街。后来，你在的地方，再也不是我眼睛的方向。
6、挥手告别，扬帆远航，别不了的，是你抛出的那根友谊的缆绳，无形中牢牢地系在我心上。揣一颗欢乐的心上路，带一份真诚前走，朋友，愿我们的友情天长地久。

7、每一次想你，我都得提醒我自己，如果你想和我聊天的话，你早就开口了。有时候，曾经的好朋友转变成陌生人了。有时候，有些人不需要说再见，就已经离开了。友谊常以爱情而结束；而爱情从不能以友谊而告终。
8、当你指望时间抚平你的伤痕时，它总是度日如年，毫无疗效。而当你把心思投入到别的事情时，回头一看，原来时光早已把你治好。时间的疗伤效果很鸡贼，你关注它时，它就罢工。而你忘掉它时，却麻利的替你包扎伤口了。所以说，时间不会替你忘，只有你忘掉了，它才会来掩埋。
9、这个世界有些事情只有经历过了才会懂得。用眼泪，留不住爱情。用恳求，留不住婚姻。有时候，该来的缘分始终都会降临。而注定要分开的，也一定会分开，无论你做什么，都很难留住一颗慢慢远去的心。
10、爱是人生的一部分，付出了爱，并不意味着就一定能收获到爱。人生中的爱情就像刷牙一样，一边是洗具，一边是杯具！
