起跑领先一小步，人生领先一大步

发表于 2018-06-15 | 分类于 励志语录

1、大海因为波澜壮阔而有气势，人生因为荆棘坎坷而有意义。拥有逆境，便拥有一次创造奇迹的机会。挫折是强者的机遇。如果人生的旅程上没有障碍，人还有什么可做的呢。
2、总有一个人要赢，为什么不能是我。解决最复杂的事情往往需要最简单的方法。时光就像一辆畜力车，它的速度取决于我们手中的鞭子。快乐不是因为拥有的多，而是因为计较的少。
3、时间是个常数，但也是个变数。勤奋的人无穷多，懒惰的人无穷少。没有退路的时候，正是潜力发挥最大的时候。不管别人脸上有没有饭粒，都请你先照照镜子。
4、相信信念能够战胜一切，这本身就是一个最伟大的信念。再长的路，一步步也能走完，再短的路，不迈开双脚也无法到达。长在我们大脑左右的耳朵，往往左右我们的大脑。
5、成功不是将来才有的，而是从决定去做的那一刻起，持续累积而成。一个细节足以改变一生。得不到的东西永远总是最好的，失去的恋情总是让人难忘的，失去的人永远是刻骨铭心的。

6、有无目标是成功者与平庸者的根本差别。给事物赋予什么样的价值，人们就有什么样的行动。把困难举在头上，它就是灭顶石；把困难踩在脚下，它就是垫脚石。
7、青春励志，起跑领先一小步，人生领先一大步。设立目标，然后把目标细化为每一步的实际行动。当你劝告别人时，若不顾及别人的自尊心，那么再好的言语都没有用的。
8、不要在你的智慧中夹杂着傲慢，不要使你的谦虚心缺乏智慧。永远不要浪费你的一分一秒，去想任何你不喜欢的人。假如我不能，我一定要；假如我一定要，我就一定能。
9、一个能从别人的观念来看事情，能了解别人心灵活动的人，永远不必为自己的前途担心。人生不过三万天，成功失败均坦然，是非恩怨莫在意，健康快乐最值钱。
10、拥有梦想的人，不做选择题，他们只做证明题。如果你不努力，一年后的你还是原来的你，只是老了一岁。人生意义何为？人生的意义在于：不断的奋斗、进取、超越自己的过程。

含泪播种的人一定能含笑收获

发表于 2018-06-15 | 分类于 励志语录

1、按照自己的活法，快乐的生活，活得像自己就好了，何必在意那么多，勇敢地走自己的路，让别人说去吧。善人行善，从乐入乐，从明入明；恶人行恶，从苦入苦，从冥入冥。
2、心慈者，寿必长；心刻者，寿必促。骨宜刚，气宜柔，志宜大，胆宜小，心宜虚，言宜实，慧宜增，福宜惜，虑不远，忧亦近。
3、青春励志，竞争，其实就是一种变相的友谊，在对手的帮助下提高你自己，害怕竞争的人已经输给了对手，注定难取得大的成就。
4、开始努力吧！在这个过程中你必须放弃很多东西，但你要明白它们都不是你最终想要的，你要相信在你成功以后，总有一天它们会再回来，而且比现在更美好！

5、含泪播种的人一定能含笑收获。不论你在什么时候开始，重要的是开始之后就不要停止。人越是高兴的事情，越爱隐藏；越是痛苦的事情，越爱小题大作。
6、钱可以帮穷人思维的人解决温饱，却可以帮富人思维的人制造财富。登高莫问顶，途中耳目新。做决定之前仔细考虑，一旦作了决定就要勇往直前坚持到底。

无须向别人证明什么，只要你能超越自己

发表于 2018-06-15 | 分类于 励志语录

1、就算你的朋友再多，人脉再广，其实真正对你好的人，你一辈子也遇不到几个。见己不是，万善之门。见人不是，诸恶之根。
2、为了向别人向世界证明自己而努力拼搏，而一旦你真的取得了成绩，才会明白：人无须向别人证明什么，只要你能超越自己。
3、青春励志，学一分退让，讨一分便宜；增一分享受，减一分福泽。我们最值得自豪的不在于从不跌倒，而在于每次跌倒之后都爬得起来。
4、明日复明日，明日何其多！日日待明日，万事成蹉跎。世人皆被明日累，明日无穷老将至。晨昏滚滚水东流。今古悠悠日西坠。百年明日能几何？请君听我《明日歌》。

5、据我观察，大部分人都是在别人荒废的时间里崭露头角的。如果青春的时光在闲散中度过，那么回忆岁月将会是一场凄凉的悲剧。
6、念头端正，福星临，念头不正，灾星照。谎言容易越说越爽，因为谎言比现实要美好，但是谎言像多米诺骨牌一样，说一个慌要十个谎来圆，最后难以自拔。

人之所以痛苦，在于追求错误的东西

发表于 2018-06-15 | 分类于 励志语录

1、有些烦恼，只有你丢掉了，才有云淡风轻的机会。现代的婚姻并不是情感的产物，更多的是竞争的结晶，选配偶其实就是变相的竞争上岗，而小三就是原配最大的竞争对手。
2、被朋友伤害了和被陌生人伤了其实是一样的，别怀疑友情，人家不欠你的，但要提防背叛你的人。工作上的执着实际上是人的一种意志。
3、青春励志，世界上只有想不到的事，没有做不成的事；世界上只有想不通的人，没有走不通的路。人之所以痛苦，在于追求错误的东西。
4、盛年不重来，一日难再晨，及时宜自勉，岁月不待人。生命是以时间为单位的，浪费别人的时间等于谋财害命，浪费自己的时间，等于慢性自杀。

5、今是生活，今是动力，今是行为，今是创作。昨天是一张作废的支票，明天是一张期票，而今天则是你惟一拥有的现金，所以应当聪明地把握。
6、盛年不重来，一日难再晨，及时当勉励，岁月不待人。唯其尊重自己的人，才更勇于缩小自己。常常责备自己的人，往往能得到他人的谅解。
