心一旦被凉透，再多的甜言蜜语也悟不热

发表于 2018-06-14 | 分类于 早安心语正能量

真正关心你的人，看的不是空间，而是你的世界；真正疼爱你的人，读的不是文字，而是你的心情。
心一旦被凉透，除丁痛，对你的事情不会再有反应。对你的一切，不会再有幻想的欲望。无论是你故意激怒，还是你无意的恶言恶语，我都不会在乎，也不会倾听你任何一句美丽的谎言，和华丽的欺骗。从此我们和陌生人并无两样。
心一旦被凉透，再多的甜言蜜语也悟不热，付出再多的感情，也只能是感动自己而已。趁还早，趁现在。趁心还是热乎的，好好的善待，好好的珍惜。千万别让他们心寒。别等心凉透了，才想到珍惜。别等人走远了，才感到清醒。
人这一生很多人不是想留就能留得住，因为有些东西失去了就再也回不来，错过了就难以拥有。这就是所谓的生活，所以说各有各的缘，是不能强结，世间万物，各有习性，花自由开放才能娇艳，鸟自由飞翔才能快乐，让来的随心，让去的随意。

在岁月中跋涉，每个人都会有自己的故事。其实看的淡了心境才会秀丽，看的开了心情才会明媚，只有忘掉曾经的不愉快，珍惜眼前的人你的生活才会幸福。
所以说，有人陪伴，就别左顾右盼，说出的话，你总是去视而不见，谁的时间都很宝贵值钱；有人眷恋，就别厌倦眼前，给予的爱，你总觉得理所当然，谁的真心都不是白白奉献。
早安心语，朋友要懂得相对，谁的帮助都不白给，交往谁都要相互回馈；缘分要懂得珍惜，谁的相识都不是注定，相处谁都要真诚面对。

别笑得太大声，会吵醒了隔壁的悲伤

发表于 2018-06-14 | 分类于 早安心语正能量

别笑得太大声，会吵醒了隔壁的悲伤。别爱得太肆意，会惊扰了对面的烦倦。要知道，每一种幸福，都有额度，挥霍的太多，消耗得就太快……
生活是一杯必须下咽的酒，酸甜苦辣各种各样的滋味；感情是一出痛并快乐的戏，喜怒哀乐千变万化的感受。生活给了你什么，即使承受不起打击，但也要接受得了风雨；感情给了你什么，纵然舍不得放弃，但也不能总委屈自己。
把苦与累当做下酒菜，才能品出生活的真滋味；把喜和悲看成必经路，才会悟出感情的入心扉。人的一生这么短，不求耀眼，只愿平安；心的容纳那么小，不求浪漫，只要简单。

无论你怎样过，开心就好；不管你哪样活，顺心就行！既然活着，就该微笑，痛了，才更坚强；疼了，才更会忍耐。你哭，没有谁替；你痛，没有谁代。路走着，要有耐力；事经着，要经得起风雨。
微笑就是阳光，它能消除人们脸上的冬色，无论是成功是失败都是人生路上一道亮丽的风景线。只要你用微笑来面对生活中的喜怒哀乐，那生活就充满阳光！
早安心语，一粒沙，也能舞出美丽轨迹；一棵草，也能渲染生命的绿意；一只鸟，也能为理想展开羽翼！人生在世，微笑面对，微笑每一天！
累了，停下来歇歇，微微一笑，继续前行；苦了，对着内心说说，微微一笑，接着来过；人生不如意十之八九，没有翻不过的山，没有过不去的坎！

日子过的是心情，生活要的是质量

发表于 2018-06-14 | 分类于 早安心语正能量

我们不想要遗憾的人生，那么就要学会在该放手的时候放手，该回头的时候回头，不要让自己一错再错下去。你要明白，已经成为定局的事情，不会管你开心与否，一切都将按照它该有的轨迹去走……
早安心语，很多时候，觉得自己很快乐，回过头，却无法克制忧伤；觉得自己很坚强，无人时，才能卸下疲惫的伪装。
有些委屈不可诉，诉了伤心；有些得意不可分享，分了伤人。有时很苦，只是无处倾诉；有时很累，只是无法拭去。谁的路途没有雪霜，谁的脚步没有薄凉，谁的心情不带着伤，谁的痛苦用别人扛。
你哭，没有谁替；你痛，没有谁代。路走着，要有耐力；事经着，要经得起风雨。活着，就该微笑；走着，就该坚强。

人，是活给自己看的。别奢望人人都懂你，别要求事事都如意。苦累中，懂得安慰自己。没人心疼，也要坚强；没人鼓掌，也要飞翔；没人欣赏，也要芬芳。
生活，是酸甜苦辣的平凡；生活，是早出晚归的奔波；生活，是聚散离合的悲欢；生活，苦乐共存，喜忧参半；生活，酸甜共味，冷热相依。
人生的道路，不是一条坦途顺到底，每个人都会遇到沟沟坎坎，只有想法跨越坎坷，才能走向自己的辉煌，若想活得洒脱，就要学会看开，日子过的是心情，生活要的是质量。

一个人不怕犯错，怕的是犯了错还执迷不悟

发表于 2018-06-14 | 分类于 早安心语正能量

在生活中，一个人不怕犯错，怕的是犯了错还执迷不悟，在错误的道路上一味执着，失去的不仅是别人，更有自己，当你发现自己犯错之后，也许会沮丧懊恼，但是这并不是你不改正的理由。
在漫漫人生路上，一个错误，或许只是个小小的考验，悬崖勒马，你会遇见更好的自己，但是继续犯错，就是不知死活了，不论什么时候，走错了路，都要记得回头！改正，永远都不晚！
走错了路，要记得回头，爱错了人，要懂得放手，陪你走过困苦的人，要记住，与你历经风雨的人，要珍惜。
生活的本真，琐琐碎碎，生活的滋味，苦苦甜甜，生活的过程，反反复复，生活的节奏，忙忙碌碌，生活的旋律，喜喜忧忧，生活的心情，起起伏伏。

人生没有谁能一世辉煌，生活没有谁能一生顺安，得到的失去的看淡，拥有的放弃的看开，人生尽心尽意的事很多，生活唉声叹气事也不少。
其实，有些事情已经发生了，我们就不要继续浪费时间去想太多。如果你发现已经是走错了路，要及时的回头，不要一错再错。无论眼前有多少美丽的风景在诱惑着自己，也不要留恋或者改变自己的方向，而是要及时回头，以免一错再错。
早安心语，一个人只有走好自己真正想走的路、走一条正确的路，才能实现自己的理想，才能让自己的奋斗有意义。
当你意识到自己走的路不对劲时，不要心存侥幸，不要期盼着有什么奇迹发生。有些等待，给我们带来的是伤痛，有些纵容会让自己无路可退。
