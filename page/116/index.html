生活也许欺骗了你，但你绝对不能欺骗生活

发表于 2018-06-15 | 分类于 心灵鸡汤

1、一个软弱的人不可能找到坚强的理由，只能找到软弱的理由；而一个坚强的人不需要坚强的理由。
2、善良是幸福的种子，痛苦是作恶的果实，如此简单的生活法则却被我们运用的复杂无比。
3、不要沉溺于过去，不要幻想未来，集中精力，过好眼下的每一分每一秒。生活也许欺骗了你，但你绝对不能欺骗生活。
4、人生最大的错误，是用健康换取身外之物，人生最大的悲哀，是用生命换取个人的烦恼，人生最大的浪费，是用生命解决自己制造的麻烦！
5、心灵禅语，孤独把时间研磨成了一点一点，所以才过的那么慢，人生之路，只需要你快快乐乐地去完成。

6、人生，需要有一些时刻，慢下来，静下来，听花开的声音，观叶绽的曼妙。告诉自己，活着，真好。
7、人生舞台的大幕随时都可能拉开，关键是你愿意表演，还是选择躲避。每天下班分钟做一下今天的整理工作。
8、能把在面前行走的机会抓住的人，十有八九都会成功。我们有什么好怕的，我们来到这个世上，就没打算活着回去！
9、宽容润滑了彼此的关系，消除了彼此的隔阂，扫清了彼此的顾忌，增进了彼此的了解。这么努力就是想让曾经看不起我的人闭嘴。
10、妒忌别人，敌视异己，就便是把生命交给别人。大肚能容，断却许多烦恼障，笑容可掬，结成无量欢喜缘。

你配不上自己的野心，也辜负了所受的苦难

发表于 2018-06-15 | 分类于 心灵鸡汤

1、昨天，略去。今天，珍惜。明天，争取。对的，坚持。错的，放弃。我的梦想就是站在万人仰慕的高度，俯视曾经看低我的人。
2、生命本该有意义，我们绝不是白来一场。梦想都是罐装的，要么拧开盖子，要么打碎罐罐，两者兼需努力。
3、人生有两条路，一条用心走，叫做梦想，一条用脚走，叫做现实。记住曾经辱骂我的人，忘记让我委屈的事，我要为我的梦想坚持，坚持为我的梦想奋斗。
4、要爱，就疯狂，两个人撑起所有梦想；不爱，就坚强，一人扛起所有的伤。你可以打败我但永远也别想让我求饶。
5、心灵禅语，有那么多人看着我等着我倒下，我拼了命也不能输。不要指望谁陪你一辈子，没光的时候连影子都会离开你。

6、无论结局怎样：总比懒惰要强。人生不能重来，青春怎敢留白。有一种落差是，你配不上自己的野心，也辜负了所受的苦难。
7、生前何必久睡，死后自会长眠。天生丽质最不励志。霸道的挽留，不如霸气的放手。天若赐我辉煌，我定比天张狂。
8、你有你的诗和远方，我有我的懒和嚣张。笑给别人看，哭给自己听。为天地立心，为生民立命，为往圣继绝学，为万世开太平。
9、子当为我击筑，我为子高歌。你觉得你过不去的坎的时候，有一天也会乐观的和别人分享。或许这就是成长。
10、成长是一种需要学会坚持的事儿，不敢怠慢。总有一个人他教会你成长，然后又独自离开。总是站在时间里等待一个催促自己成长的力量。

我有我的骄傲，凭什么为了你去卑微自己！

发表于 2018-06-15 | 分类于 心灵鸡汤

1、我要这天，再遮不住我眼，要这地，再埋不了我心，要这众生，都明白我意，要那诸佛，都烟消云散！
2、我摇晃着向前走，哪怕血与骨散落一地。男儿到死心如铁，看试手，补天裂！坚强并不只是在大是大非中不屈服，而也是在挫折前不改变自己。
3、有志者，事竟成，破釜沉舟，百二秦关终属楚；有心人，天不负，卧薪尝胆，三千越甲可吞吴。风尘三尺剑，社稷一戎衣。
4、他日若遂凌云志，敢笑黄巢不丈夫。长的漂亮是运气，活得漂亮是霸气。活着不仅需要氧气，更需要勇气，还得霸气。
5、心灵禅语，考场如战场。不一样的是，战场上你的身后还有战友，而考场上你的身边都是敌人。

6、如果结局不是我想要的，那么我宁愿不去参与这个过程。我有我的骄傲，凭什么为了你去卑微自己！
7、别把我的个性和态度混为一谈，我的个性是源于我是谁，而我的态度则取决于你是谁。
8、坚强并不只是在大是大非中不屈服，而也是在挫折前不改变自己。为天地立心，为生民立命，为往圣继绝学，为万世开太平。
9、十年如未死，卷土定重来！有一条很长很长的路叫梦想，但有一堵很高很高的墙叫现实。
10、通向梦想的路上的确有一道高墙，但它只阻挡不够热爱的人。如果累了，就看看天空遥望梦想最初的地方。

要让事情变得更好，先让自己变得更强

发表于 2018-06-15 | 分类于 心灵鸡汤

1、黑暗总会过去，黎明终会到来。谦虚的人更能认清自己。信心孕育信心，从胜利走向胜利。
2、路是自己选的，后悔的话，也只能往自己的肚子里咽。如果你的面前有阴影，那是因为你的背后有阳光。
3、面对困难挫折挑战只要你肯相信自己，不断努力的付出，哪怕你现在的人生是从零开始，你都可以做得到。
4、人生目标确定容易实现难，但如果不去行动，那么连实现的可能也不会有。自信是成功的前提，勤奋是成功的催化剂。
5、要让事情改变，先改变自己；要让事情变得更好，先让自己变得更强。顽强的毅力可以征服世界上任何一座高峰。

6、人生就有许多这样的奇迹，看似比登天还难的事，有时轻而易举就可以做到，其中的差别就在于非凡的信念。
7、天赐我一双翅膀，就应该展翅翱翔，满天乌云又能怎样，穿越过就是阳光。山顶对我们半山腰的人来说并不遥远。
8、很多时候，不快乐并不是因为快乐的条件没有齐备，而是因为活得还不够简单。目标的实现建立在“我要成功”的强烈愿望上。
9、心灵禅语，你现在的态度决定十年后你是人物还是废物。纵然霸气无限，却也脆弱难免。提笔安天下，跨马定乾坤。
10、如果有天我们湮没在人潮中，庸碌一生，那是因为我们没有努力要活的丰盛。天行健，君子以自強不息，地勢坤，君子以厚德载物。
