奔波是一种快乐，让我们真实地感受生活

发表于 2018-06-15 | 分类于 读书名言

1、伟人所达到并保持着的高处，并不是一飞就到的，而是他们在同伴们都睡着的时候，一步步艰辛地向上攀爬的。曾经拥有的不要忘记，难以得到的更要珍惜，属于自己的不要放弃，已经失去的留作回忆。
2、生活，就象一个无形的天平，站在上面的每个人都有可能走极端，但这最终都是为了寻找一个平衡的支点，使自己站的更稳，走的更好，活的更精彩！
3、读书名言，忙碌是一种幸福，让我们没时间体会痛苦；奔波是一种快乐，让我们真实地感受生活；疲惫是一种享受，让我们无暇空虚。
4、卓越人物在开始做事之前，总是具有充分信任自己能力的坚强自信心，深信所从事之事业必能成功。这样，在做事时他们就能付出全部的精力，破除一切艰难险阻，直到胜利。

5、每一日你所付出的代价都比前一日高，因为你的生命又消短了一天，所以每一日你都要更积极。与其用泪水悔恨今天，不如用汗水拼搏今天。当眼泪流尽的时候，留下的应该是坚强。
6、一个人有了决心，方能克服种种艰难，去获得胜利，这样才能得到人们的敬仰。成功也有很大的偏心，总是垂青那些有坚强决心的人。只有决心，才能增强信心，才能充分发挥才智，从而在事业上做出伟大的成就。
7、人经不住千言，树经不住千斧。希望：我喜欢的人不要滚滚而来；更希望：我不喜欢的人滚滚而去。一个人如果不能从内心去原谅别人，那他就永远不会心安理得。

知道自己目的地的人，才是旅行得最远的人

发表于 2018-06-15 | 分类于 读书名言

1、世上没有绝望的处境，只有对处境绝望的人。假如我不能，我一定要；假如我一定要，我就一定能。如果一个人不知道他要驶向哪个码头，那么任何风都不会是顺风。
2、你不能左右天气，但可以改变心情。你不能改变容貌，但可以掌握自己。你不能预见明天，但可以珍惜今天！尝试去把别人拍过来的砖砌成结实的地基，生活就不会那么辛苦了。
3、难过就是，虽然很难但总会过去。女孩要坚强，坚强为自己。女孩要努力，努力为未来。爱自己，靠自己，做自己。即使跌倒了，也要漂亮的站起来再哭。
4、读书名言，想要的东西必须自己拿，不然就算别人给你也要不起。如果错过了太阳时你流泪了，那么你也要错过群星了。路的尽头，仍然是路，只要你愿意走。

5、知道自己目的地的人，才是旅行得最远的人。耐心点，坚强点；总有一天，你承受过的疼痛会有助于你。现在不努力，将来拿什么向曾经抛弃你的人证明它有多瞎。
6、没有人能一路单纯到底，但别忘了最初的自己。比我差的人还没放弃，比我好的人仍在努力，我就更没资格说我无能为力！你配不上自己的野心，也辜负了曾经历的苦难。
7、生日是一个舞台，一次考验，一次机会。恋爱时，男人更会利用这一点；结婚后，女人更会利用这一点。每个人心里都有一段伤痕，时间才是最好的疗剂。青春如此华美，却在烟火在散场。

痛不痛只有自己知道，变没变只有自己才懂

发表于 2018-06-15 | 分类于 读书名言

1、人生没有如果，只有后果和结果。少问别人为什么，多问自己凭什么。现在不玩命，将来命玩你，现在不努力，未来不给力。我就是我，颜色不一样的烟火，天空海阔，要做最坚强的泡沫。
2、时间告诉我，无理取闹的年龄过了，该懂事了。我不是天生的王者，但我骨子里流着不服输的血液。眼泪不是答案，拼搏才是选择。只有回不了的过去，没有到不了的明天。
3、跟自己说声对不起，因为曾经为了别人难为了自己。不要指望谁陪你一辈子，没光的时候连影子都会离开你。痛不痛只有自己知道，变没变只有自己才懂。不要问我过得好不好，死不了就还好。

4、读书名言，没有一种不通过蔑视忍受和奋斗就可以征服的命运。勇气不是感觉不到恐惧而是感觉到恐惧也继续做下去。挤不进的世界就别挤了，何必为难了别人作贱了自己。
5、死亡不是失去生命，而是走出了时间。经验是由痛苦中粹取出来的。学会宽恕就是学会顺从自己的心，“恕”字拆开就是“如心”。乖，别生气了，咱甭和地球人一般见识！
6、就算全世界都说我漂亮，但你却说我不漂亮，那么我就是不漂亮。第一个青春是上帝给的；第二个青春是靠自己努力的。每个人都在自己的世界里做着只有自己才能阐释的梦。
7、头脑是日用品，而不是装饰品。在人之上，要把人当人；在人之下，要把自己当人。生活，就是生下来，活下去。寂寞的人有两种，一种是什么话都听的明白，一种则是什么都听不明白。

为学之道，莫先于穷理；穷理之要，必先于读书

发表于 2018-06-15 | 分类于 读书名言

1、生活中没有书籍，就好像没有阳光；生活中没有书籍，就好像鸟儿没了翅膀。书籍是最有耐心、最能忍耐和最令人愉快的伙伴。在任何艰难困苦的时刻，它都不会抛弃你。
2、夫读书将以何为哉？辨其大义，以修己治人之体也，察其微言，以善精义入神之用也。学如才识，不日进，则日退。
3、风声雨声读书声，声声入耳；家事国事天下事，事事关心。读书即未成名，究竟人高品雅。修德不期获报，自然梦稳心安。
4、在读书上，数量并不列于首要，重要的是书的品质与所引起的思索的程度。用心念书，是为了避免成为不中用的人。
5、读书是最好的学习。追随伟大人物的思想，是最富有趣味的一门科学。我们可以由读书搜集知识，但必须利用思考把糠和麦子分开。

6、阅读一本不适合自己阅读的书，比不阅读还要坏。我们必须学会这样一种本领：选择最有价值、最适合自己所需要的读物。
7、我读书总是以少为贵，从不贪多。不怕读得少，只怕记不牢。书籍是思想的航船，在时代的波涛中破浪前进。它满载贵重的货物，运送给一代又一代。
8、韬略终须建新国，奋发还得读良书。旦旦而学之，久而不怠焉，迄乎成。读书可以培养一个完人，谈话可以训练一个敏捷的人，而写作则可造就一个准确的人。
9、读书名言，读书使人成为完善的人。奇文共欣赏，疑义相如析。读书是最好的学习，追随伟大人物的思想，是富有趣味的事情啊。
10、少年读书，如隙中窥月；中年读书，如庭中望月；老年读书，如台上玩月。皆以阅历之深浅，为所得之深浅耳。为学之道，莫先于穷理；穷理之要，必先于读书。
