自己选择的路，跪着也要把它走完

发表于 2018-06-14 | 分类于 人生哲理

1、生活不会向你许诺什么，尤其不会向你许诺成功。它只会给你挣扎、痛苦和煎熬的过程。所以要给自己一个梦想，之后朝着那个方向前进。如果没有梦想，生命也就毫无意义。
2、每个人真正强大起来都要度过一段没人帮忙，没人支持的日子。所有事情都是自己一个人撑，所有情绪都是只有自己知道。但只要咬牙撑过去，一切都不一样了。
3、想要看清事实，必须要经历一些疼痛。在这个横冲直撞的世界，我们需要拥有强悍无比的内心。无论这个世界对你怎样，都请你一如既往的努力、勇敢、充满希望。
4、每个成功者，都要穿越不为人知的黑暗。强者不是没眼泪，只是能含着眼泪向前跑。泪水和汗水的化学成分相似，但前者只能为你换来同情，后则却可以为你赢得成功。
5、希望是生命的源泉，失去它生命就会枯萎。只有品味了痛苦，才能珍视曾经忽略的快乐；只有领略了平凡，才会收藏当初丢弃的幸福。

6、哲理签名，每个人的一生都有许多梦想，但如果其中一个不断搅扰着你，剩下的就仅仅是行动了。自己选择的路，跪着也要把它走完。
7、缺乏明确的目标，一生将庸庸碌碌。年轻是我们唯一拥有权利去编织梦想的时光。只会幻想而不行动的人，永远也体会不到收获果实时的喜悦。
8、觉得自己做得到和做不到，其实只在一念之间。要想成为强乾，决不能绕过挡道的荆棘也不能回避风雨的冲刷。现在不努力，将来拿什么向曾经抛弃你的人证明它有多瞎。
9、没有人能一路单纯到底，但别忘了最初的自己。比我差的人还没放弃，比我好的人仍在努力，我就更没资格说我无能为力！
10、你配不上自己的野心，也辜负了曾经历的苦难。人生没有如果，只有后果和结果。少问别人为什么，多问自己凭什么。

人经不住千言，树经不住千斧

发表于 2018-06-14 | 分类于 人生哲理

1、没有理所当然的成功，也没有毫无道理的平庸。生活可以漂泊，可以孤独，但灵魂必须有所归依。现在流的泪是当初喜欢你时脑子进的水。
2、如果你决定要旅行，那就别怕风雨兼程。时光的力量恰恰在于更改记忆中的模样。让生活的句号圈住的人，是无法前时半步的。
3、松软的沙滩上最容易留下脚印，但也最容易被潮水抹去。蝴蝶如要在百花园里得到飞舞的欢乐，那首先得忍受与蛹决裂的痛苦。如果命运不宠你，请你别伤害自己。
4、每个人都有潜在的能量，只是很容易：被习惯所掩盖，被时间所迷离，被惰性所消磨。只有承担起旅途风雨，才能最终守得住彩虹满天。
5、无论你觉得自己多么的不幸，永远有人比你更加不幸。失败并不意味你浪费了时间和生命，失败表明你有理由重新开始。觉得自己做得到和做不到，只在一念之间。

6、现在的我不配喊累，因为我一无所有。总有人比你好看，比你聪明，比你年轻，但没人能取代你。凡是能站在别人的角度为他人着想，这个就是慈悲。
7、人经不住千言，树经不住千斧。希望：我喜欢的人不要滚滚而来；更希望：我不喜欢的人滚滚而去。生日是一个舞台，一次考验，一次机会。恋爱时，男人更会利用这一点；结婚后，女人更会利用这一点。
8、哲理签名，跌倒，撞墙，一败涂地，都不用害怕，年轻叫你勇敢。做最好的今天，回顾最好的昨天，迎接最美好的明天。
9、一个人真正的幸福并不是呆在光明之中，而是从远处凝望光明，朝它奔去，就在那拼命忘我的时间里，才有人生真正的充实。
10、每天给自己一个希望，试着不为明天而烦恼，不为昨天而叹息，只为今天更美好！人生没有对错，只有选择后的坚持，不后悔，走下去，就是对的。走着走着，花就开了。

好好拼一把，只怕心老，不怕路长

发表于 2018-06-14 | 分类于 人生哲理

1、训练决定习惯，习惯决定性格，性格决定命运。不管多么险峻的高山，总是为不畏艰难的人留下一条攀登的路。在茫茫沙漠，唯有前时进的脚步才是希望的象征。
2、征服自己，就能征服一切。沙漠里的脚印很快就消逝了。一支支奋进歌却在跋涉者的心中长久激荡。你既然认准一条道路，何必去打听要走多久。
3、盆景秀木正因为被人溺爱，才破灭了成为栋梁之材的梦。如果把才华比作剑，那么勤奋就是磨刀石。是士滴在攀登路上的血，也是懦夫失望时流淌的泪。
4、一帆风顺，并不等于行驶的是一条平坦的航线。相信就是强大，怀疑只会抑制能力，而信仰就是力量。命是弱者的借口，运是强者的谦辞，辉煌肯定有，就看怎么走。
5、世上找不出两个完全相同的人。生得再平凡，也是限量版。不是每件事都注定会成功，但是每件事都值得一试。如果别人朝你扔石头，就不要扔回去了，留着作你建高楼的基石。

6、不是每件事都注定会成功，但是每件事都值得一试。成熟，就是某一个突如其来的时刻，把你的骄傲狠狠的踩到地上，任其开成花或者烂成泥。
7、哲理签名，如果要给美好人生一个定义，那就是惬意。如果要给惬意一个定义，那就是三五知己、谈笑风生。
8、人生中有些事是不得不做的，于不得不做中勉强去做，是毁灭，于不得不做中做的好，是勇敢。与其在风雨中逃避，不如在雷电中舞蹈，即便淋得透湿，也是领略生命的快意。
9、无须匆忙，该来的总会来，在对的时间，和对的人，因为对的理由。一个人真正的幸福并不是呆在光明之中，而是从远处凝望光明，朝它奔去，就在那拼命忘我的时间里，才有人生真正的充实。
10、每天给自己一个希望，试着不为明天而烦恼，不为昨天而叹息，只为今天更美好！所有的不甘，都是因为还心存梦想，在你放弃之前，好好拼一把，只怕心老，不怕路长。

别刚从底层混起，就混得特别没底

发表于 2018-06-14 | 分类于 人生哲理

1、总有人比你好看，比你聪明，比你年轻，但没人能取代你。凡是能站在别人的角度为他人着想，这个就是慈悲。在穷的只剩下梦想的时候，你一定要比别人更加努力。
2、你若将过去抱的太紧，怎么能腾出手来拥抱现在？不敲几下，你不可能知道一个人或一个西瓜的好坏。诚信总会给你带来成功，但可能是下一站。
3、当你对自己诚实的时候，世界上就没有人能够欺骗得了你。人生的上半场打不好没关系，还有下半场，只要努力。把自己过得像王后，你才能吸引国王。你是怎么样的人，就会吸引什么样的人。
4、我们谁都有自己的生活，谁都离不开生活，谁都不能鄙视和逃避生活，人生有很多事情是不如人意的，生活中，十之有八九也都是如此。如果每天都在抱怨，那么，自己的世界里怎么会有美好存在呢？
5、缺点也是点，点到为止。眼睛为你下着雨，心却为你撑着伞。人生来如风雨，去如微尘。妈妈说过没有人值得你为他哭，唯一值得你为他哭的那个人，永远都不会让你为他哭。

6、别刚从底层混起，就混得特别没底。我们的人生不可能永远没有遗憾或者错误，但是如果我们常常去那个老年版本的自己那里拜访一下，就不太可能错过来到这个世上应该做的事。
7、哲理签名，长得漂亮是优势，活得漂亮是本事。当一个人有了想飞的梦想，哪怕爬着，也没有不站起来的理由。充沛的精力加上顽强的决心，曾经创造出许多奇迹。
8、最好的，不一定是最合适的；最合适的，才是真正最好的。火把倒下，火焰依然向上。同样的一瓶饮料，便利店里块钱，五星饭店里块，很多的时候，一个人的价值取决于所在的位置。
9、给猴一棵树，给虎一座山。阳光落在春的枝头，日子便绿了。如果为了安全而不和大海在一起，船就失去了存在的意义。通过云端的道路，只亲吻攀登者的足迹。
10、彩云飘在空中，自然得意洋洋，但最多只能换取几声赞美；唯有化作甜雨并扎根于沃壤之中，才能给世界创造芳菲。成功者绝不放弃，放弃者绝不成功。
