有的人，连争吵都没有，却已经消失在人海

发表于 2018-06-15 | 分类于 伤感说说

1、突然很想跟自己说声对不起，对不起再也找不回原来的自己了。他身上的温暖蛊惑了我让我误以为那就是爱情。
2、最痛的不是哭出来，而是牙齿咬着嘴唇浑身颤抖却哭不出来的时候。人生最遗憾的，莫过于，轻易地放弃了不该放弃的，固执地，坚持了不该坚持的。
3、失望攒够了就离开吧，难道还要留下攒绝望吗？我拿你当成命，你把我当有病。你说你不会离开我，后来你走的比谁都洒脱。
4、有些歌，听着听着就厌了。有些事，说着说着就忘了。有些伤，想着想着就哭了。有些人，走着走着就散了。沉默是一个女孩最大的哭声。
5、真正的失望不是怒骂，不是嚎啕大哭，也不是冲你发脾气，而是沉默不语，是你做什么我都觉得和我再也没有任何关系。

6、时间告诉你什么叫衰老，回忆告诉你什么叫幼稚。突然觉得自己老了好多，像是做了一个好长好长的梦，疲倦却又醒不来，什么也抓不住。
7、伤感说说，有的人，经常和你争吵，一觉醒来后还爱着你；有的人，连争吵都没有，却已经消失在人海。原来，冷漠比争吵更可怕。
8、我们唯一的关系是没有关系。也许遗憾和青春总是绑在一起。何必赠我一朵枯萎的花，如果爱我不是你心里的话。你能看到我留在我屏幕上的字却看不到滴在键盘上的泪。

再重要的人让你失望多了也变得不重要了

发表于 2018-06-15 | 分类于 伤感说说

1、开始的时候总觉得来日方长，殊不知人生是减法，见一面少一面。孤独的人不会伤害别人，只会不断地伤害自己罢了。
2、当我确定你从我的世界消失，我才终于敢承认我喜欢过你的事实。人生真是讽刺，一个人竟然真的会变成自己曾经最反感的样子。
3、最深的孤独不是长久的一个人，而是心里没有了任何期望。每个人心里，都住着一个永远不会提，也永远不会忘的人。
4、那句“我还爱你”差点说出口，却被你一声问候堵在了心头。难过的时候，蹲下来抱抱自己，成全别人，也放开自己。
5、我知道就算我们重新认识几百遍你都不会喜欢我。以后，你会牵着谁的手，哼着怎样的歌。最怕一时温暖一世寒冰。

6、后来，那颗心，不知怎么的就安静了下来，再也不坚持什么非你不可了！再重要的人让你失望多了也变得不重要了。
7、伤感说说，眼泪真正的作用是洗净眼睛要你看清楚让你落泪的那个人！我也曾在别人面前炫耀过你的好。
8、你永远抓不住一个心里有人的人就好像你永远不会考虑一个不可能的人。没有漂亮的外表谁会给你谈真心。
9、放弃一个喜欢的人是什么感觉？就像一把大火烧了你住了很久的房子，你看着那些残骸和土灰的绝望。你知道那是你家，但已经回不去了。
10、有些人很好，但不认识更好。你越是费劲心思的去取悦一个人，那个人就越有可能让你痛彻心扉。是不是拥有以后就会开始要失去。

谢谢你让我把自作多情演得淋淋尽致

发表于 2018-06-15 | 分类于 伤感说说

1、但愿你多年以后仍能想起曾有个人爱你入骨喜你如命。有些人，你永远不必等。拥有的时候永远不懂珍惜这是每个人的通病。
2、给自己一个喘息的机会，也试着让爱有路可退。摘下倔强的面具会不会是一张哭花的脸。对不起，对方已经把你删除，请重新添加。
3、如果我用你待我的方式来待你，恐怕你早就离开了。我还是舍不得让你失去我所有的消息，哪怕你并不在乎。
4、我出席你的婚礼，该化什么妆才能告诉你，没有你我还是很好。再也没有了最初奋不顾身的念头了。
5、孤单久了，就变成了一种习惯。习惯久了想着改变，却与热闹的人群格格不入，最后还是习惯一个人呆着，习惯安静，习惯孤单。

6、伤感说说，你会怎样回忆我，带着笑或是很沉默。我在你看不见的地方想着你，你在我看不见的地方陪着谁呢。
7、有些人莫名其妙的闯进你的世界，给了你想要的温暖和陪伴，然而又莫名其妙的消失了。谢谢你让我把自作多情演得淋淋尽致。
8、旧情+复燃=重蹈覆辙+再次受伤=活该。我越来越不敢打扰你，你的冷漠让我觉得主动是那么廉价。过去那个狼狈不堪的自己是怎么一路笑着走过的。
9、好奇心不要太重，别去翻你在乎的人的说说和空间留言，你会觉得自己什么也不是。发好多好多空间动态，只为引起他的注意，殊不知他已屏蔽。
10、留不住的东西就不要再留，不心疼你的人宁愿从心里生生挖出来也别再奢求。最卑微的不过爱情，最凉薄的不过人心。

渐渐的知道了，很多东西可遇而不可求

发表于 2018-06-15 | 分类于 伤感说说

1、即使你使劲了全身的力气挣扎到全世界都颤抖，他看不见，他听不见，全世界都心酸，那又如何。后来我们各奔东西，一句“有空常联系”成了我们这段感情的墓志铭。
2、感动太少，真话太少，好友太少，笑容太少。已是故人不必再念。一旦得到了对方，爱就停止了生长。她成熟了懂事了不胡闹了可是也不爱笑了。
3、我们都在淡化这段感情，你选择新欢，我选择时间。心情不好的时候，我只想一个人安安静静的待着，不求安慰，只求有人能懂。
4、城市愈大，就愈感到孤独。这世界上最难熬的，并不是等爱的过程，而是等爱消失的过程。当时说在一起的是你，现在说到此为止的也是你。
5、后来熬夜成瘾，终于习惯了没有人关心。我放弃了太多只为了你，到头来发现你才是最应该被放弃的那一个。后来，我们连叫对方的名字都哽咽了。

6、渐渐的知道了，很多东西可遇而不可求，不属于自己的，何必拼了命去在乎。猫爱上了杯子里的鱼，却打碎了杯子。鱼离开水死了，猫终于明白了，爱一个人不一定非要和他在一起。
7、故事已走到最后，我不是你的公主你也不再是的我英雄。在漫天风沙里望着你远去，我竟悲伤的不能自已。
8、提起你还是会心酸却不再像从前一般。故事太长只有风愿意听我讲。我不想放手，可我根本就抓不住。删了一个好友，空了一个分组。
9、回忆起过去，就连呼吸都是痛的。所有的哽咽都汇成一句“没事”。你的手是我不能触及的倾城温暖，我的心是你不曾知晓的兵荒马乱。
10、伤感说说，所有的怦然心动到最后只变成泛泛之交。很多时候，你必须学会一个人去承受所有的委屈与心酸。
