前边是平安，后面是健康，祝你生日快乐！

发表于 2018-06-14 | 分类于 生日祝福语

1、今天是你的生日，愿所有的幸福都围绕着你，左边是甜蜜，右边是满足；愿所有的如意都围绕着你，前边是平安，后面是健康；祝你生日快乐！
2、用我生命的力气，给你吹色彩斑斓的气球，送你环游世界的梦想，用我心跳的动力，给你做酷炫无比的玩具，送你童心永驻的容颜，生日快乐。
3、世界之大，只愿和你做朋友，孤独寂寞我不在乎，你好我就心满意足，结识你是老天给我的幸福。今天是你的生日，千万个思念带着祝福：祝生日快乐！

4、烟雨蒙蒙，绿柳依依，春去夏至，花开花谢，沿着岁月的轨迹，我看见那些回忆被镀上一层光芒，它很美丽，只因你陪我经历了风雨！你的生日又到了，祝你生日快乐！
5、每一个人，都有属于他自己的时刻，回首过去，展望未来，接受祝福，期待明天的路；神圣的时刻，祝你永远都一切顺心。祝你生日快乐，日子没烦恼！
6、朋友生日，有限的空间，有限的文字，给你无限的祝福，大声的对你说：生日快乐！你会是天天快乐的，时时开心的。

生日快乐，永远快乐，友谊长存！

发表于 2018-06-14 | 分类于 生日祝福语

1、一年一度，每一次的生日都是新的开始，每一回的祝福都充满浓浓情谊！亲爱的朋友，愿你怀里塞满礼物，耳边充满祝福！今天，你是最灿烂的一个，生日快乐！
2、朋友生日，世界可以变化，岁月可以更改，但是每年的今天都要为你送上最诚挚的生日祝福，生日快乐，永远快乐，友谊长存！
3、生日快乐！没有最华丽的言语，没有最酷的造型，但却是发自肺腑最真诚的祝愿，愿你天天如今日般快乐，生日的幸福鸟永远停留在你的心田。

4、简单的一句话语，一个祝福，寄给一个远在他乡的你。虽然不常联系，但是不曾忘记。祝你生日快乐，更愿你天天都有个好心情！
5、日月轮转永不断，情若真挚长相伴，不论你身在天涯海角，我将永远记住这一天。祝你生辰快乐！愿你的容颜像春天般绚烂，衷心祝福你，多情的女孩，生日快乐！
6、今天是你的生日，我送你一块名牌手表，秒针走一下代表我在想你；分针走一下代表我在思念你；时针走一下代表我在祝福你，生日快乐，天天天幸福。

祝愿岁岁平平安，心里所想事事成！

发表于 2018-06-14 | 分类于 生日祝福语

1、此日一到来，年龄长一长；蛋糕来助兴，心愿来达成；发此短信有啥意？主要目的传祝福，祝愿岁岁平平安，心里所想事事成！
2、特别的日子，特别的祝福，送给生日的你：希望长大一岁的你更成熟，更顺利，更成功，要钞票多成堆，要情情溢四方，要福福满东海，祝你生日快乐！
3、知道你需要的不多，只需要一点光，那就是烛光；只需要一份情，那就是友情；只需要一颗心，那就是真心；只需要一句话，那就是：生日快乐。

4、我用真心画上一张生日贺卡，用短信编织最美好的祝福，送给最美丽的你，祝你快乐无限，平安幸福，做一个永远的幸运儿寿星，生日快乐！
5、用彩虹编织你的梦，用甜蜜围绕你的笑，用舒坦揭开你的心，用顺利装修你的路，用快乐祝福你的生日，祝你永远平安如意，美丽无比。
6、朋友生日，情人是鲜花，握在手里不愿撒；朋友是葱花，哪里需要哪里抓；爱人是麻花，饿了才会想到他。祝小葱花生日快乐！

黑发不如勤学早，白发方悔读书迟

发表于 2018-06-14 | 分类于 珍惜时间

1、如果一个结婚后的全部生活都和平共处我们一样幸福的话，那么我算是白白浪费了三十年的时光。假如一切能从头开始，那么我将会在呀呀学语的婴儿就结婚，而不会把时间荒废在磨牙和打碎瓶瓶罐罐上。
2、茫茫人海，滚滚红尘，只有时间永远铭记我们的生日，只有时间和我们的名字在墓碑上忠诚地守候着我们的亡灵，时间是孤独的，也是不幸的。
3、珍惜时间可以使生命变得更有价值。用“分”来计算时间的人，比用“时”计算时间的人，时间多五十九倍。
4、时间老人自己是个秃顶，所以直到世界末日也会有大群秃顶的徒子徒孙。喂，你可曾听说才思也许能在青春年少时获得，智慧也许会在腐朽前成熟。
5、今日复今日，今日何其少！今日又不为，此事何时了！人生百年几今日，今日不为真可惜！若言姑待明朝至，明朝又有明朝事。为君聊赋今日诗，努力请从今日始。

6、昨天只是今天的回忆，明天只是今天的梦。勤勉的人，每周七个全天；懒惰的人，每周七个早晨。花有重开日，人无再少年。
7、时间是世界上一切成就的土壤。时间给空想者痛苦，给创造者幸福。利用时间是一个极其高级的规律。莫等闲，白了少年头，空悲切！
8、你热爱生命吗？那么别浪费时间，因为时间是组成生命的材料。三更灯火五更鸡，正是男儿读书时。黑发不如勤学早，白发方悔读书迟。
9、时间的名言，辛勤的蜜蜂永没有时间的悲哀。思往事，惜流芳，易成伤。时间不能增添一个人的寿命，然而珍惜光阴可使生命变得更有价值。
10、不饱食以终日，不弃功于寸阴。你若是爱千古，你应该爱现在；昨日不能唤回来，明日还是不实在；你能确有把握的，只有今日的现在。
