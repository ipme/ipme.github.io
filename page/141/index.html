不管失败多少次，都要面对生活，充满希望！

发表于 2018-06-14 | 分类于 心灵鸡汤

1、我们经历波折，是因为上天不答允我们遏制努力。人如果没有诚信，就算有聪明能力，永远只是个危险人物。
2、人生是海，免不了会有激流暗涌，惊涛骇浪，我们需要的是鼓足干劲，满腔激情，以十足的信心驾驭出完美人生。
3、你不能改变天气，但你可以改变心情；你不能选择容貌，但你可以选择表情；你不能预支明天，但你必须用好今天！
4、女人一定要有的五样东西：扬在脸上的自信长在心底的善良融进血里的骨气两侧外泄的霸气刻进命里的坚强！
5、心灵禅语，于千万人之中，遇见你所遇见的人；于千万之中，时间的无涯荒野里，没有早一步，也不晚一步，刚巧赶上了。

6、生活不是一场赛跑，生活是一场旅行，要懂得好好欣赏每一段的风景。不要只因一次失败，就放弃你原来决心想达到的目的。
7、最足以显示一个人性格的，莫过于他所嘲笑的是什么东西。最有效的资本是我们的信誉，它小时不停为我们工作。
8、谁比同类跑的快谁就能得到最终的胜利。面对强大的敌手，明知不敌，也要毅然亮剑，纵然倒下，也要化成一座山，一道岭。
9、任何冲击都不敷以成为你犯错的借口，纵然你改变不了这个世界，你却依然可以改变自己，选择条正确的路永远走下去。
10、强者之争终有停息，弱者之战，永无休止，因为弱者从来不认可自己是弱者。不管失败多少次，都要面对生活，充满希望。

生活也许欺骗了你，但你绝对不能欺骗生活

发表于 2018-06-14 | 分类于 心灵鸡汤

1、一个软弱的人不可能找到坚强的理由，只能找到软弱的理由；而一个坚强的人不需要坚强的理由。
2、善良是幸福的种子，痛苦是作恶的果实，如此简单的生活法则却被我们运用的复杂无比。
3、不要沉溺于过去，不要幻想未来，集中精力，过好眼下的每一分每一秒。生活也许欺骗了你，但你绝对不能欺骗生活。
4、人生最大的错误，是用健康换取身外之物，人生最大的悲哀，是用生命换取个人的烦恼，人生最大的浪费，是用生命解决自己制造的麻烦！
5、心灵禅语，孤独把时间研磨成了一点一点，所以才过的那么慢，人生之路，只需要你快快乐乐地去完成。

6、人生，需要有一些时刻，慢下来，静下来，听花开的声音，观叶绽的曼妙。告诉自己，活着，真好。
7、人生舞台的大幕随时都可能拉开，关键是你愿意表演，还是选择躲避。每天下班分钟做一下今天的整理工作。
8、能把在面前行走的机会抓住的人，十有八九都会成功。我们有什么好怕的，我们来到这个世上，就没打算活着回去！
9、宽容润滑了彼此的关系，消除了彼此的隔阂，扫清了彼此的顾忌，增进了彼此的了解。这么努力就是想让曾经看不起我的人闭嘴。
10、妒忌别人，敌视异己，就便是把生命交给别人。大肚能容，断却许多烦恼障，笑容可掬，结成无量欢喜缘。

发表于 2018-06-14 | 分类于 心灵鸡汤

1、昨天，略去。今天，珍惜。明天，争取。对的，坚持。错的，放弃。我的梦想就是站在万人仰慕的高度，俯视曾经看低我的人。
2、生命本该有意义，我们绝不是白来一场。梦想都是罐装的，要么拧开盖子，要么打碎罐罐，两者兼需努力。
3、人生有两条路，一条用心走，叫做梦想，一条用脚走，叫做现实。记住曾经辱骂我的人，忘记让我委屈的事，我要为我的梦想坚持，坚持为我的梦想奋斗。
4、要爱，就疯狂，两个人撑起所有梦想；不爱，就坚强，一人扛起所有的伤。你可以打败我但永远也别想让我求饶。
5、心灵禅语，有那么多人看着我等着我倒下，我拼了命也不能输。不要指望谁陪你一辈子，没光的时候连影子都会离开你。

6、无论结局怎样：总比懒惰要强。人生不能重来，青春怎敢留白。有一种落差是，你配不上自己的野心，也辜负了所受的苦难。
7、生前何必久睡，死后自会长眠。天生丽质最不励志。霸道的挽留，不如霸气的放手。天若赐我辉煌，我定比天张狂。
8、你有你的诗和远方，我有我的懒和嚣张。笑给别人看，哭给自己听。为天地立心，为生民立命，为往圣继绝学，为万世开太平。
9、子当为我击筑，我为子高歌。你觉得你过不去的坎的时候，有一天也会乐观的和别人分享。或许这就是成长。
10、成长是一种需要学会坚持的事儿，不敢怠慢。总有一个人他教会你成长，然后又独自离开。总是站在时间里等待一个催促自己成长的力量。
