要让事情变得更好，先让自己变得更强

发表于 2018-06-14 | 分类于 心灵鸡汤

1、黑暗总会过去，黎明终会到来。谦虚的人更能认清自己。信心孕育信心，从胜利走向胜利。
2、路是自己选的，后悔的话，也只能往自己的肚子里咽。如果你的面前有阴影，那是因为你的背后有阳光。
3、面对困难挫折挑战只要你肯相信自己，不断努力的付出，哪怕你现在的人生是从零开始，你都可以做得到。
4、人生目标确定容易实现难，但如果不去行动，那么连实现的可能也不会有。自信是成功的前提，勤奋是成功的催化剂。
5、要让事情改变，先改变自己；要让事情变得更好，先让自己变得更强。顽强的毅力可以征服世界上任何一座高峰。

6、人生就有许多这样的奇迹，看似比登天还难的事，有时轻而易举就可以做到，其中的差别就在于非凡的信念。
7、天赐我一双翅膀，就应该展翅翱翔，满天乌云又能怎样，穿越过就是阳光。山顶对我们半山腰的人来说并不遥远。
8、很多时候，不快乐并不是因为快乐的条件没有齐备，而是因为活得还不够简单。目标的实现建立在“我要成功”的强烈愿望上。
9、心灵禅语，你现在的态度决定十年后你是人物还是废物。纵然霸气无限，却也脆弱难免。提笔安天下，跨马定乾坤。
10、如果有天我们湮没在人潮中，庸碌一生，那是因为我们没有努力要活的丰盛。天行健，君子以自強不息，地勢坤，君子以厚德载物。

其实一直陪着你的，是那个了不起的自己

发表于 2018-06-14 | 分类于 心灵鸡汤

1、面对机会，你要勇敢，输也罢赢也罢，都是一种经历，是一笔花钱买不来的财富。其实一直陪着你的，是那个了不起的自己。
2、一个人失败的最大原因，就是对于自己的能力永远不敢充分信任，甚至认为自己必将失败。成功的秘诀在于对目标的执着追求。
3、我是一名数学人，在这条没有鲜花与掌声的路上，我依然选择了坚守，坚守下去，坚守一生。只要你不认输，就有机会！
4、人，当你意识到自己活着的时候，就必须想到死亡，然后再努力去探寻如何才能生活得更好。
5、心灵禅语，抽出时间去学习，凡事从小做起，不怕单调和重复，长期的积累坚持，想不成功，也难。

6、决不能习惯失败，因为你要知道，身体的疲惫，不是真正的疲惫；精神上的疲惫，才是真的劳累。
7、一个人只有投身于伟大的时代洪流中，他的生命才会闪耀出光彩。只有收获，才能检验耕耘的意义；只有贡献，方可衡量人生的价值。
8、命运不是一个机遇的问题，而是一个选择问题；它不是我们要等待的东西，而是我们要实现的东西。
9、过去的习惯，决定今天的你，所以，过去的懒惰，决定你今天的一败涂地。我把苦难挫折当作自己生存的最好导师。
10、一个人要有自己的目标，只有在自己的目标中前进奋斗，才能体会到达到目标时一路上的辛酸滋味。

真正成功的人生，不在于成就的大小

发表于 2018-06-14 | 分类于 心灵鸡汤

1、人活着要呼吸。呼者，出一口气；吸者，争一口气。要想赢，就一定不能怕输。不怕输，结果未必能赢。但是怕输，结果则一定是输。
2、你要做的就是别人换不掉的，那你做不到怪谁，就是你自己没用！天塌下来，有个高的人帮你扛着，可是你能保证，天塌下来的时候，个儿高的人没在弯腰吗？之后，还不是得靠自己！
3、没有伞的孩子，必须努力奔跑！真正成功的人生，不在于成就的大小，而在于你是否努力地去实现自我，喊出自己的声音，走出属于自己的道路。
4、心灵禅语，人生充满着期待，梦想连接着未来。你不喜欢我，我一点都不介意。因为我活下来，不是为了取悦你！
5、当我觉得我倒霉了，我就会想：这是在挥洒我的霉运，挥洒之后，剩下的就全都是幸运了！你可以重重的把我打倒，但想都别想我会跟你求饶。

6、就算跑最后一名又怎样，至少我又运动精神。而且我一定会跑完全程，只要跑到终点我就成功了！不要说你不会做！你是个人你就会做！
7、懂得感恩，是收获幸福的源泉。懂得感恩，你会发现原来自己周围的一切都是那样的美好。能经得住多大诋毁，就能担起多少赞美。
8、一颗石子，阻挡不了江河的前进；一片乌云，遮挡不住太阳的光辉；一点困难，阻挡不住前进的脚步。
9、任何值得拥有的东西，一定是值得等待。每个人都在发奋努力奋不顾身，不是只有你受尽委屈。意志坚强的人，他的世界充满着无限的可能性。
10、永不言败是追究者的最佳品格。我依旧相信努力的意义，奋斗的价值，因为那是本质问题。赢了自己才能成功。

什么是家？一家人什么最重要？

发表于 2018-06-14 | 分类于 心灵鸡汤

什么是家？家，是人生的安乐窝；家，是心灵的避风港。身累了，有家可回；心累了，有家人可诉，家永远是最温暖的地方。
什么是家人？家人，是风雨同舟的守护；家人，是风霜雨雪的陪伴。富也好，穷也罢，风光也好，落魄也罢，只有家人永远不离不弃。
家，不在乎大小。大有大的舒适，小有小的贴心，一家人和和美美就好。家人，不在乎多少。多有多的热闹，少有少的温暖，有事没事常回去看看就好。

心灵鸡汤，一家人，什么最重要？要我说啊，一家人整整齐齐、和和睦睦，就比啥都重要！你说是吗？
日有笑靥，夜有香梦，人有真情，家有温馨。家和万事兴，只有家庭的和睦，才能万事兴隆，事事顺顺。
