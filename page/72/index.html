人生自古谁无死，留取丹心照汗青

发表于 2018-06-15 | 分类于 爱国名言

1、我希望在烈火与热血中得到永生！恨不抗日死，留作今日羞。国破尚如此，我何惜此头。爱国，是一种极高贵的感情活动。
2、即从巴峡穿巫峡，便下襄阳向洛阳。为了抉择真理，为了国家民族，我要回国去！牺牲小我，成功大我。
3、我们爱我们的民族，这是我们自信心的泉源。轻金钱，重义务，诚信果毅，嫉恶好善，爱乡爱国。

4、国既不国，家何能存！爱祖国，为祖国的前途而奋斗，是时代赋予我们的神圣职责。常思奋不顾身，而殉国家之急。
5、爱国名言，我不如起个磨刀石的作用，能使钢刀锋利，虽然它自己切不动什麽。人生自古谁无死，留取丹心照汗青。
6、我们波兰人，当国家遭到奴役的时候，是无权离开自己祖国的。天下兴亡，匹夫有责。我坚信任何困难都难不倒英雄的中国人民。

我是中国人，我的祖国现在正需要我！

发表于 2018-06-15 | 分类于 爱国名言

1、虽然科学没有国界，科学家却是有祖国的！我是中国人，我的祖国现在正需要我！
2、我们自古以来，就有埋头苦干的人，有拼命硬干的人，有为民请命的人，有舍身求法的人，这就是中国的脊梁。
3、爱国名言，僵卧孤村不自哀，尚思为国戌轮台。谁把中国农村搞好了，谁就是中国人民的功臣。
4、葬我于高山之上兮，望我大陆；大陆不可见兮，只有痛哭。葬我于高山之上兮，望我故乡；故乡不可见兮，永不能忘。
5、西北大风起，东南战血多。风吹铁马动，还我旧山河。祖国如有难，汝应作前锋。白日放歌须纵酒，青春作伴好还乡。
6、鞠躬尽瘁，死而后已。宁做流浪汉，不当亡国奴。爱国主义就是千百年来巩固起来的对自己祖国的一种深厚的感情。

7、要想着收咱失地，别忘了还我河山。我们现在必须完全保持党的纪律，否则一切都会陷入淤泥中。剑外忽传收蓟北，初闻涕泪满衣裳。
8、大江歌罢掉头东，邃密群科济世穷。面壁十年图破壁，难酬蹈海亦英雄。我不是为了钱，而是为了祖国的荣誉在跑。
9、我们用事实证明：中国的码头工人不比别人差，别人能干的，我们也能干，别人不能干的，我们照样能干！
10、死去原知万事空，但悲不见九州同。王师北定中原日，家祭无忘告乃翁。乡愁是一湾浅浅的海峡，我在这头，大陆在那头。

对党和人民要忠，对国家和事业要爱

发表于 2018-06-15 | 分类于 爱国名言

1、我好象一只牛，吃的是草，挤出的是奶血。爱国之心，实为一国之命脉。我爱我的祖国，爱我的人民，离开了它，离开了他们，我就无法生存，更无法写作。
2、为什么我的眼里常含泪水？因为我对这土地爱得深沉。先天下之忧而忧，后天下之乐而乐。我们知道个人是微弱的，但是我们也知道整体就是力量。
3、人民不仅有权爱国，而且爱国是个义务，是一种光荣。我有我的人格良心，不是钱能买的，我的音乐，要献给祖国，献给劳动人民大众，为挽救民族危机服务。
4、对党和人民要忠，对国家和事业要爱。只要是党和人民的事业，干什么工作都要尽心尽力，在什么岗位都要尽职尽责。

5、一个人只要热爱自己的祖国，有一颗爱国之心，就什么事情都能解决。什么苦楚，什么冤屈都受得了。苟利国家生死以，岂因祸福避趋之。
6、遗民泪尽胡尘里，南望王师又一年。祖国就是我的家，党就是我的母亲，为了他们，我决心献出我的一切！
7、我是炎黄子孙，理所当然地要把学到的知识全部奉献给我亲爱的祖国。砍头不要紧，只要主义真。杀了夏明翰，还有后来人！
8、我是你的，我的祖国！都是你的，我的这心这灵魂；假如我不爱你，我的祖国，我能爱哪一个人？
9、位卑未敢忘忧国。假如我还能生存，那我生存一天就要为中国呼喊一天。我们为祖国服务，也不能都采用同一方式，每个人应该按照资禀，各尽所能。
10、爱国名言，国耻未雪，何由成名？人生难得几回搏，此时不搏更待何时！一身报国有万死，双鬓向人无再青。

2、我们中华民族有同自己的敌人血战到底的气概，有在自力更生的基础上光复旧物的决心，有自立于世界民族之林的能力。
3、现阶段，爱国主义主要表现为献身于建设和保卫社会主义现代化的事业，献身于促进祖国统一的事业。生是为中国，死是为中国。
4、愿以我血献后土，换得神州永太平。与其忍辱生，毋宁报国死。光明的中国，让我的生命为你燃烧吧。
5、我死以后，把我的骨灰送家乡……把它埋了，上头种一棵苹果树，让我最后报答家乡的土地，报答父老乡亲。

6、商女不知亡国恨，隔江犹唱后庭花。为了国家和人民的利益不怕自己吃亏的人，才是高尚的有道德的脱离了低级趣味的人。
7、你可知“妈港”不是我的真名姓？我离开你的襁褓太久了，母亲！三百年来梦寐不忘的生母啊！请叫儿的乳名，叫我一声“澳门”！母亲！我要回来，母亲！
8、我随时等待祖国的召唤！英雄非无泪，不洒敌人前。男儿七尺躯，愿为祖国捐。起来，不愿做奴隶的人们，把我们的血肉筑成我们新的长城！
9、平生爱国不甘后人，即使把我烧成灰，我也是爱国的。为了新中国，冲啊！三万里河东入海，万千仞岳上摩天。
10、爱国名言，我荣幸地从中华民族一员的资格，而成为世界公民。我是中国人民的儿子。我深情地爱着我的祖国和人民。
