你在的地方，再也不是我眼睛的方向

发表于 2018-06-14 | 分类于 心情不好

1、其实你不用冷淡，我从未想过纠缠。哪有什么好聚好散，不过是想显出落落大方与不纠缠。总有人相濡以沫十几年，却输给天真或无邪一张脸。
2、兜兜转转，相互猜疑争吵，来来回回，不愿面对的，只是内心对失去的恐惧，有勇气去直面风雨，才会真正明白：却已经失去不再是自己的。
3、也许你并不知道在你放开风筝的那一刻，它并没有飞远而是从高空中摔到了深渊，我看着你的眼睛有些疲惫似乎也有眷恋，我知道那双眼是我永生再也遇不到的海。
4、相爱却不能相恋，相恋却不相爱。既相爱又相恋的我们却不能相守一生。我不想喜欢你了，你把你的冷漠蛮横，忽冷忽热都收回去吧，它们威胁不到我了。
5、心情不好，你是酒馆却早早打烊，满心欢喜的我冻死在长街。后来，你在的地方，再也不是我眼睛的方向。
6、挥手告别，扬帆远航，别不了的，是你抛出的那根友谊的缆绳，无形中牢牢地系在我心上。揣一颗欢乐的心上路，带一份真诚前走，朋友，愿我们的友情天长地久。

7、每一次想你，我都得提醒我自己，如果你想和我聊天的话，你早就开口了。有时候，曾经的好朋友转变成陌生人了。有时候，有些人不需要说再见，就已经离开了。友谊常以爱情而结束；而爱情从不能以友谊而告终。
8、当你指望时间抚平你的伤痕时，它总是度日如年，毫无疗效。而当你把心思投入到别的事情时，回头一看，原来时光早已把你治好。时间的疗伤效果很鸡贼，你关注它时，它就罢工。而你忘掉它时，却麻利的替你包扎伤口了。所以说，时间不会替你忘，只有你忘掉了，它才会来掩埋。
9、这个世界有些事情只有经历过了才会懂得。用眼泪，留不住爱情。用恳求，留不住婚姻。有时候，该来的缘分始终都会降临。而注定要分开的，也一定会分开，无论你做什么，都很难留住一颗慢慢远去的心。
10、爱是人生的一部分，付出了爱，并不意味着就一定能收获到爱。人生中的爱情就像刷牙一样，一边是洗具，一边是杯具！

你除了认命以外，没有更好的办法了

发表于 2018-06-14 | 分类于 心情不好

1、爱是什么？不是松开手、放你走，而是紧紧拽着你，说：不许走！卑微的向你乞讨所谓浪漫的爱情，你却无私奉献了你所谓高尚的友情。
2、喜欢一个人没有错，错就错在喜欢一个不喜欢自己的人。飞鸟等待划破云层的时刻，即使炽烈的太阳会将融化双翼，也要坚持与光相遇。
3、回家的路上我哭了，眼泪再一次崩溃孓。无能为力这样走着，再也不敢骄傲奢求了。我还能够说些什么，我还能够做些什么？我好希望你会听见，因为爱你我让你走了。
4、心情不好，你是这个秋天里最后一丝游离的秋风，而我恰是这秋天里随风飘落的最后一片枫叶！我的心中像有个洞，风扑扑的往里面吹。

5、仇恨永远不能化解仇恨，只有慈悲才能化解仇恨，这是永恒的至理。你认命比抱怨还要好，对于不可改变的事实，你除了认命以外，没有更好的办法了。
6、恋爱是想一个人的心，婚姻是栓一个人，爱情是吞一个人的心。既不回头何必不忘既然无缘何必誓言。有些人存在。是因为宿命。所以他不应该抱怨什么。
7、就算是醉生梦死，也是记忆的一种，因为你始终不能忘记什么。如果爱你是错的话，我不想对；如果对是等于没有你的话，我宁愿一辈子错。

我赢了所有人，但却输掉了你

发表于 2018-06-14 | 分类于 心情不好

1、我赢了所有人，但却输掉了你。地球是圆的，可路是直的，离开了就别妄想再遇见。妄想与你执手天涯，却留我泪染白花。
2、广告总是在最精彩的时候出现，你总是在我爱的最深的时候离开。慢慢的长大了却感觉变沉默了，懂得多了却不快乐了。
3、我们不得不分离，轻声地说声再见，心里保存着感激，感谢你曾给我那一份深厚的情谊。本以为自己可以很从容，本以为可以什么都放得下。
4、朋友，你喜欢星空吗？如果有一天你在远方流浪时感到孤独、忧郁，请抬头看看星空，那儿有我送给你的幸运星。

5、挥手告别，扬帆远航。别不了的，是你抛出的那根友谊的缆绳，无形中牢牢地系在我的心上。你有新的旅途，我却留在当初。
6、对不起，我的性格就是这样，学不会主动，学不会挽留，更学不会死缠烂打。后来，说过的话可以不算，爱过的人也可以再换。
7、心情不好，你们有谁试过半夜躲在被子里哭还要拼命捂住嘴巴不让自己发出声音的感觉么？
8、未来是什么？未来是快乐的拼搏加疲惫的身影等于成功的总点，未来是把现在当成过去加把我变成你的回忆等于梦想又增多。

凡事不要太过于执念，留不住的人就松开

发表于 2018-06-14 | 分类于 心情不好

有时候，烦恼并不是别人去害你，而是你自己太在意，到头来，别人无所谓，你却自己心太累。
心情不好，有些事，本不该计较，时间会为你证明一切；有些人，本无需理会，道不同，根本不是一类。
世界事，自己度；人间理，独自悟；面对伤害，欣然一笑是一种看开；面对误会，不去理会是一种豁达；不为不值得的事，乱心；不为不该爱的人，动情。

人生没有走不出的经历，只有不愿意离开的自己；感情没有过不去的曾经，只有不愿面对未来的心情。
凡事不要太过于执念，留不住的人就松开，两个错误的人分开，或许会成就四个人的幸福；放不下的事也放下，不让过去过去，未来又怎会敢来？
