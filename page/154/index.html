不管失败多少次，都要面对生活充满希望

发表于 2018-06-14 | 分类于 座右铭

1、自己选择的路，跪着也要把它走完。缺乏明确的目标，一生将庸庸碌碌。年轻是我们唯一拥有权利去编织梦想的时光。不管失败多少次，都要面对生活，充满希望。
2、座右铭，大胆自已地表现自已，别人无理由轻视你。看一个人的心术，看他的眼神；看一个人的身价，看他的对手；看一个人的底牌，看他的朋友。
3、不要太在乎自己的长相，因为能力不会写在脸上。竹根即使被埋在地下无人得见，也决然不会停止探索而力争冒出新笋。希望，只有和勤奋作伴，才能如虎添翼。
4、勤奋是你生命的密码，能译出你一部壮丽的史诗。对于攀登者来说，失掉往昔的足迹并不可惜，迷失了继续前时的方向却很危险。奋斗者在汗水汇集的江河里，将事业之舟驶到了理想的彼岸。
5、忙于采集的蜜蜂，无暇在人前高谈阔论。如果圆规的两只脚都动，永远也画不出一个圆。不管多么险峻的高山，总是为不畏艰难的人留下一条攀登的路。

6、稗子享受着禾苗一样的待遇，结出的却不是谷穗。骄傲是断了引线的风筝稍纵即逝。自卑是剪了双翼的飞鸟，难上青天，这两者都是成才的大忌。
7、盆景秀木正因为被人溺爱，才破灭了成为栋梁之材的梦。如果把才华比作剑，那么勤奋就是磨刀石。过去不等于未来。没有失败，只有暂时停止的成功。
8、士搏出惊涛骇流而不沉沦，懦夫在风平浪静也会溺水。志在峰巅的攀登者，不会陶醉在沿途的某个脚印之中。对自己不满是任何真正有才能的人的根本特征之一。
9、回避现实的人，未来将更不理想。给自已标价越高，自已的价值实现就越大。永不言败，是成功者的最佳品格。只有想不到的事，没有做不到的事。
10、世上没有绝望的处境，只有对处境绝望的人。如果你想攀登高峰，切莫把彩虹当作梯子。经受了火的洗礼泥巴也会有坚强的体魄。

活着不是靠泪水搏取同情，而是靠汗水获得掌声

发表于 2018-06-14 | 分类于 座右铭

1、山涧的泉水经过一路曲折，才唱出一支美妙的歌。瀑布跨过险峻陡壁时，才显得格外雄伟壮观。美丽的蓝图，落在懒汉手里，也不过是一页废纸。一时的挫折往往可以通过不屈的搏击，变成学问及见识。
2、再黑的黑夜也会迎来黎明，再长的坎坷也会出现平路，怀抱着一棵永不放弃的希望之心，明天就会有温暖的阳光雨露，坚持吧朋友，胜利就在你的下一步路！
3、要用行动控制情绪，不要让情绪控制行动；要让心灵启迪智慧，不能让耳朵支配心灵。人与人之间的差别，主要差在两耳之间的那块地方！
4、在你不害怕的时间去斗牛，这不算什么；在你害怕时不去斗牛，也没有什么了不起；只有在你害怕时还去斗牛才是真正了不起。
5、座右铭，心态决定看世界的眼光，行动决定生存的状态。要想活出尊严，展现不凡，只有改变观念，敢于和命运抗争！

6、方向比速度重要，智慧比吃苦重要，学习比学历重要，机遇比关系重要，要什么比做什么重要！如果不想做点事情，就不要想到达这个世界上的任何地方。
7、拥有远见比拥有资产重要，拥有能力比拥有知识重要，拥有人才比拥有机器重要，拥有健康比拥有金钱重要！梦想是点燃生命之火的催化剂。如果命运不宠你，请你别伤害自己。
8、趁年轻去努力，别对不起你儿时吹的牛逼。现在是我奋斗的时候因为年轻就在眼前摆着。如果放弃太早，你永远都不知道自己会错过什么。活着不是靠泪水搏取同情，而是靠汗水获得掌声。
9、耐心点，坚强点；总有一天，你承受过的疼痛会有助于你。即使你有一千个理由难过，也要有一千零一个理由欢笑。我们总是把别人安抚得太好，轮到自己就失了分寸。
10、你一个人心情不好，不代表全世界都需要陪你难过。如果你决定要旅行，那就别怕风雨兼程。时光的力量恰恰在于更改记忆中的模样。生气就好像自己喝毒药而指望别人会痛苦。

要先打败任何事情得先学会打败自己

发表于 2018-06-14 | 分类于 座右铭

1、不要质疑你的付出，这些都会一种累积一种沉淀，它们会默默铺路，只为让你成为更优秀的人。如果你想要什么，那就勇敢地去追求，不要管别人是怎么想的，因为这就是实现梦想的方式行动。
2、世界上，真正比你强的人，谁有闲工夫搭理你啊，所以不必在意别人的冷嘲热讽。很多爱不能重来我应该释怀输不起就不要输。没有创造的生活不能算生活，只能算活着。
3、只要还有明天，今天就永远是起跑线。我在奋斗在坚持在拼搏在努力你要等。知道自己目的地的人，才是旅行得最远的人。如果说我懂得的道理比别人多一点，那是因为我犯的错误比别人多一点。
4、不管前方的路有多苦，只要走的方向正确，不管多么崎岖不平，都比站在原地更接近幸福。如果你向神求助，说明你相信神的能力。如果神没有帮助你，说明神相信你的能力。
5、嘴里说的人生，就是自己以后的人生。一个华丽短暂的梦，一个残酷漫长的现实。没有人能一路单纯到底，但要记住，别忘了最初的自己。有人将你从高处推下的时候恰恰是你展翅高飞的最佳时机。

6、要生活得漂亮，需要付出极大忍耐。一不抱怨，二不解释。命是弱者的借口，运是强者的谦辞，辉煌肯定有，就看怎么走。世上找不出两个完全相同的人。生得再平凡，也是限量版。
7、不是每件事都注定会成功，但是每件事都值得一试。如果别人朝你扔石头，就不要扔回去了，留着作你建高楼的基石。不是每件事都注定会成功，但是每件事都值得一试。
8、成熟，就是某一个突如其来的时刻，把你的骄傲狠狠的踩到地上，任其开成花或者烂成泥。如果要给美好人生一个定义，那就是惬意。如果要给惬意一个定义，那就是三五知己、谈笑风生。
9、人生中有些事是不得不做的，于不得不做中勉强去做，是毁灭，于不得不做中做的好，是勇敢。与其在风雨中逃避，不如在雷电中舞蹈，即便淋得透湿，也是领略生命的快意。
10、座右铭，无须匆忙，该来的总会来，在对的时间，和对的人，因为对的理由。我会把每一次改变当做成长，哪怕是痛也值得。要先打败任何事情得先学会打败自己。

不要在这个努力拼搏的年纪去选择安逸

发表于 2018-06-14 | 分类于 座右铭

1、快乐是一种香水，无法倒在别人身上，而自己却不沾上一些。我们的报酬取决于我们所做出的贡献。给予的最需要的方面不在物质财富范围内，它存在于人性特有的领域。
2、无所不能的人实在是一无所能，无所不专的专家实在是一无所专。比我差的人还没放弃，比我好的人仍在努力，我就更没资格说我无能为力！
3、你配不上自己的野心，也辜负了曾经历的苦难。我一直以为人是慢慢变老的，其实不是，人是一瞬间变老的。成功的速度一定要超过父母老去的速度。
4、路靠自己走，就算再坎坷，也要自己过。失败并不意味你浪费了时间和生命，失败表明你有理由重新开始。将来的你，一定会感谢现在拼命的自己！
5、挫折是弱者的绊脚石，是强者的垫脚石。今天所有的不幸，是我荒废时间的报应。努力到无能为力，拼搏到感动自己。不要在这个努力拼搏的年纪去选择安逸。

6、成功就是一把梯子，双手插在口袋的人是爬不上去的。你的气质里，藏着你跑过的步和你读过的书。这世上最不会贬值的投资，就是为自己努力。
7、最怕一生碌碌无为，还安慰自己平凡可贵。不怕万人阻挡，只怕自己投降。总有人要赢，为什么不能是我？没有走不通的路，只有不敢走的人！苦难，是化了妆的祝福。
8、座右铭，人生没有绝对的公平，而是相对公平。在一个天平上，你得到越多，势必要承受更多，每一个看似低的起点，都是通往更高峰的必经之路。
9、你要学会捂上自己的耳朵，不去听那些熙熙攘攘的声音；这个世界上没有不苦逼的人，真正能治愈自己的，只有你自己。
10、时间会告诉你一切真相。有些事情，要等到你渐渐清醒了，才明白它是个错误；有些东西，要等到你真正放下了，才知道它的沉重。
