再冷的石头，坐上三年也会暖

发表于 2018-06-14 | 分类于 励志的话

1、多一分心力去注意别人，就少一分心力反省自己。人生舞台的大幕随时都可能拉开，关键是你愿意表演，还是选择躲避。能把在面前行走的机会抓住的人，十有八九都会成功。
2、再长的路，一步步也能走完，再短的路，不迈开双脚也无法到达。有志者自有千计万计，无志者只感千难万难。我成功因为我志在成功！再冷的石头，坐上三年也会暖。
3、平凡的脚步也可以走完伟大的行程。有福之人是那些抱有美好的企盼从而灵魂得到真正满足的人。如果我们都去做自己能力做得到的事，我们真会叫自己大吃一惊。
4、只有不断找寻机会的人才会及时把握机会。人之所以平凡，在于无法超越自己。无论才能知识多么卓著，如果缺乏热情，则无异纸上画饼充饥，无补于事。
5、你可以选择这样的“三心二意”：信心恒心决心；创意乐意。征服畏惧建立自信的最快最确实的方法，就是去做你害怕的事，直到你获得成功的经验。

6、励志的话，为明天做准备的最好方法就是集中你所有智慧，所有的热忱，把今天的工作做得尽善尽美，这就是你能应付未来的唯一方法。
7、人性最可怜的就是：我们总是梦想着天边的一座奇妙的玫瑰园，而不去欣赏今天就开在我们窗口的玫瑰。行动不一定带来快乐，而无行动则决无快乐。
8、世上最重要的事，不在于我们在何处，而在于我们朝着什么方向走。失去金钱的人损失甚少，失去健康的人损失极多，失去勇气的人损失一切。
9、虽然你的思维相对于宇宙智慧来说只不过是汪洋中的一滴水，但这滴水却凝聚着海洋的全部财富；是质量上的一而非数量上的一；你的思维拥有一切宇宙智慧。
10、无论才能知识多么卓著，如果缺乏热情，则无异纸上画饼充饥，无补于事。如同磁铁吸引四周的铁粉，热情也能吸引周围的人，改变周围的情况。

相信就是强大，怀疑只会抑制能力

发表于 2018-06-14 | 分类于 励志的话

1、成功不是将来才有的，而是从决定去做的那一刻起，持续累积而成。世上没有绝望的处境，只有对处境绝望的人。世界上那些最容易的事情中，拖延时间最不费力。
2、明天的希望，让我们忘了今天的痛苦。能把在面前行走的机会抓住的人，十有八九都会成功。没有哪种教育能及得上逆境。为别人鼓掌的人也是在给自己的生命加油。
3、目标的坚定是性格中最必要的力量源泉之一，也是成功的利器之一。没有它，天才也会在矛盾无定的迷径中徒劳无功。你可以用爱得到全世界，你也可以用恨失去全世界。

4、再长的路，一步步也能走完，再短的路，不迈开双脚也无法到达。在世界的历史中，每一伟大而高贵的时刻都是某种热忱的胜利。大多数人想要改造这个世界，但却罕有人想改造自己。
5、有志者自有千计万计，无志者只感千难万难。相信就是强大，怀疑只会抑制能力，而信仰就是力量。人之所以能，是相信能。不要问别人为你做了什么，而要问你为别人做了什么。
6、鼓励，如果你还可以努力可以付出，就不要轻言停止和放弃。在你停止努力的那一刻之前，一切都还没有什么真正的结果。

忍别人所不能忍的痛，吃别人所不能吃的苦

发表于 2018-06-14 | 分类于 励志的话

1、人是不能太闲的，闲久了，努力一下就以为在拼命。别说你会尽力，努力，全力，用吃奶的力，这些都不是最大的力，要使出自己的洪荒之力，一个人如果不逼自己一把，永远不知道自己有多优秀。
2、永远不要放弃自己，这也许只是生活对你的一点小考验，相信自己，你能处理好。活在当下，别在怀念过去或者憧憬未来中浪费掉你现在的精力。
3、青春不奋斗，你的青春用来干嘛？也许终点只有绝望和失败，但这绝不是停止前行的理由。其实，一件事做得到和做不到，只是在你的一念之间。
4、没有不进步的人生，只有不进取的人！相信自己能力的人，任何事情都能够做到。忍别人所不能忍的痛，吃别人所不能吃的苦，是为了收获得不到的收获。
5、真正在努力的，日子是不会舒服的。而只要自己变优秀了，其他的事情自然就会跟着好起来。岁月很长，人海茫茫，别回头也别将就，自然会有良人来爱你。

6、做有用的事，说正能量的话，想美好的事，睡安稳的觉。把时间发生在进步上，而不是抱怨上。如果事与愿违，就相信上天一定另有安排；所有失去的，都会以另外一种方式归来。
7、努力把日子都填满，别让孤单把你包围，请善待珍惜自己，给自己一个坚强的理由，生活中没有什么过不去的坎。如果你觉得生命里的每扇门都关着，那请记住这句话：关上的门不一定上锁，至少再过去推一推。
8、不要去羡慕人家过得有多好，抱怨自己的生活有多糟，也不要活在过去的光环里，要知道昨天的太阳，晒不干今天的衣裳，世上有不绝的风景，人更要有前行的心情。
9、心情就像衣服，脏了就拿去洗洗，晒晒，阳光自然就会蔓延开来。阳光那么好，何必自寻烦恼，过好每一个当下，一万个美丽的未来抵不过一个温暖的现在。
10、励志的句子，有些人与事，是你必须要路过的驿站，好坏，成败，聚散，爱恨，都无所谓，它只是帮你尝遍人生百味，看淡世事冷暖。所有的经历，都是成长的良药。

人生只有迎难而上，才能抓住艰难背后的机遇

发表于 2018-06-14 | 分类于 励志的话

1、也许你一生中走错了不少路，看错不少人，承受了许多的叛逆，落魄得狼狈不胜，但都无所谓，只要还活着，就总有盼望，余生很长，何必慌张。
2、即便遭遇了人间最大的不幸，能够处理一切困难的前提是活着。只有活着，才有盼望。无论多么痛苦、多么哀痛，只要能够勤奋地活下去，一切都会好起来。
3、只要勇于去博，英勇去闯，就可闯出一片属于自己天地，以实现人生出色。不管结局能否完美，至少你享受拼搏的过程，就是人生的成功，就是胜者。
4、改变，永久不嫌晚。无论你是几岁，也无论你目前所处的境况有多糟，只要立定目的、一步一步往前走，人生随时都有翻盘的可能性。新的一天，你要加油！
5、梦想似乎是遥远的，但是只要通过自己的勤奋，就能一步步的拉近之间的间隔，当间隔越来越短时，它就不再是梦，而成为了触手可及的想法，无论如何，不要停下走向梦想的脚步。

6、励志的句子，生活中总有阳光和雾霾，风雨和闪电，只要记住我们想要什么？该做什么？应该怎么做？以一颗善心真诚待人，做好我们该做的事，无悔人生就行。
7、生活中，欢乐和懊恼陪伴着我们一生！人这一辈子不成能风调雨顺的，总有这样那样的快乐事懊恼事或交替或同时存在，我们要做的只能是顺境时勤奋，顺境时迎难而上。
8、生活，一半是回忆，一半是继续。把所有不快给昨天，把所有的希望给明天，把所有的努力给今天。努力是人生的一种精神状态，往往最美的不是成功的那一刻，而是那段努力奋斗的过程。朋友，愿你努力后的明天更精彩。
9、现实生活中，每个人的思想都很丰富，但是能够持之以恒地坚持做一件事情的人并不多。所以，有人成功，有人失败，有人笑，有人哭，关键看你是否能坚持下去。
10、幸运只喜欢坚定执著的人。越是恶劣的环境，越是成功的契机，倘若临阵退缩，你永远不会得到幸运之神的眷顾；迎难而上，才能抓住艰难背后的机遇。
