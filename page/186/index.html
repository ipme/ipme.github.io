幸福不是靠别人来布施，而是要自己去赢取！

发表于 2018-06-15 | 分类于 人生格言励志

1、不论你在什么时候开始，重要的是开始之后就不要停止。第一个青春是上帝給的；第二个的青春是自己努力的。人生最大的挑战没过于战胜自己！
2、愚痴的人，一直想要别人了解他。有智慧的人，却努力的了解自己。生命的道路上永远没有捷径可言，只有脚踏实地走下去。
3、常求有利别人，不求有利自己。把气愤的心境转化为柔和，把柔和的心境转化为爱，如此，这个世间将更加完美。
4、一份耕耘，一份收获，付出就有回报永不遭遇过失败，因我所碰到的都是暂时的挫折。不如意的时候不要尽往悲伤里钻，想想有笑声的日子吧。
5、平静的湖面只有呆板的倒映，奔腾的激流才有美丽的浪花！幸福不是靠别人来布施，而是要自己去赢取！生命的意义在不断挑战自己，战胜自己！

6、你接受比抱怨还要好，对于不可改变的事实，你除了接受以外，没有更好的办法了。输了就是输了，没什么好去解释的，不过是一盘棋，何必去计较。
7、人生格言，开始一种恐慌，停留与不可复制的回忆。谁把安逸当成幸福的花朵，那么等到结果时节，他只能望着空枝叹息。
8、瀑布为了奔向江河湖海，即使面临百丈深渊，仍然呼啸前行，决不退。没有平日的失败，就没有最终的成功。重要的是分析失败原因并吸取教训。
9、向你的美好的希冀和追求撒开网吧，九百九十九次落空了，还有一千次呢。如果缺少破土面出并与风雪拚搏的勇气，种子的前途并不比落叶美妙一分。
10、人生如棋，走一步看一步是庸者，走一步算三步是常者，走一步定十步是智者。崇高的理想就象生长在高山上的鲜花。如果要搞下它，勤奋才能是攀登的绳索。

等时机成熟时，理想就可以实现了

发表于 2018-06-15 | 分类于 人生格言励志

1、对于攀登者来说，失掉往昔的足迹并不可惜，迷失了继续前时的方向却很危险。不要让追求之舟停泊在幻想的港湾，而应扬起奋斗的风帆，驶向现实生活的大海。
2、桂冠上的飘带，不是用天才纤维捻制而成的，而是用痛苦，磨难的丝缕纺织出来的。让自己一边成熟，一边寻找时机。等时机成熟时，理想就可以实现了。
3、就算学习和生活再艰难，也要一边痛着，一边笑着，给生活一张漂亮的脸。意志力是人的一条救生索，它可以帮助我们脱离困境，引导我们走向胜利。
4、人生格言，生活就像一面镜子，你若对她笑，她就对你笑。沙漠里的脚印很快就消逝了。一支支奋进歌却在跋涉者的心中长久激荡。
5、你既然认准一条道路，何必去打听要走多久。让生活的句号圈住的人，是无法前时半步的。松软的沙滩上最容易留下脚樱钽也最容易被潮水抹去。

6、蝴蝶如要在百花园里得到飞舞的欢乐，那首先得忍受与蛹决裂的痛苦。萤火虫的光点虽然微弱，但亮着便是向黑暗挑战。过去不等于未来。
7、没有失败，只有暂时停止的成功。士搏出惊涛骇流而不沉沦，懦夫在风平浪静也会溺水。志在峰巅的攀登者，不会陶醉在沿途的某个脚印之中。
8、如果惧怕前面跌宕的山岩，生命就永远只能是死水一潭。懦弱的人只会裹足不前，莽撞的人只能引为烧身，只有真正敢的人才能所向披靡。
9、我们这个世界，从不会给一个伤心的落伍者颁发奖牌。梯子的梯阶从来不是用来搁脚的，它只是让人们的脚放上一段时间，以便让别一只脚能够再往上登。
10、生命之长短殊不重要，只要你活得快乐，在有生之年做些有意义的事，便已足够。活在忙与闲的两种境界里，才能俯仰自得，享受生活的乐趣，成就人生的意义。

不要因为众生的无知，而痛苦了你自己

发表于 2018-06-15 | 分类于 人生格言励志

1、一个从来没有失败过的人，必然是一个从未尝试过什么的人。不要因为众生的愚疑，而带来了自己的烦恼。不要因为众生的无知，而痛苦了你自己。
2、许多时候，我们不是跌倒在自己的缺陷上，而是跌倒在自己的优势上，因为缺陷常常给我们以提醒，而优势却常常使我们忘乎所以。
3、人总爱跟别人比较，看看有谁比自己好，又有谁比不上自己。而其实，为你的烦恼和忧伤垫底的，从来不是别人的不幸和痛苦，而是你自己的态度。
4、把你的脸迎向阳光，那就不会有阴影。思想如钻子，必须集中在一点钻下去才有力量。放弃，不是生命的绝对。
5、当你快乐的时候，生活是美好的，当有人因你而快乐的时候，生活才是最美好的。与其在别人的生活里跑龙套，不如精彩做自己。

6、人生格言，路的尽头，仍然是路，只要你愿意走。你被拒绝的越多，你就成长得越快；你学的越多，就越能成功。
7、勇敢地接受挑战，不断地超越自我，这样才能发出你的无限潜能。用自己的双手去创造生活，用辛勤的汗水实现人生的梦想。
8、一份信心，一份努力，一份成功；十分信心，十分努力，十分成功。许多人缺少的不是美，而是自信的气质。

不登高山不知天之高也，不临深溪不知地之厚也

发表于 2018-06-15 | 分类于 人生格言励志

1、人生能有几回搏？现在不搏更待何时？珍惜机遇吧，让金色的年华碰撞出更加灿烂的火花！上帝从不埋怨人们的愚昧，人们却埋怨上帝的不公平。
2、以解决自己的问题为目标，这是一个实实在在的道理，正视自己的问题，设法解决它，这是成功的捷径。谁能塌下心来把目光凝集在一个个小漏洞、小障碍上，谁就先迈出了一大步。
3、高考中没有失败，它带给每个人的深刻思考、刻骨铭心的经历和感受都是不可多得的财富。我们为理想而奋进的过程，其意义远大于未知的结果。
4、记住，别轻易地放弃叩门，成功会在你下一次叩门时，微笑着迎接你的。命运之神关上一道门，必定会打开另一扇窗。
5、世间最容易的事是坚持，最难的事也是坚持。要记住，坚持到底就是胜利。每一个成功者都有一个开始。勇于开始，才能找到成功的路。

6、人生格言，吃别人吃不了的苦，忍别人受不了的气，付出比别人更多的，才会享受的比别人更多。站在巨人的肩上是为了超过巨人。
7、当你感到悲哀痛苦时，最好是去学些什么东西。学习会使你永远立于不败之地。人生舞台的大幕随时都可能拉开，关键是你愿意表演，还是选择躲避。
8、少一点预设的期待，那份对人的关怀会更自在。生命力的意义在于拚搏，因为世界本身就是一个竞技场。不登高山不知天之高也，不临深溪不知地之厚也。
