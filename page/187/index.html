做人可以不聪明，但一定要有分寸感

发表于 2018-06-15 | 分类于 人生格言励志

生活告诉我，这个世上真心不多，能谈心事的人更少。遇事的时候别想太多，别太把自己逼迫，努力过就好，尽心了就行。与人相处，要有容人的雅量，更要有沉稳的气度。
生活告诉我，人这辈子有得有失，太多东西左右不了。不管是感情还是生活，拥有的要去珍惜，这样丢失了，不至于憾恨。攀爬途中，结果不是最终目的，一路的风景才是你的人生。
不要嫌父母老、爹妈脏，因为他们是世界上最不嫌你脏的人。不要轻易评价别人，因为你没有经历过他的人生。
人生格言，女人没魅力才觉得男人花心，男人没实力才觉得女人现实。在人之上要看得起人，在人之下要看得起自己。

做人可以不聪明，但一定要有分寸感。每个人都要找到自己的位置，应该是你的，才是你的；不应该是你的，连搭腔都不要。
人这一辈子的财富，不是你拥有多少流动资产，拥有多少不动资产；而是你帮助了多少人，支持了多少人，影响了多少人，爱过多少人，在你离开这个世界的时候还有多少人想念你，怀念你，而不是让大家觉得你就是个有钱人！
男人再牛，一旦出门在外，帽子的颜色，随时由妻子决定。任何男人千万不要为了事业让女人独守空房，否则下场会很惨！当然了，想换老婆的男人除外！
女人长相再美，身材再好，个子再高，学历再高，如果不懂得矜持，没有温柔的性格，没有好的人品，就像吃美味的大餐时看见餐里的死苍蝇，不吃可惜，吃了恶心。

只要你还生活在地球上，就别把自己看得太大

发表于 2018-06-15 | 分类于 人生格言励志

失去的东西，其实从未曾真正地属于你，不必惋惜，更不必追讨。生活累，一小半源于生存，一小半源于欲望与攀比。生活本不苦，苦的是欲望过多；心本无累，累的是放不下的太多。
不要让太多昨天占据你的今天对自己好点，因为一辈子不长；对身边的人好点，因为下辈子不一定能够遇见。
所谓缘分，就是遇见了该遇见的人，望你珍惜，每一个都不会无缘无故出现在你的生命中，珍惜你的今天，珍惜你所有的遇见！
对不起是一种真诚，没关系是一种风度。如果你付出了真诚，却得不到风度，那只能说明对方的无知与粗俗。只要你的脚还在地面上，就别把自己看得太轻；只要你还生活在地球上，就别把自己看得太大。
一个人只有一个心脏，却有两个心房，一个住着快乐，一个住着悲伤，不要笑得太大声，不然会吵醒旁边的悲伤。
爱情使人忘记时间，时间也使人忘记爱情，能在一起就不能在一起吧，其实一辈子也没那么长！珍惜每一次相逢吧。

烦恼，忘了吧，痛苦，忘了吧，不满，忘了吧，委屈，忘了吧。所有让自己不开心的事，统统都忘了吧。生命只有一次，你我只活一回，与其背着烦恼活，不如带着美好过！
人生格言，人生有让步，有大度，有失望，有痛苦，我们掌控不了别人，却能控制自己。不开心的事，忘掉；不真诚的人，忘掉。
人生旅途，哪能事事顺利；人活一世，哪能样样尽心。很多事情，想多了头疼，记久了心痛，不如忘得干干净净，还自己一份宁静。有些人，该忘就忘了吧！你念念不忘，伤的是自己；你一直惦记，过得不开心。
有些事，该忘就忘了吧！过去的回不来，未来的再继续。只有忘掉不好的，才能记住最好的；只有忘掉烦心的，才能留下开心的。忘不是糊涂，而是智慧；忘不是无能，而是大度。
忘让人减少烦恼，忘让心变得宽阔。忘，不是失去记忆，而是放宽心态；忘，不是糊里糊涂，而是学会装傻；忘，不是忘记美好，而是清除悲伤。

不攀不比，才够幸福；不计不较，才够快乐

发表于 2018-06-15 | 分类于 人生格言励志

对信任你的人，永远别撒谎，对你撒谎的人，永远别信任。一些时间，总会看清一些事，一些事情，总会看清一些人。永远不要企望别人都与你相同，我们只需坦坦荡荡，问心无愧。
岁月，留不住虚幻的拥有。有心的人，不管你在与不在，都会惦念；无心的情，无论你好与不好，只是漠然。有些事，不明不白，让人猜不透；有些人，戴着面具，让人看不清；有些理，概念模糊，让人悟不出；有些路，坎坷难走，让人行不通。
世态可以炎凉，做人，不要丢掉善良；世界可以混乱，内心，不可以肮脏；有些话，能不说就沉默，藏在心里更适合；有些伤，能不揭就不放下，无声忘记更明智；活着，就是一种修行，修一颗从容达观的心，就会轻松、自在、洒脱。

人心，贵再豁达，贵在容纳，贵在平和。不气不急，才够大度；不焦不燥，才够从容；不悲不忧，才够坚强；不惆不怅，才够阳光；不嫉不妒，才够优秀；不攀不比，才够幸福；不计不较，才够快乐。
人生大舞台，风云变幻，留有余地，就是不把事情做绝，于情不偏激，于理不过头，这样才会处变不惊，从容不迫，游刃有余。处事须留余地，责善切戒尽言。利不可赚尽，福不可享尽，势不可用尽。生而为人实属不易，给自己给他人留有余地。
留有余地，使自己行不至于绝处，言不至于极端，有进有退，措置裕如，断尽别人的路径，自己路径亦危；敲碎别人的饭碗，自己饭碗也脆。不让别人为难，也是不让自己为难，让别人活得轻松，自己也活得轻松。
人生格言：不说过头话，不做心力用尽的事，则随时可以应付意外情况。因为于人本身，强中自有强中手，而于事态发展，也时有不测风云。

缘分有聚散，有人能随行，就是一种温暖

发表于 2018-06-15 | 分类于 人生格言励志

人生最难是相处，人心永远最难懂。一路走来的我们，总有遇人不淑，错把鱼目当珍珠。总有交友不慎，错把套路当深情。人这辈子，不怕真敌人，就怕假朋友。
这一生遇人无数，我们用善良，惯坏了多少不知感恩的人；我们用容忍，纵容了多少拿咱当傻子的人。用得着你时，称兄道弟；用不着你时，翻脸无情。永远记住：宁交一帮抬杠鬼，不交一个甜嘴贼。
对自私自利的人付出，是肉包子打狗，有去无回，只会索取，没有回馈。对得寸进尺的人付出，是百次帮助不记恩，半次不帮就记恨。这个世上，不一定真心就会换回真心，没良心的人只会让你寒心。
是我们太天真？还是世人城府深？是我们太顾及别人？还是狠不下来违背本心？谁懂你有求必应？谁知你情绪埋心？谁解你善良简单？谁疼你心肠很软？

只有我们自己最清楚：骗过我的人，绝不会再信，相信我的人，绝不会去骗！伤过我的人，绝不会再爱，爱着我的人，绝不会去伤！
人在旅途，总有一些不期而遇的相逢，也总有一些失之交臂的陌生。缘分有聚散，有人能随行，就是一种温暖；人心有冷暖，有人能懂得，就是一种幸福。
人生格言，有多少缘，从相濡以沫转变到相忘江湖；有多少情，从海誓山盟演变到萍水相逢。缘来是你，缘去是空；心字好写，人心难懂。
每个人的时间都不多，没有无事可做的付出；每个人的心也只有一颗，没有无缘无故的关注。看不到回报，终会退出；收不到回应，总会寒心。不要等到失去了，才去后悔；不要等到错过了，才知道遗憾。
对你好的人，并不是对谁都那么好；一直守着你的人，才是难得的一腔深情。而我们都是远视眼，忽略了离我们最近的幸福。我们也都是近视眼，模糊了离我们最真的情感。
人之相处在于真，情之相守在于心。缘不贵多，心诚则行；情不论久，心惜则浓。真心对一个人好，无所谓有所回报；默默付出一份情，只在乎重不重要。
