三更灯火五更鸡，正是男儿读书时

发表于 2018-06-15 | 分类于 珍惜时间

1、一切节省，归根到底都归结为时间的节省。利用时间是一个极其高级的规律。今天所做之事勿候明天，自己所做之事勿候他人。今天应做的事没有做，明天再早也是耽误了。
2、浪费时间是一桩大罪过。迁延蹉跎，来日无多，二十丽姝，请来吻我，衰草枯杨，青春易过。普通人只想到如何度过时间，有才能的人设法利用时间。
3、人生有一道困难，那就是如何使一寸光阴即是一寸生命。时间就是生命，时间就是速度，时间就是气力。最严重的浪费就是时间的浪费。时间是衡量事业的标准。

4、时间，天天得到的都是二十四小时，可是一天的时间给勤勉的人带来聪明和气力，给懒散的人只留下一片悔恨。落日无边江不尽，此身此日更须忙。
5、世界上最快而又最慢，最长而又最短，最平凡而又最珍贵，最轻易被人忽视，而又最令人后悔的就是时间。你热爱生命吗？那幺别浪费时间，由于时间是组成生命的材料。
6、珍惜时间，在今天和明天之间，有一段很长的时间；趁你还有精神的时候，学习迅速办事。莫轻易，白了少年头，空悲切。英语名言警句。岁往弦吐箭。

黑发不知勤学早，白发方悔读书迟

发表于 2018-06-15 | 分类于 珍惜时间

1、你热爱生命吗？那么别浪费时间，因为时间是组成生命的材料。成功＝艰苦劳动＋正确的方法＋少说空话。放弃时间的人，时间也放弃他。没有方法能使时钏为我敲已过去了的钟点。
2、我们若要生活，就该为自己建造一种充满感受思索和行动的时钟，用它来代替这个枯燥单调以愁闷来扼杀心灵，带有责备意味和冷冷地滴答着的时间。
3、人的全部本领无非是耐心和时间的混合物。任何节约归根到底是时间的节约。时间就是能力等等发展的地盘。时间是世界上一切成就的土壤。时间给空想者痛苦，给创造者幸福。

4、时间是伟大的导师。时间是一个伟大的作者，它会给每个人写出完美的结局来。时间最不偏私，给任何人都是二十四小时；时间也是偏私，给任何人都不是二十四小时。
5、忘掉今天的人将被明天忘掉。辛勤的蜜蜂永没有时间的悲哀。在所有的批评中，最伟大最正确最天才的是时间。从不浪费时间的人，没有工夫抱怨时间不够。
6、珍惜时间，时间是我的财产，我的田亩是时间。春光不自留，莫怪东风恶。抛弃今天的人，不会有明天；而昨天，不过是行去流水。抛弃时间的人，时间也抛弃他。

一寸光阴一寸金，寸金难买寸光阴

发表于 2018-06-15 | 分类于 珍惜时间

1、三更灯火五更鸡，正是男儿读书时，黑发不知勤学早，白发方悔读书迟。一寸光阴一寸金，寸金难买寸光阴。少年易学老难成，一寸光阴不可轻。吾生也有涯，而知也无涯。
2、少壮不努力，老大徒伤悲。时间的步伐有三种：未来姗姗来迟，现在像箭一样飞逝，过往永远静立不动。时间就是生命，无故的空耗别人的时间，实在无异于谋财害命的。
3、谁对时间最吝啬，时间对谁越慷慨。要时间不辜负你，首先你要不辜负时间。放弃时间的人，时间也放弃他。把活着的每一天看作生命的最后一天。

4、盛年不重来，一日难再晨；及时当勉励，岁月不待人。皇皇三十载，书剑两无成。山川满目泪沾衣，富贵荣华能几时。不见只今汾水上，唯有年年秋雁飞。
5、珍惜时间，时而言，有初中后之分；日而言，有今昨明之称；身而言，有幼壮艾之期。等时间的人，就是浪费时间的人。谁把一生的光阴虚度，便是抛下黄金未买一物。
6、时间能使隐藏的事物显露，也能使灿烂夺目的东西黯然无光。时间就是生命，无端的空耗别人的时间，其实无异于谋财害命的。少年易老学难成，一寸光阴不可轻。

与其念念不忘，不如适时放手

发表于 2018-06-15 | 分类于 爱情说说

1、别傻了，好聚好散只是你最后安慰自己的借口，他是怎么对你的，没有人能比你更清楚，你难过的时候，他没有陪着你，你挽留的时候，他没有珍惜你，说白了，他就是不爱你。
2、哪怕心里再在意，脸上总要作出一副满不在乎的样子，这样，失去的时候，就可以很酷地走开，所以我不会上前去要一个拥抱，不会说我会想你，更不会问，你可不可以留下来。
3、事事不如意，就像我一样处处不讨你喜。有人说，时间让人忘记疼，我不觉得，时间只能让人习惯疼。

4、行李我会自己扛上楼，冷了我会自己添衣服，发着烧我也可以坚持工作下班去打针，害怕麻烦别人也不爱抱怨，像我这样百毒不侵的女汉子，真的活该单身一辈子。
5、我有一万个去见你的理由，唯独没有一个去见你的身份，愿我们各自安好。如果上天再给我一次选择的机会，我还是会跟你在一起，虽然分开的时候那么痛，但是那是我自己选的，我爱你。
6、伤感爱情，分手后一定要忍住，与其念念不忘，不如适时放手，他可能爱你，只是不够深爱，好的感情不是一味的单方面付出，没有回报的感情就放弃吧，好好爱自己。
7、每个人的一生里，都会遇见一个没有办法在一起的人，很多时候，我们以为那是无法抵御的强烈爱情，最后经历悲痛分别，以为人生的遗憾不过如此了，时过境迁，再回头看看那些荒唐岁月，竟要感谢当初的选择。
8、有那么一瞬间突然觉得，我所有的等待在你眼里都没有意义，因为换不来你的任何珍惜，不是我不懂得坚持，是太久没听到你的回音，你准备好了吗，曾经那么爱你的我，这次真的要走了。
9、你没挽留，我没回头，如此余生各自安好，也没有谁不好，只是时间不凑巧。不是放弃你了，是我终于放过了自己，想你是真的，回不去也是真的。
10、最深的孤独，是你明知道自己的渴望，却得对它装聋作哑。有种人太傻，人前伪装，人后悲伤，什么都喜欢一人扛，整天让自己受伤，还在意别人怎样想。
