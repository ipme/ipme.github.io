好饭先尽爹娘用，好衣先尽爹娘穿

发表于 2018-06-14 | 分类于 短句子

1、亲善产生幸福，文明带来和谐。在子女面前，父母要善于隐藏他们的一切快乐、烦恼和恐惧。
2、一个人如果使自己的母亲伤心，无论他的地位多么显赫，无论他多么有名，他都是一个卑劣的人。父母者，人之本也。
3、年老受尊敬是出现在人类社会里的第一种特权。孝子不谀其亲，忠臣不谄其君，臣子之盛也。孝子之养也，乐其心，不违其志。
4、事其亲者，不择地而安之，孝之至也。我们体贴老人，要像对待孩子一样。孝子之至，莫大乎尊亲。开始吧，孩子，开始用微笑去认识你的母亲吧！
5、树欲静而风不止，子欲养而亲不待。贤不肖不可以不相分，若命之不可易，若美恶之不可移。老年人犹如历史和戏剧，可供我们生活的参考。

6、重资财，薄父母，不成人子。礼者，断长续短，损有余，益不足，达爱敬之文，而滋成行义之美也。慢人亲者，不敬其亲者也。
7、爹娘面前能尽孝，一孝就是好儿男；翁婆身上能尽孝，又落孝来又落贤。天地之性，人为贵；人之行，莫大于孝，孝莫大于严父。
8、出入扶持须谨慎，朝夕伺候莫厌烦。在父母的眼中，孩子常是自我的一部分，子女是他理想自我再来一次的机会。
9、慈孝之心，人皆有之。家贫知孝子，国乱识忠臣。无父无君，是禽兽也。谁言寸草心，报得三春晖。好饭先尽爹娘用，好衣先尽爹娘穿。
10、感恩父母，儿子与父母亲的关系相处得不好，不可以做人；儿子不能事事顺从父母亲的心意，便不能做为儿子。

春天是碧绿的天地，秋天是黄金的世界

发表于 2018-06-14 | 分类于 短句子

1、春天是碧绿的天地，秋天是黄金的世界。愿你用青春的绿色去酿造未来富有的金秋！衷心地祝贺你，用智慧才情胆略和毅力，开辟出一块属于你自己的土地。
2、希望昕雨在新的一年里勇敢坚强，好好学习！执著是火，用倔强点燃希望；执著是光，用痴迷寻遍梦想。生命的轨迹遍洒奋斗的光芒，成长的年轮释放着无限能量。朋友，莫再彷徨迷惘，携带执著去远航。
3、成功的花，人们只惊慕她现时的明艳，然而它当初的芽儿，浸透着奋斗的泪泉，洒遍了牺牲的血雨。黎明即起，孜孜为善。”愿你热爱生命的春天，珍惜时间的清晨，学那梅花，争作“东风第一枝”。这些都可以呀！你自己挑选吧！

4、母爱是稀释在海水中的盐分，虽然看不见，但每一朵浪花都尝到了母爱的味道。母爱是沉淀在我们骨骼中的钙质，虽然摸不着，但母爱让我们在成长中更坚强！
5、他是参天大树为你支撑起一片天空，让你无忧无虑的成长；他是一盏明灯为你照亮前程，让你不至于迷失方向；他就是父亲，您的生日别忘了给生日快乐日的问候。
6、希望滨杰能自觉学习看书，不要再让大人催促才肯完成作业，愿他健康快乐！妈妈希望你学习一定要细心、认真、踏实。做人要真诚，对老师要尊敬，对同学要关心。望你迈好人生第一步。
7、孩子，生命也是一种莫大的赐予，每人都有自己的闪耀点，你不是得天独厚的一位，上苍也不会眷顾你什么，你要把握住你自己，认识自己的长处和不足，善于选择、勇于放弃，不要盲目和别人攀比，面对挑战，你要充满自信。

新年新气象，希望新的一年里努力学习

发表于 2018-06-14 | 分类于 短句子

1、新年新气象，希望新的一年里努力学习，增长知识，在班的百花园里采撷最丰硕的花果，允吸最甜美的甘露，滋养自己，丰富自己，提高自己。
2、父母寄语，最后，我要真心感谢各科老师：感谢你们对我孩子的关心和教导。真心说一声：老师您们辛苦了。
3、儿子，短短百天的时间，好似百米赛跑最后的冲刺。希望你抓紧时间，在各科老师辛勤教育下好好学习，争分夺秒，要虚心踏实，不懂多问；学习态度要积极向上，有恒心，有毅力。望你以优异的成绩完成初中学业。我也相信，有甘老师及各科老师的关爱和培养，你们五班的每个同学都是最棒的。因为你们拥有最好的老师，希望你要珍惜这短短的百天时光。
4、孩子，不要被黑夜中的噩梦，影响你醒来时的心境，不管这世界如何的变幻，不管这时光是多么的无情，任世间坎坷，绿肥红瘦；你可明白，生命从来不是美丽的水晶云梯，所以你要学会战胜痛苦，学会享受孤独，学会让自己变成光源，点燃那胸中神灵的智慧灯，并用内心的勇敢照亮前行的方向。我会祈祷，阴霾与灰暗不再纠缠于你。

5、叮叮铛，叮叮铛，铃儿响叮当。宝宝降临多快乐，把祝福歌儿唱。祝他身体强壮，祝他平安又健康，祝他幸福快乐蹦蹦跳跳茁壮地成长。祝您喜得贵子，合家欢乐。
6、成功要靠自己努力，哪怕只有万分之一的希望也不能放弃，坚信自己我能行！要学会担当，努力做一个对得起自己，勇于面对任何困难，经得起挫折，失败的人。懂得尊重别人，才可以成为一个被社会认可的人！
7、莲莲在第一学期各门学科都取得了不错的成绩，是离不开老师的正确教育，希望第二学期学习更认真努力，争取各方面取得更大的进步！
8、我总是在你明亮的目光中起航，在你微笑的目光中欢悦，在你鼓励的目光中成长，在你凝重的目光中思索，在你深情的目光中懂得，你就是我心弦上不逝的风景，父亲。父亲节到了，愿你健康快乐。

青春永不言弃，青年节快乐，茁壮成长！

发表于 2018-06-14 | 分类于 短句子

1、感谢生活的风波，让你成长，让你进步；感谢朋友的祝福，给我温馨，给我快乐；感恩节，祝愿最挂念的你永远幸福！
2、只有你在业务上勤奋，你才会变得博学；只有你勤俭节约，你才会变得富裕；只要你积德行善，你就会心情畅快，只要你持有一颗积极向上的心，你就会迎来成功！
3、天道酬勤。人世间没有不经过勤劳而成为天才的。愿你日夜勤奋，早日成才！高考的竞争是压力和挑战，也是机遇和希望，成功属于战胜自我的人。
4、你曾拥有阳光灿烂的昨天，你将迎来美丽快乐的明天，勇敢地面对未来的道路吧！一个人的真正伟大之处就在于他能够认识到自己的渺小。
5、在青春的世界里，平凡的沙子要变成光彩夺目的珍珠，普通的石头要化作闪闪发光的金子。这是青春的魅力，青春永不言弃，青年节快乐！茁壮成长！

6、父母寄语，敬爱的老师，谢谢您以辛勤的汗水滋润我们的心田，并抚育我们茁壮成长。愿您所有的日子都充满着幸福、欢乐与温馨！
7、你是一个聪明、懂事的小朋友，上课时能遵守纪律，认真听讲，有时还能积极举手发言，你的手很能干，画得画真美，你的舞跳得棒极了！课间你能和小朋友在一起快乐地玩耍，你的上进心、自尊心都很强。就要上小学了，如果你能严格要求自己，就更棒了！
8、家长对孩子的寄语，三年前，带着爸爸妈妈无尽的爱，你来到了我们的身边，你是如此的冰雪聪明，对于新鲜的东西你总是一学就会；你非常活跃，总是爱上窜下跳，妙语连珠；你敢于探索，总是喜欢问为什么并努力去尝试各种新奇的事物；你很勇敢，摔倒的时候会自己爬起，打针也不哭；你的笑容很美，充满阳光、纯真无邪，你给我们增添了无限欢乐，感谢你的到来！
9、愿你是一棵树：春天，吐一山淡淡的香味；夏天，洒一抹如泉的凉荫；秋天，举一树甜甜的青果；冬天，做一个养精蓄锐的好梦！
10、几乎每一个人都期望在人生的旅途上一帆风顺，在成长的道路上勇往直前。许多人都说过：“成长的道路上，即使没有莺歌燕舞，没有撑开的鲜花，那最好也没有风雨，没有弯路。但是我觉得：没有弯路的人生不是真正的人生，没有荆棘的成长不会使人真正地成长。
