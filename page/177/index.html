智慧之子使父亲欢乐，愚昧之子使母亲蒙羞

发表于 2018-06-14 | 分类于 短句子

1、我们几乎是在不知不觉地爱自己的父母，因为这种爱像人的活着一样自然，只有到了最后分别的时刻才能看到这种感情的根扎得多深。我们体贴老人，要像对待孩子一样。
2、老年人犹如历史和戏剧，可供我们生活的参考。无父无君，是禽兽也。父子有亲，君臣有义，夫妇有别，长幼有叙，朋友有信。母爱是一种巨大的火焰。
3、在孩子的嘴上和心中，母亲就是上帝。慢人亲者，不敬其亲者也。意思是：不尊敬别人父母的人，肯定也不会敬重自己的父母。智慧之子使父亲欢乐，愚昧之子使母亲蒙羞。
4、世界上有一种最美丽的声音，那就是母亲的呼唤。孝顺还生孝顺子，忤逆还生忤逆儿。丑恶的海怪也比不上忘恩的儿女那样可怕。
5、孝顺父母，君子有三乐，而王天下不与存焉。父母俱存，兄弟无故，一乐也；仰不愧于天，俯不怍于人，二乐也；得天下英才而教育之，三乐也。君子有三乐，而王天下不与存焉。

6、事其亲者，不择地而安之，孝之至也。父母者，人之本也。内睦者，家道昌。孝有三：大尊尊亲，其次弗辱，其下能养。凡为父母的，莫不爱其子。
7、世俗所谓不孝者五，惰其四支，不顾父母之养，一不孝也；博奕好饮酒，不顾父母之养，二不孝也；好货财，私妻子，不顾父母之养，三不孝也；从耳目之欲，以为父母戮，四不孝也；好勇斗狠，以危父母，五不孝也。
8、老人受尊敬，是人类精神最美好的一种特权。劳苦莫教爹娘受，忧愁莫教爹娘耽。意思是：不要让父母受苦受累，不要让父母分担你的忧愁。
9、孟子曰：“亲亲，仁也；敬长，义也。”孟子说：“亲爱父母亲，便是仁；尊敬兄长便是义。”孝子事亲，不可使其亲有冷淡心，烦恼心，惊怖心，愁闷心，难言心，愧恨心。
10、大孝终身慕父母。唯孝顺父母可以解忧。仁爱和打人都先自家中开始。惟孝顺父母，可以解忧。事父母，几谏，见志不从，又敬不违，劳而不怨。

我们永远需要报答最美好的人，这就是母亲

发表于 2018-06-14 | 分类于 短句子

1、在父母的眼中，孩子常是自我的一部分，子女是他理想自我再来一次的机会。孝子之养也，乐其心，不违其志。在这个世界上，我们永远需要报答最美好的人，这就是母亲。
2、不孝的人是世界最可恶的人。年老受尊敬是出现在人类社会里的第一种特权。夫孝，天之经也，地之义也。意思是：孝是天经地义的。开始吧，孩子，开始用微笑去认识你的母亲吧！
3、一个老年人的死亡，等于倾倒了一座博物馆。共产主义不仅表现在田地里和汗水横流的工厂，它也表现在家庭里、饭桌旁，在亲戚之间，在相互的关系上。
4、还有什么比父母心中蕴藏着的情感更为神圣的呢？父母的心，是最仁慈的法官，是最贴心的朋友，是爱的太阳，它的光焰照耀、温暖着凝聚在我们心灵深处的意向！
5、作为一个人，对父母要尊敬，对子女要慈爱，对穷亲戚要慷慨，对一切人要有礼貌。父母之年，不可不知也。一则以喜，一则以惧。

6、出入扶持须谨慎，朝夕伺候莫厌烦。意思是：父母出入（门）要小心搀扶，早晚伺候父母不要厌烦。老吾老，以及人之老；幼吾幼，以及人之幼。天下可运于掌。
7、孝敬父母经常可以代替最高贵的感情。慈善行及至亲，但不应仅此为止。礼者，断长续短，损有余，益不足，达爱敬之文，而滋成行义之美也。
8、你希望子女怎样对待你，你就怎样对待你的父母。一个人如果使自己的母亲伤心，无论他的地位多么显赫，无论他多么有名，他都是一个卑劣的人。
9、世界上的一切光荣和骄傲，都来自母亲。天地之性，人为贵；人之行，莫大于孝，孝莫大于严父。母亲，是唯一能使死神屈服的力量。
10、孝顺父母，黄昏，你把清晨驱散的一切收集回来；羊群归棚，孩子回到母亲身边。家庭的基础无疑是父母对其新生儿女具有特殊的情感。

好饭先给父母吃，好衣先给父母穿

发表于 2018-06-14 | 分类于 短句子

1、做为一个人，对父母要尊敬，对子女要慈爱，对穷亲戚要慷慨，对一切人要有礼貌。仁之实，事亲是也；义之实，从兄是也。侍于亲长，声容易肃，勿因琐事，大声呼叱。
2、不得乎亲，不可以为人；不顺乎亲，不可以为子。就是在我们母亲的膝上，我们获得了我们的最高尚、最真诚和最远大的理想，但是里面很少有任何金钱。
3、重资财，薄父母，不成人子。千万经典，孝义为先。意思是：成千上万部经典上都说，孝和义是人首先应当做到的。我看见过挨饿的孩子。我看见过母亲和妻子们的悲痛。我痛恨战争。
4、父子不信，则家道不睦。父母惟其疾之忧。谁言寸草心，报得三春晖。好饭先尽爹娘用，好衣先尽爹娘穿。意思是：好饭先给父母吃，好衣先给父母穿。
5、孟武伯问孝，子曰：“父母惟其疾之忧。”父母在，不远游，游必有方。有难事，弟子服其劳；有酒食，先生馔。曾是以为孝乎？

6、爹娘面前能尽孝，一孝就是好儿男；翁婆身上能尽孝，又落孝来又落贤。意思是：孝敬父母就是好儿男，孝敬公公、婆婆，能落个既孝敬又贤惠的名声。
7、家贫知孝子，国乱识忠臣。意思是：在家庭贫困的时候，才能发现真正的孝子，在国家危难的时候，才能识别真正的忠臣。父母之年，不可不知也。一则以喜，一则以惧。
8、孝顺父母，事亲以敬，美过三牲。父母呼，应勿缓；父母命，行勿懒。意思是：父母呼唤，要赶快答应；父母有命令，应赶快去做。
9、事，孰为大？事亲为大；守，孰为大？守身为大。不失其身而能事其亲者，吾闻之矣；失其身而能事其亲者，吾未闻也。孰不为事？事亲，事之本也；孰不为守？守身，守之本也。
10、贤不肖不可以不相分，若命之不可易，若美恶之不可移。所有杰出的非凡人物都有出色的母亲，到了晚年都十分尊敬自己的母亲，把他们当作最好的朋友。

我看着妈妈疲惫的身躯，不禁地流下泪来……

发表于 2018-06-14 | 分类于 短句子

有一种爱；是那么平凡而伟大。有一种爱；是那么难忘啊。有一种爱；是那么令人怀念啊。有一种爱……那，就是父母对我们的爱。他们对我的爱是多么无私啊！那次妈妈带我去看书的经历，让我深深地感受到了无私的母爱……
暑假妈妈带我去新华书店看书，因为妈妈有急事，所以我们约定5：00准时在大厅集合。当我看书时，我看到了一个我非常熟悉的身影。于是我拍拍他的肩膀，并叫他的名字。
他也认出了我，于是他约我一起去其他地方玩玩，我答应了，因为现在才1：50，离我和妈妈约定的时间还早。我们玩着玩着就忘记了时间，竟然玩到了7：10！
我赶紧跑到大厅。站在大厅的瓷砖上，我瞥到了一个瘦小的身影疲惫地倚在门上眼皮半垂下来，脚心使劲地往里头歪。那正是我的妈妈。咦？妈妈怎么这么累？我慢慢地走到妈妈跟前。
妈妈一看是我来了便张口就说：“你……这孩……孩……子，害……得……妈找……了一……一大……会！”“对不起，妈妈……我错了，我不应该回来……这么晚……”

我的声音越来越小，还不安的抠着手指。“行了，饭都做好了，快回家吃饭吧……”不能再这样了，害得妈妈找了这么长时间……
回家后，爸爸就开始数落我，说妈妈走了一个下午，腿都要跑不动了。晚上八点多，爸爸叫妈妈洗个脚。洗脚水打来了，妈妈脱掉袜子，我看见脚心里似乎有几颗水泡。
父母的爱，我看着妈妈脚上的水泡和疲惫的身躯，不禁地流下泪来……
朦胧间，我的脑中突然闪现出一幅又一幅画面：我想起由于上学太匆忙而忘了带走作业本，老爸发现后，跑步给我送到教室里；我想起了由于打篮球而忘了回家的时间，妈妈急急忙忙四处寻找我时的身影。
我想起了给出差在外的爸爸打电话时，话筒里他急切的声音：“保险丝在阳台上工具箱里，上保险丝时一定不要忘了先拉下闸刀”。
我想起了每遇到考试，出家门前妈妈的一句叮咛：“不要慌，相信自己。”太多太多的往事一瞬间涌出，让我不觉潸然泪下。
父母的爱就像百合的清香凝成的感动，慢慢地汇拢，不再飘散。我想可能世上没有人会说不爱自己的父母，但是他们能说自己能完全地回报父母的爱吗？
也许远远不能。在父母的眼中，爱就是和你一起走，走再远的路也不会疲倦，所以我永远不会忘记爸爸拿着我的作文草稿时热切的话语：“改一下，再改一下就可以誊了。”
我也不会忘记妈妈一大清早起来为我做早点的身影，一生过于漫长，此刻就是幸福。
写到这，我不禁想起了《圣经》中的先知以利亚用手遮住脸，不敢正视上帝的面容。我想，父母是能够享受到这个权利的，他们拥有“上帝”播洒到人间的圣物。
