对待生命要认真，对待生活要活泼

发表于 2018-06-15 | 分类于 励志的话

1、胜利不是战胜敌人，而是提高自己。我们只要每天进步百分之一，那就是成功。不要因为怕被玫瑰的刺伤到你，就不敢去摘玫瑰。
2、困难里包含着胜利，失败里孕育着成功。我年轻时注意到，我每做十件事有九件不成功，于是我就十倍地去努力干下去。骄傲是胜利的敌人，努力是成功的朋友。
3、你有多努力，就有多幸运。生活不是单行线，一条路走不通，你可以转弯。任何为失败找借口的人虽然他的心灵上得到了安慰，但是他将永远的拥有失败。
4、对待生命要认真，对待生活要活泼。能把在面前行走的机会抓住的人，十有八九都会成功。再长的路，一步步也能走完，再短的路，不迈开双脚也无法到达。
5、行动力强弱决定成功快慢。人生如果错了方向，停止就是进步。生活本是痛苦，是思想和哲理使其升华。如果你觉得现在走的辛苦，那就证明你在走上坡路！

6、在你渐渐迷失在你的人生道路上的时候，记得这句话：千万不要因为走的太久，而忘记了我们为什么出发。每一天都是一个开始。深呼吸，从头再来。
7、不要只因一次挫败，就忘记你原先决定想达到的远方。挫折其实就是迈向成功所应缴的学费。天下无难事，唯坚忍二字，为成功之要诀。
8、励志的话，一个成功的人是以幽默感对付挫折的。事业的成功没有止境，它是一场无终点的追求。如果你具备开始的勇气，就有了成功的豪情。
9、成功与不成功之间有时距离很短，只是后者再向前了几步。向往成功的桂冠，就别空费每一天。如果你能成功地选择劳动，并把自己的全部精神灌注到它里面去，那么幸福本身就会找到你。
10、机遇起着重要的作用，但另一方面，即使在那些因机遇而成功的发现中，机遇也仅仅起到一部分的作用。本来无望的事，大胆尝试，往往能成功。

只有不断找寻机会的人才会及时把握机会

发表于 2018-06-15 | 分类于 励志的话

1、未曾失败的人恐怕也未曾成功过。成功的第一个条件就是要有决心；而决心要不得迅速干脆果断又必须具有成功的信心。放弃谁都可以，千万不要放弃自己！
2、最重要的就是不要去看远方模糊的，而要做手边清楚的事。征服畏惧建立自信的最快最确实的方法，就是去做你害怕的事，直到你获得成功的经验。
3、为明天做准备的最好方法就是集中你所有智慧，所有的热忱，把今天的工作做得尽善尽美。世上最重要的事，不在于我们在何处，而在于我们朝着什么方向走。
4、人性最可怜的就是：我们总是梦想着天边的一座奇妙的玫瑰园，而不去欣赏今天就开在我们窗口的玫瑰。做对的事情比把事情做对重要。
5、只有不断找寻机会的人才会及时把握机会。一个人最大的破产是绝望，最大的资产是希望。熊熊的热忱，凭着切实有用的知识与坚忍不拔，是最常造就成功的品性。

6、从胜利学得少，从失败学得多。一个人是在对周围生活环境的反抗中创造成功的。大胆的尝试只等于成功的一半。失败是坚韧的最后考验。
7、成功的花，人们只惊慕她现时的明艳！然而当初她的芽儿，浸透了奋斗的泪泉，洒遍了牺牲的血雨。别人能做到的事情，我也能做到。
8、当一个小小的心念变成成为行为时，便能成了习惯；从而形成性格，而性格就决定你一生的成败。这个世界并不是掌握在那些嘲笑者的手中，而恰恰掌握在能够经受得住嘲笑与批评忍不断往前走的人手中。
9、励志的话，成功需要成本，时间也是一种成本，对时间的珍惜就是对成本的节约。行动是治愈恐惧的良药，而犹豫拖延将不断滋养恐惧。
10、忍别人所不能忍的痛，吃别人所别人所不能吃的苦，终能收获别人收获不到的收获。做一个决定，并不难，难的是付诸行动，并且坚持到底。

酸甜苦辣都是营养，成功失败都是经验

发表于 2018-06-15 | 分类于 励志的话

1、记住：你是你生命的船长；走自己的路，何必在乎其它。不去追逐，永远不会拥有。不往前走，永远原地停留。没有所谓失败，除非你不再尝试。
2、有时可能别人不在乎你，但你不能不在乎自己。路靠自己走，就算再坎坷，也要自己过。在穷的只剩下梦想的时候，你一定要比别人更加努力。
3、我们用人生最好的年华作抵押，去担保一个说出来都会被嘲笑的梦想。累么？累就对了，舒服是留给死人的！没有口水与汗水，就没有成功的泪水。
4、才华是血汗的结晶。才华是刀刃，辛苦是磨刀石。在一个崇高的目的支持下，不停地工作，即使慢、也一定会获得成功。成功的秘诀就是每天都比别人多努力一点。
5、成功的条件在于勇气和自信，而勇气和自信乃是由健全的思想和健康的体魄而来。成功之秘诀，在始终不变其目的。按照自己的意志去做，不要听那些闲言碎语，你就一定会成功。

6、谬误有多种多样，而正确却只有一种，这就是为什么失败容易成功难，脱靶容易中靶难的缘故。能把在面前行走的机会抓住的人，十有八九都会成功。
7、酸甜苦辣都是营养，成功失败都是经验。登高必自卑，自视太高不能达到成功，因而成功者必须培养泰然心态，凡事专注，这才是成功的要点。
8、励志的话，失败就是迈向成功应付出的代价。明智的人决不坐下来为失败而哀号，他们一定乐观地寻找办法来加以挽救。
9、要是一个人，能充满信心地朝他理想的方向去做，下定决心过他所想过的生活，他就一定会得到意外的成功。无论才能知识多么卓著，如果缺乏热情，则无异纸上画饼充饥，无补于事。
10、如果在胜利前却步，往往只会拥抱失败；如果在困难时坚持，常常会获得新的成功。认真是成功的秘诀，粗心是失败的伴侣。

比我差的人还没放弃，比我好的人仍在努力

发表于 2018-06-15 | 分类于 励志的话

1、你能用金钱买来的爱，别人也能用金钱把它买去。所谓骨气，“气”由“骨”定。骨头硬者，是浩然之气；骨头软者，是颓靡之气；无骨者，无气。
2、踩着垃圾到达的高度和踩着金子到达的高度是一样的。每个人都有潜在的能量，只是很容易：被习惯所掩盖，被时间所迷离，被惰性所消磨。
3、只有承担起旅途风雨，才能最终守得住彩虹满天。无论你觉得自己多么的不幸，永远有人比你更加不幸。努力过的人总是最美的，即使最后她没赢，至少不会骗了自己。
4、这个世界到处充满着不公平，我们能做的不仅仅是接受，还要试着做一些反抗。欢喜中持有一份凝重，悲哀时多留一丝希望；对生活的负责任，谨慎对待每一个属于自己的日子。

5、宁愿跑起来被拌倒无数次，也不愿规规矩矩走一辈子。就算跌倒也要豪迈的笑。没有一种不通过蔑视、忍受和奋斗就可以征服的命运。勇气不是感觉不到恐惧而是感觉到恐惧也继续做下去。
6、挤不进的世界就别挤了，何必为难了别人作贱了自己。现在不努力，将来拿什么向曾经抛弃你的人证明它有多瞎。没有人能一路单纯到底，但别忘了最初的自己。
7、励志的话，比我差的人还没放弃，比我好的人仍在努力，我就更没资格说我无能为力！你配不上自己的野心，也辜负了曾经历的苦难。
8、人生没有如果，只有后果和结果。少问别人为什么，多问自己凭什么。现在不玩命，将来命玩你，现在不努力，未来不给力。
9、我就是我，颜色不一样的烟火，天空海阔，要做最坚强的泡沫。每件事情都必须有一个期限，否则，大多数人都会有多少时间就花掉多少时间。
10、谁若游戏人生，他就一事无成；谁不主宰自己，永远是一个奴隶。有理想在的地方，地狱就是天堂。有希望在的地方，痛苦也成欢乐。
