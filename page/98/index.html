愿以我血献后土，换得神州永太平

发表于 2018-06-14 | 分类于 爱国名言

1、我正增加百倍的勇气和信心，奋励自励，为我们伟大祖国与伟大人民继续奋斗。几日随风北海游，回从扬子大江头。
2、我们中华民族有同自己的敌人血战到底的气概，有在自力更生的基础上光复旧物的决心，有自立于世界民族之林的能力。
3、现阶段，爱国主义主要表现为献身于建设和保卫社会主义现代化的事业，献身于促进祖国统一的事业。生是为中国，死是为中国。
4、愿以我血献后土，换得神州永太平。与其忍辱生，毋宁报国死。光明的中国，让我的生命为你燃烧吧。
5、我死以后，把我的骨灰送家乡……把它埋了，上头种一棵苹果树，让我最后报答家乡的土地，报答父老乡亲。

6、商女不知亡国恨，隔江犹唱后庭花。为了国家和人民的利益不怕自己吃亏的人，才是高尚的有道德的脱离了低级趣味的人。
7、你可知“妈港”不是我的真名姓？我离开你的襁褓太久了，母亲！三百年来梦寐不忘的生母啊！请叫儿的乳名，叫我一声“澳门”！母亲！我要回来，母亲！
8、我随时等待祖国的召唤！英雄非无泪，不洒敌人前。男儿七尺躯，愿为祖国捐。起来，不愿做奴隶的人们，把我们的血肉筑成我们新的长城！
9、平生爱国不甘后人，即使把我烧成灰，我也是爱国的。为了新中国，冲啊！三万里河东入海，万千仞岳上摩天。
10、爱国名言，我荣幸地从中华民族一员的资格，而成为世界公民。我是中国人民的儿子。我深情地爱着我的祖国和人民。

国家是大家的，爱国是每个人的本分

发表于 2018-06-14 | 分类于 爱国名言

1、宁可少活二十年，拼命也要拿下大油田。想从我手中抢走火炬，除非从我尸体上爬过去。风声雨声读书声声声入耳，国事家事天下是事事关心。
2、国耳忘家，公耳忘私。我不应把我的作品全归功于自己的智慧，还应归功于我以外向我提供素材的成千成万的事情和人物。
3、每一个党员干部，都应当与人民同甘苦共命运。这样，我们党才有威信，国家才有希望。臣心一片磁针石，不指南方不肯休。
4、国家是大家的，爱国是每个人的本分。归来报命日，恢复我神州。工人阶级要用自己的模范行动来带动全国人民建设社会主义！
5、为中华之崛起而读书。我死国生，我死犹荣，身虽死精神长生，成功成仁，实现大同。我是中国人民的儿子。我深情地爱着我的祖国和人民。

6、为了中华民族的繁荣富强，我要献出全部学识智慧。瞒人之事弗为，害人之心弗存，有益国家之事虽死弗避。团结起来，振兴中华！
7、中国人民的爱国主义和自力更生精神，是社会主义现代化建设的强大力量。如果一个人没国家民族观念，即使富有，也实在令人惋惜。
8、我所谓共和国里的美德，是指爱祖国也就是爱平等而言。这并不是一种道德上的美德，也不是一种基-督教的美德，而是政治上的美德。
9、为了全人类的和平与进步，中国人来到了太空。捐躯赴国难，视死忽如归。人有人格，报有报格，国有国格，三格不存，人将非人，报将非报，国将不国。
10、爱国名言，锦城虽乐，不如回故乡；乐园虽好，非久留之地。归去来兮。我们是国家的主人，应该处处为国家着想。

你必须跳下悬崖，在坠落空中生出翅膀

发表于 2018-06-14 | 分类于 正能量

1、别紧张，深呼吸，坚持住，扛过去。你既然认准一条道路何必去打听要走多久！我将一切回忆掩埋，只想拥有一个美好的未来。最值得欣赏的风景，是自己奋斗的足迹。
2、你都没竭尽全力，哪有资格哭泣。你永远都无法借别人的翅膀，飞上自己的天空。一个人的强大在于控制情绪，而不是外在型体。努力不一定成功，不努力一定不成功。
3、你必须跳下悬崖，在坠落空中生出翅膀。不被重重的击倒，又怎么能华丽的站起。没有人可以做你的双拐，你必须学会独立去闯荡。别把自己看的那么遭，其实你很好。
4、一个人拥有什么样的性格，就拥有什么样的世界。永远不要放弃你真正想要的东西。等待虽难，但后悔更甚。必须走完必经之路，才能走自己想走的路。
5、我本微末凡尘，但也心向天空。伤痕是士兵一生的荣耀。绝望的时侯抬头看着希望的光芒其实一直存在。不管别人的嘲笑，我要走自己想走的路。

6、人生，没有永远的伤痛，再深的痛，伤口总会痊愈。不相信奇迹的人永远都不会创造奇迹。是梦别怕疼，想赢别喊停！用微笑告诉别人，今天的我，比昨天更强，今后也一样。
7、我深知我不努力什么都不是。用你的笑容去改变这个世界，别让这个世界改变了你的笑容。让坚持成为一种信仰。你可知道我拼搏不怕阻挡，就一个理由，我要得到我想要的东西。
8、走过一些漫不经心，方知人生容不得放任。态度决定高度，习惯主宰人生。大道理人人都懂，小情绪却是难以自控。不甘心失去就要去努力争取。
9、正能量，任何你的不足，在你成功的那刻，都会被人说为特色。人因梦想而伟大。弱者才会诉苦，强者永远找方法！每条堵住的路，都有一个出口。
10、自己成功才能感悟生命壮观。你什么都没有，所以你只能拼命！人生就像一杯茶，不会苦一辈子，但总会苦一阵子。有些路很远，走下去会很累，可是不走，会后悔！

青春只有一次，别让自己过得不精彩

发表于 2018-06-14 | 分类于 正能量

1、使人疲惫的不是远方的高山，而是鞋里的一粒沙子。预测未来的最好方法，就是创造未来。多用善眼看世界；不必一味讨好别人。学会忘记痛苦，为阳光记忆腾出空间。
2、在真实的生命里，每桩伟业都由信心开始，并由信心跨出第一步。目标的坚定是性格中最必要的力量源泉之一，也是成功的利器之一。
3、障碍与失败，是通往成功最稳靠的踏脚石，肯研究、利用它们，便能从失败中培养出成功。微笑是我们心灵的最真诚倾诉，是在困难面前最好的良药。
4、没有哪种教育能及得上逆境。水只有碰到石头才能碰出浪花。生活不可能像你想的那么美好，但也不会像你想的那么糟。你要记住你不是为别人而活，你是为自己而活。
5、青春只有一次，别让自己过得不精彩。生活是个吃软怕硬的东西。你弱他就强，你强他就弱。不要去预见烦恼或担心可能永远不会发生的事情，置身于明媚的阳光之中吧！

6、我想要自由飞翔，这一刻，我对自己说，我不在胆怯。我要的未来，要靠我自己去拼。乌云的背后是阳光，阳光的背后是彩虹。永远成功的秘密，就是每天淘汰自己！
7、不要被任何人打乱自己的脚步，因为没有谁会像你一样清楚和在乎自己的梦想。就算路不坦荡，也要做自己的太阳！不念过往，不为未来，珍惜现在。
8、什么时候离光明最近？那就是你觉得黑暗太黑的时候。只要有勇气，整片蓝天便是你的。每个不满意的现在，都有个不努力的曾经。你的能量超乎你想象。
9、正能量，我的青春还在继续，再多荆棘也要向前奔跑。你就是一道风景，没必要在别人风景里面仰视。地球是运动的，一个人不会永远处在倒霉的位置。
10、人生只有出走的美丽，而没有等出来的辉煌。成功是分两半的，一半在上帝手中，那是宿命；另一半在自己手中，那是拼命。别放弃你的梦想，迟早有一天它会在你手里发光。
