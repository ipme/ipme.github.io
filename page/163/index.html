太阳有时也失约，黑夜却每天必来

发表于 2018-06-14 | 分类于 名言警句

1、生活是欺骗不了的，一个人要生活得光明磊落。——冯雪峰
2、自以为聪明的人往往是没有好下场的。世界上最聪明的人是最老实的人，因为只有老实人才能经得起事实和历史的考验。——周恩来
3、不要试图给你的生命增加时间，而要给你的时间赋予生命。
4、给人幸福的不是身体上的好处，也不是财富，而是正直和谨慎。——德谟可利特
5、质朴却比巧妙的言辞更能打动我的心。——莎士比亚

6、人在得意时须沉得住傲气；失意时则要忍得住火气。
7、名言警句：太阳有时也失约，黑夜却每天必来。
8、无论乌鸦怎样用孔雀的羽毛来装饰自己，乌鸦毕竟是乌鸦。——斯大林
9、最清晰的脚印是留在最泥泞的路上的。
10、让人失去理智的，常常是外界的诱惑；让人耗尽心力的，往往是自己的欲望。

生命不在于长短，而在于活得有意义

发表于 2018-06-14 | 分类于 名言警句

1、要不断提高自身的能力，才能益己及他。有能力办实事才不会毕竟空谈何益。故事的结束总是满载而归，就是金榜题名。——方海权
2、挫折经历的太少，所以总是把一些琐碎的小事看得很重。
3、春天，不是季节，而是内心；云水，不是景色，而是襟怀。
4、好听的话容易打动人，好心的话容易得罪人。

5、奋斗是万物之父。——陶行知
6、无论做什么事情，只要肯努力奋斗，是没有不成功的。——牛顿
7、把人生一分为二，前半生不犹豫，后半生不后悔。
8、我们应当努力奋斗，有所作为。这样，我们就可以说，我们没有虚度年华，并有可能在时间的沙滩。
9、不要乱说话。话说出去之前你还是话的主人，话说出去之后你就成了话的奴隶。
10、名言警句：生命不在于长短，而在于活得有意义。

只有领略了平凡，才会收藏当初丢弃的幸福

发表于 2018-06-14 | 分类于 名言警句

1、尝试去把别人拍过来的砖砌成结实的地基，生活就不会那么辛苦了。难过就是，虽然很难但总会过去。伟大变为可笑只有一步，但再走一步，可笑又会变为伟大。
2、只要持之以恒，知识丰富了，终能发现其奥秘。在事实面前，我们的想象力越发达，后果就越不堪设想。要克服生活的焦虑和沮丧，得先学会做自己的主人。
3、世界上只有一种英雄主义，那就是了解生命而且热爱生命的人。成功的秘诀在于永不改变既定的目标。若不给自己设限，则人生中就没有限制你发挥的藩篱。

4、幸福不会遗漏任何人，迟早有一天它会找到你。时刻就像一张网，你撒在哪里，你的收获就在哪里。坚强的人，并不是能应对一切，而是能忽视所有的伤害。
5、名言警句，纽扣第一颗就扣错了，可你扣到最后一颗才发现。有些事一开始就是错的，可只有到最后才不得不承认。
6、忙时井然，闲时自然；顺多偶然，逆多必然；得之坦然，失之怡然；捧则淡然，贬则泰然。悟通“八然”，此生悠然。
7、世界上没有绝对幸福圆满的婚姻，幸福只是来自于无限的容忍与互相尊重。一个人只要强烈地坚持不懈地追求，他就能达到目的。
