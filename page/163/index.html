人生就像钟表，可以回到起点，却已不是昨天！

发表于 2018-06-15 | 分类于 励志格言

1、环境不会改变，解决之道在于改变自己。要让事情改变，先改变我自己；要让事情变得更好，先让自己变得更好。
2、不要对挫折叹气，姑且把这一切看成是在你成大事之前，必须经受的准备工作。你出生的时候，你哭着，周围的人笑着；在生命的尽头，你笑着，而周围的人在哭着。
3、伟人之所以伟大，是因为他与别人共处逆境时，别人失去了信心，他却下决心实现自己的目标。如果早上醒来，你发现自己还能自由呼吸，你就比在这一周离开人世的万人更有福气。
4、什么时候也不要放弃希望，越是险恶的环境越要燃起希望的意志。积极的人在每一次忧患中都看到一个机会，而消极的人则在每个机会都看到某种忧患。
5、许多人企求着生活的完美结局，殊不知美根本不在结局，而在于追求的过程。生命就是一个逐渐支出和利用时间的过程。一旦丧失了时间，生命也就走到了尽头。

6、也许有些人很可恶，有些人很卑鄙。而当我设身为他想象的时候，我才知道：他比我还可怜。所以请原谅所有你见过的人，好人或者坏人。
7、一句无心的话也许会点燃纠纷，一句残酷的话也许会毁掉生命，一句及时的话也许会消释紧张，一句知心的话也许会愈合伤口、挽救他人。
8、成功的法则极为简单，但简单并不代表容易。肯低头的人，永远不会撞到矮门。昨晚多几分钟的准备，今天少几小时的麻烦。拿望远镜看别人，拿放大镜看自己。
9、励志格言，一念放下，万般自在。六个不能：不能饿了才吃，不能渴了才喝，不能困了才睡，不能累了才歇，不能病了才检查，不能老了再后悔。
10、屋子修得再大也是临时住所，只有那个小木匣才是永久的家，所以，屋宽不如心宽，身安不如心安！人生就像钟表，可以回到起点，却已不是昨天！

只有你奋斗了，失败后才会问心无愧

发表于 2018-06-15 | 分类于 励志格言

1、哪怕是最没有希望的事情，只要有一个勇敢者去坚持做，到最后就会拥有希望。为别人鼓掌的人也是在给自己的生命加油。用行动祈祷比用言语更能够使上帝了解。
2、成功的人是跟别人学习经验，失败的人只跟自己学习经验。很多事先天注定，那是“命”；但你可以决定怎么面对，那是“运”！道路坎坷事不期，疾风劲草练男儿。
3、不要问别人为你做了什么，而要问你为别人做了什么。成功不是将来才有的，而是从决定去做的那一刻起，持续累积而成。你一天的爱心可能带来别人一生的感谢。

4、山不辞土，故能成其高；海不辞水，故能成其深！奋斗没有终点，任何时候都是一个起点。没有热忱，世间便无进步。觉得自己做的到和不做的到，其实只在一念之间。
5、励志格言，没有什么事情有象热忱这般具有传染性，它能感动顽石，它是真诚的精髓。失去金钱的人损失甚少，失去健康的人损失极多，失去勇气的人损失一切。
6、人生是需要奋斗的，只有你奋斗了，失败后才会问心无愧；人生是单行路，只有奋斗了，才会有光明的前途；人生中有许多的竞争对手，正因有这么多的竞争对手，所以我们更得奋斗！

成功在优点的发挥，失败是缺点的累积

发表于 2018-06-15 | 分类于 励志格言

1、让奋斗之花遍地盛开；让奋斗的清泉在泉眼中喷涌而出；让奋斗的雄鹰在蓝天下翱翔；让奋斗的果实丰硕地结满大树下吧！奋斗就是路，世上本没有路，走的人多了，也便成了路，人生需要奋斗，去开辟属天你的那条新航路吧！
2、生命需要奋斗，奋斗与不奋斗，造就的结果截然不同。生无所息，保持奋斗的姿态，让世界变得如此灿烂，让你的人生绚烂多姿。千万不能满足小溪的平缓，否则你也就满足了自己的平庸，只有努力奋斗，才有机会欣赏自己。
3、也只是前行路上的插曲，没有任何一条路可以风风顺顺，只有在风雨中不怕失败的打拼才会看到最美的彩虹，只有奋斗没有失望不会迷失，鼓励自己还要加油，要奋发不垂头不丧气保持的信念依然坚定！

4、励志格言，梦想从来不会选择人，它是上天赋予每个人构建未来蓝图的神奇画笔。梦想不分高低贵贱，只要你心中有梦，乐观充实地过好每一天。
5、困难与折磨对于人来说，是一把打向坯料的锤，打掉的应是脆弱的铁屑，锻成的将是锋利的钢刀。幸福不会遗漏任何人，迟早有一天它会找到你。
6、时刻就像一张网，你撒在哪里，你的收获就在哪里。与其祈求生活平淡点，不如祈求自己强大点。成功在优点的发挥，失败是缺点的累积。

世界上只有想不通的人，没有走不通的路

发表于 2018-06-15 | 分类于 励志格言

1、金钱难买健康，健康大于金钱，金钱难买幸福，幸福必有健康，生命的幸福不在名利在健康，身体的强壮不在金钱在运动！假如我不能，我一定要；假如我一定要，我就一定能。
2、如果一个人不知道他要驶向哪个码头，那么任何风都不会是顺风。你不能左右天气，但可以改变心情。你不能改变容貌，但可以掌握自己。你不能预见明天，但可以珍惜今天！
3、励志格言，尝试去把别人拍过来的砖砌成结实的地基，生活就不会那么辛苦了。难过就是，虽然很难但总会过去。世界上只有想不通的人，没有走不通的路。

4、把你的脸迎向阳光，那就不会有阴影。在我们了解什么是生命之前，我们已将它消磨了一半。障碍与失败，是通往成功最稳靠的踏脚石，肯研究利用它们，便能从失败中培养出成功。
5、没有人富有得可以不要别人的帮助，也没有人穷得不能在某方面给他人帮助。如果说我懂的道理比别人多一点，那是因为我犯的错误比别人多一点。
6、任何人都可以变得狠毒，只要你尝试过嫉妒。纽扣第一颗就扣错了，可你扣到最后一颗才发现。有些事一开始就是错的，可只有到最后才不得不承认。
