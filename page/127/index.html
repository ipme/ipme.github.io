夏天桑拿，冬天冰箱，晴天火焰山

发表于 2018-06-14 | 分类于 夏天的古诗句

1、清风吹过，一阵阵花雨飘落下来，山野像铺上了一床彩色的大锦被。夏日里，清清的小河成了孩子们避暑的好去处，你看，他们在水中嬉闹着，一会儿打水仗，一会儿扎猛子摸鱼，玩得十分开心。
2、满塘的荷花荷叶，远远望去就像碧波上荡着点点五颜六色的帆，煞是好看。水牛早就躲到了池塘里，整个身子埋在了池水中，只露出一个头在水面上透气。
3、当花翅膀的麦雀子把乡村每个日子叫甜的时候，把黑土地的渴盼叫成金黄色，叫成起伏不平的波浪形，一个最最撩拨庄稼人的时令，又从镰刀上和红缨鞭上熟落了，又从那些古铜色的胸大肌肱二头肌和醉意朦胧的微笑上熟落了！

4、形容夏天，望着渐变黝黑的大海，想银河的浩瀚，想宇宙的奥秘，想人生的目的，想人类最后的归处……唉！睡吧！仲夏夜之梦应是甜美的！
5、每个在河南上学的孩子，上辈子都是折翼的天使：夏天桑拿，冬天冰箱，晴天火焰山，雨天水帘洞……四年以后都炼成孙悟空，到哪都死不了！

刚还是晴空万里，一下子便乌云滚滚

发表于 2018-06-14 | 分类于 夏天的古诗句

1、夏天的味道，多少人在这光芒的时刻染上永恒，滚烫的空气，指间的微凉，思念如泼墨，泛起层层渲染，而今的彼此，隔着时光看彼岸，对岸夏天里的青春正一天一天地成熟，你微笑的脸庞，我浅唱着婉词，生如夏花般绽放，空气中的絮语，那一抹纯白的似水柔情，诠释着七月流火最美的纪念。
2、雨后空气清新，仰望天空，云朵们又开始了玩耍，时而变成鲨鱼，想一口把我们吃了；时而又变成小船，在空中航行；时而又变成气球，在空中飘来荡去；时而也会变成小孩，在空中奔跑，他们的高兴劲感动了整个天，天越变越蓝，这时我似乎听到了云朵们的欢笑声。

3、深夜里，静悄悄的，调皮的星星哥哥月亮姐姐偷偷地跑出来玩耍。星星像一枚枚天灯，美丽极了。星星是恬静的，是安详的，像天空的眼睛。星星仿佛和我捉迷藏，东藏西躲，弄的我眼花缭乱。月亮像一个大玉盘，又像一块圆饼干。
4、关于夏天，夏，是一个多么爽气的名字呀！夏小伙把桃花画的粉中带紫，把梨花擦的晶莹剔透，把荷花抹的银光闪闪，让人看了心旷神怡！刚还是晴空万里，一下子便乌云滚滚，只见天空中一道闪电穿过，便是轰轰的雷声了，声音令人感到刺骨。

让我们一起去聆听夏天来到的声音

发表于 2018-06-14 | 分类于 夏天的古诗句

1、黄昏十分，垂柳轻柔地挥着手，送别了恋恋不舍的夕阳，天空做好了梦的绿色襁褓，等待着第一颗星星的诞生，最后一抹晚霞也在西方的天空中悄然隐去，暖风来回地在每一个人的耳畔喃喃低语，宣布着夜的来临。人们也在这个时候出来透透气，深蓝的天空，又仿佛有几颗星在闪烁。
2、温柔恬静的春姑娘迈着轻盈的步伐悄悄离开了，可爱而率真的夏姑娘坐着春季的末班车来到大地，向世间万物宣告“：夏天，就要来临了！”我走进了六月，一个童话般热闹美丽的世界，走吧，让我们一起去聆听夏天来到的声音。
3、漫不经心地走在路上，身边的树木上的叶子有的被烤干了；有的刚抽出嫩芽。蝉在树上吱吱喳喳的叫着，蜜蜂在耳边嗡嗡的叫着。蝉好像是大自然的演唱家为大自然演唱出一首又一首悦耳动听的歌曲。

4、落红还没完全化作春泥，石榴已有了笑意。接踵而来的是销声匿迹已久的第一声蝉鸣。夏天，就这样来了。夏天像一个青年人，灼热的阳光是他灿烂的笑脸，迸发的热量是他蓬勃的气息，葱茏的草木是他厚密的头发，水涨潮急的山洪是他的力量，速来忽去的骤雨是他的脾气。
5、关于夏天，繁茂的大树下，有一个清澈的小溪，挨挨挤挤的荷叶像一个绿色的盘子，又像一把碧绿的伞，荷叶给小溪给覆盖了，小溪像换了件翠绿的衣裳。还有几朵五颜六色的荷花笑开了脸。小溪美丽极了！

火辣辣的太阳挂在天空，白云自由地飘着……

发表于 2018-06-14 | 分类于 夏天的古诗句

1、看，透明的露珠在绿叶上翻滚着，迎风吹来，就像一颗颗晶莹的珍珠。如绿地毯的青草上，长着一棵棵高耸入云的大树。路旁的花儿争奇斗艳，都绽放出了笑容。火辣辣的太阳挂在天空，白云自由地飘着……
2、夏天，火辣辣的太阳毫不留情地炙烤着大地上的一切。大地被晒得发焦发烫，仿佛被一个巨大的蒸笼罩住，使人透不过气来。柳树没精打采地垂着枝条，碧绿的叶子打着卷，像一个挨了批评的小孩，垂头丧气地耷拉着脑袋。风儿也不知躲到哪儿去了，到处都是热的。连柳树上的蝉也拉长噪音拼命叫：“热死了！热死了！”
3、夏天的河流，水势很急。经过雨天的积累，顺流而下。遇到悬崖峭壁，形成万丈的瀑布，倾天而下。河边的石头，被清洗的格外干净，岸上的小孩，不时的捡起石头往水里扔。河水蜿蜒的前行，有的水流，顺着渠沟，流入千家万户的水田里。

4、夏天是多姿多彩的，沉静的是湖蓝，纯洁的是乳白，高贵的是米黄，热烈的是大红，典雅的是银灰，庄重的是墨黑……缤纷的色彩把温煦的夏日画满了。清雅的雏菊、馥郁的郁金香、娇艳的山茶花、妖娆的牡丹……芬芳的香气把幽幽的庭院溢满了。脆亮的蛙鼓、知了的聒噪、蚊蚋的低吟、婉转的鸟叫……丰富的声响把轻飘的空气充满了，这就是夏的神韵。让冬苏醒，让春灿烂，让夏多姿，让秋辉煌，让生活多彩。有了天地间的滋润，才有了旺盛的生命力。因为有了彼此的关爱，才有了春华秋实。
5、关于夏天，我喜欢雨，喜欢粗犷好放的夏雨，看着窗外将天地交织在一起的雨，使人浮想联翩。想起生活的恬趣，想起犹如战场之上的气概。雨日观雨，实是有一种别样的情趣。
