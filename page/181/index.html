努力抒写今天，让明天的怀念多一些亮色

发表于 2018-06-15 | 分类于 伤感说说

1、生命的人流总是穿梭不息，以一种别样的风景装扮着我们的生活。
2、总有一些人悄然的来，默然而去，如灯盏照亮了我们的行程，温暖着我们的人生。
3、伤感的句子，人生如流星，有瞬间的光辉，却没有永恒。
4、渐渐懂得，人生没有什么是不能放弃的，没有什么是不能割舍的，背负的越多，活的就会越沉重。
5、生活总有一些时候，一些地方，不需要我们执着，转身之后就该遗忘，挥手之后就该作别，不属于自己的，再美的都是镜花水月。

6、当尘埃落定，你也只是我生命中的过客，过去的既然已经定格，那就让它尘封，努力抒写今天，让明天的怀念多一些亮色。
7、生命的长度不可预期，我们丈量不了生命的距离，但是我们可以把握住生命的质量。
8、期待着在我们还不老的时光里，极力展示生命的色彩，努力让人生变得丰盈一些、快乐一点。
9、也许我们都在孤寂的时刻邂逅了今生的相遇，不管如何，还是让我们守护在真实与感动的世界里，默默无语的让这份情意澹然在心间。

有人说不懂得珍惜，失去后会很疼

发表于 2018-06-15 | 分类于 伤感说说

1、两人一起时间长了，新鲜感就淡了，难免就会产生裂痕，其实大家都没变，只是时间变了。
2、有人说不懂得珍惜，失去后会很疼。可是我却觉得，珍惜后的失去最疼最疼。
3、你我之间，就像乘客到站了就要下车，也要谢谢你捎我一路，没有缘分了就两两相忘好自为之。
4、最怕的就是我一直觉得很重要的人，而他（她）最重要的人却不是我。
5、就因为凡事都想了解透彻，知道太多往往最难开心，我知道糊涂才会快乐些。
6、我们坚持不和对方说话，不表示我没有想你。试着疏远你，因为我知道我不能拥有你。

7、人生不过如此，且行且珍惜，自己永远是自己的主角，不要总在别人的戏剧里充当着配角。
8、每个人都有自己的路要走，你们或许曾经陪伴着走过一段路。就像知道前面就是家门口，但一定要放手了，因为那是你的家，不是你们的家。
9、伤感的句子，有人说，时间让人忘记疼，我不觉得，时间只能让人习惯疼。

在这个世界上，还有什么东西是不会过期的

发表于 2018-06-15 | 分类于 伤感说说

1、我这辈子最遗憾的事，就是推我入地狱的人，也曾带我上天堂。是不是，一旦得到对方，爱就停止生长。算了吧，被我搞砸的关系又不止这一段。
2、有人稍稍靠近你，你就产生幻觉，殊不知她对每个人都一样，而你却把她当特别，这就是孤独太久的表现。公子不必觉得愧疚，怎么说也该谢谢你，好歹赠我一场空欢喜。
3、从来都是独行者，没有同行人，没有庇护所。离开你这件事，一直不得善终。早点睡吧，把夜熬穿了他也不会来找你。你从不拒绝，不让我走开，也没让我留下来。
4、当眼泪流下来，才知道，分开也是另一种明白。就让往事随风，你别回头，往事也别将就。你是我的定格，我是你的过客。一个人自以为刻骨铭心的回忆，别人也许早已经忘记了。
5、看上就爱，处腻就甩。你曾是我唯一的骄傲，是我炫耀的资本，也是你，拿起刀刺得我最痛。你说，在想哭的时候把眼泪包住不让它掉下来，有什么用呢？

6、我还是会笑，只是笑中不再温暖，带着冷漠与疏离。竟然选择了奋不顾身，就别怕重蹈覆辙。听说后来你中意了一人，我背着你，浪荡了一生。
7、千好不抵一过。离开的时候别回头，以后活着也别来找我。已有岁月可回首，却无情深共白头。你说你终于可以放下那段感情了，然后偷偷哭了很久对吧。
8、委屈什么，世上又不止你一个人爱而不得，几乎每个人都会遇到一个，不能结婚却深爱过的人。我开始怀疑，在这个世界上，还有什么东西是不会过期的。
9、枕头里藏满了发了霉的梦，梦里住满了无法拥有的人。那个费劲心思逗你笑的人，终究比不上你一见面就开心的人。心若没有良人，眼前全是路人。
10、伤感说说，准备好了么，那么爱你的我，终于舍得走了。让一个人成长的所有事，都是孤独的。所以，“长大”两个字连偏旁部首都没有。
