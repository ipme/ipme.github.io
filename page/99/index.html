只有千锤百炼，才能成为好钢

发表于 2018-06-14 | 分类于 正能量

1、在比夜更深的地方，一定有比夜更黑的眼睛。人生就像一个大舞台，每个人都有自己所要扮演的角色。至于要表演甚么角色，自己去决定。如果你很聪明，为什么不富有呢？
2、只有傻瓜才用双脚去试河水的深浅。如果要挖井，就要挖到水出为止。成功不是将来才有的，而是从决定去做的那一刻起，持续累积而成。
3、成功的信念在人脑中的作用就如闹钟，会在你需要时将你唤醒。成功是一种观念，致富是一种义务，快乐是一种权力。
4、强者向人们揭示的是确认人生的价值，弱者向人们揭示的却是对人生的怀疑。智者一切求自己，愚者一切求他人。两点之间未必直线最短，有时迂回曲折能够更快地抵达终点。
5、行动是治愈恐惧的良药，而犹豫、拖延将不断滋养恐惧。在人生舞台上，从不给落伍者颁发奖牌。不要太在乎自己的长相，因为能力不会写在脸上。

6、正能量，一个人只有亲眼看到自己伤疤的时候才知道什么是痛，什么是对与错。忙于采集的蜜蜂，无暇在人前高谈阔论。
7、走得最慢的人，只要他不丧失目标，也比漫无目的地徘徊的人走得快。一个最困苦、最卑贱、最为命运所屈辱的人，只要还抱有希望，便无所怨惧。
8、只有千锤百炼，才能成为好钢。人生最大的错误是不断担心会犯错。不经巨大的困难，不会有伟大的事业。要无条件自信，即使在做错的时候。
9、命运，你残忍的诉说着我的悲痛。望着别人幸福的笑容心里总觉得不安！乐观者在灾祸中看到机会；悲观者在机会中看到灾祸。如果耐不住寂寞，你就看不到繁华。
10、因为年轻我们一无所有，也正因为年轻我们将拥有一切。每一个成功者都有一个开始。勇于开始，才能找到成功。鹰击天风壮，鹏飞海浪春。

改变能改变的，接受不能改变的

发表于 2018-06-14 | 分类于 正能量

1、时光就像一辆畜力车，它的速度取决于我们手中的鞭子。快乐不是因为拥有的多，而是因为计较的少。补救一个错误，与时间无任何的关系，主动寻求一个新的开始永不为迟。
2、生命如自助餐厅，要吃什么菜自己选择。真正的快乐来源于宽容和帮助。记住该记住的，忘记该忘记的。改变能改变的，接受不能改变的。
3、我们唯一的悲哀是生活于愿望之中而没有希望。逆风的方向更适合飞翔，我不怕千万人阻挡只怕自己投降。你若不想做，会找一个或无数个借口；你若想做，会想一个或无数个办法。
4、还能冲动，表示你还对生活有激情，总是冲动，表示你还不懂生活。没有目标的人永远为有目标的人去努力。别说你最爱的是谁，人生还很长，无法预知明天。
5、要克服生活的焦虑和沮丧，得先学会做自己的主人。肯承认错误则错已改了一半。现代社会已不是大鱼吃小鱼的年代，而是快鱼吃慢鱼的年代。

6、正能量，谁不曾一意孤行，怒发冲冠过怕只怕少了那份执着。世界上那些最容易的事情中，拖延时间最不费力。人生如天气，可预料，但往往出乎意料。
7、每天晚上疲惫地坐到椅子上时，才觉得真真切切地过了一天。命运如同手中的掌纹，无论多曲折，终掌握在自己手中。人生没有理想，生命便只是一堆空架子。
8、把你的脸迎向阳光，那就不会有阴影。流过泪的眼睛更明亮，滴过血的心灵更坚强！不是境况造就人，而是人造就境况。比别人多一点执着，你就会创造奇迹。
9、超乎一切之上的一件事，就是保持青春朝气。少说多做，句句都会得到别人的重视；多说少做，句句都会受到别人的忽视。不经巨大的困难，不会有伟大的事业。
10、勤学的人，总是感到时间过得太快；懒惰的人，却总是埋怨时间跑得太慢。每个人都有潜在的能量，只是很容易：被习惯所掩盖，被时间所迷离，被惰性所消磨。

你要让你的实力配得上自己的野心

发表于 2018-06-14 | 分类于 正能量

1、最大的遗憾，是被命运安排！放弃自己，相信别人，这就是失败的原因。世上只有想不通的人，没有走不通的路。自己打败自己是最可悲的失败，自己战胜自己是最可贵的胜利。
2、人的活动如果没有理想的鼓舞，就会变得空虚而渺小。再长的路，一步步也能走完，再短的路，不迈开双脚也无法到达。人最大的改变就是去做自己害怕的事情。
3、自我控制是最强者的本能。只有你爬到山顶了，这座山才会支撑着你。乐观者在灾祸中看到机会；悲观者在机会中看到灾祸。昨天下了雨，今天刮了风，明天太阳就出来了。
4、别潇洒了自己，却苦了父母。没有遇到挫折，永远不会懂得自己的力量有多大。不拼一把你怎么知道自己是人物还是废物。承载了太多期许的目光，怎能轻易辜负。
5、正能量，今天的努力就是为了铺垫未来的幸福大道。你要记住你不是为别人而活，你是为自己而活。没人能让我输，除非我不想赢！永远不抱怨，一切靠自己。

6、累累的创伤就是生命给你最好的礼物，因为在每个创伤上面都标志着前进的一步。不苦不累，生活无味；不拼不搏，等于白活。要想人前显贵，必须人后受罪！
7、当你决定坚持一件事情，全世界都会为你让路。不逼自己一把，怎么知道自己不行。今天所有的懒惰都会换来明天的卑微。你要让你的实力配得上自己的野心。
8、我讨厌什么都不做，却有所期待的自己！如果努力就可以做到的事情，为什么不去做。父母给的那叫背景，自己打的才叫江山。花开不是为了花落，而是为了开的更加灿烂。
9、往事不必遗憾，若是美好，叫做精彩。若是糟糕，叫做经历。没有耐心和耐力，就别怪现实太现实。炫耀是需要观众的，而炫耀恰恰让我们失去观众。
10、理论是一码事，实际又是一码事。知识玩转财富。有人在光明中注视着阴影，有人在阴影中眺望着光明。解决最复杂的事情往往需要最简单的方法。

与其当一辈子乌鸦，莫如当一次鹰

发表于 2018-06-14 | 分类于 正能量

1、与其当一辈子乌鸦，莫如当一次鹰。我成功因为我志在成功。顶天立地奇男子，要把乾坤扭转来。雄心志四海，万里望风尘。一切的成就，一切的财富，都始于一个意念。
2、历史上伟大的变革总是悄悄地发生，我们要跟上时代的步伐！雄心壮志是茫茫黑夜中的北斗星。大丈夫处世，不能立功建业，几与草木同腐乎？人的价值，在遭受诱惑的一瞬间被决定。
3、年轻是我们唯一拥有权利去编织梦想的时光。拒绝严峻的冶炼，矿石并不比被发掘前更有价值。人无远虑，必有近忧。只要能收获甜蜜，荆棘丛中也会有蜜蜂忙碌的身影。

4、正能量，人生的成败往往就在一念之间。因害怕失败而不敢放手一搏，永远不会成功。每天告诉自己一次，“我真的很不错”。没有志向的人，就像失去了领航舵的航船。
5、器大者声必闳，志高者意必远。才自清明志自高。只要你相信，奇迹一定会实现。智者的梦再美，也不如愚人实干的脚印。儿童有无抱负，这无关紧要，可成年人则不可胸无大志。
