没法爱你一万年，只求能照顾你这一辈子

发表于 2018-06-15 | 分类于 表白的话

1、这个世界上，也许有人比我更优秀，但没人比我更懂你；也许有人比我更适合你，但没人比我更爱你；我永远是陪你一生的人！
2、遇见你是偶然的，喜欢你是自然的，爱上你是毅然的，得到你是欣然的，伴你一生是必然的。我是你夏天里的雪糕，冬天里的棉袄，黑暗里的灯泡，饥饿时的面包！我好想对你说：我爱你！
3、我希望你能给我一个机会，照顾你。爱护你。平平淡淡和你一起慢慢变老！我没有多的言语！只有一句话要告诉你：和你在一起，你是一切！没有你在身边，一切是你！

4、你我相见是一种缘份，我们彼此珍惜着这份来之不易的感情。希望在以后的路上能陪伴着你一生一世。如果对你的爱无法用言语来表示，我愿用生命来证明。
5、是因为爱才悄悄的躲开，躲开的是身影，躲不开的是默默的情怀；今天我终于鼓起勇气，向你表达我的爱。我不要短暂的温存，只要你一世的陪伴。
6、表白的句子，爱让我们相见，爱让我们相恋，爱让我们相爱，爱让我们牵挂，让我在有生之年把我全部的爱都送给你！没法爱你一万年，只求能照顾你这一辈子。

不要企图无所不能，否则你将一无所能

发表于 2018-06-15 | 分类于 经典说说

1、总有一天会面目全非。时光没有教会我任何东西。却教会了我不要轻易相信神话。没有辛勤的汗水，就没有成功的泪水；没有艰辛的付出，就没有丰硕的果实；没有刻苦的训练，就没有闪光的金牌。
2、要改造世界，得先改造自己；要成就事业，得先劳苦自身；要胜利登顶，得先奋力攀登。和自己相处，就是让我们静静坐下来，卸下负担，抚慰心灵，清楚自己身在何处心往何方。
3、人生是一张没有回程的车票，一场无法回放的电影，一扇走出无回的门。过去的不必后悔，把握眼前所有值得珍惜的机会，勇敢的面对生活。将奋斗进行到底！
4、人生总是多磨难，世事无常难抉择。逃避只是短暂瞬间，命运不会从此改写；缘分需要珍惜，强求不是好手段；放弃有时也是必然，愿你人生辉煌呈现。
5、心灵，为自己的事业日夜兼程，总想着出人头地；为自己喜爱的人身心疲惫，总想着朝夕相处；为自己的兄弟在所不辞，总想着手足情深。

6、虽然现实生活中，不是所有的梦想都能开花结果，也不是所有的人都能梦想成真。但每一个梦想都是绚烂多姿，每一个人都因追逐梦想而生活得更加精彩。
7、从未失败就意味着从未成功；从未挫折就意味着从未胜利；从未反思就意味着从未经历；从未跌倒就意味着从未登顶。路由“足”与“各”组成，所以才“人各有路”；正因为“人各有路”，所以才人各有成。
8、遇到一件事，喜欢它就享受它；不喜欢就避开它；避不开就改变它，改不了就接受它；接收不下，就处理它；难以处理，就放下它。其实，人最难的，是放下。
9、同时追两只兔子的人一只也逮不到。不要贪图无所不有，否则你将一无所有；不要试图无所不知，否则你将一无所知；不要企图无所不能，否则你将一无所能。
10、任何一件事情，只要心甘情愿，总是能够变得简单。一觉醒来，开始憧憬爱情，而爱情刚好做了场诡异的梦。千万别说直到永远，因为你压根不知道永远有多远。

人生最大的悲哀是无知，人生最大的忧虑是生死

发表于 2018-06-15 | 分类于 经典说说

1、人生最大的敌人是自己，人生最大的毛病是自私，人生最大的悲哀是无知，人生最大的忧虑是生死，人生最大的困扰是是非，人生最大的美德是慈悲。
2、有三种人是我的朋友：爱我的人，恨我的人，以及对我冷漠的人。爱我的人教我温柔；恨我的人教我谨慎；对我冷漠的人教我自立。
3、生活是首歌爱情是轻音乐，工作是交响乐，奋斗是摇滚乐，应酬是流行乐，朋友是通俗曲，事业是进行曲。勿满足于眼前的小小成就。你要问自己，我这辈子只有这样吗？
4、心灵，要做的事情总找得出时间和机会，不要做的事情总找得出借口。劳动一日，可得一夜的安眠，勤劳一生，可得幸福的长眠。
5、忘掉你的一次次失败，但不要忘记你获得的胜利；忘掉你遭遇的不幸，但不要忘记你的时来运转。没有谁瞧不起你，因为别人根本就没有瞧你，大家都很忙的。

6、没有人可以回到过去重新开始，但谁都可以从今日开始，书写一个全然不同的结局。爱情就像是一杯美味香浓的咖啡；婚姻则是剩余咖啡渣的咖啡杯。
7、只有你的未来，才能挥霍我的现在；只有我的最爱，给我最致命的伤害。有些事遇见了就无法放手，有些事放开了手就不必再回首。
8、生活中会发生什么，我们无法选择，但至少，我们可以选择怎样面对。所谓的幸福，就是一个笨蛋遇到一个傻瓜，引来无数人的羡慕和嫉妒，一起风风雨雨，平平淡淡。
9、似乎是没有欲求的开端，总是以备受打击而结束。人生是没有穷尽的，所以也就没有什么所谓的归宿。人必须像天上的星星，永远很清楚地看出一切希望和愿望的火光，在地上永远不熄地燃烧着火光。
10、所谓爱，就是将感觉热情和浪漫统统拿掉之后，你仍然珍惜对方。多站在别人的立场考虑问题，如果一件事情以你来说是种痛苦，对别人来说也是。

没有清醒的头脑，再快的脚步也会走歪

发表于 2018-06-15 | 分类于 经典说说

1、人生：一半是现实，一半是梦想；爱情：一半是激情，一半是执着；幸福：一半是金钱，一半是满足；工作：一半是马屁，一半是能力；友谊：一半是牵挂，一半是提醒；家庭：一半是依恋，一半是责任。
2、要处理世间的各种矛盾，自己的内心首先不矛盾，这样就会越理越顺；内心矛盾重重，就会越理越乱，越想越没有章程。大度大幅，小心眼无福。
3、心灵，最快的脚步不是跨越，而是继续；最慢的步伐不是小步，而是徘徊；最好的道路不是大道，而是小心；最险的道路不是陡坡，而是陷阱。
4、没有清醒的头脑，再快的脚步也会走歪；没有谨慎的步伐，再平的道路也会跌倒。一个人的成就，不是以金钱衡量，而是一生中，有多少人怀念你。生意人的帐薄，收入支出两数相减，便是盈利。人生的账薄，爱与被爱两数相加，就是成就。
5、在外面与朋友嬉游的时候，不要忘记家中孤独的老人。工作要像红色那样热烈，邻里需要蓝色那样温馨，爱情要像粉色那样迷人，做人要像白色那样纯洁，环境要像绿色那样环保，生活要像彩云那样多姿，生命要像花朵那样绚烂。

6、含蓄是一种美，也是对他人的一种尊重。初恋是初开的花朵浓郁芬芳，使人终身回味，失恋是初秋的果实，不成熟就采摘，品尝到的是酸楚和苦涩。
7、人生的奔跑，不在于瞬间的爆发，而在于途中的坚持。不敢说真话是个人的耻辱，不能说真话是时代的耻辱。蓝天下便是阳光；艰苦后便是甘甜；失败了就当经验；成功时便是灿烂。
8、人生是有许多偶然的，所以，也就多了很多的机会。对于工作，努力但不痴狂；对于家庭，忠诚但不刻板；对于金钱，喜爱但不贪婪；对于爱情，相信但不迷失。
9、蔷薇开出的花朵没有芬芳，想念一个人，怀念一段伤，不流泪，不说话。房子和老婆一样，没有时候，前半辈子为得到而忙活；有了以后，后半辈子要为拥有而忙活。
10、想是问题，做是答案，输在犹豫，赢在行动。再好的路也有不平坦的时候，在顺利的人生也有不如意的瞬间，要坚信我们可以迎战困难，要树立必胜的信念，人活只一世，让我们奋力拼搏！
