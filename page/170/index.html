学而不思则罔，思而不学则殆

发表于 2018-06-14 | 分类于 名言名句

1、一个人最大的破产是绝望，最大的资产是希望。好的想法是十分钱一打，真正无价的是能够实现这些想法的人。
2、好问的人，只做了五分种的愚人；耻于发问的人，终身为愚人。行动是治愈恐惧的良药，而犹豫拖延将不断滋养恐惧。
3、学到很多东西的诀窍，就是一下子不要学很多。自己打败自己的远远多于比别人打败的。学问是异常珍贵的东西，从任何源泉吸收都不可耻。
4、有教养的头脑的第一个标志就是善于提问。不知道自己无知，乃是双倍的无知。学而不思则罔，思而不学则殆。
5、非淡泊无以明志，非宁静无以致远。饭可以一日不吃，觉可以一日不睡，书不可以一日不读。青年是学习智慧的时期，中年是付诸实践的时期。

6、非静无以成学。少年读书，如隙中窥月；中年读书，如庭中望月；老年读书，如台上玩月。皆以阅历之深浅，为所得之深浅耳。
7、立志宜思真品格，读书须尽苦功夫。治学有三大原则：广见闻，多阅读，勤实验。黑发不知勤学早，白发方悔读书迟。
8、没有一个大学，是比拥有我们从未使用过的能力的大自我和人类意志与理智所创造的现实，更能包罗万象的了。
9、正确的道路是这样：吸取你的前辈所做的一切，然后再往前走。莫等闲，白了少年头，空悲切。如果把生活比喻为创作的意境，那么阅读就像阳光。
10、关于学习的名言，一个爱书的人，他必定不致于缺少一个忠实的朋友，一个良好的老师，一个可爱的伴侣，一个温情的安慰者。

学而时习之，不亦说乎？

发表于 2018-06-14 | 分类于 名言名句

1、在寻求真理的长河中，唯有学习，不断地学习，勤奋地学习，有创造性地学习，才能越重山跨峻岭。三人行，必有我师也。择其善者而从之，其不善者而改之。
2、学而时习之，不亦说乎？好好学习，天天向上。人的知识愈广，人的本身也愈臻完善。与肝胆人共事，无字句处读书。
3、学习这件事不在乎有没有人教你，最重要的是在于你自己有没有觉悟和恒心。生而知之者上也；学而知之者次也；困而学之又其次也；困而不学，民斯为下矣。
4、关于学习的名言，人是活的，书是死的。活人读死书，可以把书读活。死书读活人，可以把人读死。韬略终须建新国，奋发还得读良书。
5、旧书不厌百回读，熟读精思子自知。非学无以广才，非志无以成学。读书不趁早，后来徒悔懊。鸟欲高飞先振翅，人求上进先读书。
6、学习有如母亲一般慈爱，它用纯洁和温柔的欢乐来哺育孩子，如果向它要求额外的报酬，也许就是罪过。无限相信书籍的力量，是我的教育信仰的真谛之一。

7、阅读的最大理由是想摆脱平庸，早一天就多一份人生的精彩；迟一天就多一天平庸的困扰。科学研究好象钻木板，有人喜欢钻薄的；而我喜欢钻厚的。
8、学习文学而懒于记诵是不成的，特别是诗。一个高中文科的学生，与其囫囵吞枣或走马观花地读十部诗集，不如仔仔细细地背诵三百首诗。
9、虚假的学问比无知更糟糕。无知好比一块空地，可以耕耘和播种；虚假的学问就象一块长满杂草的荒地，几乎无法把草拔尽。
10、智力决不会在已经认识的真理上停止不前，而始终会不断前进，走向尚未被认识的真理。情况是在不断的变化，要使自己的思想适应新的情况，就得学习。

富贵必从勤苦得，男儿须读五车书

发表于 2018-06-14 | 分类于 名言名句

1、任何倏忽的灵感事实上不能代替长期的功夫。千教万教教人求真，千学万学学做真人。富贵必从勤苦得，男儿须读五车书。
2、少壮不努力，老大徒伤悲。业精于勤，荒于嬉；行成于思，毁于随。求学将以致用；读书先在虚心。学习知识要善于思考，思考，再思考。
3、敏而好学，不耻下问。我学习了一生，现在我还在学习，而将来，只要我还有精力，我还要学习下去。愚昧从来没有给人带来幸福；幸福的根源在于知识。
4、你应该小心一切假知识，它比无知更危险。古今来许多世家，无非积德。天地间第一人品，还是读书。路漫漫其修道远，吾将上下而求索。
5、唯一能持久的竞争优势是胜过竞争对手的学习能力。青年最主要的任务是学习。谁游乐无度，谁就没有功夫学习。勿以恶小而为之，勿以善小而不为。

6、研卷知古今；藏书教子孙。吾生也有涯，而知也无涯。生活的全部意义在于无穷地探索尚未知道的东西，在于不断地增加更多的知识。
7、关于学习的名言，学如逆水行舟，不进则退。科学的自负比起无知的自负来还只能算是谦虚。知之者不如好之者，好之者不如乐之者。
8、学习专看文学书，也是不好的。先前的文学青年，往往厌恶数学理化史地生物学，以为这些都无足轻重，后来变成连常识也没有。
9、学和行本来是有机联着的，学了必须要想，想通了就要行，要在行的当中才能看出自己是否真正学到了手。否则读书虽多，只是成为一座死书库。
10、现在，我怕的并不是那艰苦严峻的生活，而是不能再学习和认识我迫切想了解的世界。对我来说，不学习，毋宁死。
11、你们要学习思考，然后再来写作。发奋识遍天下字，立志读尽人间书。惜时专心苦读是做学问的一个好方法。夫学须志也，才须学也。非学无以广才，非志无以成学。

钱就像内裤，你得有，但不必逢人就证明你有

发表于 2018-06-14 | 分类于 名人名言警句

1、钱，就像内裤，你得有，但不必逢人就证明你有。
2、干事业，就像怀孕，得够月，着急了就容易流产。
3、你若成功了，放屁都有道理，你若失败了，再有道理都是放屁。任何不能置你于死地的，都将让你更坚强！所以不要因为眼前的困难就选择屈服。
4、猫，喜欢吃鱼，可惜猫不会游泳。鱼，喜欢吃蚯蚓，可鱼又不能上岸。老天爷给了你很多诱惑，却不让你轻易得到。
5、不要随便把自己心里的伤口晒给别人看，因为这个社会上你根本就分不清，哪些人给你撒的是云南白药，哪些人给你撒的是辣椒和咸盐……

6、心灵鸡汤，可以哭，可以恨，但是不可以不坚强，因为后面还有一群人等着看笑话！
7、人生是很累的，你现在不累，以后就会更累；人生是很苦的，你现在不苦，以后就会更苦。
8、这世界上没有人能真正帮到你，除了你自己。别人帮了你，那是情分；不帮你，才是本分。
