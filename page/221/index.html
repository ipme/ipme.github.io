我们老得太快，却聪明得太迟

发表于 2018-06-14 | 分类于 人生哲理

光阴易逝如流水，一去就再也不复回。我们老得太快，却明白得太晚。我们总是在等待，偶然猛一回头才发现，已经蹉跎了那么多岁月。也只有到此刻才明白，时间不会为快乐停留，也不会为逆境停顿。
我们总是老得太快，却明白得太晚。把钱省下来，等待退休后再去享受。结果退休后，因为年纪大，身体差，行动不方便，哪里也去不成。
人生哲理，钱存下来等养老，结果孩子长大了，要出国留学，要创业做生意，要花钱娶老婆，自己的退休金都被要走了。
我们老得太快，却聪明得太迟。不管你是否察觉，生命都一直在前进。人生并未售来回票，失去的便永远不再得到。将希望寄予“等到方便的时间才享受”，我们不知失去了多少可能的幸福！
不要再等待有一天你“可以松口气”，或是“麻烦都过去了”，再去你喜欢的地方，见你想见的人，看你想看的书，过你想过的日子。把握当下，不要等待来日方长。

人这一辈子，无非就是个过程，荣华花间露，富贵草上霜。生不带来，死不带去。得意些什么？失意些什么？顺其自然、随遇而安，如行云般自在，像流水般洒脱，才是人生应有的态度。
现在过的每一天，都是余生中最年轻的一天。请不要老得太快，却明白得太迟，很多错误不必亲自试验，在别人的经历中吸取营养也是一种智慧！
所以，当自己有足够的能力善待自己时，就立刻去做，老年人有时候是无法做中年，或是青少年人可以做的事，年纪和健康就是一大因素。

本本分分做事，无愧天地良心，无愧朋友家人

发表于 2018-06-14 | 分类于 人生哲理

我问心：生活为何不快乐？心告诉我，想得太多，才会负累，追逐太多，才会疲惫。人世间谁真谁假？其实心里清楚就好。心清目明，知足长乐，穷善其身，达济天下。
我问心：情为何会让人疼？心告诉我，人若多情，必然自苦，心若不动，则不会痛。唯千般滋味尝过，才算是成全自己。若，付出有人珍惜，那么一切都是值得。
我问心：身体为何总疲惫？心告诉我，生而为人，就得奔波，酸甜苦辣，才是生活。累，证明在付出，难，证明在进步，受得住，扛得住，经得住，吃得苦中苦，方为人上人。
人生为何难抉择？心告诉我，心有不甘，想要改变，人有贪欲，不愿割舍。人生岔路口太多，总有适合你的那条。前方没不后悔的路，只有选择后的坚持。

我问心：如何能做好自己？心告诉我，好是相对，没有绝对，偶有进步，便要知足。善心不能丢，坏心不能有，信心不能失，歹心不能留！
自己的路，想怎么走就怎么走，别人的嘴，爱怎么说就怎么说，圈子不同的人，何必跟他浪费口舌，素质不高的人，何必和他斤斤计较。
悠悠之口堵不住，做好自己才不怒，嘴长在别人脸上，随他们去说吧。乱嚼舌根的人，多是素质不高的小人，挑破离间的人，多是心术不正的坏人，和这样的人一般见识，别污了你的身份。
人生哲理，好人，上天自会眷顾；坏人，上天不会饶恕谁好谁坏，谁真谁假，时间帮你验证。
这个世界就是这样，你做得好，没人看得见；你做得差，人人都埋怨。你做百件好事，没人称赞；你做一件坏事，人人指点。人，争不过一张嘴算不过一颗心。
别人的嘴，爱咋说咋说，你只管堂堂正正做人，本本分分做事，无愧天地良心，无愧朋友家人。

懂得珍惜，才配拥有；知道感恩，才能长久！

发表于 2018-06-14 | 分类于 人生哲理

有些命里遇见，从相识到相知，似乎都是一场缘分，缘分到了，自然有人向你走来。有些情缘不问因果，有些只是一场烟花易逝。相遇，不会有刻意。相识，不该去矫情。
有种相识，在路上，有种相知，早已封存在心，似曾相识，淡然一笑。人之相识，贵在相知。人之相知，贵在知心。不是每个擦肩过的人都会相识，也不是每个相识的人都会让人牵挂。
生命中有一些人擦肩了，却来不及遇见；遇见了，却来不及相识；相识了，却来不及熟悉；熟悉了，却还是要说再见。
不曾后悔相遇，也不曾惋惜相识，感受过关怀，也信奉过承诺，不论是悲是喜，感谢曾经陪伴！对自己好点，因为一辈子不长；对身边的人好点，因为下辈子不一定能遇见。

真心是什么，可以说是世上最珍贵，也最不值钱的东西。如果真心能换来真心，世上就不会有那么多的放弃；如果不离能换取不弃，生命就不会有千疮百孔的心灵。
多少主动被视作自作多情，多少深情到最后痛不欲生。真心，有时候是不值钱，可没了真心，所有的感情全都不值钱。一个人对你好，叫真心。一个人帮助你，也叫真心。一个人惦记你，心疼你，更叫真心。
人生哲理，真心有多珍贵，谁都买不起。如果你真的得到了，请拿出自心用一辈子的期限，去换，去还，去珍惜！
平坦的路，是人一步一步走出来的；真挚的情，是心一点一点换回来的。没有无缘无故的离开，只因没被善待；没有毫无缘由的放手，只因少了关怀。
缘分，不是一个人的挽留，而是两个人的坚守；真情，不是一颗心单方面的付出，而是两颗心共同的呵护。人心不是一天走远的，感情不是一天变淡的。
一个人说再多，若少了应答，心情总会慢慢低落；一颗心爱再深，若缺了回应，感情难免悄悄淡化。懂得珍惜，才配拥有；知道感恩，才能长久！

死犹未肯输心去，贫亦其能奈我何！

发表于 2018-06-14 | 分类于 个性签名

1、在年轻人的颈项上，没有什么东西能比事业心这颗灿烂的宝珠。不为穷变节，不为贱易志。褴褛衣内可藏志。志气和贫困是患难兄弟，世人常见他们伴在一起。
2、死犹未肯输心去，贫亦其能奈我何！师傅领进门，学艺在自身。成功是努力的结晶，只有努力才会有成功。向竞争挑战，向压力挑战，更要向自己挑战时间就是一切。
3、人若有志，就不会在半坡停止。眼泪终究流不成海洋，人总要不断成长。在经过岁月的磨砺之后，每个人都可能拥有一对闪闪发光的翅膀，在自己的岁月里化茧成蝶。

4、个性签名，有梦就该勇敢去追一腔孤勇你别怕单枪匹马又如何这一路你可以哭但你一定不能怂。失败的定义：什么都要做，什么都在做，却从未做完过，也未做好过。
5、生活总是让我们遍体鳞伤，但到后来，那些受伤的地方一定会变成我们最强壮的地方。人生就像赛跑，不在乎你是否第一个到达尽头，而在乎你有没有跑完全程。
