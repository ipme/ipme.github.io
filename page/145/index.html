吃一堑长一智，伤害就是明白，失败就是懂得

发表于 2018-06-14 | 分类于 心灵鸡汤

1、占据别人回忆的最好方法就是：活的更好。生活不可能像你想象得那么好，但也不会像你想象得那么糟。我觉得人的脆弱和坚强都超乎自己的想象。有时，我可能脆弱得一句话就泪流满面，有时，也发现自己咬着牙走了很长的路。
2、因为喜欢，所以情愿，没有那么多为什么。生命，是一场放逐和流浪；偏执，能让我们更接近幸福，或者自由。做决定是最难的，一旦把决定实施，事情倒也没想象的那样困难重重了。想一千次，不如去做一次。华丽的跌倒，胜过无谓的徘徊。
3、有些事情，只有经历了，才有穿透心扉的体验；曾经的人，只有从心底放下了，心灵才会真正地解脱。如果没有盲目的期待，就不会有失望。如果能了解一切都是无常，就不会攀缘执著；如果不攀缘执著，就不会患得患失，也才能真正完完全全地活着。
4、每天告诉自己：你的生命非常有限，所以，不要浪费在重复他人的生活上；不要被教条束缚，不要被他人喧嚣的声音掩盖你内心；你要有勇气，听从你心灵和直觉的指示，你的内心知道你想要成为什么样子的。其他事情都是次要的。

5、生命短促，没有时间可以再浪费，一切随心自由才是应该努力去追求的，别人如何想我便是那么的无足轻重了。苦难与幸福一样，都是生命对我们的奖赏。艰难活过来的今天，就是明天活下去的希望，活着就是幸福。
6、一件事，想通了是天堂，想不通就是地狱。既然活着，就要活好。有些时候我们由于太小心眼，太在意身边的琐事而因小失大，得不偿失。有些事是否能引来麻烦和烦恼，完全取决于我们如何看待和处理它。别总拿什么都当回事，别去钻牛角尖，别太要面子，别小心眼。不在意，就是一种豁达、一种洒脱。
7、心灵禅语，人生如果是个圆满的圆，对于一个时刻准备着和不懈努力着的人而言，那么他奇幻旅程的彼岸，必然还是那一原点，心灵的童真。
8、爱情有一个极为突出的特点：人们有了爱情，掩饰不住；没有爱情，又装不出来。喜欢你的人你怎么样都行，不喜欢你的人你怎么样都不行。
9、让我们坚强的是伤害，使我们成熟的是失败。不论受到多大的伤害，经受了多大的失败，我们依然会慢慢站起来。伤害让我们明白，失败使我们懂得，在挫折中我们渐渐成长，慢慢成熟。人生，吃一堑，长一智，伤害就是明白，失败就是懂得。我们会从失利中慢慢走出，重拾信心，继续我们的努力与追求。

世间所有的好，莫过于如愿以偿四个字

发表于 2018-06-14 | 分类于 心情说说

1、如果一个人真的想你，他会来找你，没有例外。聪明的人嘴甜，讨厌的人会演，而你什么都不会，所以只能被骗。
2、世界上有两个我，一个假装快乐，一个真心难过。总有一天，你会站在最亮的地方，活成自己曾经最渴望的样子。
3、心情说说，世间所有的好，莫过于如愿以偿四个字。我的评论从来不会火，就像我爱的人从来不爱我。
4、别去打扰那些已活在你记忆中的人，也许这才是最适合你们的距离。守护公主的永远是骑士，而公主选择的却是王子。
5、以后见面别再躲了，不然不像普通朋友。什么是无奈：就是被狗咬了一口，却无法反咬一口的感觉。
6、成熟就意味着你要做到难过不声张，有事自己扛。我要怎么跟你解释，在我毫不犹豫爱你时，恐惧同样无边无际。

7、除了想让自己快乐一点，我也没有什么更伟大的梦想。希望以后我是个很棒的人，不是的话，是一个像现在一样温暖可爱的人也好。
8、那些被困在痛苦里反复折磨的日子，只有自己知道。我要很多很多的爱，如果没有，那么就要很多很多的钱，我就是这么庸俗。
9、谁没为生活煎熬过，努力过，忍耐过，委曲求全过。脸大有什么关系，脸美就行了，大点儿美得清楚。
10、你假装不懂，我也故作轻松，错过就错过，人嘛，总要有点性格。愿你可以精致到老，眼里长着太阳，笑里全是坦荡。

有些人谈不上原谅，老死不相往来就好！

发表于 2018-06-14 | 分类于 心情说说

1、雨会停，心会晴，没有什么会永远糟糕透顶。不抱怨，靠自己，因为久了你就会发现，抱怨除了扰乱你和别人的心情，屁用都没有。
2、有些人谈不上原谅，老死不相往来就好！不要以为有朝一日自己会变成熟，每个年龄段都有他该有的自讨苦吃。
3、心情说说，如果我不曾见过太阳，我本可以忍受黑暗，然而阳光已使我的荒凉，成为更新的荒凉。
4、睡的时候，不辜负床，忙的时候，不辜负路，爱的时候，不辜负人，饿的时候，不辜负胃。
5、我不是话少，也不是冷漠，只是我没必要对每个人都有说有笑。人总要找点事情做，让自己忙起来，忙起来才知道生活不易，才明白平时的忧伤都是矫情。
6、你要积蓄你的能量，眷顾你的善良，变得勇敢，当这个世界越来越坏，我只希望你能越来越好。

7、看见朋友哭不想递她纸巾，想给她墨镜让她依然很酷。现在的我，除了快递谁都不等。
8、对你的喜欢就像七八岁的英雄梦，不可一世又不堪一击。你并没有多糟糕，只是他看不到你的好。
9、终于到了小时候最羡慕的年龄，可我并没有活成小时候最喜欢的样子。这个世界上不如愿的人很多，我只是其中一个。
10、我们都没错，错就错在年轻毁了承诺。越长大越不敢依赖别人，因为人心易变，自己给的，才叫安全感。

别对我忽冷忽热，这样你累，我也反胃

发表于 2018-06-14 | 分类于 心情说说

1、遇到恶心事，你要学会淡定和从容，这个世界上的人渣是千姿百态的。我们都跑不过时间，我们只能跑过昨天的自己。
2、千万不要对任何事感到后悔，因为它曾经一度就是你想要的，后悔没用，要么忘记，要么努力。
3、我相信你不是不爱了，只是我们的爱情实在走不下去了。每次想找个人陪的时候，就发现有的人不能找，有的人不该找，还有的人找不到。
4、你要堕落神仙也救不了，你要成长绝处也能逢生。不要做一个被情绪左右的傻瓜，情绪可能会使你为一个错误的人或为一件错误的事奋不顾身。
5、一切的矫情和装逼，咆哮和压抑，都是源于很缺钱和很缺爱。别对我忽冷忽热，这样你累，我也反胃。
6、心情说说，我能想到最浪漫的事，就是和你一起吃吃吃吃吃吃，然后你付钱你付钱你付钱。

7、人常犯的一个错，就是花费时间去在乎不怎么在乎你的人。你的态度，决定你十年后是人物还是废物；你才二十岁，你可以成为任何你想成为的人。
8、除非你改变了交往的人和阅读的书，否则，你的五年之后和现在完全一样。
9、没有艰辛就没有收获，上帝不会辜负一个努力过的人。越长大越发现一句歪理，善良的人没好报，安静点做个混蛋比较好。
10、每个人都有一个不堪的过往，别人都过得去了，你又何必拿出来议论，抓着不放，斤斤计较。
