只有不断地取长补短，才会不断地进步

发表于 2018-06-15 | 分类于 励志格言

1、没有利益的战斗我绝不参与，没有任何好处的朋友我绝不结交，只要对生存有利，哪怕是敌人，也可以成为暂时的朋友。靠山山会倒，靠水水会流，靠自己永远不倒。
2、我的对手只有自己，要想爬得更高，只应该自己往上走，而不应该把别人往下压。最强壮的狼也有它脆弱的地方，要想做到最好，仅仅靠自己是不行的。只有不断地取长补短，才会不断地进步。
3、励志格言，短暂的争吵是为了再次握手，一时的争斗是为了永远的和平。分歧是一种个性存在的必然，但是绝对不是消极的借口。不是境况造就人，而是人造就境况。

4、一个狼群就是一支训练有素，纪律严明的部队，统一行动，绝对服从，协同作战，这就是狼的纪律。一时的忍耐是为了更广阔的自由，一时的纪律约束是为了更大的成功。
5、忘掉失败，不过要牢记失败中的教训。未曾失败的人恐怕也未曾成功过。大多数人想要改造这个世界，但却罕有人想改造自己。轰轰烈烈是一生，平平淡淡也是一生，人活一次，没有一次拼搏，那还叫什么人生？

其实真正对你好的人，你一辈子也遇不到几个

发表于 2018-06-15 | 分类于 励志格言

宁愿跑起来被拌倒无数次，也不愿规规矩矩走一辈子，就算跌倒也要豪迈的笑。把艰辛的劳作看作是性命的必然，即使没有收获的期望也心平气和的继续。
苦心人天不负，卧薪尝胆，三千越甲可吞吴。有志者事竞成，破釜沉舟，百二秦川终属楚。不好生气要争气，不好看破要突破，不好嫉妒要欣赏，不好托延要用心，不好心动要行动。
人生格言，一个人想要创业，先要学习并领悟打工。咱们什么都没有，唯一的本钱就是青春。梦想让我与众不一样，奋斗让我改变命运！
这个社会是存在不公平的，不好抱怨，正因没有用！人总是在反省中进步的！普通人成功并非靠天赋，而是靠把寻常的天资发挥到不寻常的高度。
我走得很慢！但我从不后退！一件事，会干了，好好干；不会干了，乱干。干的次数多了，自然会干。但是，首先，你要干。愉悦和幸运是需要代价的，天下没有免费的午餐！

自暴自弃便是命运的奴隶，自强不息是性命的天使；我不想用别人的汗水浇灌自我的心灵，我愿意用别人的棉袄，来温暖自我的躯体。我只想堂堂正正的做人，我只愿光明磊落做事，该记得的我不会遗忘，该遗忘的我不会存放。
志在峰巅的攀登者，不会陶醉在沿途的某个脚印之中。就算你的朋友再多，人脉再广，其实真正对你好的人，你一辈子也遇不到几个。
按照自己的活法，快乐的生活，活得像自己就好了，何必在意那么多，勇敢地走自己的路，让别人说去吧。人越是高兴的事情，越爱隐藏；越是痛苦的事情，越爱小题大作。
见己不是，万善之门。见人不是，诸恶之根。为了向别人向世界证明自己而努力拼搏，而一旦你真的取得了成绩，才会明白：人无须向别人证明什么，只要你能超越自己。

相信你做得到，你一定会做到

发表于 2018-06-15 | 分类于 励志格言

学一分退让，讨一分便宜；增一分享受，减一分福泽。念头端正，福星临，念头不正，灾星照。善人行善，从乐入乐，从明入明；恶人行恶，从苦入苦，从冥入冥。
人生的道路都是由心来描绘的。所以，无论自己处于多么严酷的境遇之中，心头都不应为悲观的思想所萦绕。你的选择是做或不做，但不做就永远不会有机会。
有些人累积金钱换取财富，智者累积快乐，与人分享仍取之不竭。快乐在于行动，不只是拥有。剥夺别人的快乐不能使自己快乐。微笑使人更美丽更愉快，却不费分文。热情比怨恨更得人心。慷慨的给予快乐，自己更快乐。
明确的了解自己想要什么，致力追求。一个人没有明确的目标，就象船没有罗盘一样。最遗憾的事，你配不上自己的雄心，也辜负了自己的受过的苦难。

智者都有清晰思考的习惯。意志力缘于持续的行动自动自发明确的目标。诚实与努力的工作，需要明确的目标引导才能成功。缺乏明确的目标，一生将庸庸碌碌。坚定的目标是成功的首要原则。
相信你做得到，你一定会做到。不断告诉自己某一件事，即使不是真的，最后也会让自己相信。思想如钻子，必须集中在一点钻下去才有力量。
得意时应善待他人，因为你失意时会需要他们。对于最有能力的领航人风浪总是格外的汹涌。世上最蠢的行为，就是有效率地做一件根本不该做的事。
人生格言，珍惜自己的时间，利用好每一份每一秒。做事不放过没一个细节，小心谨慎，细致，能够做到这些，还有什么是不可能的呢？

只有保持奋斗的姿态，才能证明生命的存在

发表于 2018-06-15 | 分类于 励志格言

生命需要保持一种激情，这激情能让别人感到你是不可阻挡的时候，就会为你的成功让路！一个人内心不可屈服的气质是可以感动人的，并且能够改变很多东西。
第一个青春是上帝给的；第二个的青春是靠自己努力的。生命太过短暂，今天放弃了明天不一定能得到。礼貌像只气垫，里面可能什么都没有，却能奇妙地减轻颠簸。
只有尝试过，努力过，坚持过，才能有收获。一分耕耘，一分收获，只有努力了，才能绽放出成功的花朵。只要功夫深，铁杵磨成针。我相信，只要我朝着这个理想努力奋斗，坚持不懈，那么，我就一定会成功！
老虎向来都是一个一个的，绵羊从来都是一群一群的。最惨的破产就是丧失自己的热情。有些事情，在当时你不一定能看个透彻，我们只需要等待，时间会告诉我们结果。

人生格言，要是散布阳光到别人心里，先得自己心里有阳光。发光并非太阳的专利，你也可以发光。成大功者，不顾小嫌；建远略者，不期近效。
十几年的奋斗，给了我崭新的生活；十几年的奋斗，给了我冲向顶峰的动力。正如富强的中国一样，我，与许多不计辛劳的人，一起跨入梦想的殿堂。
人生像激流中行驶的帆船，必定会有奋斗，为了战胜激流而奋斗，而“激流”代表的则是理想。自信的人，懂得“天生我才必有用”的道理，依靠着必胜的信心，在艰苦的搏击中，驶向成功的彼岸。
战胜自己，终有一天，你会扇动着梦想的翅膀，飞翔在蔚蓝的天空！生命中遇到最美的景致，并不需要浓墨重彩去描绘，而是平常心踩出的一串淡淡的足迹。
人生中要走很多路，有一条路不能回头，就是放弃的路；有一条路不能拒绝，就是成长的路；有一条路不能迷失，就是信念的路；有一条路不能停滞，就是奋斗的路；有一条路不能忘记，就是回家的路；信心满满的走好脚下的每条路，祝你生命中的每一天都很精彩。
