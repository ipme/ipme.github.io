人生不在年龄，贵在心理年轻

发表于 2018-06-14 | 分类于 人生感悟

1、人，早晚要习惯孤独，没有谁会把你当宝护着，再大的难事也要微笑面对，哪怕是装的。对人要学会绝情，该滚的滚，该留的留，但你也要学会珍惜，知心朋友已不多，难道你真要只剩自己？
2、养活自己只需要勤劳，让自己幸福却需要智慧。幸福是一种感受，处境不同，追求不同，追求的结果不同，心态不同，感受就不同。真正的智慧，不仅是让你懂得追求，更是让你明白何时该执着，何时该放手。
3、感悟人生，现实生活中，人人都有梦想，都渴望成功，都想找到一条成功的捷径。其实，捷径就在你的身边，那就是勤于积累，脚踏实地，积极肯干。

4、在对的时间遇上对的人，是一生幸福。在对的时间遇上错的人，是一种悲哀。在错的时间遇上对的人，是一生叹息。在错的时间遇上错的人，是一是荒唐！
5、梦，不能做得太美，美了难以实现；话，不能说得太满，满了难以圆通；调，不能定得太高，高了难以合声；事，不能做得太绝，绝了难以进退；情，不能陷得太深，深了难以自拔。
6、人生不在年龄，贵在心理年轻；衣着不在时尚，贵在舒适合体；膳食不在丰富，贵在营养均衡；居室不在大小，贵在整洁舒畅；养生不在刻意，贵在顺其自然；锻炼不在夏冬，贵在持之以恒。
7、感情从来都不是一种交换，而是一种取悦和被取悦。你想要取悦人，就是被人吃定了；而别人讨好你，就是被你吃定了。恋爱要么是享受虐人，要么是安于被虐。
8、人生像洋葱：你只能一层一层地把它剥开，有时你还得流泪。如果抱怨成习惯，心灵就像上了枷锁，没有一刻解脱。唯有放下抱怨，才能体会到生命的自在与幸福。

珍惜那个愿意无怨无悔对你好的人

发表于 2018-06-14 | 分类于 人生感悟

这世上，没有谁的好是平白无故，珍惜那个愿意无怨无悔对你好的人。一辈子能遇上这样的人，是一种福气。
一个对你好的人，懂你好在哪里。别人眼里你平淡无奇，在他眼里却独有风情。这种欣赏这份懂得，已经是礼物。
一个对你好的人，能包容你的不好。他知道你好在哪里，更明白人无完人。别人只强求你事事完美，他却懂你也有自己的缺点和软肋。真正爱你的人，是看到了你的缺点，依然愿意和你一起走下去。

一个对你好的人，是对你不求回报的人。也总是安安静静地，接受你的冷漠和不理不睬。他也会难过也会感觉挫败，但是他仍然一心一意对你好。对于你，他别无所求；对于你，他奉献全部真心。
一个对你好的人，是真正尊重爱护你的人。一个真正对你好的人，不会强你所难，也不会令你难堪。他懂得尊重你呵护你，最愿意你凡事都选择幸福快乐的方式。
一个对你好的人，会想要努力让你开心，满足你的一切要求。但一个人被忽冷忽热对待太多次，终究会怀疑自己。若你只在需要帮忙的时候才想起他，他也会慢慢明白你不是真的爱他。
人生感悟，一颗心被伤了太多次，慢慢地就会麻木；一个人被冷落太多次，渐渐地就想离开。不要等到失去才学会珍惜，不要弄丢一个对你好的人！

真正的懂得，不必言语，不必刻意

发表于 2018-06-14 | 分类于 人生感悟

懂，是一种简简单单的感动，懂得你的眼泪，给你最暖的安慰；深知你的憔悴，怪你为何不知疲惫。
懂，是一种难言的柔情，入心入肺，入骨入髓；懂，即使不言不语，即使山高水远，彼此的心依然贴近，惺惺相惜没有远离。
懂，是一种没有距离的陪伴。相隔万水千山，也泯灭不了深切的牵挂；纵使海角天涯，也阻碍不了敞开的心扉。
懂，是一种触碰心灵的相对，也许一个眼神，一个哀叹便能了解你内心的喜悲；无需热烈拥抱的交汇，便能明白你的一切。

真正的懂得，不是相邀，也不是牵引，更不是逼迫，而是实实在在自然而然的明白，这样的明白，无关风月，无关功利，甚至无关风雨也无关晴天。
真正的懂得，不必言语，不必刻意，有时，只需浅浅一个微笑。真正的懂得，不必解释，不必逃避，有时，只需轻轻的一声呼唤。
人生感悟，真正的懂得是一种心情，一种欣赏，更是一种心灵的默契。
懂你，何需千言万语，因为彼此懂得，所以心怀感激；因为彼此眷恋，所以格外珍惜。真情的相伴在彼此心里，是生命的一种交集，是灵魂的一种相依。
世界之大，并非每个人都能相遇；茫茫人海，并非每颗心都能相通。有个人经过你的生命，也许可以和亲情相媲美；这份情在你眼里最为珍贵，能够和你的生命相依相随。
懂你，无需千言万语。

好好地活着，别在乎太多！

发表于 2018-06-14 | 分类于 人生感悟

好好地活着，别在乎太多；认真地活着，别奢求更多。活着不能不要命，拼命不要命那是把自己愧对。
每当疲惫的时候，就请停下脚步。遥想追逐的远方，汲取力量再上路。人活一世，能认识的人不少，能深交的人不多；能陪你笑的人不少，能陪你哭的人不多；能看见脸的人不少，能看清心的人不多。
一辈子很长也很短，很多我们无能为力，我们唯一能掌控，就是要好好地活着。山有山的高度，水有水的深度，没必要攀比，每个人都有自己的长处；风有风的自由，云有云的温柔，没必要模仿，每个人都有自己的个性。
你认为快乐的，就去寻找；你认为值得的，就去守候；你认为幸福的，就去珍惜。没有不被评说的事，只有不被猜测的人。做最真实最漂亮的自己，依心而行，无憾今生！
路是自己选的，就不怕走远；情是自己珍惜的，就不怕付出。人生一条路，走自己的路。活着不能太累，活着活着就不知道自己为谁；活着不能太苦，过着过着只剩自己受罪。
别羡慕他人大富大贵，谁的死后不是一把黄土，一缕灰；别羡慕他人豪车豪房，有个小窝安然入睡谁能说不美。我们都爱笑，但过得好不好只有自己知道；我们都挺累，但人只活一回又何苦身心疲惫。

饿了就吃，渴了就喝，困了咱就倒头睡；得意就笑，失意就跳，摔几个跟头没什么大不了。为人咱坚强无畏，是把那残酷的现实防备；处事咱问心无愧，做不到完美就尽力而为。生命有多珍贵，哪有心情总叹息后悔。
人活着有多不易，哪有时间总咀嚼伤悲。父母，爱人，别辜负；风雨，坎坷，别颓废；好时，好景，别浪费。好好地活着，别在乎太多；认真地活着，别奢求更多。只要没死，咱就好好地活，有滋有味地活。人生苦短，好好活着，别在乎太多。
人生感悟的句子，人生的路，总是深一脚，浅一脚，悲伤在路上，希望也在路上；生活的味总是苦一阵，甜一阵，疲惫在心里，欢乐也在心里。人生无常，人心善变。
内心宁静才能够处世不惊；足够淡定才不会患得患失。清清浅浅人生路，简简单单随缘行。平和于心态，充实于生活。人间三千事，淡然一笑间。
