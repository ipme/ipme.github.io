命运有一半在你手里，另一半在上帝的手里

发表于 2018-06-14 | 分类于 正能量

1、命运有一半在你手里，另一半在上帝的手里。你既然认准一条道路，何必去打听要走多久。如果为了安全而不和大海在一起，船就失去了存在的意义。
2、蝴蝶如要在百花园里得到飞舞的欢乐，那首先得忍受与蛹决裂的痛苦。萤火虫的光点虽然微弱，但亮着便是向黑暗挑战。拒绝严峻的冶炼，矿石并不比被发掘前更有价值。
3、你就像是一只暗夜里的灵魂，潜伏在我的周边。山路不象坦途那样匍匐在人们足下。用平常心来生活，用惭愧心来待人，心来处事，用菩提心契佛心。

4、正能量，你什么时候放下，什么时候就没有烦恼。不是某人使我烦恼，而是我拿某人的言行来烦恼自己。人生没有十全十美，如果你发现错了，就重新再来。
5、一个女人可以长得不漂亮，但是一定要活得漂亮，活的精彩。人，穷时简单，富了复杂；落魄时简单，得势了复杂。淡泊是人生的一种坦然，坦然面对生命中的得失。
6、爱死去时，通常都死得很艰难。既不是消失无踪，也不是寿终正寝。命运一半掌握在上帝手中，另一半掌握在自己手中。成功就是用自己手中的一半去赢得上帝手中的另一半。

一个人至少拥有一个梦想

发表于 2018-06-14 | 分类于 正能量

1、正能量，福报不够的人，就会常常听到是非；福报够的人，从来就没听到过是非。你永远要感谢给你逆境的众生。每一种创伤，都是一种成熟。
2、当你知道迷惑时，并不可怜，当你不知道迷惑时，才是最可怜的。狂妄的人有救，自卑的人没有救。激流勇进者方能领略江河源头的奇观胜景。如果圆规的两只脚都动，永远也画不出一个圆。
3、如果你想攀登高峰，切莫把彩虹当作梯子。脚步怎样才能不断前时？把脚印留在身后。要想成为强乾，决不能绕过挡道的荆棘也不能回避风雨的冲刷。行路人，用足音代替叹息吧！

4、假如你从来未曾害怕受窘受伤害，好就是你从来没有冒过险。耕耘者最信得过自己的汗水，每一滴都孕育着一颗希望的种子。只有脚踏实地的人，才能够说：路，就在我的脚下。
5、同样的一瓶饮料，便利店里块钱，五星饭店里块。很多时候，一个人的价值取决于所在的位置。不要害怕诱惑，你抵制了它，说明你是个好人；抵制失败了，说明你曾经是个好人……
6、一个人至少拥有一个梦想，有一个理由去坚强。你不要一直不满人家，你应该一直检讨自己才对。不满人家，是苦了你自己。不要浪费你的生命，在你一定会后悔的地方上。

冰冻三尺，非一日之寒

发表于 2018-06-14 | 分类于 正能量

1、憎恨别人对自己是一种很大的损失。逆境是成长必经的过程，能勇于接受逆境的人，生命就会日渐的茁壮。保持激情；只有激情，你才有动力，才能感染自己和其他人。
2、正能量，人生没有彩排，每天都是直播，不仅收视率低，而且工资不高。不要刻意去猜测他人的想法，如果你没有智慧与经验的正确判断，通常都会有错误的。
3、河流之所以能够到达目的地，是因为它懂得怎样避开障碍。不可压倒一切，但你也不能被一切压倒。一个人的成败，与他的心量有很大关系。山路曲折盘旋，但毕竟朝着顶峰延伸。

4、只有登上山顶，才能看到那边的风光。海浪为劈风斩浪的航船饯行，为随波逐流的轻舟送葬。有努力就会成功！书山有路勤为径，学海无崖苦作舟。亡羊补牢，为时未晚。
5、冰冻三尺，非一日之寒，成大事者不拘小节。智者千虑，必有一失；遇者千虑，必有一得。是金子总会发光。人在职场应该是一个扁豆，任何人怎么踩，结果还是一样，扁的。
6、再高的人有时也需踮足，再矮的人有时也需屈身。你永远要宽恕众生，不论他有多坏，甚至他伤害过你，你一定要放下，才能得到真正的快乐。这个世界本来就是痛苦的，没有例外的。

处事不必求功，无过便是功

发表于 2018-06-14 | 分类于 正能量

1、当你快乐时，你要想，这快乐不是永恒的。当你痛苦时你要想这痛苦也不是永恒的。说金钱是罪恶，都在捞；说美女是祸水，都想要；说高处不胜寒，都在爬；说天堂最美好，都不去。
2、世界是大舞台，公司是小舞台，家庭是后台，生活是连续剧，我们就是主角。处事不必求功，无过便是功；为人不必感德，无怨便是德。
3、正能量，当你劝告别人时，若不顾及别人的自尊心，那么再好的言语都没有用的。不要在你的智慧中夹杂着傲慢。不要使你的谦虚心缺乏智慧。

4、毁灭人只要一句话，培植一个人却要千句话，请你多口下留情。请你用慈悲心和温和的态度，把你的不满与委屈说出来，别人就容易接受。
5、我们确实有如是的优点，但也要隐藏几分，这个叫做涵养。幸福不是被致命的错误扼杀的，而是被不断重复出现的小错误一点点分解掉的。过错是暂时的遗憾，而错过则是永远的遗憾。
6、我始终坚信，即使再黑暗的夜也会走到尽头，并迎来曙光。钱有两种：花掉的是钱，是财产；没花掉的是“纸”，是遗产。失去的东西，其实从来未曾真正地属于你，也不必惋惜。
