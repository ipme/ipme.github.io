人总是珍惜未得到的，而遗忘了所拥有的

发表于 2018-06-15 | 分类于 经典美文摘抄

1、加倍努力，证明你想要的不是空中楼阁。胜利是在多次失败之后才姗姗而来。做最好的今天，回顾最好的昨天，迎接最美好的明天。沒有了爱的语言，所有的文字都是乏味的。
2、就算你留恋开放在水中娇艳的水仙，别忘了山谷中寂寞的角落深处，野百合也有自己的春天！生命太过短暂，今天放弃了明天不一定能得到。
3、梦想是一个天真的词，实现梦想是个残酷的词。只有轮回继续转，日升月沉草木枯荣。是谁说过：时间仍在，是我们在飞逝。人总是珍惜未得到的，而遗忘了所拥有的。
4、不登高山不知天之高也，不临深溪不知地之厚也。上帝从不埋怨人们的愚昧，人们却埋怨上帝的不公平。积极向上的人总是把苦难化为积极向上的动力。
5、美句摘抄，热情和欲望可以突破一切难关。相信你行，你就活力无穷。你要决心成功就要做自已的国王。昨天下了雨，今天刮了风，明天太阳就出来了。

6、不会生气的人是愚者，不生气的人乃真正的智者。千万人的失败，都有是失败在做事不彻底，往往做到离成功只差一步就终止不做了。信念是一把无坚不摧的利刃。
7、爬上最高的境界，你会陡然发现：那里的景色竟然是你司空见惯的。你在学习上这种尝试精神很可贵。通过云端的道路，只亲吻攀登者的足迹。决心是成功的开始。
8、当你没有借口的那一刻，就是你成功的开始。命运是可以改变的。信心、毅力、勇气三者具备，则天下没有做不成的事。如果你想得到，你就会得到，你所需要付出的只是行动。
9、明白事理的人使自己适应世界，不明事理的人硬想使世界适应自己。当困苦姗姗而来之时，超越它们会更有余味。你要求的次数愈多，你就越容易得到你要的东西，而且连带地也会得到更多乐趣。
10、再好的种子，不播种下去，也结不出丰硕的果实。改变自我，挑战自我，从现在开始。一切皆有因果。希望，是一种甜蜜的等待；想念，是一份温馨的心情；朋友，是一生修来的福分；爱情，是一世难解的缘分。

路在自己脚下，没有人可以决定我的方向

发表于 2018-06-15 | 分类于 经典美文摘抄

1、只有不断找寻机会的人才会及时把握机会。做一个决定，并不难，难的是付诸行动，并且坚持到底。旁观者的姓名永远爬不到比赛的计分板上。同样的瓶子，你为什么要装毒药呢？同样的心理，你为什么要充满着烦恼呢？
2、放弃谁都可以，千万不要放弃自己！人生是一场旅行，在乎的不是目的地，是沿途的风景以及看风景的心情。再长的路，一步步也能走完，再短的路，不迈开双脚也无法到达。
3、多一分心力去注意别人，就少一分心力反省自己。一个缺口的杯子，如果换一个角度看它，它仍然是圆的。宁愿跑起来被拌倒无数次，也不愿规规矩矩走一辈子。就算跌倒也要豪迈的笑。
4、世上没有绝望的处境，只有对处境绝望的人。真正的爱，应该超越生命的长度、心灵的寬度、灵魂的深度。泪水不代表着谁的失败，微笑不意味着谁的成功。
5、驾驭命运的舵是奋斗。不抱有一丝幻想，不放弃一点机会，不停止一日努力。目光能看多远，你就能走多远。改变自我，挑战自我，从现在开始。要想改变命运，首先改变自己。

6、今天的成功是因为昨天的积累，明天的成功则依赖于今天的努力。成功需要一个过程。命运掌握在自己手里，命运的好坏由自己去创造。
7、如果我们投一辈子石块，即使闭着眼睛，也肯定有一次击中成功。人生就像一个大舞台，每个人都有自己所要扮演的角色。至于要表演甚么角色，自己去决定。
8、美句摘抄，让我们将事前的忧虑，换为事前的思考和计划吧！不论你在什么时候开始，重要的是开始之后就不要停止。一个今天胜过两个明天。
9、是勇士滴在攀登路上的血，也是懦夫失望时流淌的泪。无论你觉得自己多么的不幸，永远有人比你更加不幸。路在自己脚下，没有人可以决定我的方向。
10、待人退一步，爱人宽一寸，人生自然活得很快乐。经验不是发生在一个人身上的事件，而是一个人如何看待发生在他身上的事。

父母给的那叫背景，自己打的才叫江山

发表于 2018-06-15 | 分类于 经典美文摘抄

1、人生，没有永远的伤痛，再深的痛，伤口总会痊愈。不相信奇迹的人永远都不会创造奇迹。是梦别怕疼，想赢别喊停！用微笑告诉别人，今天的我，比昨天更强，今后也一样。
2、我深知我不努力什么都不是。用你的笑容去改变这个世界，别让这个世界改变了你的笑容。让坚持成为一种信仰。你可知道我拼搏不怕阻挡，就一个理由，我要得到我想要的东西。
3、走过一些漫不经心，方知人生容不得放任。态度决定高度，习惯主宰人生。大道理人人都懂，小情绪却是难以自控。别人能行的，我也能行！努力吧，你要对得起你所受过的苦。
4、打击与挫败是成功的踏脚石，而不是绊脚石。太阳升起的瞬间，对于昨天已成永远！承载了太多期许的目光，怎能轻易辜负。今天的努力就是为了铺垫未来的幸福大道。
5、你要记住你不是为别人而活，你是为自己而活。没人能让我输，除非我不想赢！永远不抱怨，一切靠自己。累累的创伤就是生命给你最好的礼物，因为在每个创伤上面都标志着前进的一步。

6、不苦不累，生活无味；不拼不搏，等于白活。要想人前显贵，必须人后受罪！当你决定坚持一件事情，全世界都会为你让路。不逼自己一把，怎么知道自己不行。今天所有的懒惰都会换来明天的卑微。
7、美句摘抄，你要让你的实力配得上自己的野心。我讨厌什么都不做，却有所期待的自己！如果努力就可以做到的事情，为什么不去做。父母给的那叫背景，自己打的才叫江山。
8、花开不是为了花落，而是为了开的更加灿烂。往事不必遗憾，若是美好，叫做精彩。若是糟糕，叫做经历。没有耐心和耐力，就别怪现实太现实。努力与幸运成正比。
9、我可能会失败，但我不会一直失败。请享受无法回避的痛苦。没有伞的孩子，必须努力奔跑。运气就是机会碰巧撞到了你的努力。得之坦然，失之淡然，顺其自然，争其必然。
10、时间是治疗心灵创伤的大师，但绝不是解决问题的高手。天道酬勤。也许你付出了不一定得到回报，但不付出一定得不到回报。逆境是成长必经的过程，能勇于接受逆境的人，生命就会日渐的茁壮。

人活着一天，就是有福气，就应该珍惜

发表于 2018-06-15 | 分类于 经典美文摘抄

1、人活着一天，就是有福气，就应该珍惜，人生短短几十年，不要给自己留下更多的遗憾，日出东海落西山，愁也一天，喜也一天；遇事不钻牛角尖，人也舒坦，心也舒坦。
2、无论生活过的是多么的严肃认真，你总是能找到一个不嫌你二的人。清醒时做事，糊涂时读书，大怒时睡觉。许多人费劲爬到了梯子的顶端，才发现梯子架错了墙。
3、美文摘抄，人，只要拥有微笑，就能够豁达乐观地活着；人，只要时刻保持微笑，就能够在漫漫人生道路上拥有一张永久的通行证。

4、人情，人情，人之常情，要乐善好施，长于交往，“平时不烧香，急时抱佛脚”是行不通的，所以，“人的情绪要储存”，就象银行存款，存的越多，时间越长，红利就越大。
5、修心当以净心为要，修道当以无我为基。过去事，过去心，不可记得；现在事，现在心，随缘即可；未来事，未来心，何必劳心。
6、学会把自己的心态放低放平，多看自己的缺点，多看别人的优点，让傲慢的心变得谦虚恭敬，这样您所处的环境自然就融洽了。
