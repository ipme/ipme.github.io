在你穷的时候，要少在家里，多在外面

发表于 2018-06-14 | 分类于 名言警句

1、心如止水，笑对人生，坦然会让我们的生活快乐而美丽；微笑，会让我们敢于迎接人生的挑战。不要慨叹生活底痛苦，慨叹是弱者。
2、世界上没有一个人是只靠自己一个人的力量就可以达到成功的，成功的背后一定有许多人在有意或无意识地帮助你，你要永远感激他们。
3、生活就是一种积累，你若储存的温暖越多，你的生活就会越阳光明媚，你若储存太多寒凉，你的生活就会阴云密布。
4、如果说失败是人生的一种经历，那么这种经历会使我们的人生走向成熟；如果说一个人的成熟，必须历经沧桑的话，沧桑就能够成为一种奇特的美丽。
5、人活在世上，无非是面对两大世界，身外的大千世界和自己的内心世界。总有些事放不下，让人忘了时间。总有些事必须放下，因为还有明天。

6、或许我们走那么远，不是为了看风景，而是为了去天地的尽头会一会自己。因为只有在那样遥远的地方，你才能把喧嚣的人世抛在身后。
7、在你穷的时候，要少在家里，多在外面。在你富有的时候，要多在家里，少在外面。这就是生活的艺术。穷得时候，钱要花给别人，富的时候，钱要花给自己。很多人，都做颠倒了。
8、名言警句，通往光明的道路是平坦的，为了成功，为了奋斗的渴望，我们不得不努力。对自己不满是任何真正有才能的人的根本特征之一。
9、希望是生命的源泉，失去它生命就会枯萎。只有品味了痛苦，才能珍视曾经忽略的快乐；只有领略了平凡，才会收藏当初丢弃的幸福。
10、善待自己，幸福无比，善待别人，快乐无比，善待生命，健康无比。任何的限制，都是从自己的内心开始的。忘掉失败，不过要牢记失败中的教训。

你不能预见明天，但可以珍惜今天！

发表于 2018-06-14 | 分类于 名言警句

1、当所有人都低调的时候，你可以高调，但不能跑调。人生最大的错误是不断担心会犯错。目标的坚定是性格中最必要的力量源泉之一，也是成功的利器之一。
2、金钱难买健康，健康大于金钱，金钱难买幸福，幸福必有健康，生命的幸福不在名利在健康，身体的强壮不在金钱在运动！
3、如果你希望成功，以恒心为良友，以经验为参谋，以小心为兄弟，以希望为哨兵。当我们眺望远方的时候，近处的风景便看不清了。
4、如果说我懂的道理比别人多一点，那是因为我犯的错误比别人多一点。任何人都可以变得狠毒，只要你尝试过嫉妒。

5、金钱难买健康，健康大于金钱，金钱难买幸福，幸福必有健康，生命的幸福不在名利在健康，身体的强壮不在金钱在运动！
6、假如我不能，我一定要；假如我一定要，我就一定能。如果一个人不知道他要驶向哪个码头，那么任何风都不会是顺风。
7、名言警句，你不能左右天气，但可以改变心情。你不能改变容貌，但可以掌握自己。你不能预见明天，但可以珍惜今天！

诚实是人生的命脉，是一切价值的根基

发表于 2018-06-14 | 分类于 名言名句

1、没有一处遗产像诚实那样丰富的了。诚者，天之道也；思诚者，人之道也。少说空话，多做工作，扎扎实实，埋头苦干。要我们买他的诚实，这种人出售的是他的名誉。
2、当信用消失的时候，肉体就没有了生命。说谎话的人所得到的，就只即使觉说直话也没有人相信。不信不立，不诚不行。
3、诚信名言，信用就像一面镜子，只要有了裂缝就不能像原来那样连成一片。与朋友交，言而有信。一言而适，可能却敌，一言而得，可以保国。
4、坦白使人心地轻松的妙药。小信诚则大信立。信用是一种现代社会无法或缺的个人无形资产。诚信的约束不仅来自外界，更来自我们的自律心态和自身的道德力量。
5、言必信，行必果。老老实实最能打动人心。失去了诚信，就等同于敌人毁灭了自己。信用既是无形的力量，也是无形的财富。

6、诚实是人生的命脉，是一切价值的根基。诚实的人从来讨厌虚伪的人，而虚伪的人却常常以诚实的面目出现。忠诚需要完完全全的真实。
7、诚信是一种心灵的开放。一切的美德都包含在自我信赖里。靠着忠诚，你能与伟大的心灵为伴，有如一个精神上的家庭。
8、语言犹如枝叶，行为犹如果实。一言既出，驷马难追。失信就是失败。宁可失钱，不可失信。诚实是人生永远最美好的品格。
9、失足，你可能马上复站立，失信，你也许永难挽回。有一种真话是人所需要的，它以羞耻的火焰烧掉人心头的污秽和庸欲。这就是真话万岁！
10、坦白真爽最得人心。当信用消失的时候，肉体就没有生命。诚实的人必须对自己守信，他的最后靠山就是真诚。说谎话的人所得到的，就只是即使说了真话也没有人相信。

失去信用等于碎了的镜子，不可能修复

发表于 2018-06-14 | 分类于 名言名句

1、信犹五行之土，无定位，无成名，而水金木无不待是以生者。坦白是使人心地轻松的妙药。人之所助者，信也。不宝金玉，而忠信以为宝。
2、失去信用等于碎了的镜子，不可能修复。一两重的真诚，其值等于一吨重的聪明。没有诚实何来尊严。
3、诚信名言，不须犯一口说，不须着一意念，只凭真真诚诚行将去，久则自有不言之信，默成之孚。千教万教，教人求真；千学万学，学做真人。言必诚信，行必忠正。
4、以诚感人者，人亦诚而应。遵守诺言就象保卫你的荣誉一样。诚实和勤勉应该成为你永久的伴侣。世界未有比真诚人更为可贵的。
5、人无忠信，不可立于世。始终不渝地忠实于自己和别人，就能具备最伟大才华的最高贵品质。巧诈不如拙诚。才者，德之资也；德者，才之帅也。

6、精诚所至，金石为开。人以实，虽疏必密；与人以虚，虽戚必疏。信不足，安有信。走正直诚实的生活道路，必定会有一个问心无愧的归宿。
7、信用是难得易失的，费十年功夫积累的信用，往往由于一时的言行而失掉。诚实是人生的命脉，是一切价值的根基。
8、君子养心，莫善于诚。人类最不道德订户，是不诚实与懦弱。言无常信，行无常贞，惟利所在，无所不倾，若是则可谓小人矣。至诚而不动者，未之有也；不诚，未有能动者也。
9、甚至上帝也助诚实勇敢者一臂之力。没有诚信，何来尊严。失足，你可以马上恢复站立；失信，你也许永难挽回。
10、诚实是艺术的最大要素。只要我们总对自己诚实，艺术总在那里，它决不会让我们失望的。信用是无形的资产。坦白是诚实与勇敢的产物。
