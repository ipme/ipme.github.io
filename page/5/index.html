这一秒不放弃，下一秒就有希望！

发表于 2018-06-15 | 分类于 人生哲理

1、人生便如一块香料，是只有在坚持的信念为柴，坚守的行动为火炙烤中才能散发出最浓郁的芬芳。人在旅途，难免会遇到荆棘和坎坷，但风雨过后，一定会有美丽的彩虹。
2、人生没有彩排，每天都是现场直播。带着感恩的心启程，学会爱，爱父母，爱自己，爱朋友，爱他人。用爱生活，你会使自己幸福！用爱工作，你会使很多人幸福！
3、流过泪的眼睛更明亮，滴过血的心灵更坚强！大路走尽还有小路，只要不停地走，就有数不尽的风光。生命不在长而在于好，只要每一次尽力的演示，都值得鼓励与喝采。
4、坚持是雄壮的，因为坚持是由于百般地敲打而磨练出来的；坚持是甘甜的，因为无畏的与不倒的毅力早已在心中播下了胜利的种子。
5、哲理人生，你的努力，也许有人会讥讽；你的执着，也许不会有人读懂。在别人眼里你也许是小丑，在自己心中你就是国王！

6、有了坚持不一定成功；但没有坚持，就注定失败。对于成功，坚持的塑造必不可少！这一秒不放弃，下一秒就有希望！坚持下去就会成功！
7、不要轻言放弃，否则对不起自己！很多时候，不快乐并不是因为快乐的条件没有齐备，而是因为活得还不够简单。有目标的人生才有方向有规划的人生才更精彩。
8、失败是什么？没有什么，只是更走近成功一步，成功是什么？就是走过了所有通向失败的路，只剩下一条路，那就是成功的路。
9、坚持是一种意志的体现，运动员用坚持的精神获得优异的成绩，科研工作者用坚持的精神创造出一个又一个的成果。
10、有了坚持不一定成功；但没有坚持，就注定失败。对于成功，坚持的塑造必不可少！现实会告诉你，不努力就会被生活踩死。无需找什么借口，一无所有，就是拼的理由。

成功与失败之间，说简单也很简单

发表于 2018-06-15 | 分类于 人生哲理

1、丢下一些包袱后，你会惊讶地发现，原来自己可以飞的那么高。要冒一险！整个生命就是一场冒险，走得最远的人常是愿意去做、愿意去冒险的人。
2、在世界的历史中，每一伟大而高贵的时刻都是某种热忱的胜利。恐惧自己受苦的人，已经因为自己的恐惧在受苦。生命太过短暂，今天放弃了明天不一定能得到。
3、遇到困难时不要抱怨，既然改变不了过去，那么就努力改变未来。生活不是等待风暴过去，而是学会在雨中翩翩起舞。快乐不是因为得到的多而是因为计较的少！
4、行动是治愈恐惧的良药，而犹豫、拖延将不断滋养恐惧。没有天生的信心，只有不断培养的信心。得意时应善待他人，因为你失意时会需要他们。心若没有栖息的地方，到哪里都是流浪。
5、自己拥有的一切都是自己的失去所换来的！笑着面临，不去抱怨。悠然，随心，随性，随缘。肯定终身改动的，只在百年后，那一朵花开的时分。

6、只要不让年轻时美丽的梦想随岁月飘逝，成功总有一天会出现在你的面前。做任何事都要经受得挫折，要有恒心和毅力，满怀信心坚持到底。智慧是一把刀，一磨就亮。
7、难过时，吃一粒糖，告诉自己生活是甜的！心里的痛，总是借助文字来化解。也许你真的很有实力，但我未必看的起你。长得漂亮是你们的优势，活得漂亮是我的本事。
8、不大可能的事也许这天实现，根本不可能的事也许明天会实现。同在一个环境中生活，强者与弱者的分界就在于谁能改变它。咱们最值得自豪的不在于从不跌倒，而在于每次跌倒之后都爬起来。
9、哲理人生，成功与失败之间，说简单也很简单，说复杂也很复杂。总而言之一句话，只要做人不失败，那就是最大的成功！梦想太遥远遥远到我想放弃。
10、别人能够违背因果，别人能够害咱们，打咱们，毁谤咱们。但是咱们不能因此而憎恨别人，为什么？咱们必须要保有一颗完整的本性和一颗清净的心。

付出就要赢得回报，这是永恒的真理

发表于 2018-06-15 | 分类于 人生哲理

1、征服自己，就能征服一切。沙漠里的脚印很快就消逝了。一支支奋进歌却在跋涉者的心中长久激荡。你既然认准一条道路，何必去打听要走多久。让生活的句号圈住的人，是无法前时半步的。
2、松软的沙滩上最容易留下脚印，但也最容易被潮水抹去。蝴蝶如要在百花园里得到飞舞的欢乐，那首先得忍受与蛹决裂的痛苦。萤火虫的光点虽然微弱，但亮着便是向黑暗挑战。
3、拒绝严峻的冶炼，矿石并不比被发掘前更有价值。要想成为强者，决不能绕过挡道的荆棘，也不能回避风雨的冲刷。行路人，用足音代替叹息吧！
4、假如你从来未曾害怕、受窘、受伤害，好就是你从来没有冒过险。耕耘者最信和过自己的汗水，每一滴都孕育着一颗希望的种子。只有脚踏实地的人，才能够说：路，就在我的脚下。
5、只要能收获甜蜜，荆棘丛中也会有蜜蜂忙碌的身影。一个人除非自我有信心，否则带给别人信心。让咱们将事前的忧虑，换为事前的思考和计划吧！

6、人是为了期盼而活着的，当你疲倦了，心痛了，想想你的期盼你就会有继续前进的念想，因此不好抛弃任何完美的期盼期望念想。
7、付出就要赢得回报，这是永恒的真理，自古以来很少有人能突破它。然而，如果有人能够超越它的限制，付出而不求回报，那么他必须会得到得更多。
8、目标不是都能到达的，但它能够作为瞄准点。让咱们将事前的忧虑，换为事前的思考和计划吧！坚持不懈。待人对事不要太计较，如果太计较就会有悔恨！
9、哲理人生，给自我一点掌声，让我战胜内心的怯懦；给自我一点掌声，无畏的心更加的坚定；给自我一点掌声，温暖我独自前行的路。
10、人生舞台的大幕随时都可能拉开，关键是你愿意表演，还是选取躲避。所有的胜利，与征服自我的胜利比起来，都是微不足道。所有的失败，与失去自我的失败比起来，更是微不足道。

骄傲是断了引线的风筝稍纵即逝

发表于 2018-06-15 | 分类于 人生哲理

1、泉水，奋斗之路越曲折，心灵越纯洁。如果缺少破土面出并与风雪拚搏的气，种子的前途并不比落叶美妙一分。不要让追求之舟停泊在幻想的港湾，而应扬起奋斗的风帆，驶向现实生活的大海。
2、耕耘者的汗水是哺育种子成长的乳汁。不去耕耘，不去播种，再肥的沃土也长不出庄稼，不去奋斗，不去创造，再美的青春也结不出硕果。让珊瑚远离惊涛骇浪的侵蚀吗？那无异是将它们的美丽葬送。
3、再好的种子，不播种下去，也结不出丰硕的果实。没有失败，只有放弃。每个人学习中都会遇到各种困难、挫折和失败，不同的心态，是成功者和普通者的区别。只有奋斗才会有结果。
4、事常与人违，事总在人为。上帝的延迟并不是上帝的拒绝。任何事情的发生必有其目的，并且总会从某一角度有助于你的成长。重要的不是发生了什么事，而是要做哪些事来改善它。
5、哲理人生，驾驭命运的舵是奋斗。勤奋是你生命的密码，能译出你一部壮丽的史诗。对于攀登者来说，失掉往昔的足迹并不可惜，迷失了继续前时的方向却很危险。

6、奋斗者在汗水汇集的江河里，将事业之舟驶到了理想的彼岸。忙于采集的蜜蜂，无暇在人前高谈阔论。骏马是跑出来的，强兵是打出来的。假如我不能，我一定要；假如我一定要，我就一定能。
7、成功者绝不放弃，放弃者绝不成功。训练决定习惯，习惯决定性格，性格决定命运。不管多么险峻的高山，总是为不畏艰难的人留下一条攀登的路。稗子享受着禾苗一样的待遇，结出的却不是谷穗。
8、骄傲是断了引线的风筝稍纵即逝。自卑是剪了双翼的飞鸟，难上青天，这两者都是成才的大忌。盆景秀木正因为被人溺爱，才破灭了成为栋梁之材的梦。如果把才华比作剑，那么勤奋就是磨刀石。
9、过去不等于未来。没有失败，只有暂时停止的成功。士搏出惊涛骇流而不沉沦，懦夫在风平浪静也会溺水。志在峰巅的攀登者，不会陶醉在沿途的某个脚印之中。
10、如果惧怕前面跌宕的山岩，生命就永远只能是死水一潭。懦弱的人只会裹足不前，莽撞的人只能引为烧身，只有真正敢的人才能所向披靡。我们这个世界，从不会给一个伤心的落伍者颁发奖牌。
