石看纹理山看脉，人看志气树看材

发表于 2018-06-15 | 分类于 努力奋斗

1、越来越感觉，年轻人有目标，有理想，有梦想的重要性了。这是他们前进的动力，这是他们行动的方向，这是他们人生奋斗的灯塔。唯有这样，我们的青春才能不老不死不休。
2、坚持做自己，不是一意孤行，而是在市场竞争中，随缘而不变，稳定的贯彻自己的棋风棋路，非达目标，绝不中止。
3、失去后才知道珍惜，那是因为不懂得尊重；得到了再失去，那是因为不属于你；属于的要珍惜，才会永远。
4、不要急躁，要冷静；不要盲目，要思考；不要坐等，要主动；不要依赖，要自主！一定要摒弃一夜成富翁的幻想正视需要克服的困难，稳扎稳打，活学活用，这才是实际的经营。
5、人，应该要有自知之明，该得的就得，不该得的不必勉强。来的偶然，去的必然。该来的来，该去的去。来去之间，能留下多少就算多少。

6、鸟贵有翼，人贵有志。三军可夺帅也，匹夫不可夺志也。志，气之帅也。石看纹理山看脉，人看志气树看材。
7、志之所向，金石为开，谁能御之？志坚者，功名之柱也。登山不以艰险而止，则必臻乎峻岭。心志要坚，意趣要乐。
8、一人立志，万夫莫敌。把意念沉潜得下，何理不可得，把志气奋发得起，何事不可做。立志是事业的大门，工作是登门入室的旅程。
9、努力奋斗，壮志与毅力是事业的双翼。在年轻人的颈项上，没有什么东西能比事业心这颗灿烂的宝珠。不为穷变节，不为贱易志。
10、褴褛衣内可藏志。志气和贫困是患难兄弟，世人常见他们伴在一起。死犹未肯输心去，贫亦其能奈我何！困，你是人类艺术的源泉，你将伟大的灵感赐予诗人。

让努力成为一种习惯，让奋斗成为一种享受

发表于 2018-06-15 | 分类于 努力奋斗

1、贫穷是一切艺术职业的母亲。知识给人重量，成就给人光彩，大多数人只是看到了光彩，而不去称量重量。只要热情犹在，哪怕青春消逝。
2、让我们将事前的忧虑，换为事前的思考和计划吧！只有一条路不能选择，那就是放弃的路；只有一条路不能拒绝，那就是成长的路。
3、命运总是光临在那些有准备的人身上。人格的完善是本，财富的确立是末。只要路是对的，就不怕路远。
4、征服困难，就是荣耀。你一天的爱心可能带来别人一生的感谢。旁观者的姓名永远爬不到比赛的计分板上。

5、强烈的信仰会赢取坚强的人，然后又使他们更坚强。只要我们能梦想的，我们就能实现。平凡的脚步也可以走完伟大的行程。
6、最重要的就是不要去看远方模糊的，而要做手边清楚的事。那些尝试去做某事却失败的人，比那些什么也不尝试做却成功的人不知要好上多少。
7、努力奋斗，做对的事情比把事情做对重要。最足以显示一个人性格的，莫过于他所嘲笑的是什么东西。
8、最有效的资本是我们的信誉，它小时不停为我们工作。让努力成为一种习惯，让奋斗成为一种享受。
9、如果你已经尽了最大的努力，就不要为任何的失败而气馁；若没有，就不要找任何借口。生命必须有裂缝，阳光才能照得进来。路上有坎坷，人才变得坚强起来。
10、你只有走完必须走的路，才能过想过的生活。有的路，你必须一个人走，这不是孤独，而是选择。

谁不是一边受伤，一边学会坚强

发表于 2018-06-15 | 分类于 努力奋斗

1、不要活在别人的嘴里，不要活在别人的眼里，而是把命运把握在自己的手里。才能一旦让懒惰支配，它就一无可为。
2、阻止你前行的，不是人生道路上的一百块石头，而是你鞋子里的那一颗石子。积木搭起的房子看似很美，却会在不经意间轰然倒塌。
3、当你握着两手沙子时，一定就拿不到地上那颗珍珠了。每一种创伤，都是一种成熟。用鞭子抽着，陀螺才会旋转。
4、努力奋斗，成功需要付出代价，不成功需要付出更高的代价。你的负担将变成礼物，你受的苦将照亮你的路。

5、人都有梦想，但不同的是，有的人只梦，有的人敢想。认定的事情，就义无反顾，即使结果会受伤，也在所不惜。
6、握住的手，始终要放，谁不是一边受伤，一边学会坚强。生活的舞台，不是每一面都精彩，而是要发现精彩的那一面。
7、大器不必晚成，趁着年轻，努力让自己的才能创造最大的价值。最快乐的人并不是一切东西都是最好的，但他们会充分享受自己已有的东西。

人生是很累的，你现在不累，以后就会更累

发表于 2018-06-15 | 分类于 努力奋斗

1、出门走好路，出口说好话，出手做好事。积极的人在每一次忧患中都看到一个机会，而消极的人则在每个机会都看到某种忧患。
2、努力奋斗，我们总是希望自己快快长大，却又担心再也回不去无忧无虑的时光。生活从未变得轻松，是你在一点一点变强大。
3、习惯是一个很可怕的东西，因为习惯，会觉得理所当然。因为习惯，没有人去想如果失去是什么模样。别总把悲伤挂在嘴上，每个人都有自己的故事……
4、心若改变，你的态度跟着改变；态度改变，你的习惯跟着改变；习惯改变，你的性格跟着改变；性格改变，你的人生跟着改变。

5、人生是很累的，你现在不累，以后就会更累。人生是很苦的，你现在不苦，以后就会更苦。唯累过，方得闲。唯苦过，方知甜。
6、当朋友不开心的时候，你只需要静静地坐在他身边，默默陪伴。即使什么也不说，他也会感觉好很多很多。只有千锤百炼，才能成为好钢。
7、你脑子里东西的多寡，就关系着你将来的前途。你希望掌握永恒，那你必须控制现在。宁可慢些，不要太急而错误；宁可笨些，不要太巧而败事。
