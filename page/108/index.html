不要让一个女人孤独，懂得珍惜才配拥有！

发表于 2018-06-14 | 分类于 早安心语正能量

不要让一个女人孤独，一旦她适应了就真的不再需要你了，当女人真的生气了，不是哭也，不是闹，而是选择沉默，一个男人最大的失败，就是把爱你的女人让给了别人，懂得珍惜才配拥有！
曾经拥有的不要忘记，难以得到的更要珍惜，属于自己的不要放弃，已经失去的留作回忆。别拿尊严去作践自己，与其卑微里犯贱，不如高傲的放手。
不是每个你在乎的人都要去争取，要看值不值；但是每个在乎你的人都要去珍惜，因为不离不弃，谁做起来都不容易！无论何种感情，要走的人你留不住。等待关心，等待到关上了心；期盼拥有，期盼到没有回首。
人生那么短暂，凭什么让不看重你的人来影响心情；人心如此矜贵，干嘛让眼里没你的人来亵渎你的感情。不要为不愿在你身上花费时间的人，而浪费你的时间。

更不要错误地认为你放不下的人，同样也放不下你。你用谎言去验证诺言得到的一定是欺骗，宁可高傲发霉，也不委曲求全。拿出点姿态：我可以惯着你，也可以换了你！
早安心语，走错了路，要记得回头；看错了人，要懂得放手。谁都不傻，谁也不笨，用眼看人，会走眼，用心感受，才是真。
世上的事，多是这样，有顺，有逆；人生的情，也是如此，有喜，有悲。顺境，不要得意，狂妄傲慢，喜悦能够持续；逆境，不要气馁，绝望悲戚，事情还有转机。
得不到的别要，够不着的别够，先做好该做的，再做想做的，每一次挫折都有值得坚强的原因，每一天都有值得欢乐的理由。想开、看开，才会放开。

真心对你的人，会把你牵挂在心

发表于 2018-06-14 | 分类于 早安心语正能量

有人不在意你，你却在珍惜；有人在伤你，你却在原谅。再多的关心，感动不了不爱你的心；再长的等待，等不来不想你的人。情可以无价，但不能廉价；可以放宽底线，但不允许践踏底线。
明知道爱你是不对的，可是你一笑，我又义无反顾的扑了上去。世界上不爱的理由有很多：忙，累，不合适，而爱的表现只有一个：就是，想和你在一起。
为爱可放弃所有，唯独不能放弃自尊。真心对你的人，会把你牵挂在心，绝不会因忙碌而去忽略；真正在乎你的情，会把你捧在手心，绝不会忽冷忽热让你去猜测感觉。
早安心语，人就是这样，越是得不到的东西就越想争取，而拥有之后却不懂得珍惜。简单的陪伴，你却毫不在意，一直的给予，你却漫不经心。

其实触动人心的东西，往往是最平凡中的不离不弃。感人肺腑的情意，往往是最真实就是同行。感情中最可贵的，莫过于就是有人一直在乎你。
人心都是相对的，以真换真；感情都是相互的，用心暖心。要走的人留不住，装睡的人叫不醒，不喜欢你的人感动不了。别拿尊严去作践自己，与其卑微里讨好，不如高傲里撒手。
所谓温暖，从来不是单向的。你把自己抱的再紧，一样还是冷。你把别人照顾的再好，没有反应，一样是苦涩的。人在孤独时，渴望的只是一份陪伴，心脆弱时，需要的仅是一种懂得。
别让你爱的女人掉眼泪，无论老婆还是知己，让女人失望的不是你没钱而是在身上感受不到温暖、看不到未来，永远不要低估一个女人，只要你拿心来换，她便会用命赌。

你开心，这一天过了，不开心，一天也过了

发表于 2018-06-14 | 分类于 早安心语正能量

心宽，意味着对外在世界的相容，也意味着内在世界开放，表明自己与周边世界相处和谐，宽厚多恕地对人对事。
心善，是养生的一大营养素，一个人行善事，能经常帮助弱者，一个乐善好施的人，心理会处于最佳状态，这样的人大多健康长寿。
俗话说得好：心有多宽，命有多好！放下自己的固执己见，宽心做人，舍得做事；多一份平和，多一点温暖，生活才有阳光。
人心如路，越计较，越狭窄；越宽容，越宽阔。不与君子计较，他会加倍奉还；不与小人计较，他会拿你无招。不计较，貌似是让别人，实际是给自己的心开拓道路。

生活、工作中遇到不顺的事，对自己说一声：今天会过去，明天会到来，新的一天会开始。得意时要看淡，失意时要看开。人生有许多东西是可以放下的。只有放得下，才能拿得起。
早安心语，多一些宽容，多一些大度，挥挥手，笑一笑，一切的不愉快都会成为过去。心宽，路才宽；心宽，命才好。
当一个男人不开心的时候，想想自己还剩下多少天可以折腾，还有多少时间可以浪费，你开心，这一天过了，不开心，一天也过了，想明白了，你就不会再生气了。

不忘人恩，不念人过，不思人非，不计人怨

发表于 2018-06-14 | 分类于 早安心语正能量

当一个男人烦恼的时候，想想人生就是减法，见一面少一面，活一天少一天，还有什么好烦恼的呢，不忘人恩，不念人过，不思人非，不计人怨，不缺谁，不少谁，对得起良心就好。
当一个男人忧伤的时候，想想生活就是一场路过，赤裸裸来，匆匆而去，带不来一丝尘土，带不走一片云彩，什么功名利禄，人情世故，什么你都带不走，想明白了，还有什么好难过的。
当一个男人不满的时候，想想和富人比，我们活着知足就是幸福，和病人比，我们活着健康就是幸福，和死人比，我们还活着就是幸福，人这辈子要活好，人要看淡，心要简单。
作为女人，很多时候，我们不能被别人了解和理解，痛苦的时候只能自己默默的忍受着，受伤的时候悄悄的承受。人生中不是所有的痛苦都能够倾诉，不是心中所有的苦都能够流露出来。

作为女人，累了就让心吹吹风，伤了就从梦里想来，痛了停下脚步。一辈子是一场修行，短途是旅行，长途是人生，或许，旅行才能让自己遇到最好的自己。不管多么残酷，一定要记住坚强，付出了多少努力就会收获多少光芒。
早安心语，作为女人年，要做一个平静的人，做一个善良的人，常常把微笑挂在嘴边，让生活充满快乐。
作为女人，每天都有琐碎的事情，要把心放宽。很多事情不会都圆满，笑着对待琐碎的事情。万事忧，贵在心放宽。
女人要心善，但千万要有底线！因为，很多人就是得寸进尺的。若是不设定底线，别人就会当没有底线，从而一踩再踩、一进再进，甚至堂而皇之的反客为主，从被动接受变成主动索取。
心善的女人应该像是玫瑰花，既有花的香，花的美，也该有花的刺。活得善良美丽其实并无不可，但也要尽量保护自己不受伤害。
