最怕深交后的陌生，认真后的痛苦

发表于 2018-06-14 | 分类于 爱情说说

1、我不知道自己到底在执着什么，但我知道，我一直都在为难自己。别动不动就付出真心，他给你的糖可能也给过别人。
2、这世上根本不存在感同身受，只因你给的嬉笑太盛，无人在意你的心酸认真，所以别傻傻的摊开伤口向别人诉苦，这世上多的是撒盐的人。
3、有些人会一直刻在记忆里的，即使忘记了他的声音，忘记了他的笑容，忘记了他的脸，但是，每当想起他时的那种感受是永远都不会改变的。
4、饿了吃东西，困了去睡觉，伤了看医生，痛了你忍着，路途自己走，眼泪自己擦，请不要到处宣扬，因为没人心疼你。

5、伤感爱情，你要明白，再烫手的水还是会凉，再饱满的热情还是会退散，再爱的人还是会离开，所以你要乖，要长大，不再开口就是来日方长，我们要习惯人走茶凉，那么从此你信你的神明，我守我的地狱。
6、最怕，深交后的陌生，认真后的痛苦，信任后的利用，温柔后的冷漠……别折腾了，一个人使劲，维持不了两个人的感情，你忙着靠近他忙着离开。
7、有些事，想多了头疼，想通了心疼。曾经说好的永远，曾经说好的不放手，曾经说好的要一起走下去，曾经的一切，原来都只是曾经。
8、一起走过的路，一起的所有一切，一个人会傻笑，然后是无比的心痛，或许，最美的事不是留住时光，而是留住记忆，但愿，时光，只如初见。
9、有那么一段时间，我觉得自己无话可说。就像是平时，见到陌生人，硬生生打完招呼就卡在那里，不会找话题不会聊天，甚至自己一个人开始发呆，不知道说什么。
10、当你全力以赴，打算对一个人好的时候，你就变成了傻子，聋子，眼里除了他，什么都没有，就连伤害，都变成了一场爱情的考验。

我打了很多字，又默默的删了

发表于 2018-06-14 | 分类于 爱情说说

1、真正的感情压抑不了的，别谈卑微的暗恋，似是而非的暧昧，友情以上恋人未满这种话，嘴巴堵住了眼睛会泄漏，眼睛闭上了心跳会泄漏，心跳掩饰住了，小动作就露出马脚。
2、有时候不善于表达的人挺吃亏的，付出多用情深，却没人说你好。从陌生人，到相见，到相识，最终离别，然后忘记，不如从不相见。
3、我打了很多字，又默默的删了，因为有些东西说不清。往事就此不提了，夜还很长，余生也是，需要留点力气重新等待，等等后来的人……
4、有时候如果和很重要的人发QQ或微信，他一直没有回我，我就会删了那个对话框，总感觉看见了那个对话框，就好像看见了自己的卑微和讨好。

5、我对你的喜欢，三句根本写不完，可惜你一句也不会看。开始的时候，你怎样任性，他都可以包容，可到后来，你连脾气都不敢有了。
6、世界太黑，童话太假，人心太暗，我们太傻。主动久了才发现，心已随着那不冷不热的话语渐渐冷却；在乎多了才感觉，情已伴着那不痛不痒的态度慢慢瓦解。
7、伤感爱情，有没有那么一瞬间，在大街上看到一个熟悉的背影，心突然就跳乱了节拍，直到发现原来只是陌生人，于是一整天，全是回忆。
8、喜欢你是真的，想在一起也是真的，但喜欢没能升华成爱情，唯有把它变长长久久的友情。人和人终究是不同的，有人风雨中送来的晚餐，却不及某人一句随手的晚安。
9、总该长大，总该试着去成熟，放弃一些曾经你以为不可分离的东西，即使这个过程漫长且难捱，当做睡前关上的灯。
10、咖啡没放糖你嫌苦，放了糖你又嫌甜，苹果没削皮你嫌不干净，削了皮你又嫌肉太少，开始你说想喝热开水，我打来了你放久了冷了又不喝，我不主动你嫌不够爱你，我主动多了你又嫌我烦。

忽冷忽热最折磨人，反反复复最伤人

发表于 2018-06-14 | 分类于 爱情说说

1、你什么时候回来啊，没有你的日子，我真的快撑不下去了。见过你爱我的样子，所以你不爱我的时候，真的很明显。
2、有一种爱，叫白首，有一种伤，叫放手，就像你的一去不回头，便是永久，可是我只能在黑夜里，偷偷泪流。或许，我和你，只是一场梦，梦醒后，便是孤独一人。
3、一个人晕着也能去看病打针，一个人难受也能爬起来去买药，一个人没带雨伞也能淋雨走回家，一个人能挣钱买想要的东西，一个人也会哭。
4、我主动来找你，是因为我太喜欢你，我已经不主动找你，因为你每一句回复都敷衍得那么不走心，不是不喜欢你了，只是不敢再那样喜欢了。

5、有的人，失去后时常想起，有的情，不在后追悔莫及。遇见了你，从来没有想过放弃，失去了你，浓浓思念依旧继续。
6、爱情说说，长颈鹿的脖子那么长，哽咽的时候是不是很难受？章鱼有三颗心脏，心痛的时候是不是很疼？壁虎有一条会断的尾巴，逃跑的时候是在欺骗对方还是在伤害自己？
7、忽冷忽热最折磨人，反反复复最伤人。我已经越来越像一个木偶，面无表情信仰孤独，越来越懒散于交际，该怎么办呢我真焦躁，我呀，除了自渡，他人爱莫能助。
8、我一直盼望你来找我，发型凌乱，胡子拉碴，带着满身的悔意敲我的门，一下恶狠狠一下充满深情地看着我，不应该是现在这样穿着得体，英俊有礼，不急不慌地介绍挽着别的女人，不应该是这样。
9、这些年来，离我最近的是你，离我最远的也是你，后来我终于明白，所有的悲欢都是我一个人的灰烬。
10、刚开始的时候我对你其实只是好感，后来就变成了喜欢，再后来也就成为了一种习惯，然后在放弃喜欢你的时候还是忍不住喜欢，你就像一种坏习惯，想改也很难改，强忍着却还是想要喜欢你，谢谢你从我的全世界路过。

一往情深深几许，深山夕照深秋雨

发表于 2018-06-14 | 分类于 爱情说说

1、有些伤只适合放在心里，就像有些人只适合忘记。想了一万种再见到你的情形，却从未想过不会再见到你的这一种可能。
2、后来你没有挽留我，也没有回头。你每天熬夜的时候有没有想过，你喜欢的那个人已经打呼噜了，而且梦里没有你，醒了也不会爱你。
3、一个漫游聊天记录的功能，毁了多少情人，看透了多少人。世界上最远的距离，不是爱，不是恨，而是熟悉的人，渐渐变得陌生。
4、我记得你爱我，或者我记反了。有时候，我们就像仙人掌，防备了别人，孤单了自己。

5、再见了，我终于要放弃你了，可我怎么还是想哭呢？时隔多年，你突然出现在访客记录里，我呆坐了一夜，哭着笑，笑着哭。
6、伤感爱情，你要明白，再烫手的水还是会凉，再饱满的热情还是会退散，再爱的人还是会离开，所以你要乖，要长大，不再张口就是来日方长，而要习惯走人茶凉。
7、他熄灭了烟，说起了以前，眼泪在眼眶里打转硬是不让它掉下来，他说，前半生就这样吧，余生也别将就，大不了孤独一生。
8、情难舍，心难留，花朝月夜，转眼便成指间沙。不如生生世世，两两相忘，且归去，看青山隐隐，流水迢迢，望断天涯。
9、谁解相思味，谁盼良人归，谁捧胭脂泪，谁描柳月眉，谁将曲中情怨，谁思红袖轮回，谁相思错付，一江东流水！
10、一往情深深几许，深山夕照深秋雨。既不回头，何必不忘。既然无缘，何须誓言。今日种种，似水无痕。明夕何夕，君已陌路。
