秋天到了，秋风也是人们所盼望的

发表于 2018-06-15 | 分类于 秋天的古诗句

秋天到了，秋风也是人们所盼望的。不是落叶漫舞的惊喜，也不是秋日凄美的遐想，而是收获时的甜美与怡悦。它带着喜悦攀上颗颗成熟的果实，拥着成功跳上填满果实的竹筐，伴随甜蜜，添上快乐。给果园呈现出无尽的甜美与芳馨，展示着“秋日果实遍地熟”的丰收景象。

对于秋天的阳光，我有着独特的感受

发表于 2018-06-15 | 分类于 秋天的古诗句

对于秋天的阳光，我有着独特的感受。它不像春天里的那样撩人，也不像夏天里的那样热烈，更不像冬天里的那样凝重。尤其，午后的阳光，鹅黄妩媚、暧昧含蓄、教人迷恋、令人神往。尽避，秋天的阳光带着微微的凉意，确依然沁人心脾、恬静温馨。

秋分至，秋风起，记得早晚添衣裳！

发表于 2018-06-15 | 分类于 秋天的古诗句

一场秋雨一场寒，有人惦念心就暖；一个转换一季秋，几多牵挂致朋友！秋分至秋意浓，注意身体要保重；天涯远念心中，记得添衣多珍重。
真正的朋友无需多客套，只要一声问候就足够；真正的感情无需说承诺，只要一直陪伴在身边；真正的缘分无需多深情，只要一生牵手一起走！

几多惦念，送给忙碌的你，工作再忙，也要抽时间休息！几多关心，送给不易的你，生活再累，也要照顾好自己！几多想念，送给知心的你，距离再远，心里也一直有你！
这个世上最幸福的事，莫过于有人想着你，念着你，发自内心的牵挂着你！
这世上最感动的情，莫过于有心装着你，疼着你，一生一世都在乎着你！
这世上最难得的人，莫过于：一个真心的朋友，一个贴心的爱人，一辈子都珍惜着你的人！
秋分至，秋风起，记得早晚添衣裳！秋雨落，一场寒，喝杯热茶暖身体！秋意浓，情更浓，有个牵挂你的朋友在远方！

秋天风起，秋天雨落，致牵挂的朋友！

发表于 2018-06-15 | 分类于 秋天的古诗句

十月，又是一年深秋。秋风渐凉，秋雨飘落，冷意渐渐袭来，凉风吹过枝头。
一声暖意的问候，致我牵挂的朋友；一份真心的惦记，伴你走过深秋。
天冷了，出门多加一件外套。天冷了，回家多喝一杯热茶。天冷了，早晨要记得吃早餐。天冷了，晚上要记得关门窗。
给不了拥抱，给不了牵手，陪不了病痛，陪不了左右，这一句句叮嘱，千万要记在心头！你的安好，便是我最大的心愿。

工作再忙碌，也不会忽略了真正的朋友。生活再劳累，也不会忘记了深刻的友情。
因为在乎，所以放在心上。因为惦记，所以关心冷暖。
有人陪伴的路，是平坦的；有人同看的景，是绚烂的；有人懂得的情，是眷恋的；有人惦念的秋，是温暖的。
一辈子最幸福的时刻，不是捡钱，不是中奖，而是身在寒秋，收到朋友暖心的问候。
好好保重身体，早晚记得加衣；好好照顾自己，晚上早点休息；好好心疼自己，别太奔波疲惫。
远方的你，一直在我的心里；记挂的你，一定要幸福如意。
