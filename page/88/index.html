南京市区一老头，在家热得受不了了

发表于 2018-06-15 | 分类于 搞笑段子

01
南京市区一老头，在家热得受不了了，去加拿大的女儿家。刚到加拿大第一天上街，看到很多人在买彩票抽奖，有人抽到三等奖LV包，有人抽到二等奖劳力士手表……老头也上去买彩票碰碰运气，打开奖票，众人立即欢呼，一等奖出现了！
工作人员上前告诉他：先生，恭喜您，请准备行李，明天出发，一等奖是中国5A级旅游区江苏南京夫子庙三日游，老头大喊一声：我日尼玛卖比……02
搞笑段子，四点起的是送奶工，五点起的是菜贩子，六点起的是小孩子和爸妈，七点起的是教授，八点起的是公务员，九点起的是小秘，十点起的是厨子，十一点起的是作家，十二点起的是主持人，十三点起的是二奶，十四点起的是小姐，十五点起的是病人，十六点以后还没起的请打120！
03
昨晚，在出租上问：师傅，您这车也太坑人了吧，虽然打车费上调了，也不能跑了才十多分钟就103.9块钱了，你丫非逼我打电话报警吗？！司机：傻碧，瞧好了，那是收音机，下面才是计价器。04
去饭店吃饭，一推开门一股热浪扑面而来。我：你们怎么不开空调？服务员：老板不让开，说费电。我：那你们把窗开开呀，热死了！服务员：老板说关上窗户，外头的人就会以为我们开空调了。我：……

记得小时候，我们农村一般谁家结婚

发表于 2018-06-15 | 分类于 搞笑段子

01
搞笑段子，高三，一女生骂一老实男生，我日你娘，我日你娘……男生开始不理，后来被骂急了，一拍桌子站起来说了句：我日你娘俩！女生顿时安静了。02
记得小时候，我们农村一般谁家结婚，办喜事都会选择星期日。亲戚朋友有工作的也能这天来，不用耽误工作，人也来的多，就图个热闹。
一到星期日，我就换上干净的衣服，到镇上，看哪家饭店办喜事，就找个座位一坐，默默地吃起来。一个小孩子也没人注意，婆家，娘家都以为是对方的亲属，遇到好的还能给个红包。就这样蹭着蹭着，渐渐的就长大了……
03
晚上10：30，接到一个陌生来电，对方问：“是王刚吗？”我说不是。晚上11：36，那人又打来电话：“王刚，睡了吗？”我说你打错了！0：21，电话又响了：“王刚，在干嘛？”我发火了：“我不叫王刚！”
凌晨4点，我怎么也睡不着，想起那个电话，于是拨了过去。对方迷迷糊糊地问：“谁呀？什么事？”我冷冷地说：“没什么事，就是想问你找到王刚没有？”04
去应聘司机，老板是个女的，四十多岁，也没聊什么，说了一会家常，和工作经历，然后老板给我拿了瓶营养快线，我说不喝，然后老板又给我拿了瓶冰糖雪梨，我也说不喝，最后老板给我拿了一支烟，我接了过来，然后老板说不招司机了，就把我请离办公室，弄得我一头雾水。

果遇到一个女的，对你有意思，你上不上？

发表于 2018-06-15 | 分类于 搞笑段子

01
搞笑段子，宿舍六个人聊起了三国，李哥问我：兄弟，三国里你最喜欢哪个？我想了想说：肯定最喜欢诸葛亮，夜观天象，知天下大事。说完我问他们，他们五个都喜欢貂蝉……02
吃饭的时候问老公：你们如果遇到一个女的，对你有意思，你上不上？老公说，如果女的不丑，干净没病，还不要钱，上上也无妨，我心里一阵激动，老公，我上班的地方有一个小帅哥，长的不丑，白白净净，青春活力，他约我今天晚上看电影，我晚上就晚点回来啊……
03
早上挤地铁的时候手肘不小心碰到一个女孩子的胸部，很抱歉的和她说无意的，结果她胡搅蛮缠，非说我是故意的，实在是懒得争论，直接伸手摸了一把她的胸部，告诉她这次是故意的，然后她就安静了。04
晚上下班回家，在一楼等电梯，随后走来两个男人，他们满身酒气，我下意识有些害怕，生怕他们耍酒疯。电梯来了我进去，见他俩没动，就问：“不进来吗？”其中一个男人红着脸道：“我们一身酒气，怕熏着你，也怕吓着你，我们等下一趟，姑娘你先上去吧。”05
今天跟一个投资大鳄聊天，他神秘兮兮跟我说，经过多年商海浮沉研究出了一款保本保收益又有机会冲击高额回报的理财产品，我一听赶紧请他吃饭虚心求教，他说存三万到余额宝，每天拿两块钱利息买双色球……握草。

闺密陪着我去相亲，对方为了缓解尴尬的气氛

发表于 2018-06-15 | 分类于 搞笑段子

01
搞笑段子，闺密陪着我去相亲，对方为了缓解尴尬的气氛，讲了一个有点黄的笑话，我正低着头假装害羞，二货闺密大声对我说道：靠，还没有你昨天讲的那个黄呢。我……02
晚上刚躺下，听见小区楼下有个男人撕心裂肺大喊：“打！打！打！打死！！打死！！往死里打！！！臭婆娘，你反了！反了！”我当即感觉马上要出大事的节奏，人命关天！瞬间睡意全无，穿上一只拖鞋就冲下了楼。和我一样冲下楼的好几十人。。。远远看见一个男人正指挥媳妇在倒车。
03
刚才我洗澡，合计逗逗出差的老公，于是就拍了两张果照微信发给老公，结果一不小心发到了单位群里！我脑袋一热点撤回没点好，点了删除……卧槽！卧槽！还好我比较机智，在群里又说了一句：“不好意思，网速太慢，2G网，衣服没传上去……”04
还记得初中有个同学要转走，（她人缘不错），我们班在暑假给他开了送行会，大家都哭的稀里哗啦的，每个人都给他送了礼物，交换联系方式，晚上还请她吃饭，然后各种舍不得。结果开学她因为父母的事没走，然后，一学期都没怎么跟我们讲话。05
公司一女同事经常带着自己的小儿子来上班，大伙没事总爱逗他：你把裤子脱了，拿小弟弟给我们看，我们就拿棒棒糖给你吃。于是小家伙一次次地脱裤子。今天小家伙又来了，想吃糖但又不好意思了，就说道：阿姨，你们可不可以先给我糖吃，回头我叫我爸爸脱裤子给你们看？呃……
