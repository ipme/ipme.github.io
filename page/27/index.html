坚持到底不会错的，坚持到底就是成功！

发表于 2018-06-14 | 分类于 人生哲理

1、生活不要太劳累，弄得自己很疲惫，快乐幸福多体会，烦恼忧愁不用追，吃点好的别嫌贵，联系朋友别欠费，天空辽阔任你飞，再多困难别后退！
2、成功根本没有秘诀可言，如果有的话，就有两个：第一个就是坚持到底，永不言弃；第二个就是当你想放弃的时候，回过头来看看第一个秘诀，坚持到底，永不言弃。
3、斧头虽小，但经历多次劈砍，终能将一棵最坚硬的橡树砍刀。要有把路走绝的精神，激流勇退是一种境界，把路走绝是一种精神。坚持到底，不会错的；坚持到底，就是成功！
4、掌握坚持的人是成功的，是永不言弃的。穷则思变，差则思勤！没有比人更高的山没有比脚更长的路。行动是治愈恐惧的良药，而犹豫、拖延将不断滋养恐惧。
5、女人一定要有的五样东西：扬在脸上的自信、长在心底的善良、融进血里的骨气、两侧外泄的霸气、刻进命里的坚强！哪怕是最没有期望的事情，只要有一个勇敢者去坚持做，到最后就会拥有期望。

6、有志者自有千计万计，无志者只感千难万难。业精于勤而荒于嬉，行成于思而毁于随。难免和痛苦不期而遇，内心不好背叛自我，快乐的人不是没有痛苦，而是不被痛苦所左右！
7、一个彻底诚实的人，如同黑夜中也健步如飞的人，内心只有一种声音。他人的劝诫世俗的虚荣生活的诱惑权力的胁迫，这一切都无法入耳入心。当一个人有所追寻时，他只会看到他所追寻的东西。
8、每个人都有自我的剧场，他集经理演员提词编剧换景包厢侍者看门人诸职于一身，此外还是观众。平凡的脚步也能够走完伟大的行程。当你珍惜自我的过去，满意自我的此刻，乐观自我的未来时，你就站在了生活的最高处。
9、明天是世上增值最快的一块土地，因它充满了期望。嘲讽是一种力量，消极的力量。赞扬也是一种力量，但却是积极的力量。你从来都没有发现，在我的背后，还有一个你在默默支持。
10、哲理人生，一个人背两人的债，受多少苦我也能挨。如果我不坚强，那就等着别人来嘲笑。早晨给自己一个微笑，种下一天旳阳光。强者会哭，但绝不会认输。

在成功的道路上，坚持是非常重要的

发表于 2018-06-14 | 分类于 人生哲理

1、骏马是跑出来的，强兵是打出来的。有时候，执着是一种负担，放弃是一种解脱。别人不会在意你的一切，只有自己成就自己，才能在别人面前抬得起头。
2、宁可慢些，不要太急而错误；宁可笨些，不要太巧而败事。场法则，晋升渠道：直面上司领导，主动推销自己，积极脱颖而出，建立人脉通道，全力以赴助上司。生活有度，职场有道，认真学习，提升自己。
3、开启中考成功之门，钥匙有三！其一：勤奋的精神；其二：科学的方法；其三：良好的心态！胜利贵在坚持，要取得胜利就要坚持不懈地努力，饱尝了许多次的失败之后才能成功。
4、哲理人生，我相信所有坚韧不拔的努力，迟早会得好报酬。功到自然成，成功之前难免有失败，然而只要能克服困难，坚持不懈地努力，那么，成功就在眼前。
5、朋友，人生之路并非只有坦途，也有不少崎岖与坎坷，甚至会有一时难以跨越的沟坎儿。在这样的紧要关头我们只有一种选择：再向前跨出一步！尽管可能非常艰难，但请相信：只要坚持下去，你的人生会无比绚丽！

6、逆境总是有的，人生总要进击。愿你不要屈从于命运的安排，坚韧不拔，锲而不舍！做永远的生活强者！骆驼走得慢，但终能走到目的地。男人的帅不在脸蛋，而在岁月积淀下来的睿智与淡定。
7、在成功的道路上，坚持是非常重要的，面对挫折时，要告诉自己：饱尝甜头。在攀登人生阶梯的旅途中，我们不怕慢，只怕停。只要不停止，再慢的速度也能达到顶峰。
8、人生要敢于接受挑战，经受得起挑战的人才能够领悟人生非凡的真谛，才能够实现自我无限的超越，才能够创造魅力永恒的价值。黑暗总会过去，黎明终会到来。
9、在各自岗位上尽职尽责，无需豪言壮语，默默行动会诠释一切。如果心胸不似海，又怎能有海一样的事业。命运如同手中的掌纹，无论多曲折，终掌握在自己手中。
10、真正的强者，不是流泪的人，而是含泪奔跑的人。只做第一个我，不做第二个谁。只有经历过地狱般的折磨，才有征服天堂的力量。只有流过血的手指才能弹出世间的绝唱。

人要学会自己照顾自己，自己激励自己

发表于 2018-06-14 | 分类于 人生哲理

1、所有的胜利，与征服自己的胜利比起来，都是微不足道。所有的失败，与失去自己的失败比起来，更是微不足道。没有人会让我输，除非我不想赢。有多少永远值得坚持，有多少永远配得上永远。
2、只有一条路不能选择那就是放弃的路；只有一条路不能拒绝那就是成长的路。目标和信念给人以持久的动力，它是人的精神支柱。困难，激发前进的力量；挫折，磨练奋斗的勇气；失败，指明成功的方向。
3、一个真正的将军是拼出来的。吃得苦中苦，方为人上人。每个人在成功之初都会遇到各种困难。但失败是成功之母，只有经历失败的洗礼，才能有丰富的成功。要珍惜每个人的态度，再平凡的人也有自己的主见，也会决定你的质量。
4、人的一生中都会有机会获得一些或多或少的掌声，但仅仅是别人的掌声，那是远远不够的。人要学会自己照顾自己，自己激励自己。要做到这一点，你必须在生活中不断地为自己鼓掌。
5、哲理人生，一个人承受孤独的能力有多大，他的能力就有多大。人既不是天使，又不是禽兽；但不幸就在于想表现为天使的人却表现为禽兽。

6、不要因为一点瑕疵而放弃一段爱情，毕竟在爱情里，需要的是真情，而不是完美。很多时候，我们视为刻骨铭心的记忆，而别人却早已忘记，与其纠结于心，不如看淡，看轻。
7、人生最重要的一点是，永远不要迷失自己。人生最精彩的不是成功的那一瞬间，而是回头看，那段漆黑看似没有尽头、苦苦摸索的过程。其实，我只是很在意，在意在我所在意的人的心里，我，在哪个位置。
8、把重要的观念，方法写下来，随时提示自己。任何值得拥有的东西，一定是值得等待。不要因为他的离开而哭泣，应当为他的出现而笑。
9、命运常常是一种折磨。不论是谁，在人生中有时总难免身陷逆境，一时又无力扭转面临的颓势，那么最好的选择就是暂时的忍耐，事物总是在不断的运动和变化，在忍耐中等待命运转折的时机。
10、当你决定了一辈子干什么以后，你就要坚定不移地干下去，就不要随便地换。没有天生的信心，只有不断培养的信心。时光匆匆流逝过，平平淡淡才是真。忍耐任由风雨过，守得云开见月明。

生命的道路上永远没有捷径可言

发表于 2018-06-14 | 分类于 人生哲理

1、不要指望谁陪你一辈子，没光的时候连影子都会离开你。痛不痛只有自己知道，变没变只有自己才懂。不要问我过得好不好，死不了就还好。第一个青春是上帝給的；第二个的青春是自己努力的。
2、在你穷的时候，要少在家里，多在外面。在你富有的时候，要多在家里，少在外面。这就是生活的艺术。穷得时候，钱要花给别人，富的时候，钱要花给自己。很多人，都做颠倒了。
3、不要慨叹生活底痛苦，慨叹是弱者。通往光明的道路是平坦的，为了成功，为了奋斗的渴望，我们不得不努力。我大好的一个人，凭什么跑到别人的生命里去当插曲。
4、如果我坚持什么，就是用大炮也不能打倒我。希望是引导人成功的信仰。如果没了希望，便一事无成。希望是惟一所有的人都共同享有的好处；一无所有的人，仍拥有希望。
5、乌云再密集，天也有放晴的时候；海洋再广阔，船也有到岸的时候。只要用乐观的心态来看待事物，面对生活，就会发现这个世界真的好美。

6、哲理人生，少年为未知的前途彷徨，老年为已知的寂寞而发愁；独独中年，江山已定，前途在握，分分秒秒都可以自行支配，充分享受。
7、不要感叹人生苦短。拾起梦想的种子，用一生的时间去播种，在最后一刻去收获。你会发现，你的一生，其实很精彩、很充实！青年是人生的骄傲，也是时代未来的希望。
8、宁可笑着流泪，绝不哭着后悔。无论你昨晚经历了怎样的泣不成声，早上醒来这个城市依旧车水马龙。没有人能一路单纯到底，但别忘了最初的自己。比我差的人还没放弃，比我好的人仍在努力，我就更没资格说我无能为力！
9、生命的道路上永远没有捷径可言，只有脚踏实地走下去。只要还有明天，今天就永远是起跑线。人人都会错过，人人都曾经错过，真正属于你的，永远不会错过。
10、有没有人爱，我们也要努力做一个可爱的人。不埋怨谁，不嘲笑谁，也不羡慕谁，阳光下灿烂，风雨中奔跑，做自己的梦，走自己的路。
