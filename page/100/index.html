小狗热得吐出舌头不停地喘气

发表于 2018-06-15 | 分类于 夏天的古诗句

1、七月已经过去了，跟着来到的是暑月，这仿佛是自然一方面，看着塔布篱厂里的情人那样热烈，特为和他们斗胜争强似的。这块地方上的空气，在春天和初夏的时候，本来非常清新，现在却变得停滞不动，使人困懒了。浓厚的气息老压在他们上面；正午的时候，一片大地好像都昏昏晕去。草原上较高的山坡，都叫跟埃塞俄比亚那样一样地灼热的太阳晒成黄色，不过这里水声淙淙的地方，却还有鲜明青绿的草色。
2、夏天是一个突飞猛进的季节。一切都在肆无忌惮地疯长，尤其是那夏天的绿色，又浓又深，霸占得漫山遍野，虽然是映衬着花朵，但事实上却是绿肥红瘦。雨后春笋一夜间冒出大地，竹子快速成长，关节胀得直响，原上芳草萋萋离离，林间参天大树又增加了一圈年轮。
3、八月的骄阳像张火伞，中午，天空亮得耀眼，好像一大张烧烫了的白马口铁板。路边垂柳的细枝一动不动，树影缩成了一团，蒙着一层尘土的叶子都蔫蔫地打卷了。柏油路面也被晒得软软的，向远方望去，寂静无人的马路上，似乎有一片透明的蒸气在升腾。

4、月季花的枝叶是绿油油的，上面有许多又细又尖的小刺，只要你不小心碰着，就会立刻出血，而且很痛。夏日炎炎，小鸟不知躲藏到什么地方去了；草木都低垂着头；小狗热得吐出舌头不停地喘气。
5、形容夏天，望着色彩艳丽的一大片樱花，我真怀疑是九天仙女把撕碎的彩缎撒向人间。夏天，草木特别茂盛，冬青树的叶子油亮油亮的，老榆树枝繁叶茂，给人们撑起了一片浓浓的绿阴。

整个城市像烧透了的砖窑，使人喘不过气来

发表于 2018-06-15 | 分类于 夏天的古诗句

1、风暴一停，黄尘歇在大地上。枯黄的沙冈子，像一条死龙似的，横卧在草原里。夏末的太阳，从一块青石板似的薄云后一露头，就像从空中抛下无数火球，滚落在草原上，几乎将草原上的一切都烧焦了。河边的杨柳，垂着叶子，低着梢头。才盛开的野菊花，也皱卷起花瓣，萎缩在绿草下。热，热得人们几乎喘不过气来。
2、七月，透蓝的天空，悬着火球似的太阳，云彩好似被太阳烧化了，也消失得无影无踪。整个城市像烧透了的砖窑，使人喘不过气来。狗趴在地上吐出鲜红的舌头，骡马的鼻孔张得特别大。
3、我喜欢，我喜欢满目苍翠的夏天，因为夏天可纵情泳池游泳。夏天给孩子们带来欢乐，他们是夏季里最快乐的天使。我喜欢，我喜欢狂风暴雨的夏天，因为夏雨是那么豪爽干脆。夏天的荷花给我们露出笑脸，夏天的荷叶向我们展示魅力。

4、形容夏天，那天，天热得发了狂。太阳刚一出来，地上已经着了火，一些似云非云似雾非雾的灰气，低低地浮在空中，使人觉得憋气。知了不住地在枝头发着令人烦躁地叫声，像是在替烈日呐喊助威。
5、看这朵小花两片天蓝色的花瓣舒展，两根细细的花蕊高高地翘起，淡黄色的蕊头微微晃动，多像翩翩起舞的小蝴蝶。月季花可以美化环境，它不怕风吹雨打，日晒虫叮。只要有土壤和水分，它就可以生存下去，给我们化环境。

烈火般的阳光，扫尽清晨晶莹的露珠

发表于 2018-06-15 | 分类于 夏天的古诗句

1、天气是那样炎热，仿佛一点星火就会引起爆炸似的。与那山峦之上的森林，那赫然耸立的山峦。组成了道道屏障，将独立家屋围困其间。空气和太阳一起燃烧。
2、盛夏的天气，烈火般的阳光，扫尽清晨晶莹的露珠，统御着宇宙一直到黄昏后，这是怎样沉重闷人的时光啊！人们在这种的压迫下，懒洋洋的像是失去了活跃的生命力，尤其午后那更是可怕的蒸闷；马路上躺着的小石块发出孜孜的响声，和炙人脚心的灼热。
3、形容夏天，一切处于静止状态。偶尔一声鹿鸣虎啸，这里的世界暂时停止了进程。划破了这沉沉的静寂，那长长的尾音滞留不散，里面含着死的恐惧与生的艰辛。

4、七月盛夏，瓦蓝瓦蓝的天空没有一丝云彩，火热的太阳炙烤着大地，河里的水烫手，地里的土冒烟。躺在凉席上，感觉自己就像是铁板牛柳。白色的月季花，洁白如玉，人们爱把它作装饰品。
5、夏天是一个燃烧的季节。核聚变的火球烈日中天，狗吐着舌头，蝉烦躁地鸣叫，柏油马路被晒得软软的，鱼浮出水面出来换气。玫瑰花整个夏天一直都在开花。

盛夏则是一年中最煎熬难捱的日子

发表于 2018-06-15 | 分类于 夏天的古诗句

1、红色的月季花的颜色最丰富，有红色，有粉红，还有大红。人们爱把它装饰在家里和商场里。月季花的花一片一片的，每片是椭圆形的。花瓣里是黄色的花蕊，透漏出醉人的芳香。
2、六月十五那天，天热得发了狂。太阳刚一出来，地上已像下了火，一些似云非云，似雾非雾的灰气低低的浮在空中，使人觉得憋气。月季红艳艳的花儿在枝头怒放，颜色是那么浓，那么纯，没有一点杂色，简直像一团燃烧的火焰。
3、夏天一到，这青山一天一个样，经过烈日的暴晒，骤雨的浇淋，那草木就窜枝拔节很快地长起来，变得葱茏青黑了。这时，山地里一片青纱帐起，那些狼呀山猫呀野兔子呀……逍遥自在地活跃在里面，就像鱼儿游在海洋里那样。

4、这一年的夏天，天气特别热，我们溪岭乡虽说是个山乡，白天也闷热得叫人受不了，你浑身脱个精光，只穿条短裤，汗水还是直淌。要到傍晚太阳落了山，方才有风打北干山那边吹来，凉飕飕的，夹着苦艾和松树脂的气息。
5、形容夏天，盛夏，则是一年中最煎熬难捱的日子，令人一筹莫展。盛夏的阳光已是近于“刻毒”了，似乎它有意要与人类作对，报复人类，气得人们诅咒它，甚至于要学后羿射掉这个“毒日”了。
