白天想多了，头疼，晚上想多了，难眠

发表于 2018-06-14 | 分类于 个性签名

人生一世，都不容易！人活一世，总活在别人的眼中，总想别人觉得你好才行，这样往往最苦的就是自己。真心对你的人要珍惜，自己的身体更要爱惜！失意了，不要放弃；走远的，不要在意。
为了生计，每天奔波劳碌，累了，苦了，没人替你受，你总为别人着想，总为别人而活，却不为自己好好活。好好善待自己，生命只有一次，别浪费了！人这一辈子，自己最宝贵！
社会越来越现实，人情也越来越冷淡，身边的人来来去去，珍惜你的却寥寥无几。很多人，因利靠近你，也因利远离你，谁是真心，谁是假意，日久见人心。人这辈子，真心最宝贵！
人活着是为了什么？开心是一天，烦恼也是一天，何不学会放下，心情好一切都好。想太多庸人自扰，想开了就没有烦恼，计较多了心就乱了，其实放开一切都好。人这一辈子，心态最珍贵！

心简单就快乐，生活简单就幸福。什么都可以丢，但尊严丢不得；什么都可以输，但信任输不得。好好爱自己，认真生活，开心生活。风雨一辈子，珍惜眼前最重要！
人生一世，除了生死，其他的都是小事，别想没用的，别想过去的，别想烦心的，别想生气的。开开心心过日子，轻轻松松享生活！
生活，太难，每个人都有心事背负。生活，太累，每个人都有难言之隐。生活，太苦，每个人都有苦闷烦恼。
个性签名，我们只有一次生命，闭上眼，就是永远，没呼吸，就是死亡。好好活着，别在乎太多！谁没有经历过打击，谁没有遭遇过挫折。
生活已经很累，别让心更疲惫。人际已经很复杂，何必事事都在乎！该放的事，必须放下，该忘的人，别再牵挂。
失去的，就算了，离开的，就忘了别赖着；过去不肯走，别缠着往事不放手。心累了，静一静，不要胡思乱想。白天想多了，头疼，晚上想多了，难眠。
用加法看待快乐，用减法消除恩怨。处的来就好好处，处不来也别记恨。心就那么大，容不下太多复杂。感情易变化，要珍惜就在当下！
好好活着，别在乎太多！珍惜拥有的，才能减少更多的遗憾。忘记失去的，才能露出更多的笑容。

人和人，并不相同，心和心，无法比较

发表于 2018-06-14 | 分类于 个性签名

人和人，并不相同，心和心，无法比较。小人，喜欢算计，好人，行善积德，真心，待人真诚，假意，对人奉承。有些事，何必解释，时间会给出满意的答复。有些人，何必揭穿，时间会帮他慢慢的改正。事不由心，才会烦恼，人不由己，才会悲哀。
近朱者赤，近墨者黑，和什么样的人交往，就会变成什么样的人。如果你想像苍鹰一样展翅高飞，就和群鹰一起翱翔天空，千万不要和麻雀挤枝头。如果你想像骏马一样驰骋大地，就要和马群一起狂奔，千万不要学猪吃饱就睡。
道不同不相为谋，志不同不必为友。酒肉朋友，和你吃喝玩乐，会拖你后腿。真心朋友，和你一起奋斗，会带你成功。如果你想变得优秀，就和优秀的人在一起，如果你想变成强者，就和有本事的人交往，他们会激励你，鼓励你，指点你，耳濡目染，你也会变得更好！

个性签名，与其追马，不如种草吸引马，与其拉拢，不如逼自己进步。当你强大了优秀了，身边都是有本事有能力的朋友。
当你尽我双手，里面什么都没有，当你打开双手，世界就在你手中！人生总是活在得与失之间，舍得笑你会得到快乐，舍得宽容你会得到是大气！
舍得舍得，你会渐渐懂得，你的舍是一种收货一种得到，人生总是很奇妙，你有何苦去苦苦追寻一个目标，在那边庸人自扰，舍的小，就可能得到大，舍得近就可能得到远，舍得物质你可以收获精神。
放得下，往往你会走的更远，有所放弃你才会有所追求和奋斗，什么都不愿意去放弃的人，反而会失去更珍贵的东西，舍得诚实，你得到了真心的朋友，舍得面子你得到是真实的自己，舍得酒色，你得到的是健康。

世界很大，好像一转身，就不知道谁会消失

发表于 2018-06-14 | 分类于 个性签名

有人说，世间所有的相遇，都是久别重逢。在人海茫茫，众生芸芸中，彼此能够相遇，就像是在赴那个偶然中必然的缘，萍水相逢是缘，错误的遇见也是缘。无论相知与错过，皆是缘分，没有好与坏，没有对与错。
人的一生会遭遇无数次相逢，有些人是你看过便忘了的风景，有些人则在你的心里生根抽芽。那些无法诠释的感觉，都是没有来由的缘分，缘深缘浅，早有分晓。
相遇与别离，每一天都在上演。世界很小，好像一转身，就不知道会遇见谁；世界很大，好像一转身，就不知道谁会消失。
个性签名：有错过，才会有新的相遇；有旧人离开，才会有新人进入我们的生活。至于我们一生会遇见多少人，没有人知道，也没有人在意。
有些相遇，能够彼此陪伴走一程，已然是最好的结局。所以，无论缘长缘短，相遇本是一场偶然，懂得惜缘，便不负相识一场；离散是一场必然，懂得释怀，才不会深陷其中。

生活中，很多人喜欢站在自己的视角，以为很了解别人，自以为是地随便谴责批判别人，以为全世界就自己是对的。其实，你不知道别人的生活，更无法对别人的酸甜苦辣感同身受，所以不要轻易地去指责别人或者批判别人。
别人有着怎样的生活，走什么样子的道路，旁人永远无法真正了解。所以，任何人都没有资格去指责评论别人，因为未曾真正的经历，就无法体会其中的一切。
千人千样，每个人都有自己的故事，每个人都是自己故事里的主角，不管故事是平淡无奇，还是曲折坎坷，每个人都已经历不同的故事，或悲伤或幸福。
人生无常，谁都会有眼泪有悲伤，我们要学会欣赏和悲悯，学会善待他人，毕竟人生一世谁都不容易。你不知道别人的生活，请不要随意去指责。
一花一世界，一叶一菩提。这是何等的境界！其实做到这一点并不难，做到善待他人、体谅他人，便能在这世俗的世界中得到内心的宁静。
有句话说的好：幸福并不取决于财富、权利和容貌，而是取决于你和周围人的相处。这辈子，你到底误解了多少人？
