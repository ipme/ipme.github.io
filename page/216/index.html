人活着，能忘就忘，不能忘，就放一放！

发表于 2018-06-14 | 分类于 人生感悟

人活着，记性不能太好。很多事情，想多了，心神不宁；记久了，作践心灵。不如把往事清零，还自己一分清净。能忘就忘，不能忘，就放一放！
忘记是是非非，不能因一次的矛盾，否定了别人的品行。忘记恩恩怨怨，记住别人的好，宽恕别人的错。有时候真的挺恨自己记性太好。活得太明白的人总过得太累。
吵架的时候，记不住你撂的所有狠话，记不住你对我毫不留情的责骂。好像这段感情里都是歌舞升平，只要看不到华丽衣袍下面的虱子，就好像不存在似的。
分手的时候，记不住你手指的温度，记不住你充满爱意的眼神。这些会让我胸膛响起玻璃碎裂的声音，让我意识到从此以后这些都不是为我，而是给了另一个人。

为人心胸宽阔，别人佩服，自己快乐！忘，不是失忆，而是不想记起！忘，不是无情，而是删除多余！忘，不是伤感，而是心态乐观！忘，不是糊涂，而是活得理智！
人生苦短，烦恼，忘了吧，痛苦，忘了吧，不满，忘了吧，委屈，忘了吧。所有让自己不开心的事，统统都忘了吧。
生命只有一次，你我只活一回，与其背着烦恼活，不如带着美好过！与其在纠结焦虑中渡过，不如在乐观洒脱中活着！
人生感悟，谁的人生，没有点困难；谁的感情，没有点遗憾；谁的心情，没有点烦恼；谁的生活，没有点辛酸。
人生苦短，与其在纠结焦虑中渡过，不如在乐观洒脱中活着！该放就放了吧，该弃就弃了吧，该舍就舍了吧，该忘就忘了吧！

万物都在改变，人又凭什么不变！

发表于 2018-06-14 | 分类于 人生感悟

这个世上，没有任何东西是一成不变的。人会老，树会黄，有地老天荒，也有人走茶凉；月亮有圆缺，四季有更替。有不离不弃，也有各奔东西。万物都在改变，人又凭什么不变！
再熟悉的朋友，彼此不联系也会慢慢陌生；再亲密的感情，彼此不关心也会渐渐疏远。世道在变，人心也在变。多少兄弟姐妹，因为钱财也会翻脸；多少难兄难弟，能同苦却不能同甘。
有些人耐得住寂寞，却经不起诱惑；有些人扛得起贫穷，却经不起富贵；有些人走得过风雨，却经不起平淡。
人，都是会变的！从个性单纯到成熟老练；从快言快语到含蓄内敛；从不会说谎到学会伪装；从渲染伤感到酸楚隐藏。
社会，是一个大染缸。改变了我们最初的模样。你不愿改变，也得改变，你不改变，无法适应社会。现实，是一个大熔炉。锤炼了我们最终的坚强。你不想坚强，也得坚强，你不坚强，就得向现实投降。

人总是会变的，从年少无知到沉稳成熟；从喜形于色到宠辱不惊；经历过喜怒哀乐，也品尝过酸甜苦辣；知道了什么是人情世故，也明白了什么叫冷暖自知。
感悟人生，人生本就是一场历练，经历的多了，也就慢慢开始改变。有的人变好了，变得善解人意；有的人变坏了，变得虚情假意。
人总是会变的，你不变，就追不上这个时代；你不变，就注定要被他人伤害。但无论如何，请保持好自己的初心，不要让在意你的人寒心；不管怎样，请保护好原本的模样，爱你的人希望你永远别来无恙……
我们不是想要改变，而是必须改变，一个人如此，而人心，更是一样！其实，人，还是当成的那个人，只是心，变了！可能会变得我们不认识，但是，最初的那个自己，其实还在！只是社会不允许最初的自己生存，为此，变，变，变，才有了现在的自己！
这个人世间，每个人都会改变，谁都不能例外。以前挑嘴的，如今爱吃了，以前厌恶的，如今看淡了，以前喜欢的，渐渐疏远了。
大事不糊涂，小事装糊涂，变其实没什么不好，变才能让自己活得更好！我们变了，但依旧保持干净的灵魂，我们变了，但依旧秉承善良的本心。

总是有些路，你要独自坚强，然后继续前行

发表于 2018-06-14 | 分类于 人生感悟

人生不如意十之八九，事事都不会顺心。所以才有了拼搏，努力，向上这些代名词。不以物喜，不以已悲。值得我们用一辈子来修炼！
所以，再艰难的困境，走下去就是柳暗花明。关键是，你要挺住，不要停。细想一下即使失败又能怎么样？本来我们一无所有的来到这个世界，大不了也就是个一无所有。不是吗？
人一辈子，总有几次让生活打得头晕目眩的时候，总有几次陷在黑暗中孤立无援的时候。但无论处境多么糟糕，只要你扛过这一秒，下一秒就是柳暗花明。
所以，能够伤害你，打败你的人，只有你自己。没有谁的人生不困难，不挣扎，倘若人生路上没有障碍，人还有什么可做的呢？又怎能正确认识自我，突破自我呢？
老话说“天无绝人之路”，人生最怕的不是上天给你设了道坎，而是你连坎都不肯过就举手投降。所有对你有帮助的东西，都不是唾手可得的。
人生感悟，可能你正被生活搞得焦头烂额，但你一定要相信，所有的苦难都是生活的必须。

生活其实是欺软怕硬的，常常，你越怂，它越不停招惹你、跟你过不去。你要是强硬起来，它就会乖乖低眉顺眼听你的话。
看似粗糙地迟钝地麻木地坚持，却仍是不停止，不放弃，就算脸上全是泪，也闷着头固执向前。然后，那些艰难，不知何时，就真的过去了。
无论境况多糟糕，只要不认怂，生活就没办法撂倒你。再艰难的困境，走下去便是柳暗花明。
人生难免经历挫折与失败。面对失败，有的人选择放弃，有的人则选择继续挑战。放弃的人把成功的机会拒之千里，只有坚持挑战、不服输的人，才能拿到通向成功的钥匙！
其实在很多时候，人都一样。生活中的点点滴滴，那些苦和累只有自己了，不能说给身边的人听，因为他们不是你，不会了解你的辛酸，就算再苦再累，也要自己扛住。
生活中，谁都想取得令人羡慕的成绩，谁都想拥有让人称赞的生活。可是要知道：每一个光鲜的背后，总有不为人道的心酸；每一个淡然的表面，总有难以言说的苦衷。
总是有些泪，你要学会擦干，然后保持微笑；总是有些路，你要独自坚强，然后继续前行。

不把人生目标定得太高，自己就会欢乐常在

发表于 2018-06-14 | 分类于 人生感悟

受挫一次，对生活的理解加深一层；失误一次，对人生的醒悟增添一阶；不幸一次，对世间的认识成熟一级；磨难一次，对成功的内涵透彻一遍。从这个意义上说，想获得成功和幸福，想过得快乐和欢欣，首先要把失败，不幸，挫折和痛苦读懂。
被人误解的时候能微微的一笑，这是一种素养；受委屈的时候能坦然的一笑，这是一种大度；吃亏的时候能开心的一笑，这是一种豁达；无奈的时候能达观的一笑，这是一种境界；危难的时候能泰然一笑，这是一种大气。
不与别人盲目攀比，自己就会悠然自得。不把人生目标定得太高，自己就会欢乐常在。不刻意追求完美，自己就会远离痛苦。不是时时苛求自己，自己就会活的自在。不每每吹毛求疵，自己就会轻轻松松。
人生感悟，你改变不了环境，但你可以改变自己；你改变不了事实，但你可以改变态度；你改变不了过去，但你可以改变现在；你不能控制他人，但你可以掌握自己；你不能预知明天，但你可以把握今天。

有时候，你难免多心。心眼一多，许多小事就跟着过敏，於是别人多看你一眼，你便觉得他对你有敌意；少看你一眼，你又认定他故意对你冷落。多心的人注定活得辛苦，因为太容易被别人的情绪所左右。有时候，与其多心，不如少根筋。
以平常之心，接受已发生的事。以宽阔之心，包容对不起你的人。以不变之心，坚持正确的理念。以喜悦之心，帮助须帮助的人。以放下之心，面对难割舍的事。以美好之心，欣赏周遭的事物。以真诚之心，对待每一个人。
人生没有十全十美，如果你发现错了，就重新再来；没有机会重新来过的，就让它永远沉入时间的大海。千万不要用一个错误去掩盖另一个错误。总是浸泡在注满过错的池子里，只会让自己越来越虚弱。
不要太心急，不用太焦虑，所有事物的流动，都自有它的顺序。很多的时候，只要我们等一等，就安全了，让一让就过去了，忍一忍就和谐了，看一看就明白了，尝一尝就知道了，问一问就清楚了，想一想就悟出了。
