在比夜更深的地方，一定有比夜更黑的眼睛

发表于 2018-06-14 | 分类于 励志语录

1、先知三日，富贵十年。创造机会的人是勇者。等待机会的人是愚者。多用心去倾听别人怎么说，不要急着表达你自己的看法。
2、同样的瓶子，你为什么要装毒药呢？同样的心理，你为什么要充满着烦恼呢？活着一天，就是有福气，就该珍惜。当我哭泣我没有鞋子穿的时候，我发现有人却没有脚。
3、多一分心力去注意别人，就少一分心力反省自己，你懂吗？时间才是每个人的终极资源。在比夜更深的地方，一定有比夜更黑的眼睛。
4、人生就像一个大舞台，每个人都有自己所要扮演的角色。至于要表演甚么角色，自己去决定。伟人之所以伟大，是因为他与别人共处逆境时，别人失去了信心，他却下决心实现自己的目标。
5、如果早上醒来，你发现自己还能自由呼吸，你就比在这一周离开人世的万人更有福气。什么时候也不要放弃希望，越是险恶的环境越要燃起希望的意志。

6、积极的人在每一次忧患中都看到一个机会，而消极的人则在每个机会都看到某种忧患。许多人企求着生活的完美结局，殊不知美根本不在结局，而在于追求的过程。
7、也许有些人很可恶，有些人很卑鄙。而当我设身为他想象的时候，我才知道：他比我还可怜。所以请原谅所有你见过的人，好人或者坏人。
8、一句无心的话也许会点燃纠纷，一句残酷的话也许会毁掉生命，一句及时的话也许会消释紧张，一句知心的话也许会愈合伤口、挽救他人。
9、既然一切都会过去，那我们一定要抓住现在的。阻碍我们飞翔的力量，是来自我们内心的恐惧。本性的苏醒，往往在遭遇真实之后。
10、青春励志，才能一旦让懒惰支配，它就一无可为。相信朋友的忠诚。相信自己的勇气。相信敌人的愚蠢。生气是拿别人做错的事来惩罚自己。

你不能改变容貌，但可以掌握自己

发表于 2018-06-14 | 分类于 励志语录

1、命运就像自己的掌纹，虽然弯弯曲曲，却永远掌握在自己手中。不要浪费你的生命，在你一定会后悔的地方上。
2、激情，这是鼓满船帆的风。风有时会把船帆吹断；但没有风，帆船就不能航行。人是可以快乐地生活的，只是我们自己选择了复杂，选择了叹息！
3、不管从什么时候开始，重要的是开始以后不要停止；不管在什么时候结束，重要的是结束以后不要后悔。
4、你不能左右天气，但可以改变心情。你不能改变容貌，但可以掌握自己。你不能预见明天，但可以珍惜今天。
5、尝试去把别人拍过来的砖砌成结实的地基，生活就不会那么辛苦了。行动不一定带来快乐，而无行动则决无快乐。你可以一无所有，但绝不能一无是处。

6、苦到极处回甘，冰到极处回温。不要配不上自己的野心，也不要辜负了曾经所受的苦难！哪有那么多的一夜成名，其实都是百炼成钢。
7、不要让未来的你，讨厌现在的自己，困惑谁都有，但成功只配得上勇敢的行动派。我们最大的错误，就是把最差的脾气和糟糕的一面都给了最亲近的人，却把宽松和耐心给了陌生人。
8、青春励志，人一辈子不长不短，走着走着，就进了坟墓，你是要轰轰烈烈地风光下葬，还是一把骨灰撒向河流山川。
9、梦想也许今天无法实现，明天也不行，但是只要今天的自己比昨天的自己足够努力，就会距离梦想近一步。心之所向，披荆以往。
10、你脚踩的地狱是天堂的幻影，你所向往的伊甸是地狱的幻影。一无所有，才是拼的理由。生活是一面镜子。你对它笑，它就对你笑；你对它哭，它也对你哭。

我们无法改变人生，但可以改变人生观

发表于 2018-06-14 | 分类于 励志语录

1、如果你能像看别人缺点一样，如此准确的发现自己的缺点，那么你的生命将会不平凡。人生是一条没有回程的单行线，上帝不会给你一张返程的票。
2、无论你觉得自己多么的了不起，也永远有人比你更强；无论你觉得自己多么的不幸，永远有人比你更加不幸。
3、背负着过去的痛苦，夹杂着现实的烦恼，这对于人的心灵而言是无任何益处。坚持最难，但成果也最大。
4、青春励志，只有你学会把自己已有的成绩都归零，才能腾出空间去接纳更多的新东西，如此才能使自己不断的超越自己。
5、对待生活中的每一天若都像生命中的最后一天去对待，人生定会更精彩。活在昨天的人失去过去，活在明天的人失去未来，活在今天的人拥有过去和未来。

6、忍别人所不能忍的痛，吃别人所别人所不能吃的苦，是为了收获得不到的收获。大部分人往往对已经失去的机遇捶胸顿足，却对眼前的机遇熟视无睹。
7、困难是一块顽石，对于弱者它是绊脚石，对于强者它是垫脚石。笑对人生，能穿透迷雾；笑对人生，能坚持到底；笑对人生，能化解危机；笑对人生，能照亮黑暗。
8、人生最大的悲哀不是失去太多，而是计较太多，这也是导致一个人不快乐的重要原因。我们总是对陌生人太客气，而对亲密的人太苛刻。
9、人的一生就像一篇文章，只有经过多次精心修改，才能不断完善。虽然我们无法改变人生，但可以改变人生观。虽然我们无法改变环境，但我们可以改变心境。
10、如果你还认为自己还年轻，还可以蹉跎岁月的话，你终将一事无成，老来叹息。当你快乐时，你要想，这快乐不是永恒的。当你痛苦时，你要想，这痛苦也不是永恒的。

得之坦然，失之淡然，顺其自然，争其必然

发表于 2018-06-14 | 分类于 励志语录

1、一份耕耘，一份收获。我本微末凡尘，可也心向天空。青春不奋斗，你的青春用来干嘛？过错是暂时的遗憾，而错过则是永远的遗憾！
2、环境不会改变，解决之道在于改变自己。勇气是控制恐惧心理，而不是心里毫无恐惧。还能冲动，表示你还对生活有激情，总是冲动，表示你还不懂生活。
3、在实现理想的路途中，必须排除一切干扰，特别是要看清那些美丽的诱惑。人一生下就会哭，笑是后来才学会的。所以忧伤是一种低级的本能，而快乐是一种更高级的能力。
4、两个人共尝一个痛苦只有半个痛苦，两个人共享一个欢乐却有两个欢乐。放弃该放弃的是无奈，放弃不该放弃的是无能，不放弃该放弃的是无知，不放弃不该放弃的是执著！
5、行动是治愈恐惧的良药，而犹豫、拖延将不断滋养恐惧。你把周围的人看作魔鬼，你就生活在地狱；你把周围的人看作天使，你就生活在天堂。

6、人之所以痛苦，在于追求错误的东西。与其说是别人让你痛苦，不如说自己的修养不够。如果你不给自己烦恼，别人也永远不可能给你烦恼，烦恼都是自己内心制造的。
7、好好管教自己，不要管别人。你硬要把单纯的事情看得很严重，那样子你会很痛苦。一杯清水因滴入一滴污水而变污浊，一杯污水却不会因一滴清水的存在而变清澈。
8、青春励志，运气就是机会碰巧撞到了你的努力。得之坦然，失之淡然，顺其自然，争其必然。时间是治疗心灵创伤的大师，但绝不是解决问题的高手。
9、天道酬勤。也许你付出了不一定得到回报，但不付出一定得不到回报。逆境是成长必经的过程，能勇于接受逆境的人，生命就会日渐的茁壮。
10、只有不断找寻机会的人才会及时把握机会。做一个决定，并不难，难的是付诸行动，并且坚持到底。一个懒惰的少年将来就是一褴褛的老人。
