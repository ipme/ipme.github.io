别人眼里的云淡风轻，在我心里却是狂风骤雨

发表于 2018-06-14 | 分类于 心情说说

1、以前我爱你，你说什么是什么；现在我不爱你了，你说你是什么。
2、你们整天说不想恋爱只想发财，说的好像有人想和你恋爱一样，说的好像你不恋爱就能发财一样。
3、千万别在最好的年龄里，吃得最胖，用得最差，活得最便宜。时间过得真快，一眨眼你就喜欢上别人了。
4、我希望我喜欢的人也喜欢我，我不想理的人也不理我，满世界清净，皆大欢喜。
5、心情说说，若你听见我的哽咽声，请你捂上耳朵，让我一直孤傲，一直漂亮。
6、以前总以为，人生最美好的是相遇，后来才明白，其实难得的是重逢。如果我们能够久别重逢，我希望你别来无恙。

7、不要和我讲道理，你只需要不分青红皂白站在我身边。也许，遗憾总是和年轻绑在一起。
8、没人没给我回消息，一定是他们收到我的消息时，开心的昏倒啦。
9、人生的不如意莫过于，橡皮筋扎头发两圈太松，三圈太紧，洗澡时开关往左掰一点冻死，往右掰一点烫死。
10、心软的时候，什么都可以原谅；心硬的时候，老子就是全世界。别人眼里的云淡风轻，在我心里却是狂风骤雨。

感情都会沉淀，爱一个人怎么可能日日新鲜

发表于 2018-06-14 | 分类于 心情说说

1、所谓压力，其实是自身的能力不足；所谓困难，其实是自己的本事不够。你要长大，不要总是依赖，你不累，别人也会烦。讨厌酒的味道，却爱极了醉的模样。
2、这世上95%的男生都会喜欢身材好长得漂亮的女生，剩下的4％注重内在，假如你外貌内在皆无，别灰心，还有1％的男生，他们喜欢男的。
3、愿我们所愿终能实现，一切都不辜负如今的委屈求全。感情都会沉淀，爱一个人怎么可能日日新鲜。
4、心情说说，成熟，大概是做一个简单的人，怀着少女心，享受被爱，而不是一味单向付出。
5、别人都同情你无依无靠，只有你自己知道你自在如风。抽烟的人有故事，喝酒的人有心事，不要冒然的评价我，你只是知道我的名字，却不知道我的故事。
6、没有优点，一身坏毛病，但也轮不到你来指指点点。别人可以替你做事，但不能替你感受，人生的路要靠自己行走。

7、你这一副收不了心还忘不了过往的模样好窝囊。好追的姑娘不是因为廉价，而是她爱你。不好追的姑娘不是因为傲娇，而是她不愿将就。
8、有人可以包容你的不成熟，但没人受得了你一直长不大。别在难过的时候喝酒，别在要走的时候回头。
9、小时候以为早睡早起身体好是一句口号，长大后才发现，那是三个愿望。
10、鞋要买第一眼就喜欢的那双，人要选每天都想着的那个。别随便评价我，你所看到的，也只是我选择让你看到的。希望你等到的消息提示，刚好来自你喜欢的人。

生活就像淋浴，方向转错，水深火热

发表于 2018-06-14 | 分类于 心情说说

1、有些感情就像废纸篓里的文件，不想看到，不忍清空。你愿意与我为友吗？哪怕我不善言辞，不苟言笑，无利可图。
2、你之所以过得太累，主要源于你太过于敏感，又太过于心软。事事为别人着想，即使有一天你撑不住了，累了，也没有人会心疼你，同情你。因为在他们眼里，这都是你愿意做的。
3、每次崩溃的理由别人看来是小题大做，只有自己心里清楚这根稻草到底压垮了多少千斤重的难过。
4、心情说说，因为生性敏感也玻璃心，想得太多不够洒脱，所以活得这么难过怪不了谁。太过迁就别人，别人就会变本加厉的为难你；太过忍让别人，别人就会得寸进尺的伤害你。
5、当你烦恼不断的时候，就走出去看看。看看不同的风景，接触不同的人和事，你就知道，你的烦恼有时只是你自己想多了。
6、为值得的人赴汤蹈火，对闲杂人等别在乎太多。如此，你的善良才显矜贵。

7、生活就像淋浴，方向转错，水深火热。知道你聊天软件为什么那么安静吗？因为你曾经为了一个人淡了所有人。
8、就像鞋一样，如果一个人注定是你的，他就会完全合适你，没有勉强，没有挣扎，也没有痛苦。
9、遇到恶心事，你要学会淡定和从容，这个世界上的人渣，千姿百态，防不胜防。
10、或许太重感情的人，日子终究不会好过。信任，依赖，念旧，分分钟把你虐的万劫不复。

我假装无所谓，却发现你是真的不在乎

发表于 2018-06-14 | 分类于 心情说说

1、街角有人祝福，巷口有人哭，是不是感情就该有个胜负。有时候挺讨厌自己的，不会说话，太过善良，没什么心眼，还这么可爱。
2、我知道你最终不会是那个归人，如果只能成为过客也没关系，至少现在你停在我这里，而我把最好的给了你。
3、那些让我不开心的人，我都觉得他们有病。现在你觉得难以忘记的人或是刻在骨子里的人，都会忘记的，只是你还没经历过大起大落。
4、我也并非与生俱来就带刺，知道我不好就别碰。总有一天你会遇上那么一个人，他让你的欢笑和泪水都有意义，他善待你，把你当成生命中最重要的那个。
5、如果再见不能红着眼，是否能让我抽红你的脸。我仍然记得很多细节，只是那种感觉再也不会有了，就像讲故事的人不在故事里。
6、不要愁，十年后所有的事都是下酒菜。女生不用明白太多道理，只要知道体重越来越轻，钱包越来越鼓，越来越快乐就好了。

7、人总是把最好的东西糟蹋以后，才开始感慨人生若只如初见。有的关系断了，是因为旁人的嘴。
8、心情说说，朋友圈里的负能量越来越少，身边的朋友越来越沉迷于赚钱，恭喜你长大了。
9、做个明白人，分得出友情，拎得清爱情。没什么好抱怨的，你现在的一切遭遇都是在为自己的曾经买单。
10、现在有些人有点成就就沾沾自喜，觉得自己很厉害，我不一样，我什么也不做也觉得自己厉害暴了。
11、我假装无所谓，却发现你是真的不在乎。班主任就是那个破坏完你友情，再破坏你爱情，还不放过你亲情的人。
12、你必须靠自己去克服现实获得物质，只有拥有独立自由的生活后，才能在选择爱情的时候不被现实羁绊。
