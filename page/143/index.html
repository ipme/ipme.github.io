世间最好的默契，是有人懂你的欲言又止

发表于 2018-06-14 | 分类于 心灵鸡汤

生活常和我们开着玩笑，你期待谁，谁就会离你越远；你执着谁，谁就会伤你最深。谁不虚伪，谁不善变，谁都不是谁的谁。又何必把一些人，一些事看的那么重要。
你什么都藏不住，你什么事情都忍不住要说出来，后来慢慢地发现，这样做真的好傻，安全感和安慰都是自己给的，别人只会看热闹不嫌事大。
做人，管好自己的嘴，有时候做哑巴，是一种境界。对感情我们都像小孩，胡闹，是因为依赖；礼貌，是因为是陌生；主动，是因为在乎；不联系，是因为觉得自己多余。
如果有缘，错过了还会重来；如果无缘，相遇了也会离开。聚有聚的理由，离有离的借口，人生没有如果，只有后果和结果。
现在的人，你跟他说真话，他以为你说假话；你跟他说假话，他以为你说笑话；你跟他说笑话，他竟然当真了。

心灵鸡汤，很现实的一句话：你得有足够的实力，你的原则和底线才会被人尊重。
时间真的是这个世界上最好的跨度，让惨痛变得苍白，让执着的人选择离开，然后历经沧桑人来人往，你会明白，万般皆是命，半点不由人。
能放下自己的脾气，向你的坏脾气低头的人，并不是因为怕你，而是比你更懂得珍惜。
所以，爱你的人，一定要好好善待；疼你的心，一定要装在心间。世间最好的默契，并非有人懂你的言外之意，而是有人懂你的欲言又止。

一辈子只活一回，千万别把自己委屈！

发表于 2018-06-14 | 分类于 心灵鸡汤

我们，一点点变老…皱纹深了，白发多了；腿脚慢了，身心累了。年复一年，为生计努力；日复一日，为生活打拼。
曾经，我们也是活泼的少年，我们也有无忧的童年。从小时候的单纯，到长大后的成熟，经历了太多的坎坎坷坷，见过了太多的是是非非。
日子一天天过，时间一秒秒走，我们也一点点地老去。总恨岁月无情，感叹时间飞快，一岁接一岁，一年又一年，腰杆不再挺直，肩膀日渐发酸。
心灵鸡汤，人生的路，越走越顺；抬起的脚，越走越稳。在变老的路上，我们增长的是年龄，不变的是心态。

在变老的路上，善待自己：忘记所有的烦恼，用乐观的心态面对生活，用宽阔的心胸善待他人，不计较，不抱怨，开心过好每一天。
在变老的路上，善待自己：努力活成自己喜欢的样子。不用在意别人的眼光，不必在乎别人的议论，活给自己看，才能笑的灿烂。
在变老的路上，善待自己：想爱就爱吧，想见就见吧。感情不怕告白，最怕等待；真心别怕辜负，就怕空白。宁可做错，也别错过，至少证明自己拥有过。
在变老的路上，善待自己：前半生过去了，后半生在继续，该吃吃，该喝喝，该玩玩，该歇歇，一辈子只活一回，千万别把自己委屈！

我不玩心眼，送给简单正直的你！

发表于 2018-06-14 | 分类于 心灵鸡汤

站在人群，我毫不起眼，一个普通而又平凡的我。活在世上，我不玩心眼，一个简单而又明了的我。即使我容貌平平，也不去讨好不喜欢我的人，我只会在喜欢我的人面前露微笑。
就算我生活贫瘠，也不会巴结看不起我的人，我只想让看重我的人。因我而自豪，人活一世，我不想戴着伪装的面具，那样活得太累。
懂你的人不用解释，不懂你的人不必解释，爱你的人不会放弃，不爱的人迟早离去，一心一意做好你自己，即使不完美，也是最美。
我习惯有话就直来直去，圆滑我不曾学会，我习惯做事就尽心尽力，糊弄不是我的原则，处世，凭的就是真性情。

守着最初的纯真，心若干净人就快乐，沿着最后的底线，人不犯我我不犯人，不管你怎么做，怎么说，总会有人议论评说，何不做个真实的我。
心灵鸡汤，心情珍贵不要发霉，必须要笑得最真最美，生命宝贵不要浪费，一定要过得有滋有。

突然间便懂了，说得很现实！

发表于 2018-06-14 | 分类于 心灵鸡汤

1、突然间便懂了：一个人想要幸福，就不能太聪明，也不能太傻，这种介于聪明与傻之间的状态叫做智慧。
感悟：太聪明了让人防备，太傻了让人摆弄，难得糊涂。
2、突然间便懂了：不想爱了就放手，不想讲了就闭嘴，不想恨了就释怀。
感悟：一切都会过去的，事事不必太执着。
3、突然间便懂了：所有负面的情绪和心态，对于你所不满的人和事情不起一点作用，而对于你自己和你的亲人友人起大作用。
感悟：人要乐观一点，一辈子很短暂，有时一转身就是尽头…

4、突然间便懂了：说真话的好处是你不必记得你曾经说过什么。
感悟：通常一句谎言，要用十句甚至更多的谎言来解释，人不要活的那么辛苦。
5、突然间便懂了：山盟海誓是一种经常让高山和海洋领受尴尬的重量级承诺。
感悟：不要轻易相信承诺，一句“拿着”胜过十句“我会给你的”。
6、突然间便懂了：时间是个可怕的东西，随着它的流逝，总有一些真相被永远掩盖，也总有一些真相慢慢被揭开。有时，不经意间，一些真相就来到你身边……
感悟：无语…做任何事一定不要总抱侥幸心理，“若要人不知，除非己莫为”也是真理。
7、突然间便懂了：咖啡的苦与甜，不在于怎么搅拌，而在于是否放糖；一段伤痛，不在于怎么忘记，而在于是否有勇气重新开始。
感悟：心灵鸡汤，真正的痛苦是没有人替你分担，怎么走出阴影只有靠自己。
8、突然间便懂了：真正的爱情不是知道对方有多好才在一起，而是明知道对方有多不好还舍不得离开。
感悟：爱一个人，不光要爱对方的优点，也要接受对方的缺点，人无完人，不离不弃是爱情最高境界。
9、突然间便懂了：两个人在一起久了，就像左手和右手，即使不再相爱也会选择相守，因为放弃这么多年的感情需要很大的勇气。也许生命中会出现你爱的人，但最终你还是会牵着你的左手或右手一直走下去……
感悟：“执子之手，与子偕老”。当初是誓言，后来是责任，再后来是习惯。
10、突然间便懂了：女人会记得让她笑的男人，男人会记得让他哭的女人，可是女人总是留在让她哭的男人身边，男人却留在让他笑的女人身边。
感悟：哭或者笑都是真情的流露。但对于大多数人来说，笑比哭好。
11、突然间便懂了：能够说出的委屈便不算委屈，能够抢走的爱人便不算爱人。
感悟：不说出委屈就只能委屈自己；不放走不爱你的人就得不到你爱的人。
