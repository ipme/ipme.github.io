最好的爱情，没有天荒，也没有地老

发表于 2018-06-14 | 分类于 心情不好

1、感情有时候只是一个人的事情。和任何人无关。爱或者不爱，只能自行了断。从不知道爱情可以让人这么脆弱。
2、如果可以，我希望装疯卖傻过一生，可以听不到那些刺耳，看不见那些揪心，感觉不出那些疼。曾经以为只要我等你，坚定对你的爱，你便不会离开。可你的突然消失，还是让我伪装的坚强跌入谷底。
3、懂得太多，看的太透，你就会变成世界的孤儿。每个人一生之中心里总会藏着一个人，也许这个人永远都不会知道，尽管如此，这个人始终都无法被谁所替代。
4、我们能做的，仅仅是在（生活和梦想）这条路上走得更远，绝不能回头。天堂未必在前方，但地狱一定在身后。
5、幸福就是开心时有人陪着吃大餐，难过时有人拍拍自己的肩膀递上一杯温暖的咖啡，犹豫不决时有人像灯塔一样照亮我前进的方向，累了倦了时发现有人一直在不远处默默守候。

6、一个人的感动，就像是打喷嚏，不是你想遇到就能遇到。生活往往会这样，用时常来让你明白偶然的美丽，用寂寞来让你懂得热闹的珍贵，用缺失来评价存在的意义。
7、不管你学什么专业，找工作一定要找个你喜欢的，这样你每天早晨六点到晚上八点都是高兴的。再找个喜欢的人在一起，这样晚上八点到早晨六点就是开心的，这就是生活。
8、真正爱我的男人，不管我打他骂他叫他滚甚至说分手，他都会赖在我身边，因为他知道，没了他的我会独自哭泣。
9、不要等我流泪，你才明白我的悲伤；不要等我消失，你才知道我的存在。
10、心情不好，最好的爱情，没有天荒，也没有地老，只是想在一起，仅此而已。

人生短短几十年，想笑就笑，想哭就哭

发表于 2018-06-14 | 分类于 心情不好

1、有时候，莫名的心情不好，不想和任何人说话，只想一个人静静的发呆。有时候，夜深人静，突然觉得不是睡不着，而是固执地不想睡。
2、有时候，听到一首歌，就会突然想起一个人。有时候，别人突然对你说，我觉得你变了，然后自己开始百感交集。丢了的自己，只能慢慢捡回来。
3、假如有一天我去了那个世界，请不要伤心，我会把在这个世界没能给你的，在那个世界都做备好，但是你不要来的太快，我怕我没准备好。
4、曾经总以为自己是你的唯一，可现在却只成了你的那万分之一；曾经总以为自己站在那万众瞩目的光芒之下，可现在却跌入了那万丈深崖。
5、生活中，很多细小的问题，换个角度，或者稍微忍一忍，你都能笑着处理好。心态是调整出来的，笑着看世界，世界会还你一份好心情。

6、我们再也不会像以前那样，以彼此为不可替代；我们再也不会像以前那样，那样用力的爱，直到哭了出来。
7、人生短短几十年，不要给自己留下了什么遗憾，想笑就笑，想哭就哭，该爱的时候就去爱，无谓压抑自己。人生的苦闷有二，一是欲望没有被满足，二是它得到了满足。
8、心情不好，那个人，突然不联系你了，很正常；那个人，突然又联系你了，也很正常，这什么也不说明。
9、假如你想要一件东西，就放它走。它若能回来找你，就永远属于你；它若不回来，那根本就不是你的。
10、不要去害怕做一件事，不要害怕触景伤情，不要害怕说错话，不要害怕想起过去，不要害怕面对未来。

你不在时，我哈着气，独自取暖

发表于 2018-06-14 | 分类于 心情不好

1、隔着屏幕说情话只能感受到百分之二十，隔着屏幕说狠话却能感受到百分之二百，距离之所以可怕，是因为你不知道对方是在想念你，还是在忘记你。
2、如若不是你，我怎舍得伤了自己。懵懂的年纪说不得爱，受伤害的一定是爱做梦的女孩。你不在时，我哈着气，独自取暖。
3、有多少人故意赌气关掉手机忍不住打开后，发现什么也没有，自作多情以为自己在他心里有多重要，到最后才发现自己原来是个笑话，别再傻了，人家不在乎你。
4、从不后悔遇见谁，只是后悔怎么成了现在的模样。是小生不才，无福得姑娘芳许，无奈此生无缘厮守，经此一别望姑娘一切安好，笑语如初。
5、人前段子手，人后孤独狗。他可能没做过什么，也可能不小心做多了什么，就无辜地被你大爱了一场。别总是无限放大坏情绪自己为难自己。

6、时间不是问题，我才是问题。谈了一场恋爱，对着手机哭，对着手机笑，最后分手了也没能抱一抱。越是殷勤越被怠慢。一句来日方长让我目睹了人走茶凉。
7、心情不好，真想联系你啊，可总缺一个身份。你哭的最惨的那一天你一定长大了不少吧。地狱是怎样，你让我知晓。我一直在等你，等死一样。
8、梦是个很可怕的东西，总在你毫无防备的时候让你记起你费尽心思想要忘记的人。也许他只是暧昧成瘾，而你却走了心。不在一起就不在一起吧，反正一辈子也没多长。
9、小女子不才，不知公子已有意中人，公子大可放心，小女子一定竭尽全力忘掉公子。你不用担心失去我的，我才是小心翼翼诚惶诚恐的那个，我才是一直都被抛弃的那个。
10、如今我们仅有的默契，就是安静的躺在对方的好友列表里。别指望着一切都会维持原样，任何事情都是变化的，人更是如此。

其实我很好，不知不觉的眼泪就留出来了

发表于 2018-06-14 | 分类于 心情不好

1、其实我很好，只是不习惯，只是会偶尔难受一下，只是会在某一瞬间突然很想某个人，只是会在听到某一句熟悉的话时很难过。
2、对于异地恋的人来讲，通常距离真的不是问题，钱才是问题。好像下一秒天就要塌下来了，每天都在泪水中度过，不知不觉的眼泪就留出来了。
3、青春像玻璃，在经历伤痛、背叛、谎言以及跟多不安的之后，变得不堪一击。曾经我们都守护着一个诺言不变。即使一切都在变，我们都还深信不疑。就像那些一起曾经说过，要一辈子的人，都散落在天涯。
4、你不来都曾给我一次回眸，可我却始终在对你微笑，等一次转身。不要说机会从来没有出现，它出现过多次，只是你舍不得放下自己握着的东西，而没有手去抓住他。
5、当你年老岁月将近白发苍苍，沉思漫想，陷入往事的回忆，你是一度的当年的柔情与美彩缤纷，多少人是爱你昙花一现的身影，爱的是你的容貌于虚情假意之中，只有一人爱你不会因那岁月无情至始所终。

6、人生就是一次次心情的穿越，每个人都在承受着命运的安排，演绎着自己的角色，品尝着自己酿制的酒，接受着经历的洗礼，当痛苦一次次擦肩，当迷雾一次次消散，经历让我们逐渐学会了坚强，必要的放手过往，又何尝不是一种智慧与洒脱？
7、人生旅途，有些事情是分担的，有些事情却是一个人承受的，我们能够不去理会世俗的偏见，却无法逃离琐碎生活中的是非恩怨，隐藏在生活中的纠葛，总会给心情掀起一个波澜，更多的时候我们是孤独的，总是希望有个人能够读懂自己，但很多事情都因为彼此太过于自我，而忽略了他人的感受，把最有效的沟通演变成了缄默，给完美的画布上涂上了一条鸿沟，而承受无语的折磨。
8、有些路需要有人同行，我们都知道孤掌难鸣，脱离了群体，就失去了相互搀扶的可能，有些高度还是需要有个扶梯才能达到顶峰；有些路却要自己去走，因为没有哪个人是你依靠的永久，不锻炼独闯风雨的韧性，就会处于生存的被动，人生是一个不停被强化的过程，事态的瞬息转变，只有练就独立的本能，才能不被击倒而尽快适应。
9、心情不好，爱到分才显珍贵，很多人都不懂珍惜拥有。只到失去才看到，其实那最熟悉的才是最珍贵的。我真的爱你，闭上眼，以为我能忘记，但流下的眼泪，却没有骗到自己。
10、不要轻易说爱，许下的承诺就是欠下的债！恋爱，在感情上，当你想征服对方的时候，实际上已经在一定程度上被对方征服了。首先是对方对你的吸引，然后才是你征服对方的欲望。
