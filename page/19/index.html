若不被珍惜，又何必苦苦去维系

发表于 2018-06-14 | 分类于 人生感悟

若总被忽视，又何必作贱了自己；若不被珍惜，又何必苦苦去维系。
学会放弃，是洒脱离开的美丽，更是自信的魅力。

时间，不要浪费在没有价值的事情上；感情，不要倾注在不懂珍惜的人身上。
有些主动，别人不理就算了；有些在乎，他人不觉就罢了。

何必用真心换来伤心，最后只剩寒心；何苦用重视收回漠视，最终只有无视。
心里有你的人，不会让你等到心痛；心里没你的人，等再久也是没有用。

没有人值得，你把身段一再放低。
你是唯一，必须要有自信的魅力；你很珍贵，所以要活得很高贵！

我们，都需要一个知己

发表于 2018-06-14 | 分类于 人生感悟

一生中，来来去去很多人，却只有极少数可以成为知己，大多数都是观众。观众只是一个旁观者，笑看你起落沉浮，知己却是你人生的参与者，分享你的快乐、分担你的悲伤。

肤浅的人，要的是观众；成熟的人，要的是知己。
知己不是天天见面，吃喝玩乐、相互吹捧；而是懂你，在精神上支持你、鼓励你、帮助你；在你有所不足时，指正你；在你失落时，安慰你。
知己是灵魂深处的一个伴，陪我们度过孤独的岁月。
真正的知己，是一份懂得，一份相知，一种淡淡的陪伴与共鸣。犹如一杯清茶，淡然中沁入心田。
有时候只要一个拥抱，一个眼神，便一切尽在不言中；有时候只要一份关心，一次疼惜，便留下永久的感动。
我们，都需要一个知己。
在疲惫时，有人心疼；在委屈时，有人安慰。在满腹心事堆积如山时，能有个耐心倾听的伴；在压力增加无处释放时，能有人默默陪在身边。
知己，胜似亲人，总是能对你默默付出。知己，犹如爱人，总是能为你排忧解难。如若人生有幸得一知己，请千万要珍惜！

想要活得舒服，记住这些话

发表于 2018-06-14 | 分类于 人生感悟

人生感悟，一天又要过去了，此刻的你是否还有烦恼呢？有几句话想送给你，要想活得舒服，就记住这几句话吧。

1、常与同好争高下，不与傻瓜论短长。
与层次较高的同好相处，就算有意见不合的地方，彼此讨论一来二去也能收获满满。而费尽口舌与那些固执且迂腐的人去争论，不仅没有收获，还会让自己的心情变得很差。所以完全不必与那些不值得的人较劲，有用的话要说给懂的人听。
2、永远不要在生气时和深夜做决定。
生气时的决定，往往充满了意气用事与冲动，等到气消了再看待事情，观点可能截然相反。而深夜，最容易让人感到孤独从而思绪万千，因此深夜里的决定，大多过于感性和执拗，容易不够客观。一旦做了错误的决定，除了后悔之外还要花费更多的精力去弥补，得不偿失。
3、在你没有鞋子的时候，别忘了还有人没有脚。
人生很多烦恼，都是因为盲目比较而徒然生出的。这世上总有人比你优秀，也总有人不如你，每个人都有自己的位置，要想提高应该和过去的自己比较，而不是别人。学会知足，别忘了在你抱怨没有鞋子的同时，还有人没有脚。
4、如果失去了，那就假装它们从未发生过。
不是你的你终将失去，是你的总会留在你身边。而那些失去的，总会以另一种方式归来。当你失去了一样宝贵的东西，挽留无果，不如就假设它从未来过。一本旧书，一段时光，一个爱过的人……既然离开的追不回，那就大大方方地遗忘吧。

你若不坚强，那些苦痛谁替你去扛

发表于 2018-06-14 | 分类于 人生感悟

人活在世上靠别人总会没底气，要想活得硬气始终还得靠自己。总是把希望寄托在别人身上，像菟丝花一样攀附吸取别人的营养，只会让人渐渐窒息，渐渐把胸口的爱意随着氧气一起消磨光。

当你把生活的重心倚靠在别人身上，当你习惯了依赖一个人，一旦失去，便是天崩地陷，生命都开始黯淡无光。你只能在日复一日的担心里彷徨，在年复一复的等待里绝望，最后变成你曾经最看不起的模样。
生老病死，世事无常，谁都不能保证一切都会保持最初的模样。每个人都会有累的时候，没有人能无时无刻陪在你身旁。生活里的雨雪风霜到底还是需要你自己去抵挡，你若不坚强，那些苦痛谁替你去扛。
比起娇弱无力的菟丝花，自信独立的玫瑰更让人欣赏。想要飞翔，就得自己长出翅膀，哪怕最初会跌跌撞撞，哪怕要独自面对风霜，你的翅膀也会越来越强壮，等到阳光灿烂雨过天晴，你定能折射出最绚丽的光。
