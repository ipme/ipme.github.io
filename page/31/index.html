不必刻意去改变他人，只需自己做好就行

发表于 2018-06-15 | 分类于 读书名言

人人有自尊，个个有苦衷，想法、做法和活法都不同。理念不同，做法不同，活法也不一样，不必刻意去改变他人，只需自己做好就行。
01、越有本事的人越没脾气。因为素质，修为，涵养，学识，能力财力会综合一个人的品格。
02、世界上最美妙的一件事是，当你拥抱一个你爱的人，他竟然把你抱得更紧。
03、信任就像一张纸，皱了，即使抚平，也恢复不了原样了。
04、成功根本没有秘诀，如果有的话，就只有两个词：谦虚，坚持。
05、能开口说出的委屈，便不是委屈；能离开的人，便不算是爱人。

06、女人最可悲的，不是年华老去，而是迷失自我。女人最可叹的，不是红颜不再，而是自信全无。一个有灵魂的女人，是一个有魅力的女人。一个有信仰的女人，是一个有能量的女人。
07、往事不必遗憾。若是美好，叫做精彩。若是糟糕，叫做经历。把握眼前，当下才重要。
08、顺利，只是一种平庸的人生。
09、如果一个男人真的爱你，他只会怕没本事照顾好你而努力。如果他事事与你计较，随便对你发脾气，那只是说明他爱自己，没有爱你的本事和能力。
10、心态不好，说穿了，就是心太小了。心态的“态”字，拆解开来，就是心大一点。
人与人之间，就是一种缘分；心与心之间，就是一种交流；爱与爱之间，就是一种感情；情与情之间，就是一颗真心；错与错之间，就是一个原谅。
读书名言，人人有自尊，个个有苦衷，想法、做法和活法都不同。理念不同，做法不同，活法也不一样，不必刻意去改变他人，只需自己做好就行。

生活其实很简单，过了今天就是明天

发表于 2018-06-15 | 分类于 读书名言

1、自己那么差又不努力，还有什么资格唉声叹气。为了自己想要的未来，无论现在有多难熬，都必须得满怀信心的坚持下去。即使你有一千个理由难过，也要有一千零一个理由欢笑。
2、没有人能一路单纯到底，但要记住，别忘了最初的自己。要生活得漂亮，需要付出极大忍耐。一不抱怨，二不解释。只要面对阳光，阴影就在背后。
3、孤独足够让人有无所畏惧的勇气。生活其实很简单，过了今天就是明天。靠山山会倒，靠人人会跑，只有自己最可靠。如果放弃太早，你永远都不知道自己会错过什么。

4、如果机遇总是不曾垂青他他总是找不到自己喜欢做的事，那他就承认自己不是环境的主人，他不得不向困难低头，因为他没有足够的力量。房屋别墅只是虚无坟墓才是永远的家。
5、那些只看到困难的人有一个致命弱点，就是没有坚强的意志去驱除障碍。他没有下定决心去完成艰苦工作的意愿。他渴望成功，却不想付出代价。他习惯于随波逐流，浅尝辄止，贪图安乐，胸无大志。
6、读书名言，他们善于夸大困难，缺少必胜的决心和勇气。即使为了赢得成功，也不愿意牺牲一点点安乐和舒适作为代价。总是希望别人能帮助他们，给他们支持。

人生就是一场旅行，不在乎目的地

发表于 2018-06-15 | 分类于 读书名言

1、人生就是一场旅行，不在乎目的地，在乎的应该是沿途的风景以及看风景的心情。人生最难吃的三碗面：人面情面场面。嘴里说的人生，就是自己以后的人生。
2、很多爱不能重来我应该释怀输不起就不要输。没有创造的生活不能算生活，只能算活着。只要还有明天，今天就永远是起跑线。愚痴的人，一直想要别人了解他。有智慧的人，却努力的了解自己。
3、有多大的思想，才有多大的能量。觉得自己做得到和做不到，其实只在一念之间。要想成为强乾，决不能绕过挡道的荆棘也不能回避风雨的冲刷。

4、人的一生，可以有所作为的时机只有一次，那就是现在。生命的道路上永远没有捷径可言，只有脚踏实地走下去。常求有利别人，不求有利自己。
5、不管失败多少次，都要面对生活，充满希望。奋斗的双脚在踏碎自己的温床时，却开拓了一条创造之路。只有想不到的事，没有做不到的事。
6、读书名言，只有创造，才是真正的享受，只有拚搏，才是充实的生活。别拿自己的无知说成是别人的愚昧！泪水不代表着谁的失败，微笑不意味着谁的成功。

年轻不为梦想买单，老了拿什么话说当年

发表于 2018-06-15 | 分类于 读书名言

1、人生最精彩的不是实现梦想的瞬间，而是坚持梦想的过程。以平常心来面对一些事物，用现实的眼光来看待一些人物，你会收获很多的。
2、左手提着的是事业，肩上扛着的是风雨，右手撑起的是希望。这就是男人！年轻不为梦想买单，老了拿什么话说当年。横跨在你和你的梦想之间的唯一的东西就是奋力拼搏。
3、读书名言，这世上，没有谁活得比谁容易，只是有人在呼天抢地，有人在默默努力。生活就是要把你折腾的死去活来，你要做的就是咬牙坚持下去。

4、梦想看上去遥不可及，你要去实现它，就要坚持，坚持和梦想缺一不可。你真正喜欢想要的，没有一样，是可以轻易得到的。这就是努力的理由。
5、无论昨天今天明天，只要心中有梦永远是春天；不管阴天雨天晴天，只要有好心情永远是艳阳天。生活就是这样，有时只要鼓足勇气跨出一步，一切都会变得与先前想象的好！
6、当一个小小的心念变成成为行为时，便能成了习惯；从而形成性格，而性格就决定你一生的成败。不求与人相比，但求超越自己，要哭就哭出激动的泪水，要笑就笑出成长的性格。
