骄傲是断了引线的风筝稍纵即逝

发表于 2018-06-14 | 分类于 人生哲理

1、泉水，奋斗之路越曲折，心灵越纯洁。如果缺少破土面出并与风雪拚搏的气，种子的前途并不比落叶美妙一分。不要让追求之舟停泊在幻想的港湾，而应扬起奋斗的风帆，驶向现实生活的大海。
2、耕耘者的汗水是哺育种子成长的乳汁。不去耕耘，不去播种，再肥的沃土也长不出庄稼，不去奋斗，不去创造，再美的青春也结不出硕果。让珊瑚远离惊涛骇浪的侵蚀吗？那无异是将它们的美丽葬送。
3、再好的种子，不播种下去，也结不出丰硕的果实。没有失败，只有放弃。每个人学习中都会遇到各种困难、挫折和失败，不同的心态，是成功者和普通者的区别。只有奋斗才会有结果。
4、事常与人违，事总在人为。上帝的延迟并不是上帝的拒绝。任何事情的发生必有其目的，并且总会从某一角度有助于你的成长。重要的不是发生了什么事，而是要做哪些事来改善它。
5、哲理人生，驾驭命运的舵是奋斗。勤奋是你生命的密码，能译出你一部壮丽的史诗。对于攀登者来说，失掉往昔的足迹并不可惜，迷失了继续前时的方向却很危险。

6、奋斗者在汗水汇集的江河里，将事业之舟驶到了理想的彼岸。忙于采集的蜜蜂，无暇在人前高谈阔论。骏马是跑出来的，强兵是打出来的。假如我不能，我一定要；假如我一定要，我就一定能。
7、成功者绝不放弃，放弃者绝不成功。训练决定习惯，习惯决定性格，性格决定命运。不管多么险峻的高山，总是为不畏艰难的人留下一条攀登的路。稗子享受着禾苗一样的待遇，结出的却不是谷穗。
8、骄傲是断了引线的风筝稍纵即逝。自卑是剪了双翼的飞鸟，难上青天，这两者都是成才的大忌。盆景秀木正因为被人溺爱，才破灭了成为栋梁之材的梦。如果把才华比作剑，那么勤奋就是磨刀石。
9、过去不等于未来。没有失败，只有暂时停止的成功。士搏出惊涛骇流而不沉沦，懦夫在风平浪静也会溺水。志在峰巅的攀登者，不会陶醉在沿途的某个脚印之中。
10、如果惧怕前面跌宕的山岩，生命就永远只能是死水一潭。懦弱的人只会裹足不前，莽撞的人只能引为烧身，只有真正敢的人才能所向披靡。我们这个世界，从不会给一个伤心的落伍者颁发奖牌。

努力一点，现在想要的未来都会有

发表于 2018-06-14 | 分类于 人生哲理

1、当你懈怠了，想一想父母期待的眼神！我本可以坠入深渊，我偏要一鸣惊人。努力一点，现在想要的未来都会有。人生的道路上，有些事只能自己面对，我很想依赖，但我必须坚强。
2、认真的人改变自己，坚持的人改变命运。你花时间做什么事，你就会成为什么样的人！那些打不垮你的东西，最终必将成就你。喜欢就坚持，爱就别放弃！
3、努力不一定成功，但不努力一定不成功。别人的质疑就是我进步的动力，总有一天我可以证明自己。欲带皇冠，必承其重。欲握玫瑰，必承其痛。
4、反正我来到这个世界上就没打算活着回去。你生而有翼，为何一生匍匐前进，形如蝼蚁。现在不努力拼搏进取，将来你该拿什么跟别人比。

5、哲理签名，连自己的心都不相信的人，才是最可怜的弱者。生活不是用来度过的，而是用来突破的。你所有的借口不过你给拖延和懒惰找的理由，哪有那么多未来等着你，你只有现在。
6、当一个小小的心念变成成为行为时，便能成了习惯；从而形成性格，而性格就决定你一生的成败。这个世界并不是掌握在那些嘲笑者的手中，而恰恰掌握在能够经受得住嘲笑与批评忍不断往前走的人手中。

平凡不平庸，做最好的自己！

发表于 2018-06-14 | 分类于 人生哲理

1、站在巨人的肩上是为了超过巨人。希望未来我的名字能成为你们口中的骄傲。没有失败，只有放弃。后悔跟努力哪个让你更痛苦？
2、哲理签名，成长的过程其实就是世界观不断崩塌重建的过程。能力是练出来的，潜能是逼出来的，习惯是养成的，成功是一步步走出来的。
3、忘掉所有那些不可能的借口，去坚持那一个可能的理由。无论你认为自己行，还是不行，你的想法都决定了你的人生。心若在，梦就在。平静的湖面练不出精悍的水手，安逸的生活造不出时代的伟人！
4、不苦不累，人生无味。输在犹豫，赢在行动。来路无可眷恋，值得期待只有远方。不要因为没有最好的回报你就不付出，也不要因为没有完美的结果你就不开始。
5、这世界不单单是靠聪明伶俐支撑起来的，还有天道酬勤。相信自己，成就未来！你要信仰别人的光芒，也要相信自己的力量。人都应该有梦，有梦就别怕痛。

6、平凡不平庸，做最好的自己！天生傲骨，永不服输。每个人都在奋不顾身都在加倍努力，你没有理由一边委屈，一边抱怨人世寒凉。
7、冷静分析过去的错误，设法从中获益，再忘掉它，这是惟一让过去有建设性意义的做法。说人是非者，已是是非人。潜龙怎能久卧于深水，总要翱翔于九天。
8、一世其实并不长，既然来了，就要活得漂亮。最大的决心出自于你最大的不甘。最艰难的日子终将过去，此后你将一身荣光屹立于世。
9、这个世界，不会因为你的懦弱无能，而停下脚步，你若不勇敢，没人替你坚强。为得钢骨洒热血，我命由我不由天。在我的天空，做我的英雄。
10、别人都在努力，你有什么资格懒惰，当你在赞叹别人的成功为何不看看别人的付出，与其赞叹不如超越。天赐我一双翅膀，就应该展翅翱翔，满天乌云又能怎样，穿越过后就是阳光。

没光的时候连影子都会离开你

发表于 2018-06-14 | 分类于 人生哲理

1、现在不玩命，将来命玩你，现在不努力，未来不给力。我就是我，颜色不一样的烟火，天空海阔，要做最坚强的泡沫。
2、时间告诉我，无理取闹的年龄过了，该懂事了。我不是天生的王者，但我骨子里流着不服输的血液。眼泪不是答案，拼搏才是选择。只有回不了的过去，没有到不了的明天。
3、哲理签名，跟自己说声对不起，因为曾经为了别人难为了自己。不要指望谁陪你一辈子，没光的时候连影子都会离开你。
4、痛不痛只有自己知道，变没变只有自己才懂。不要问我过得好不好，死不了就还好。若现在就觉得失望无力，未来那么远你该怎么扛。
5、我一直以为人是慢慢变老的，其实不是，人是一瞬间变老的。成功的速度一定要超过父母老去的速度。没有人能替你承受痛苦，也没有人能抢走你的坚强。

6、不要拿我跟任何人比，我不是谁的影子，更不是谁的替代品，我不知道年少轻狂，我只懂得胜者为。如果你看到面前的阴影，别怕，那是因为你的背后有阳光。
7、宁可笑着流泪，绝不哭着后悔。无论你昨晚经历了怎样的泣不成声，早上醒来这个城市依旧车水马龙。做自己生命的主角，而不是别人生命中的看客。
8、成功不是得到多少东西，而是看你把身上多余的东西扔掉多少。现在的我不配喊累，因为我一无所有。总有人比你好看，比你聪明，比你年轻，但没人能取代你。
9、凡是能站在别人的角度为他人着想，这个就是慈悲。只有不想做的，没有做不到的。回避现实的人，未来将更不理想。
10、世上所有美好的感情加在一起，也抵不上一桩高尚的行动。奋斗的双脚在踏碎自己的温床时，却开拓了一条创造之路。
