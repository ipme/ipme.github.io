2、人生就像一场戏，演好自己的角色才是戏成功的关键。
3、有财富的人追求优裕的生活，有智慧的人追求优质的生活。
4、很多时候人们不是因为失败而烦恼，而是因为失败之后找不到任何借口而烦恼。
5、孤独和勇敢是探路者的游戏。——冯宇学

6、走自己的路，让别人说去吧！——但丁
7、伟大的精力只是为了伟大的目的而产生的。——斯大林
8、多一分心力去注意别人，就少一分心力反省自己。
9、你在公交车上看到骑自行车的人在寒风中拼命蹬车不由庆幸，骑自行车的人看到在公交车上挤成沙丁罐头的你不由窃喜，原来短处都是从别人身上发现的。
10、把拳头收回来是为了更有力地还击。

生命不在于长短，而在于活得有意义

发表于 2018-06-15 | 分类于 名言警句

1、要不断提高自身的能力，才能益己及他。有能力办实事才不会毕竟空谈何益。故事的结束总是满载而归，就是金榜题名。——方海权
2、挫折经历的太少，所以总是把一些琐碎的小事看得很重。
3、春天，不是季节，而是内心；云水，不是景色，而是襟怀。
4、好听的话容易打动人，好心的话容易得罪人。

5、奋斗是万物之父。——陶行知
6、无论做什么事情，只要肯努力奋斗，是没有不成功的。——牛顿
7、把人生一分为二，前半生不犹豫，后半生不后悔。
8、我们应当努力奋斗，有所作为。这样，我们就可以说，我们没有虚度年华，并有可能在时间的沙滩。
9、不要乱说话。话说出去之前你还是话的主人，话说出去之后你就成了话的奴隶。
10、名言警句：生命不在于长短，而在于活得有意义。

只有领略了平凡，才会收藏当初丢弃的幸福

发表于 2018-06-15 | 分类于 名言警句

1、尝试去把别人拍过来的砖砌成结实的地基，生活就不会那么辛苦了。难过就是，虽然很难但总会过去。伟大变为可笑只有一步，但再走一步，可笑又会变为伟大。
2、只要持之以恒，知识丰富了，终能发现其奥秘。在事实面前，我们的想象力越发达，后果就越不堪设想。要克服生活的焦虑和沮丧，得先学会做自己的主人。
3、世界上只有一种英雄主义，那就是了解生命而且热爱生命的人。成功的秘诀在于永不改变既定的目标。若不给自己设限，则人生中就没有限制你发挥的藩篱。

4、幸福不会遗漏任何人，迟早有一天它会找到你。时刻就像一张网，你撒在哪里，你的收获就在哪里。坚强的人，并不是能应对一切，而是能忽视所有的伤害。
5、名言警句，纽扣第一颗就扣错了，可你扣到最后一颗才发现。有些事一开始就是错的，可只有到最后才不得不承认。
6、忙时井然，闲时自然；顺多偶然，逆多必然；得之坦然，失之怡然；捧则淡然，贬则泰然。悟通“八然”，此生悠然。
7、世界上没有绝对幸福圆满的婚姻，幸福只是来自于无限的容忍与互相尊重。一个人只要强烈地坚持不懈地追求，他就能达到目的。

人一简单就快乐，一世故就变老

发表于 2018-06-15 | 分类于 名言警句

1、爱的力量大到可以使人忘记一切，却又小到连一粒嫉妒的沙石也不能容纳。别想一下造出大海，必须先由小河川开始。一个人只要强烈地坚持不懈地追求，他就能达到目的。
2、你在希望中享受到的乐趣，比将来实际享受的乐趣要大得多。不要太兴奋，也不要太忧伤，这世界，没你想象的好，也没有如你想的那样糟糕。
3、没有人陪你走一辈子，所以你要适应孤独；没有人会帮你一辈子，所以你要一直奋斗。说有上辈子的人是在骗自我；说有下辈子的人是在骗别人。
4、谁有历经千辛万苦的意志，谁就能达到任何目的。任何人都能够变得狠毒，只要你尝试过嫉妒。属于自己的，不要放弃；别人得到的，切莫妒忌。
5、就算你瘦了，变好看了，你什么都好了，不爱你的人还是不爱你。即使你再胖，再难看，再怎么不好，爱你的人永远不会嫌弃你。

6、名言警句，如果你独自一人笑了，那是真心的笑。世界上有两种东西是怎样都无法挽回的，一种是过去的时间；另一种就是说过的话。
7、生活不是一场赛跑，生活是一场旅行，要懂得好好欣赏每一段的风景。人生有几件绝对不能失去的东西：自制的力量，冷静的头脑，希望和信心。
8、坦然，其实就是平淡中的一份自信，也是平静中的一份快乐，更是一种潇洒。伟大变为可笑只有一步，但再走一步，可笑又会变为伟大。
9、只要持之以恒，知识丰富了，终能发现其奥秘。在事实面前，我们的想象力越发达，后果就越不堪设想。人一简单就快乐，一世故就变老。
10、走得最慢的人，只要他不丧失目标，也比漫无目的地徘徊的人走得快。如果你希望成功，以恒心为良友，以经验为参谋，以小心为兄弟，以希望为哨兵。
