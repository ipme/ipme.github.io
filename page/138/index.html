心里有一些牵挂，有些爱却不得不各安天涯

发表于 2018-06-14 | 分类于 情感美文日志

流星，就像短暂的爱情，美的是一刹那，怀念的是一辈子。青春有价，给自己一个期限吧。未来的每一步一脚印，相知相惜相依为命，别忘记之间的约定，我会永远在你身边陪着你。
即便那爱情没什么复杂的调调但深爱的人之间也能感到它的甜蜜也会甜在心头。幸福的驿站，有你就永远不会打烊。斜阳的灿烂穿透云层包围着消散的群楼。我的生命里、永远活着一个你。
情感语录，那年的岁月，天蓝云白，你我没有故事。那轮傍晚的落日，哀怨地为隔世的爱情画上一个句号。心里有一些牵挂，有些爱却不得不各安天涯……纸钱折成的花在火焰下格外的耀眼。

与你缠绵的每一秒，都是我生命里的永远。在精神的废墟上聚拢起零碎的希望之光。傻与不傻，要看你会不会装傻。人生有两大悲剧：一个是得不到想要的东西，另一个是得到了不想要的东西。
如果说我懂的道理比别人多一点，那是因为我犯的错误比别人多一点。生活可以将就，生活也可以讲究。痛苦来临时不要总问：为什么偏偏是我？因为快乐降临时你可没有问过这个问题。
忙碌是一种幸福，让我们没时间体会痛苦；奔波是一种快乐，让我们真实地感受生活；疲惫是一种享受，让我们无暇空虚。如果我能够看到自己的背影，我想它一定很忧伤，因为我把快乐都留在了前面。

在生命长河中能遇见你，我何其有幸

发表于 2018-06-14 | 分类于 情感美文日志

女人用友情来拒绝爱情，男人用友情来换取爱情。当我们搬开别人架下的绊脚石时，也许恰恰是在为自己铺路。任何人都可以变得狠毒，只要你尝试过嫉妒。在生命长河中能遇见你，我何其有幸。
流星，就像短暂的爱情，美的是一刹那，怀念的是一辈子。请不要失望，平凡是为了最美的荡气回肠。想着过去的记忆，品着现在的心酸，念着未来的坎坷。安静地阅读一本书，嘴角带着静如止水的笑。
再烦，也别忘微笑，再苦，也别忘坚持，再累，也别忘爱惜自己。那些流年、那些颠狂、那些悲伤，在某个十字路口、已经尘埃落定。我不能忍受自己活着—用自己不喜欢的姿态。

人，长得漂亮不如活的漂亮。同样的一瓶饮料，便利店里2块钱，五星饭店里60块，很多的时候，一个人的价值取决于所在的位置。浪漫是一袭美丽的晚礼服，但你不能一天到晚都穿着它。
忍无可忍，就重新再忍。成熟不是心变老，而是眼泪在眼里打转却还保持微笑。美丽让男人停下，智慧让男人留下。想完全了解一个男人，最好别做他的恋人，而做他的朋友。
情感语录，漂亮只能为别人提供眼福，却不一定换到幸福。有时候，不是对方不在乎你，而是你把对方看的太重。绝口不提不是因为忘记，而是因为铭记。低头要有勇气，抬头要有底气。

一个人有多成功，要看他有谁相伴

发表于 2018-06-14 | 分类于 情感美文日志

人生最精彩的不是实现梦想的瞬间，而是坚持梦想的过程。一个人能走多远，要看他有谁同行；一个人有多优秀，要看他有谁指点；一个人有多成功，要看他有谁相伴。
人生的冷暖取决于心灵的温度。有些事，明知是错的，也要去坚持，因为不甘心；有些人，明知是爱的，也要去放弃，因为没有结局；有时候，明知没路了，却还在前进，因为习惯了。
爱情就像攥在手里的沙子，攥的越紧，流失的越快。理想和现实总是有差距的，幸好还有差距，不然，谁还稀罕理想？时间就像一张网，你撒在哪里，你的收获就在哪里。

问候不一定要郑重其事，但一定要真诚感人。做与不做的最大区别是：后者拥有对前者的评论权。如果你为自己定的所有目标都已达到，那么说明你定的目标还不够远大。
让梦想成真的最好办法就是醒来。微小的幸福就在身边，容易满足就是天堂。常常告诫自己不要在一棵树上吊死，结果？在树林里迷路了。朋友就是把你看透了，还能喜欢你的人。
情感语录，就算不快乐也不要皱眉，因为你永远不知道谁会爱上你的笑容。你永远看不见我眼里的泪，因为你不在时我才会哭泣。当别人开始说你是疯子的时候，你离成功就不远了？

不要见一个爱一个，爱的太多，你的爱就要贬值

发表于 2018-06-14 | 分类于 情感美文日志

不要见一个爱一个，爱的太多，你的爱就要贬值。说真话的最大好处就是你不必记得你都说些什么。天使之所以会飞，是因为她们把自己看得很轻？理想很丰满，现实很骨感。
走的最急的是最美的景色，伤的最深的是最真的感情。当大部分人都在关注你飞的高不高时，只有少部分人关心你飞的累不累，这就是友情。上天决定了谁是你的亲戚，幸运的是在选择朋友方面它给你留了余地。
情感语录，叹气是最浪费时间的事情，哭泣是最浪费力气的行径。你以为最酸的感觉是吃醋吗？不是，最酸的感觉是没权吃醋。女人的眼泪是没用的液体，但你让女人流泪说明你很没用。

虽然终于遇到了对的人，但是却偏偏在错误的时间里，那只能徒留遗憾，不如没有遇见。其实真的累了，好想好想就这样直接消失了，让谁也找不到。也就不用面对某些人某些事，面对这充满谎言的现实。
就是寻一温暖的人，渡此余生。爱是相伴。原谅我的贪心，总是想留住身边每个对我好的人，最后才发现，有些人，无论怎样也留不住。那年夏天，我无比憧憬大学；今年夏天，我却无比憧憬那年。
总有一些事，我们不愿它发生，却必须接受；总有些东西，我们不想知道，却必须了解；总有一些人，我们不能没有，却必须学着放手；总有些时间，我们不愿它流逝，却发现根本无能为力。
