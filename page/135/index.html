睡也是你，醒也是你，无孔不入也是你

发表于 2018-06-15 | 分类于 唯美的短句子

1、别聊，别习惯，别开始。你最钟我意，也唯独你最不识抬举。以后见面，记得抱我，揉揉我的脑袋，告诉我，不好意思，让你久等了。
2、梦真是个可怕的东西，总在你毫无防备的时候，让你记起你费尽心思想要忘记的人。温柔都给你，别丢下我好不好。
3、你是我今生渡不过的劫，多看一眼就心软，拥抱一下就沦陷。看你时眼里只有你，看别人时余光全是你，无药可救。
4、以前啊，羡慕那些敢爱敢恨的人，爱就爱了，走就走了，就算心里在流泪扭头也走的干脆，现在啊，终于变成了那样的人，最酷，也是真的最难过。
5、睡也是你，醒也是你，无孔不入也是你，吃饭想你，走路想你，望穿秋水还是你。你永远也不会知道，背后的我忍得有多苦，无论委屈，还是想念。

6、我羡慕那些和你在同一座城市的人，可以和你擦肩而过，乘坐同一辆地铁，走同一条路，看同一处风景，他们甚至还可能在汹涌的人潮中不小心踩了你一脚说对不起，再听你温柔道声没关系，他们那么幸运，而我只能从心里对你说：我想你。
7、练习了千百遍的笑容和言语，见到你却词不达意胡言乱语，我很想你，我喜欢你。想抱抱你，隔着屏幕，委屈的要死。
8、有时候想念就像突然闻到跟你一样的味道，没闻到的时候没那么深刻，突然味道跑出来却没来由的想哭，好想你也真的好想忘了你。
9、唯美的句子，你知道思念一个人的滋味吗？就好像独自喝下一大杯冰水，然后化成热泪流出来。我的心很痛，只有你才是解药。
10、我嫉妒你身边每一个无关紧要的人，他们就那样轻而易举，见到我朝思暮想的你。越汹涌越克制，想念百遍不说一遍。

我曾十步杀一人，却败给你一个眼神

发表于 2018-06-15 | 分类于 唯美的短句子

1、其实我偷偷记住了很多你以为我会忘记的事。我送你此心至诚，百年不变，千岁未寒。希望你从不后悔遇到我，也是真的快乐过。
2、始于脸红，终于眼红。我不知道我有多喜欢你，我只知道你出现后，我就失去了和别人说话的兴趣。如果我脑子进水了，你也会被淹死。
3、我所有的快乐都是伴随你，而来同样这些年所有的悲伤也将和你一起消失。希望我这个小朋友，让你牵挂让你难忘。
4、答案很长，我准备用一生时间来回答，你准备听了吗。一个抱抱就能解决的事，为什么总要跟我讲大道理。
5、别和我聊那些天南地北，也别和我一醉方休，我怕我的声音带着颤，我怕我喝多了没忍住抱了抱你，说了声我还爱你。

6、我曾十步杀一人，却败给你一个眼神。你看，我一想到你，便眼带笑意，我怎么能放过你。想被你宠成三秒不回信息就委屈到哭的娇气包。
7、最棒的感觉大概是我打开你的对话框开始打字的时候突然收到你发来的消息。喜欢一个人，是一万次心跳呼吸，也是一万次低头叹息。
8、以前觉得爱情是一见钟情，后来认为爱情是细水长流，之后认为爱情是安全感，是陪伴，是放不下，是舍不得，现在觉得，爱情就是我爱你时你也爱我。
9、我就像是幼儿园里想被接走的小朋友，而你呢你是幼儿园的校车，你可以接走整个幼儿园里的小朋友，在你的眼里，我就像是一个过街小丑，你是一位盖世英雄。不怕你怜香惜玉，就怕你来者不拒。
10、唯美的句子，你说我爱人的心尤为动人，我笑你不理我的样子最为情深。再美的爱情经不起等待，再深的感情经不起伤害。

眼一睁一闭就是一天，人一起一落便是一生

发表于 2018-06-15 | 分类于 唯美的短句子

一辈子，有挣不够的钱，有做不完的事。一生，很短，一笑一哭，一悲一喜，就过去了。没有一个人，一生没有坎坷，没有一个人，一世没有痛苦。
人生，很短，聚散不由己，悲喜不由人。眼一睁一闭就是一天，人一起一落便是一生。人生，没有什么大不了，命中有的终须有，命中无的莫强求短暂的一生，匆匆的一生，忙碌的一生。
人啊，幸福喜乐才是最重要的。钱，永远挣不够，事，永远做不完，时光匆匆，生命太短暂，对于生活，不要安排得太满，人生，更不要设计得太挤。
无论上帝让你拥有多少，这一切不是为了担忧，而是透过给予、奉献、享用，使自己心安坦然。人要懂得，知足才能常乐，真诚才能拥有，吃得饱、睡得着；乐得出、走得动。不用预支明天的忧虑，一天的难处一天当就够了。

唯美的句子，人活一辈子，真的不容易！挣不够的人民币，忙不完的苦日子，看不透的虚情意，说不出的小情绪，赶不走的不如意，解不开的心中结！
人这一生，没有多少岁月可挥霍！什么都可以舍弃，但不可舍弃快乐；什么都可以输掉，但不可输掉微笑。再烦，也别忘记想开；再急，也要注意语气；再苦，也别告别坚持；再累，也要爱惜自己！做好眼前事，珍惜身边人，走好脚下路，人生，就是这么简单！
一辈子，忙忙碌碌。柴米油盐，穿衣吃饭；人情世故，喜怒悲欢。有操不完的心，有受不完的累。一生很短，在自己的哭声中来，在别人的哭声中走。没有一个人，一生没有波折；没有一个人，一生没有遗憾，没有一个人，一生没有伤痛。
一生很短，有身不由己的活，有言不由衷的说，有迫不得已的做。人生如风絮，聚散不由人；人生如场梦，悲喜不由己；人生如手中沙，得失不由心。

太阳总是新的，每天都是美好的日子

发表于 2018-06-15 | 分类于 唯美的短句子

01、抱最大的期望，尽最大的努力，做最坏的打算，持最好的心态，记住该记住的，忘记该忘记的，改变能改变的，接受成事实的，太阳总是新的，每天都是美好的日子。
02、现状是没那么容易改变的，即便是足够努力，也很难在短时间内看出效果。所以有时你认为的无法改变，也可能只是暂时没看出效果而已，而不是不够努力。改变是需漫长的努力的，而不是一时的急功近利。还是那句话，除却人类无法改变的事情，你所做的每一次努力，都在潜移默化的影响着你的人生轨迹。
03、生命是在一次次的突围中再生的。每一个太阳的升起又落下，都是人冲破自我的一种催促。阳光下，你容貌是这样的，到了月色里，你已是另一个崭新的自己。
04、你想过普通的生活，就会遇到普通的挫折。你想过上最好的生活，就一定会遇上最强的伤害。这世界很公平，你想要最好，就一定会给你最痛。能闯过去，你就是赢家，闯不过去，那就乖乖退回去做个普通人吧。所谓成功，并不是看你有多聪明，也不是要你出卖自己，而是看你能否笑着度过难关。
05、在平凡的日子里，不必苛求轰轰烈烈，但求恬静如诗，不做刻意的挽留和守候。人生的道路不可能一帆风顺，每个人都可能遇到一些挫折和失败，也会遇到不尽人意的事情，问题是要看你怎么去面对。

06、最能让人感到快乐的事，莫过于经过一番努力后，所有东西正慢慢变成你想要的样子。保持一个乐观向上的心态，有个平常的心，对于名利和金钱不要看得太重，应该抱着得之淡燃，失之泰然的态度。
07、所谓坚持，不是四处寻求安慰后的坚持，不是需求鼓励后的坚持，不是被人说服后的坚持。而是无论如何，牙碎自己吞，流泪自己擦，摔了站起来继续走。真正的坚持，和别人永远发生不了关系，全靠自己每日擦拭。不要逢人便说：请鼓励我，我会坚持下去的。那不是坚持，是乞讨。
08、很喜欢的一句话：“喷泉之所以漂亮是因为她有了压力；瀑布之所以壮观是因为她没有了退路；水之所以能穿石是因为她有了目标……”人生亦是如此，早安，献给每一位睁开眼就要努力奋斗的人！
09、点击崭新的日历，翻开新的日记，保存真心的祝福，删除所有的委屈，复制阳光的魔力，粘贴到我的信息，愿你早起就能收到我的祝福，感受阳光的神奇魔力，清晨如此清新美丽，心情永远快乐无敌！！
10、大部分的人都是过着年复一年的重复生活，然后幻想着不一样的今年。除掉睡眠，人的一辈子只有一万多天，你是真的活了一万多天，还是仅仅生活了一天，却重复了一万多次。
11、再苦再累，只要坚持往前走，属于你的风景终会出现。只要是自己选择的，那就无怨无悔，青春一经典当，永远无法赎回。
12、唯美的句子，微笑吧，因为你笑起来很美丽；保持坚强吧，因为这样谁都不可以打倒你。
