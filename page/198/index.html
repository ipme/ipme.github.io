你知道别人都在拼命，你怎么好意思在浪费时间

发表于 2018-06-14 | 分类于 努力奋斗

1、你知道别人都在拼命，你怎么好意思在浪费时间。你的脸是为了呈现上帝赐给人类最贵重的礼物是微笑，一定要成为你工作最大的资产。人生的烦恼，多在于知道的太多，而做的太少。
2、能把在面前行走的机会抓住的人，十有八九都会成功。征服畏惧建立自信的最快最确实的方法，就是去做你害怕的事，直到你获得成功的经验。
3、于千万人之中，遇见你所遇见的人；于千万之中，时间的无涯荒野里，没有早一步，也不晚一步，刚巧赶上了。人如果没有诚信，就算有聪明能力，永远只是个危险人物。
4、你的努力，终将成就无可替代的自己。努力从来不会白费，今日撒下种子，正在你看不见想不到的某处，悄悄地生根发芽。你必须非常的努力，才能够看起来毫不费力。

5、只要你足够努力，连命运都不忍心辜负你。为了对得起来未来的自己，你不能贪图安逸。努力不是为了现在得到别人的夸耀，而是为了将来活得有底气的尊严。
6、在自己的世界里努力，在别人的世界里发光。活出精彩的自己。一无所有就是拼博的理由。现在你不拼，老了你拿什么跟孙子说你曾经牛逼的故事。
7、累吗？累就对了，舒服是留给死人的。做的技艺来自做的过程。自己打败自己是最可悲的失败，自己战胜自己是最可贵的胜利。每一发奋努力的背后，必有加倍的赏赐。
8、知识决定竞争力。每一个成功者都有一个开始。勇于开始，才能找到成功的路。自弃者扶不起，自强者击不倒。昨晚多几分钟的准备，今天少几小时的麻烦。
9、努力奋斗，如果在胜利前却步，往往只会拥抱失败；如果在困难时坚持，常常会获得新的成功。过错是暂时的遗憾，而错过则是永远的遗憾！
10、凡过于把幸运之事归功于自己的聪明和智谋的人多半是结局很不幸的。我们最值得自豪的不在于从不跌倒，而在于每次跌倒之后都爬起来。

只有相信自己，才能让别人相信你

发表于 2018-06-14 | 分类于 努力奋斗

1、在你不害怕的时间去斗牛，这不算什么；在你害怕时不去斗牛，也没有什么了不起；只有在你害怕时还去斗牛才是真正了不起。
2、不大可能的事也许今天实现，根本不可能的事也许明天会实现。永不言败是追究者的最佳品格。我依旧相信努力的意义，奋斗的价值，因为那是本质问题。
3、谦虚的人更能认清自已。只有相信自己，才能让别人相信你。伟大的成就，来自为远大的目标所花费的巨大心思和付诸的最大努力。
4、实现梦想比睡在床上的梦想更灿烂。真正能让你倒下的，不是对手，而是你绝望的内心。乐观的人能重整旗鼓东山再起，悲观的人因缺乏自信，往往一败涂地。
5、生活总会给你另一个机会，这个机会叫明天。永不放弃是你梦想实现的唯一秘诀。意志坚强的人，他的世界充满着无限的可能性。

6、世界上只有一个人阻碍你的成长，那就是你自己。人生最大的敌人是自己怯懦。总是在疼过之后，才能学会做一个全新的自己。
7、看得见的伤口，迟早有一天会痊愈的。一个人只有在全力以赴的时候才能发挥最大的潜能。现在不玩命，将来命玩你，现在不努力，未来不给力，致自己。
8、人生只有走出来的美丽，没有等出来的辉煌。平凡的脚步也可以走完伟大的行程。诚心诚意，“诚”字的另一半就是成功。
9、因为奋斗，我变得坚强；因为奋斗，我学会拼搏；因为奋斗，我懂得自豪；因为奋斗，我学会走好人生之路；因为奋斗，我努力奔向理想的彼岸；因为奋斗，我选择完美人生。
10、努力奋斗，相信自己，坚信自己的目标，去承受常人承受不了的磨难与挫折，不断去努力去奋斗，成功最终就会是你的！

与其在悲哀中死亡，何不努力去拼出精彩

发表于 2018-06-14 | 分类于 努力奋斗

1、要记住舒服是留给死人的！活着的心就别总留在黑暗里。成功与不成功之间有时距离很短，只要后者再向前几步。
2、别人都在努力，我有什么资格停下来休息。现在的拼命努力，不过是让自己日后有想要就买的魄力。
3、当你为自己想要的东西而忙碌的时候，就没有时间为不想要的东西而担忧了！人生只有回不去，没有什么过不去。水不撩不知深浅，人不拼怎知输赢。
4、撑不住的时候可以对自己说声“我好累”，但永远不要在心里说“我不行”。在你想要放弃的那一刻，想想为什么当初坚持走到了这里。
5、努力奋斗，你不去拼博，你怎么会知道自己有多大的潜力。提高自己水平的最好方法就是勇敢地接受挑战。

6、人只能活年，不论是好是坏都得死，与其在悲哀中死亡，何不努力去拼出精彩。走下去，凭着耐心和拼搏，生活自会给予你全部答案。
7、一朵花凋零荒芜不了整个春天，一次跌倒也荒废不了整段旅程。未来属于那些明知有风雨，还能坚持走下去的家伙。
8、不管多险峻的高山，总会给勇敢的人留一条攀登的路。没有等出来的美好，只有拼出来的幸福。强大的信心，能克服来自内心的恶魔，产生无往不胜的勇气。
9、一生中你唯一需要回头的时候，是为了看自己到底走了多远。一旦立下目标，不达目标绝不罢手，方可成功。
10、生活中的许多事，并不是我们不能做到，而是我们不相信能够做到。无人理睬时，坚定执着。万人羡慕时，心如止水。
