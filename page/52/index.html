失败了就当经验，成功时便是灿烂

发表于 2018-06-15 | 分类于 经典台词

1、不敢说真话是个人的耻辱，不能说真话是时代的耻辱。蓝天下便是阳光；艰苦后便是甘甜；失败了就当经验；成功时便是灿烂。
2、人生是有许多偶然的，所以，也就多了很多的机会。人人都以为自己是世上唯一，因此世上再无唯一。微小的幸福就在身边，容易满足就是天堂。
3、不是只有你一个人在努力，所以不要轻易地就放弃。对于工作，努力但不痴狂；对于家庭，忠诚但不刻板；对于金钱，喜爱但不贪婪；对于爱情，相信但不迷失。
4、蔷薇开出的花朵没有芬芳，想念一个人，怀念一段伤，不流泪，不说话。房子和老婆一样，没有时候，前半辈子为得到而忙活；有了以后，后半辈子要为拥有而忙活。
5、想是问题，做是答案，输在犹豫，赢在行动。谈话要切题，谈话要有逻辑性，谈话要有力，谈话要充满自信，要使你推销时的谈话直接，自然，而且尽可能的简洁。

6、一个人如果学会始终让自己的大脑充满积极，进取，乐观，愉快和有希望的想法，那么他就己经解决了人生的一大奥秘。
7、如果你认为每一位成功者都只有成功的经验，那就错了。其实，没有比成功者拥有更多的是失败经验。有些事遇见了就无法放手，有些事放开了手就不必再回首。
8、莫言，只有尝试的人才会获得成功：尝试不会失去什么，如果尝试成功，就可以获得很多东西，所以尽管去尝试。
9、当对方越冷淡时，你就越以明朗，动人的笑声对待他，这样一来你在气势上会居于优势，容易击倒对方。似乎是没有欲求的开端，总是以备受打击而结束。
10、集中你的精力去学习一件小事，而成功一名专家；比你分散精力去学习许多小事要更加节省时间，而且容易成功。

无论眼下是好是坏，都只是暂时的

发表于 2018-06-15 | 分类于 经典台词

1、生活本来很不易，不必事事渴求别人的理解和认同，静静的过自己的生活。心若不动，风又奈何。你若不伤，岁月无恙。
2、人生四然：来是偶然，去是必然，尽其当然，顺其自然。生活就像坐过山车，有高峰，也有低谷，这意味着，无论眼下是好是坏，都只是暂时的。
3、清醒时做事，糊涂时读书，大怒时睡觉，独处时思考。我们总是把陌生人给的一些小恩小惠当做是大恩大德，却把身边那些死心塌地对你好的人当做理所当然。
4、若无其事，原来是最好的报复。何必向不值得的人证明什么，生活得更好，是为了自己。不管你去往何方，不管将来迎接你的是什么，请你带着阳光般的心情启程。
5、莫言，因为没有，从不担心失去；因为想要，才会患得患失。人生中最艰难的两场考验：等待时机到来的耐心，和面对一切际遇的勇气。

6、种子牢记着雨滴献身的叮嘱，增强了冒尖的勇气。山路曲折盘旋，但毕竟朝着顶峰延伸。劳动一日，可得一夜的安眠，勤劳一生，可得幸福的长眠。
7、忘掉你的一次次失败，但不要忘记你获得的胜利；忘掉你遭遇的不幸，但不要忘记你的时来运转。没有谁瞧不起你，因为别人根本就没有瞧你，大家都很忙的。
8、当世界给草籽以重压时，它总是会用自己的方式破土而出。经常往上看你会长高，经常低着头捡便宜你就会驼背。年轻的时候，如果爱，别说永远，说珍惜。
9、有的家庭，几根面条就能撑起热腾腾的日子；有的家庭，一堆金币反而把日子折腾得七颠八倒。不要活在别人的嘴里，不要活在别人的眼里，而是把命运握在自己手里。
10、初恋是初开的花朵浓郁芬芳，使人终身回味，失恋是初秋的果实，不成熟就采摘，品尝到的是酸楚和苦涩。人生的奔跑，不在于瞬间的爆发，而在于途中的坚持。

失败缘于忽视细处，成功始于重视小事

发表于 2018-06-15 | 分类于 经典台词

1、如果你找不到一个坚持的理由，你就必须找到一个重新开始的理由。船放在港口最安全，但那不是造船的目的。瀑布是江河走投无路时创造的奇迹。
2、我们最强的对手，不一定是别人，而可能是我们自己！在超越别人之前，先得超越自己！真正的坚韧，应该是哭的时候要彻底，笑的时候要开怀，说的时候要淋漓尽致，做的时候不要犹豫。
3、一个人知道自己为什么而活，就能忍受任何生活。一无所有就是拼搏的理由。你得先感动自己，才能感动别人。成功的速度，要超过爸妈老去的速度。
4、生命如同一根火柴，只有磨砺才会跳跃出灿烂的火花。每个人都会累，没人能为你承担所有的伤悲和疲惫，所以人总有那么一段时间要学会自己长大。
5、只要站起来比倒下去多一次就是成功。如果敌人让你生气，那说明你还没有胜他的把握。如果你坚信自己最优秀，那么你就最聪明。

6、后悔是一种耗费精神的情绪，后悔是比损失更大的损失，比错误更大的错误，所以不要后悔。失败缘于忽视细处，成功始于重视小事。
7、人生的冷暖取决于心灵的温度。人最可悲的是自己不能战胜自己。勤奋可以弥补聪明的不足，但聪明无法弥补懒惰的缺陷。狂妄的人有救，自卑的人没有救。
8、生气是拿别人做错的事来惩罚自己。快乐是一种能力。世界让我遍体鳞伤，但伤口长出的却是翅膀。请看护好你曾经的激情和理想。在这个怀疑的时代，我们依然需要信仰。
9、莫言，如果别人朝你扔石头，就不要扔回去了，留着做你建高楼的基石。你讨厌我，我不介意，我活着不是为了取悦你。
10、当我们搬开别人脚下的绊脚石时，也许恰恰是在为自己铺路。很多时候都是自己挖了个坑，然后义无反顾的跳进去。坑是自己挖的，跳也是自己跳的，最后爬不出来的也是自己。

心简单，世界就简单，幸福才会生长

发表于 2018-06-15 | 分类于 经典台词

1、年轻是我们唯一拥有权利去编织梦想的时光。觉得自己做得到和做不到，其实只在一念之间。要想成为强乾，决不能绕过挡道的荆棘也不能回避风雨的冲刷。
2、人的一生，可以有所作为的时机只有一次，那就是现在。行动不一定带来快乐，而无行动则决无快乐。绝大多数人，在绝大多数时候，都只能靠自己。
3、莫言，只有不想做的，没有做不到的。昨天下了雨，今天刮了风，明天太阳就出来了。不会生气的人是愚者，不生气的人乃真正的智者。
4、千万人的失败，都有是失败在做事不彻底，往往做到离成功只差一步就终止不做了。信念是一把无坚不摧的利刃。爬上最高的境界，你会陡然发现：那里的景色竟然是你司空见惯的。
5、你在学习上这种尝试精神很可贵。通过云端的道路，只亲吻攀登者的足迹。我为你今天的表现感到骄傲。不要埋怨别人让你失望了，怪你自己期望太多。

6、当一个人真正觉悟的一刻，他放弃追寻外在世界的财富，而开始追寻他内心世界的真正财富。炫耀是需要观众的，而炫耀恰恰让我们失去观众。
7、理论是一码事，实际又是一码事。知识玩转财富。有人在光明中注视着阴影，有人在阴影中眺望着光明。解决最复杂的事情往往需要最简单的方法。
8、有些人、有些话、说不说理不理都无所谓；因为看清了，也就看轻了。每个人的性格中，都有某些无法让人接受的部分，再美好的人也一样。所以不要苛求别人，也不要埋怨自己。
9、幸福并不需要奢侈和豪华，有时要的越多反而越难幸福。小人物，小欲望，小满足，才是大幸福。心简单，世界就简单，幸福才会生长；心自由，生活就自由，到哪都有快乐。
10、不懂时，别乱说；懂得时，别多说；心乱时，慢慢说；没话时，就别说。不必太过斤斤计较，愿意吃亏的人，终究吃不了亏，吃亏多了，总有厚报；爱占便宜的人，定是占不了便宜，赢了微利，却失了大贵。
