累就对了，舒服是留给死人的！

发表于 2018-06-15 | 分类于 人生哲理

1、在穷的只剩下梦想的时候，你一定要比别人更加努力。我们用人生最好的年华作抵押，去担保一个说出来都会被嘲笑的梦想。
2、累么？累就对了，舒服是留给死人的！如果我们一直告诫自己要开心过每一天，就是说我们并不开心。天气影响身体，身体决定思想，思想左右心情。
3、不论你在什么时候结束，重要的是结束之后就不要悔恨。你把思考交给了电视，把联系交给了手机，把双腿交给了汽车，把健康交给了药丸。
4、不是某人使你烦恼，而是你拿某人的言行来烦恼自己。不要因为众生的愚疑，而带来了自己的烦恼。不要因为众生的无知，而痛苦了你自己。
5、人生要用简单的心境，对待复杂的人生，最无情的不是人，是时间；最珍贵的不是金钱，是情感；最可怕的不是失恋，是心身不全；最舒适的不是酒店，是家里；最难听的不是脏话，是无言；最美好的不是未来，是今天。

6、在一切变好之前，我们总要经历一些不开心的日子，这段日子也许很长，也许只是一觉醒来。有时候，选择快乐，更需要勇气。
7、经典哲理，只要站起来的次数比倒下去的次数多，那就是成功。活得失败的原因就是：讨厌自己明明不甘平凡，却又不好好努力。
8、答应自己，不要去在乎一些么必要的人和事。失因为贪，说真的，老实人很少上当。人生应该树立目标，否则你的精力会白白浪费。
9、山涧的泉水经过一路曲折，才唱出一支美妙的歌。时间告诉我，无理取闹的年龄过了，该懂事了。我不是天生的王者，但我骨子里流着不服输的血液。
10、眼泪不是答案，拼搏才是选择。只有回不了的过去，没有到不了的明天。跟自己说声对不起，因为曾经为了别人难为了自己。

不要问我过得好不好，死不了就还好

发表于 2018-06-15 | 分类于 人生哲理

1、努力过的人总是最美的，即使最后她没赢，至少不会骗了自己。没有一种困难，是成为你放弃的理由。正当壮年，怎能暮气沉沉；台风口上，何不迎风飞翔。
2、何必站在回忆里不肯出去，阳光还在外面等着你。活着就意味着必须要做点什么，请好好努力。别放弃太早，咬牙撑过，才有回忆的价值。
3、人生没有如果，只有后果和结果。少问别人为什么，多问自己凭什么。现在不玩命，将来命玩你，现在不努力，未来不给力。
4、我就是我，颜色不一样的烟火，天空海阔，要做最坚强的泡沫。时间告诉我，无理取闹的年龄过了，该懂事了。
5、我不是天生的王者，但我骨子里流着不服输的血液。眼泪不是答案，拼搏才是选择。只有回不了的过去，没有到不了的明天。

6、经典哲理，跟自己说声对不起，因为曾经为了别人难为了自己。不要指望谁陪你一辈子，没光的时候连影子都会离开你。
7、痛不痛只有自己知道，变没变只有自己才懂。不要问我过得好不好，死不了就还好。若现在就觉得失望无力，未来那么远你该怎么扛。
8、我一直以为人是慢慢变老的，其实不是，人是一瞬间变老的。成功的速度一定要超过父母老去的速度。没有人能替你承受痛苦，也没有人能抢走你的坚强。
9、不要拿我跟任何人比，我不是谁的影子，更不是谁的替代品，我不知道年少轻狂，我只懂得胜者为。如果你看到面前的阴影，别怕，那是因为你的背后有阳光。
10、宁可笑着流泪，绝不哭着后悔。总有人比你好看，比你聪明，比你年轻，但没人能取代你。凡是能站在别人的角度为他人着想，这个就是慈悲。

头脑是日用品，而不是装饰品

发表于 2018-06-15 | 分类于 人生哲理

1、每个人都在自己的世界里做着只有自己才能阐释的梦。头脑是日用品，而不是装饰品。在人之上，要把人当人；在人之下，要把自己当人。
2、生活，就是生下来，活下去。寂寞的人有两种，一种是什么话都听的明白，一种则是什么都听不明白。乖，别生气了，咱甭和地球人一般见识！
3、人经不住千言，树经不住千斧。希望：我喜欢的人不要滚滚而来；更希望：我不喜欢的人滚滚而去。生日是一个舞台，一次考验，一次机会。恋爱时，男人更会利用这一点；结婚后，女人更会利用这一点。
4、经典哲理，一个人如果不能从内心去原谅别人，那他就永远不会心安理得。每个人心里都有一段伤痕，时间才是最好的疗剂。
5、青春如此华美，却在烟火在散场。历史，只有人名真的；小说，只有人名是假的。妈妈说过没有人值得你为他哭，唯一值得你为他哭的那个人，永远都不会让你为他哭。

6、你能用金钱买来的爱，别人也能用金钱把它买去。所谓骨气，“气”由“骨”定。骨头硬者，是浩然之气；骨头软者，是颓靡之气；无骨者，无气。
7、踩着垃圾到达的高度和踩着金子到达的高度是一样的。每天告诉自己一次：我真的很不错。人生最大的挑战没过于战胜自己！
8、愚痴的人，一直想要别人了解他。有智慧的人，却努力的了解自己。生命的道路上永远没有捷径可言，只有脚踏实地走下去。
9、只要还有明天，今天就永远是起跑线。火把倒下，火焰依然向上。认真可以把事情做对，而用心却可以做到完美。宁可自己去原谅别人，莫等别人来原谅自己。
10、拼尽全力，逼自己优秀一把，青春已所剩不多。最孤独的时光，会塑造最坚强的自己。亲爱的你，前路迷茫而艰辛，愿你能披荆斩棘，无所畏惧，勇敢前行。

与其给鱼一双翅膀，不如给鱼一方池塘

发表于 2018-06-15 | 分类于 人生哲理

1、泪水不是为了排除外在的悲伤，而是为了自由的哭泣。使人疲惫的不是远方的高山，而是鞋里的一粒沙子。成功就是你被击落到失望的深渊之后反弹得有多高。
2、生活，就象一个无形的天平，站在上面的每个人都有可能走极端，但这最终都是为了寻找一个平衡的支点，使自己站的更稳，走的更好，活的更精彩！
3、难过时，吃一粒糖，告诉自己生活是甜的！生活原则：往高处立、朝平地坐、向阔处行。生活态度：存上等心、结中等缘、享下等福。生活目标：为社会奉献经典，为人生创造精彩！
4、有时候对一个作家而言，真正的奖赏不是诺贝尔奖，而是盗版。青椒肉丝和肉丝青椒是同一道菜，关键是谁来炒。如果错过了太阳时你流泪了，那么你也要错过群星了。
5、生活中促使我们努力向前的往往就是痛苦，因为情有所系，心有不甘，因为不想痛苦，不愿难受。或许，痛苦的后面就是美好，关键是你怎样看待与理解痛苦，并赋予怎样的行动。

6、经典哲理，为了照亮夜空，星星才站在天空的高处。上苍不会给你快乐、也不会给你痛苦，它只会给你真实的生活。有人忍受不了生活的平淡而死去，却不知道生命本身就是奇迹！
7、我不去想是否能够成功，既然选择了远方，便只顾风雨兼程；我不去想身后会不会袭来寒风冷雨，既然目标是地平线，留给世界的只能是背影。
8、与其给鱼一双翅膀，不如给鱼一方池塘。如果你的生活已处于低谷，那就，大胆走，因为你怎样走都是在向上。生活中没有退步。只要我们肯学，无论什么经验都是我们需要领悟的章节。
9、人一旦不存希望，生命也就休止。希望的具体表现就是欲，它形成具体的目标和实践的动力，而令自己乐于完成它，乐于实现它，从而缔造幸福的生活。
10、活着一天，就是有福气，就该珍惜。当我哭泣我没有鞋子穿的时辰，我发觉有人却没有脚。大家都不急，我急什么？大家都急了，我能不急？这就是工作与奖金的区别。
