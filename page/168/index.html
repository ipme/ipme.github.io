现在的努力，是为了小时候吹过的牛逼！

发表于 2018-06-15 | 分类于 励志名言

1、人生就像一道漫长的阶梯，任何人也无法逆向而行，只能在急促而繁忙的进程中，偶尔转过头来，回望自己留下的蹒跚脚印。
2、励志名言，人只要不失去方向，就不会失去自己。怕走崎岖路，莫想登高峰。现在的努力，是为了小时候吹过的牛逼！
3、决定你的人生高度的，不是你的才能，而是你的人生态度！人生就像一场旅行，不必在乎目的地，在乎的只有沿途的风景，以及看风景的心情。
4、明白事理的人使自己适应世界，不明事理的人硬想使世界适应自己。人最可悲的是自己不能战胜自己。每一个成功者都有一个开始。勇于开始，才能找到成功的路。
5、世界上只有想不通的人，没有走不通的路。如果说我懂得的道理比别人多一点，那是因为我犯的错误比别人多一点。

6、不管前方的路有多苦，只要走的方向正确，不管多么崎岖不平，都比站在原地更接近幸福。如果你向神求助，说明你相信神的能力。如果神没有帮助你，说明神相信你的能力。
7、嘴里说的人生，就是自己以后的人生。一个华丽短暂的梦，一个残酷漫长的现实。过了做梦的年纪，就只能往前走。
8、只要内心不乱，外界就很难改变你什么。不要艳羡他人，不要输掉自己。信心毅力勇气三者具备，则天下没有做不成的事。
9、有人将你从高处推下的时候恰恰是你展翅高飞的最佳时机。要生活得漂亮，需要付出极大忍耐。一不抱怨，二不解释。
10、命是弱者的借口，运是强者的谦辞，辉煌肯定有，就看怎么走。世上找不出两个完全相同的人。生得再平凡，也是限量版。

永远别放弃自己，哪怕所有人都放弃了你

发表于 2018-06-15 | 分类于 励志名言

1、当机会呈现在眼前时，若能牢牢掌握，十之八九都可以获得成功而能克服偶发事件，并且替自己找寻机会的人，更可以百分之百的获得胜利。
2、走过一些漫不经心，方知人生容不得放任。选对事业可以成就一生，选对朋友可以智能一生，选对环境可以快乐一生，选对伴侣可以幸福一生，选对生活方式可以健康一生。
3、快乐总和宽厚的人相伴，财富总与诚信的人相伴，聪明总与高尚的人相伴，魅力总与幽默的人相伴，健康总与豁达的人相伴。
4、人生又如同猎豹捕食，它蓄积力量，向猎物狂奔而去，可离猎物越来越近，它发现自己的力量也消失殆尽了，最终放弃了奔跑追逐。
5、影响我们人生的绝不仅仅是环境，其实是心态在控制个人的行动和思想。同时，心态也决定了一个人的视野事业和成就，甚至一生。

6、人往往有时候为了争夺名利，有时驱车去争，有时驱马去夺，想方设法，不遗余力。压力挑战，这一切消极的东西都是我取得成功的催化剂。
7、任何人都不可能随随便便地成功，它来自完全的自我约束和坚韧不拔的毅力。永远别放弃自己，哪怕所有人都放弃了你。
8、行为胜于言论，对人微笑就是向人表明：”我喜欢你，你使我快乐，我喜欢见到你”。最值得欣赏的风景，是自己奋斗的足迹。
9、出发，永远是最有意义的事，去做就是了。当一个人真正觉悟的一刻，他放弃追寻外在世界的财富，而开始追寻他内心世界的真正财富。
10、励志名言，无论你觉得自己多么的了不起，也永远有人比你更强；无论你觉得自己多么的不幸，永远有人比你更加不幸。

没有迈不过的坎，没有闯不过的关

发表于 2018-06-15 | 分类于 励志名言

1、要克服生活的焦虑和沮丧，得先学会做自己的主人。人的一生，没有一味的苦，没有永远的痛；没有迈不过的坎，没有闯不过的关。
2、相信自己能力的人，任何事情都能够做到。我命由我，不由天！如果你希望成功，以恒心为良友，以经验为参谋，以小心为兄弟，以希望为哨兵。
3、勤俭节约，可以使人更高尚更美好。一个不注意小事情的人，永远不会成就大事业。如果你真的愿意为自己的梦想去努力，最差的结果，不过是大器晚成。
4、愚昧者怨天尤人，无能者长吁短叹，儒弱者颓然放弃。想交朋友，就要先为别人做些事，那些需要花时间体力体贴奉献才能做到的事。
5、一个不注意小事情的人，永远不会成功大事业。有什么别有病，没什么别没钱，缺什么也别缺健康，健康不是一切，但是没有健康就没有一切。

6、励志名言，当你知道你不在是你的时候，你才是真正的你！你要努力成为梧桐，让喜鹊在这里栖息。你要努力成为大海，让百川在这里聚积。
7、人是可以快乐地生活的，只是我们自己选择了复杂，选择了叹息！人生就像一块调色板，有明亮，也有灰暗，谁都希望五彩缤纷，有的结果并不尽然。
8、带着自己的梦，以一种骄傲的姿态走下去。伟人之所以会成功，原因是当他和别人一样的处于逆境时，别人丧失了信心和进取心，他却坚定的向目标前进。
9、行动好像是跟随感觉走的，其实不是如此，行动是与感觉平行的，我们能使直接受意志支配的行动有规律，也能间接的使不直接受意志制约的支配有规律。
10、世俗有“时间是金钱”这句话，所以窃取他人时间的小偷，当然该加以处罚，即使是那些愉快的好人，还是该如忌讳疾病地躲避他们。

你不能预见明天，但可以珍惜今天！

发表于 2018-06-15 | 分类于 励志名言

1、励志名言，当所有人都低调的时候，你可以高调，但不能跑调。人生最大的错误是不断担心会犯错。
2、绝口不提不是因为忘记，而是因为铭记。目标的坚定是性格中最必要的力量源泉之一，也是成功的利器之一。
3、金钱难买健康，健康大于金钱，金钱难买幸福，幸福必有健康，生命的幸福不在名利在健康，身体的强壮不在金钱在运动！
4、如果你希望成功，以恒心为良友，以经验为参谋，以小心为兄弟，以希望为哨兵。当我们眺望远方的时候，近处的风景便看不清了。
5、如果说我懂的道理比别人多一点，那是因为我犯的错误比别人多一点。任何人都可以变得狠毒，只要你尝试过嫉妒。

6、金钱难买健康，健康大于金钱，金钱难买幸福，幸福必有健康，生命的幸福不在名利在健康，身体的强壮不在金钱在运动！
7、假如我不能，我一定要；假如我一定要，我就一定能。如果一个人不知道他要驶向哪个码头，那么任何风都不会是顺风。
8、你不能左右天气，但可以改变心情。你不能改变容貌，但可以掌握自己。你不能预见明天，但可以珍惜今天！
9、尝试去把别人拍过来的砖砌成结实的地基，生活就不会那么辛苦了。难过就是，虽然很难但总会过去。伟大变为可笑只有一步，但再走一步，可笑又会变为伟大。
10、只要持之以恒，知识丰富了，终能发现其奥秘。在事实面前，我们的想象力越发达，后果就越不堪设想。
