人活着一天，就是有福气，就应该珍惜

发表于 2018-06-14 | 分类于 经典美文摘抄

1、人活着一天，就是有福气，就应该珍惜，人生短短几十年，不要给自己留下更多的遗憾，日出东海落西山，愁也一天，喜也一天；遇事不钻牛角尖，人也舒坦，心也舒坦。
2、无论生活过的是多么的严肃认真，你总是能找到一个不嫌你二的人。清醒时做事，糊涂时读书，大怒时睡觉。许多人费劲爬到了梯子的顶端，才发现梯子架错了墙。
3、美文摘抄，人，只要拥有微笑，就能够豁达乐观地活着；人，只要时刻保持微笑，就能够在漫漫人生道路上拥有一张永久的通行证。

4、人情，人情，人之常情，要乐善好施，长于交往，“平时不烧香，急时抱佛脚”是行不通的，所以，“人的情绪要储存”，就象银行存款，存的越多，时间越长，红利就越大。
5、修心当以净心为要，修道当以无我为基。过去事，过去心，不可记得；现在事，现在心，随缘即可；未来事，未来心，何必劳心。
6、学会把自己的心态放低放平，多看自己的缺点，多看别人的优点，让傲慢的心变得谦虚恭敬，这样您所处的环境自然就融洽了。

有一种境界叫超然，有一种达观叫了然

发表于 2018-06-14 | 分类于 经典美文摘抄

1、法本法无法，无法法亦法。今付无法时，法法何曾法。令一切众生生欢喜者，则令一切如来欢喜。生活磨砺了人生，这一路走来，学会了接受。
2、人生四季，寒暑交替，静心体味生活的悲与喜，笑看起起落落。抱怨，是一块坚硬的石头，横在心间如果不放下，久而久之你会发现，因为这些怨恨，自己的人生便处处有绊脚的石头。
3、美文摘抄，不管人遇到怎样的磨难，不管面临怎样的困境，我们都要坦然面对，因为今天过后就是明天，每一个明天都给人带来新的希望。
4、有一种境界叫超然，有一种达观叫了然，有一种天真叫自然，有一种态度叫悠然，有一种成就叫斐然，有一种繁盛叫蔚然，有一种愉悦叫欣然，有一种释怀叫惠然，而我要的那种幸福，是淡然。

5、将生活变得更丰富更有意义更有价值。体验成功的喜悦，这是每个人最基本的愿望。生而为人，活着，活出风度，风采，风格。这些皆为外表，而气度则是内涵，是质地，是根本。
6、坚强的人把困难当成朋友，软弱的人把困难当成敌人。不要只看到别人的一举成名，更要看到别人的十年寒窗。没有十年寒窗的冷寂，就没有一举成名的荣光。易钉进去的钉子，也易拔出来。困时多想更困事，难时多想更难人。

与其生气埋怨，不如积极快乐的去面对

发表于 2018-06-14 | 分类于 经典美文摘抄

1、凡事三思而后行，以避免发生后悔的事，是一种正确的处世之道。然而，这种态度有时会显得过于消极。相反地，若不加考虑即鲁莽，然后再加以反省检讨的功夫，可能其收获会来得更大。
2、对于人际往来上的第一次接触，仍然要保持年轻人的尝试精神，亦即付出关心并接受挑战，那么，即使会遭遇后悔的事，这种经验即成为往后面对同样问题时的参考。
3、如果抱怨成习惯，心灵就像上了枷锁，没有一刻解脱。唯有放下抱怨，才能体会到生命的自在与幸福。心若简单，生活就简单；心若复杂，生活就充满痛苦。

4、别把工作当负担，与其生气埋怨，不如积极快乐的去面对，当你把工作当作生活和艺术，你就会享受到工作的乐趣。当眼泪流尽的时候，留下的就应是坚强。
5、如果咱们做与不做都会有人笑，如果做不好与做得好还会有人笑，那么咱们索性就做得更好，来给人笑吧！人生就像一杯没有加糖的咖啡，喝起来是苦涩的，回味起来却有久久不会退去的余香。
6、美文摘抄，你永远是最棒的，必须能行！靠山山会倒，靠人人会跑，只有自我最可靠。信念不是到处去寻找顾客的产品推销员，它永远也不会主动地去敲你的大门。

欲望以提升热忱，毅力以磨平高山

发表于 2018-06-14 | 分类于 经典美文摘抄

1、如果惧怕前面跌宕的山岩，生命就永远只能是死水一潭。一个讯息从地球这一端到另一端只需秒，而一个观念从脑外传到脑里却需要一年，三年甚至十五年。
2、美文摘抄，学会忘记是是生活的技术，学会微笑是生活的艺术。志得意满时享受成就，身处低潮时享受人生，有心思时干有意义的活，没心情时做有意思的事。
3、欲望以提升热忱，毅力以磨平高山。泪水和汗水的化学成分相似，但前者只能为你换来同情，后则却可以为你赢得成功。我们总是像智者一样劝慰别人，像傻子一样折磨自己。

4、成功呈概率分布，关键是你能不能坚持到成功开始呈现的那一刻。学会选择，学会放弃；耐得住寂寞，经得起诱惑。昨晚多几分钟的准备，今天少几小时的麻烦。
5、积极的人在每一次忧患中都看到一个机会，而消极的人则在每个机会都看到某种忧患。路漫漫其修远兮，吾将上下而求索。山涧的泉水经过一路曲折，才唱出一支美妙的歌。
6、只有一条路不能选择，那就是放弃的路；只有一条路不能拒绝，那就是成长的路。职场不是走秀场，而是斗兽场，比的不是谁更厉害，而是谁能生存下去，能活得久。
