世上没有绝望的处境，只有对处境绝望的人

发表于 2018-06-15 | 分类于 正能量

1、强行者有志。志坚者，功名之柱也。登山不以艰险而止，则必臻乎峻岭。骄傲是断了引线的风筝稍纵即逝。人无志向，和迷途的盲人一样。胸有凌云志，无高不可攀。
2、正能量，胸无大志，枉活一世。人惟患无志，有志无有不成者。让自己的内心藏着一条巨龙，既是一种苦刑，也是一种乐趣。立志难也，不在胜人，在自胜。
3、一人立志，万夫莫敌。世上无难事只要肯登攀。销售是从被别人拒绝开始的。未遭拒绝的成功决不会长久没有失败，只有放弃。人生不得行胸怀，虽寿百岁犹为无也。

4、人之所以异于禽者，唯志而已矣！莫为一身之谋，而有天下之志。让生命的每一天都成为收获的小时。你的选择是做或不做，但不做就永远不会有机会。
5、志正则众邪不生。人总是珍惜未得到的，而遗忘了所拥有的。所有的胜利，与征服自己的胜利比起来，都是微不足道。千金难买早知道，万金难买的是后悔药。
6、明天的希望，让我们忘了今天的痛苦。世上没有绝望的处境，只有对处境绝望的人。把活着的每一天看作生命的最后一天。要克服生活的焦虑和沮丧，得先学会做自己的主人。

挑战无处不在，自己才是真正的敌人

发表于 2018-06-15 | 分类于 正能量

1、人生各有志。心志要坚，意趣要乐。鱼跳龙门往上游。把意念沉潜得下，何理不可得，把志气奋发得起，何事不可做。丈夫四海志，万里犹比邻。
2、进取用汗水谱烈军属着奋斗和希望之歌。拥有梦想只是一种智力，实现梦想才是一种能力。想要什么，想清楚，选择，珍惜，一如对你的婚姻。
3、正能量，如果你想攀登高峰，切莫把彩虹当作梯子。比别人更爱自己。鸟贵有翼，人贵有志。褴褛衣内可藏志。年轻是我们唯一有权力去编织梦想的时光。

4、远大的希望造就伟大的人物。寄言燕雀莫相唣，自有云霄万里高。丈夫志气薄，儿女安得知？萤火虫的光点虽然微弱，但亮着便是向黑暗挑战。
5、有志者，事竟成。生无一锥土，常有四海心。不为穷变节，不为贱易志。丈夫志不大，何以佐乾坤。用最少的悔恨面对过去。挑战无处不在，自己才是真正的敌人。
6、生命太过短暂，今天放弃了明天不一定能得到。治天下者必先立其志。母鸡的理想不过是一把糠。心随朗月高，志与秋霜洁。要铭记在心；每天都是一年中最美好的日子。

每次跌倒后能再站起来，才是最大的荣耀

发表于 2018-06-15 | 分类于 正能量

有压力，但不会被压垮；迷茫，但永不绝望。有压力，但不会被压垮；迷茫，但永不绝望。从来不跌倒不算光彩，每次跌倒后能再站起来，才是最大的荣耀。
人只要不失去方向，就不会失去自我。失败只有一种，那就是半途而废。人生寂寞是一种力量。人经得起寂寞，就能获得自由；耐不住寂寞，就会受人牵制。
有没有人爱，咱们也要发奋做一个可爱的人。不埋怨谁，不嘲笑谁，也不羡慕谁，阳光下灿烂，风雨中奔跑，做自我的梦，走自我的路。
愤怒有百害而无一利，它只会增添个人的忧虑。一个人愤怒时，心脏泵出的血比平常要多，因此容易导致高血压，还可能导致心脏病。

开始发奋吧！在这个过程中你务必放下很多东西，但你要明白它们都不是你最终想要的，你要坚信在你成功以后，总有一天它们会再回来，而且比此刻更完美！
天将降大任于斯人也，必先苦其心智劳其筋骨饿其体肤空乏其身，行弗乱其所为，因此动心忍性，增益其所不能。
正能量，目标的坚定是性格中最必要的力量源泉之一，也是成功的利器之一。没有它，天才会在矛盾无定的迷径中徒劳无功。
成功没有快车道，愉悦没有高速路。所有的成功，都来自不倦的发奋和奔跑；所有愉悦，都来自平凡的奋斗和坚持。

命运有一半在你手里，另一半在上帝的手里

发表于 2018-06-15 | 分类于 正能量

1、命运有一半在你手里，另一半在上帝的手里。你既然认准一条道路，何必去打听要走多久。如果为了安全而不和大海在一起，船就失去了存在的意义。
2、蝴蝶如要在百花园里得到飞舞的欢乐，那首先得忍受与蛹决裂的痛苦。萤火虫的光点虽然微弱，但亮着便是向黑暗挑战。拒绝严峻的冶炼，矿石并不比被发掘前更有价值。
3、你就像是一只暗夜里的灵魂，潜伏在我的周边。山路不象坦途那样匍匐在人们足下。用平常心来生活，用惭愧心来待人，心来处事，用菩提心契佛心。

4、正能量，你什么时候放下，什么时候就没有烦恼。不是某人使我烦恼，而是我拿某人的言行来烦恼自己。人生没有十全十美，如果你发现错了，就重新再来。
5、一个女人可以长得不漂亮，但是一定要活得漂亮，活的精彩。人，穷时简单，富了复杂；落魄时简单，得势了复杂。淡泊是人生的一种坦然，坦然面对生命中的得失。
6、爱死去时，通常都死得很艰难。既不是消失无踪，也不是寿终正寝。命运一半掌握在上帝手中，另一半掌握在自己手中。成功就是用自己手中的一半去赢得上帝手中的另一半。
