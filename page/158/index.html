青春去时不告别，老年来时不招手

发表于 2018-06-15 | 分类于 励志语录

1、志气和贫困是患难兄弟，世人常见他们伴在一起。一年之计在于春，一日之计在于晨。天地最有情，少年莫浪投。
2、时间是一位可爱的恋人，对你是多么的爱慕倾心，每分每秒都在叮嘱：劳动，创造！别虚度了一生！时间是我的财产，我的田亩是时间。
3、当许多人在一条路上徘徊不前时，他们不得不让开一条大路，让那珍惜时间的人赶到他们的前面去。把握住今天，胜过两个明天。
4、每天不浪费或不虚度或不空抛的那一点点时间，即使只有五六分钟，如得正用，也一样可以有很大的成就。游手好闲惯了，就是有聪明才智，也不会有所作为。

5、青春励志，黄河走东溟，白日落西海，逝川与流光，飘忽不相待。时间不能增添一个人的生命，然而珍惜光阴却可使生命变得更有价值。
6、青春去时不告别，老年来时不招手。丝染无复白，鬓白无重黑，努力爱青春，一失不再得。运气永远不可能持续一辈子，能帮助你持续一辈子的东西只有你个人的能力。
7、必须记住我们学习的时间是有限的。时间有限，不只由于人生短促，更由于人的纷繁。我们应该力求把我们所有的时间用去做最有益的事。

活总是干不完，钱总是不够花

发表于 2018-06-15 | 分类于 励志的话

活，总是干不完！每天睁开眼就忙忙碌碌，盼着能有休息的一天，什么也不用干；一生在打拼总是奔奔波波，想着能有安逸的晚年，什么也不用惦念。
谁又想得到，要做的打算一桩挨着一桩，要干的事情一件接着一件。人这一辈子，一刻也不得闲！该干的，一点不能少干，该担的，一分不能少担。
钱，总是不够花！一日要三餐，四季要衣穿，涨价你也得买，太贵你也得换；孩子要养育，老人要孝敬，学费你也得交，医院你也得跑。
挣的没有花的快，存的没有出的多，人这一辈子，追着钞票跑，最后剩手里的真没多少！

人，总是看不穿！表面上对你笑，背后里做手脚，谁没遇到过几个笑里藏刀；嘴上夸着你好，心里说着你孬，谁没赶上过几个酒肉之交。
看脸看不见心，听音听不见真。人这一辈子，就想看清了人心，好人拿命交，真情用力抱！
情，总是弄不懂！说着一心一意的，不知怎么就变了；聊着有情有义的，不知怎么就淡了；许着不离不弃的，不知怎么就散了。
等你缓个神来，一切都远了，再找也不见了。人这一辈子，感情的事最难琢磨了，心里面清楚谁走谁在，就行了！
其实人这辈子，千万别想太多了。想多了：身就疲惫了，心就憔悴了，人就遭罪了；想多了：睡也睡不着了，吃也吃不好了，干也干不了了！
励志一生，以后就要该吃吃，该喝喝，活好了才是王道；以后就要该玩玩，该乐乐，过好了才有微笑！

这一秒不放弃，下一秒就有希望！

发表于 2018-06-15 | 分类于 励志的话

1、大路走尽还有小路，只要不停地走，就有数不尽的风光。
2、坚持是雄壮的，因为坚持是由于百般地敲打而磨练出来的；坚持是甘甜的，因为无畏的与不倒的毅力早已在心中播下了胜利的种子。
3、你的努力，也许有人会讥讽；你的执着，也许不会有人读懂。在别人眼里你也许是小丑，在自己心中你就是国王！
4、生命不在长而在于好，只要每一次尽力的演示，都值得鼓励与喝采。

5、有了坚持不一定成功；但没有坚持，就注定失败。对于成功，坚持的塑造必不可少！
6、励志的话，这一秒不放弃，下一秒就有希望！坚持下去就会成功！
7、人生便如一块香料，是只有在坚持的信念为柴，坚守的行动为火炙烤中才能散发出最浓郁的芬芳。
8、人在旅途，难免会遇到荆棘和坎坷，但风雨过后，一定会有美丽的彩虹。
9、斧头虽小，但经历多次劈砍，终能将一棵最坚硬的橡树砍刀。
10、要有把路走绝的精神，激流勇退是一种境界，把路走绝是一种精神。坚持到底，不会错的；坚持到底，就是成功！

有一种努力，叫靠自己！

发表于 2018-06-15 | 分类于 励志的话

励志的句子，人生，哪有那么多时间用来浪费。过去的，就算了；伤心的，就忘了，别一直活着悲伤的记忆中，对眼前的不珍惜，对未来的没兴趣。
什么叫做人？宁可吃亏，也不占小便宜；宁可付出，也不辜负人心。不会为了金钱，泯灭了自己的良心；不会为了利益，欺骗了他人的信任。

靠自己，才能踏实安心；靠自己，才能笑得漂亮。钱财不多，自己赚；没车没房，自己拼；累了痛了，自己忘；苦了哭了，自己藏。别人给你的只是一片叶，自己做大树才可以乘凉。
人生如同攀岩，靠别人只会一起坠落，松开手，独自翻越，脚步越来越轻，胜利才越来越近。好好努力，靠自己闯出一片天；好好坚持，靠自己赢得万人颂！
