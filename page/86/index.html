百年明日能几何，请君听我明日歌

发表于 2018-06-14 | 分类于 珍惜时间

1、你热爱生命吗？那就别浪费时间，因为时间是组成生命的材料。年难留，时易损。正当利用时间！你要理解什么，不要舍近求远。
2、每天不浪费或不虚度或不空抛的那一点点时间，即使只有五六分钟，如得正用，也一样可以有很大的成就。游手好闲惯了，就是有聪明才智，也不会有所作为。
3、青春啊，难道你始终囚禁在狭小圈子里，你得撕破老年的蛊惑人心的网。青春是阵偶尔滑过的风，不经意间，已吹得我泪流满面。
4、明日复明日，明日何其多，我生待明日，万事成蹉跎。世人若被明日累，春去秋来老将至。朝看水东流，暮看日西坠。百年明日能几何，请君听我明日歌。
5、利用寸阴是任何种类的战斗中博得胜利的秘诀。生命是短暂的，空余时间很少，因此我们不应把一刻空余时间耗费在阅读价值不大的书籍上。

6、在你青春期的某一个时间点，你都会明白，学习的重要性高过所有。你要无欲则刚，你要学会孤独，你要把自己逼出最大的潜能。没有人会为了你的未来买单，你要么努力向上爬，要么烂在社会最底层的泥淖里，这就是生活。
7、时间的名言，合理安排时间，就等于节约时间。时间是世界上的一切成就的土壤。时间给空想者痛苦，给创造者幸福。
8、人们说生命是很短促的，我认为是他们自己使生命那样短促的。由于他们不善于利用生命，所以他们反过来抱怨说时间过得太快；可是我认为，就他们那种生活来说，时间倒是过得太慢了。
9、时间，每天得到的都是二十四小时，可是一天的时间给勤勉的人带来智慧与力量，给懒散的人只能留下一片悔恨。
10、勤勉的人，每周七个全天；懒惰的人，每周七个早晨。万事须已运，他得非我贤。青春须早为，岂能长少年。光景不待人，须叟发成丝。

时间带来的不只是岁月的沧桑

发表于 2018-06-14 | 分类于 珍惜时间

1、历史孕育了真理，它能和时间抗衡，把遗闻旧事保藏下来。它是往昔的迹象，当代的鉴戒，后世的教训。
2、当我们为一去不复返的青春叹息时，我们应该考虑将来的衰老，不要到那时再为没有珍惜壮年而悔恨。选择时间就是节约时间。
3、勿谓寸阴短，既过难再获。世界上最快而又最慢，最长而又最短，最平凡而又最珍贵，最易被忽视而又最令人后悔的就是时间。
4、时光不能倒退阳光，是免费的，时光，是不能倒退的；爱情，是免费的，婚姻，是不能凑合的；友情，是免费的，相遇，是很不容易的；梦想，是免费的，努力，一定需要的。
5、凭着日规上潜私的阴影，你也能知道时间在偷偷地走向亘古。日月忽其不淹今，春与秋其代序。不守时间就是没有道德。

6、即将来临的一天，比过去的一年更为悠长。古人轻尺璧而重寸阴，惧求时之过已。抛弃时间的人，时间也抛弃他。
7、消磨时间消磨时间是一种多么劳累，多么可怕的事情啊，这只肉眼看不见的秒针无时不在地平线下转圈，你一再醉生梦死地消磨时间，到头来你还得明白，它仍在继续转圈，无情地继续转圈。
8、于千万人之中遇见你所遇见的人，于千万年之中，时间的无涯的荒野里，没有早一步，也没有晚一步，刚巧赶上了，那也没有别的话可说，惟有轻轻地问一声：“噢，你也在这里吗？”
9、今天做不成的，明天也不会做好。一天也不能虚度，要下决心把可能的事情，一把抓住而紧紧抱住，有决心就不会任其逃去，而且必然要贯彻实行。光阴潮汐不等人。
10、时间的名言，时间带来的不只是岁月的沧桑，时间还让人成熟，面对迷茫的世界不再恐慌，面对坎坷的人生不再畏惧。

合理的安排时间，就等于节约时间

发表于 2018-06-14 | 分类于 珍惜时间

1、昨天是一张作废的支票，明天是一张期票，而今天则是你惟一拥有的现金，所以应当聪明地把握。浪费时间是一桩大罪过。
2、我只惋惜一件事，日子太短，过得太快。一个人从来看不出做成什么，只能看出还应该作什么？神龟虽寿，犹有竞时。
3、时间的印记，带走的只是断断续续的回忆，有些许的定格也是在那特殊的日子里。合理的安排时间，就等于节约时间。
4、劝君须惜少年时，劝君莫惜金缕衣，有花堪折直须折，莫待无花空折枝。人误地一时，地误人一年。
5、时而言，有初中后之分；日而言，有今昨明之称；身而言，有幼壮艾之期。时间是人能消费的最有价值的东西。
6、我的艺术应当只为贫苦的人造福。啊，多么幸福的时刻啊！当我能接近这地步时，我该多么幸福啊！

7、时间的名言，充实闲暇时间能聪明地充实闲暇时间是人类文明最新成果。没有一种不幸可与失掉时间相比了。
8、如果有什么需要明天做的事，最好现在就开始。一个人越知道时间的价值，越倍觉失时的痛苦！合理安排时间，就等于节约时间。
9、在所有的批评中，最伟大最正确最天才的是时间。落日无边江不尽，此身此日更须忙。尽头时间有没有尽头？
10、最聪明的人是最不愿浪费时间的人。吾生也有涯，而知也无涯。在一切与天俱来的天然赠品中，时间最为宝贵。

人生天地之间，若白驹过隙，忽然而已

发表于 2018-06-14 | 分类于 珍惜时间

1、善于利用时间的人，永远找得到充裕的时间。普通人只想到如何度过时间，有才能的人设法利用时间。
2、不教一日闲过。时间就像海绵里的水，只要愿挤，总还是有的。活在当下，别在怀念过去或者憧憬未来中浪费掉你现在的生活。
3、我并没有什么方法，只是对于一件事情很长时间很热心地去考虑罢了。人生天地之间，若白驹过隙，忽然而已。
4、凡是想获得优异成果的人，都应该异常谨慎地珍惜和支配自己的时间。庸人费心将是消磨时光，能人费尽心计利用时间。
5、我曾经认为自己是个太空人，我花了好长一段时间，才发现我只是一名星球之间的寂寞舞者。

6、生命是短促的，然而尽管如此，人们还是有时间讲究礼仪。三更灯火五更鸡，正男儿立志时。黑发不知勤学早，白首方悔读书迟。
7、幸福有它的两重性：一方面在于福至心灵，时来运至……另一方面，也是最实际的方面，就是知足常乐地安度日常生活，这也就是说，头脑清醒，不干蠢事。
8、即使一动不动，时间也在替我们移动。而日子的消逝，就是带走我们希望保留的幻想。时间好比河水，只能流去不能回。
9、时间的名言，痛苦和寂寞对年轻人是一剂良药，它们不仅使灵魂更美好，更崇高，还保持了它青春的色泽。
10、生命是以时间为单位，浪费别人的时间等于谋财害命；浪费自己的时间，等于慢性自杀。在每个国家，知识都是公共幸福的最可靠的基础。
