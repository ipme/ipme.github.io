是你选择了这个世界，不是这个世界选择了你！

发表于 2018-06-14 | 分类于 人生感悟

生活终究是自己的，活给别人看的人生，终究不快乐，别人又不是你自己，他们怎么知道你走过的路，他们怎么了解你心中的苦与乐。
我们总是活给别人看，也在盯着别人怎么活，不断地揣摩着别人的想法，照顾着别人的情绪。忘记了为自己好好活一次！
人这辈子，有人羡慕你，有人讨厌你，有人嫉妒你，有人看不起你，没关系，他们都是外人。什么样的路适合你，只有你自己知道，喜欢什么样的生活，只有你自己心里明白。
生活就是这样，你所做的一切，不能让每个人都满意。不要为了讨好别人，而丟失自己的本性，因为每个人都有原则，别人嘴里的你，不是真实的你。你是活给自己看的，只要自己觉得幸福就行。
我们用不着向别人证明什么；也不要光顾着看别人，走错了自己脚下的路。是你选择了这个世界，不是这个世界选择了你！所以这辈子，请为自己而活。

人生感悟，因别因为别人的评价，丢掉了多少独一无二的性格；在乎的时间愈久，我们就会分不清生命。究竟是活给自己看还是活给别人看。
佛说一切唯心造，当我们有烦恼时，心就在地狱了。生活中的不顺、烦忧、睡不好做噩梦，皆是地狱发通知书的征兆，此时虽然人还没进地狱，但是心已经靠近了。
佛说心在哪，命就在哪。念佛就是把心放在了极乐，一切恶疾、烦恼、恐惧、怨恨、所有不顺心、所有障碍生命欢喜的，都可在佛号中转化消失。
我们过去因贪嗔痴，犯过杀盗淫妄酒，过去不信因果、不敬师长、不孝父母，所有大麻烦、大罪业，通过念佛忏悔，统统交给佛，恭请佛菩萨把这一切转化过来。
佛说苦难是上天另一种形式的爱，提醒我们从错误中回来。生命中的灾厄、苦难、麻烦，其实是上天善神、诸佛菩萨在提醒我们改变，不能再继续造业。
无论我们遇到什么灾难、烦恼、问题，只要遇到佛法，就能转化，把丑恶转成美善，把烦恼转成菩提，把苦难转成财富，把罪业转成功德。

别说狠话，让自己后悔；别留遗憾，让心灵愧对

发表于 2018-06-14 | 分类于 人生感悟

人生如戏，演的是自己，没有指定的剧本，没有一样的剧情，剧本是自己的经历，剧情靠自己去演绎。悲也好，喜也罢，我们都是自己的主角，和别人的戏，没有关系。
人生一场戏，全都是天意，相遇，是上天的指派，离开，是上天的安排。分分合合，恩恩怨怨，都是戏里的片段。喜怒哀愁，生离死别，都是注定的情节。有缘的相聚，无缘的分开，相欠的偿还，不欠的不见。
茫茫人海中，谁是你擦肩的客？大千世界里，谁是你牵挂的人？缘分的尽头，有人在你心上停留，是疼痛是欢喜，他都是那绝无仅有。
相遇了，请你珍惜，因为没有预见的结局。陪伴了，请你用心，因为得一真心太不易。给予就会被给予，剥夺就会被剥夺。多去疼爱，收获真爱多去付出，内心满足。
人生一场戏，全都是天意，本以为一辈子，结果却只剩下，一句偶尔的关心，一声简单的问候。遇见不容易，分别不由己，这世间有多少场戏，都成了没结局的剧。

人生感悟，人生一场戏，全都是天意，缘来缘又去，情是局中局，何必再强留，人留心不留，好好做自己，难得是开心！
我们只活这一回，我们只爱这一次，我们只陪这一程，我们只有这一生。来世，谁还记得清谁的面容；来世，谁还分得清谁的姓名；来世，谁还陪得了谁的身旁；来世，我们再也不是谁的谁了。
生命轮回，来世不知道还能否遇见。人一旦死去，再也没有重来。感情，真的没有彩排。一旦失去，再也没有机会。遇到有缘的人：能爱就认真爱，能聚就多聚聚，能守就别离开！
今生，彼此多一点善待！谁错谁对，互相包容；谁是谁非，互相理解；不要争吵，不要计较，不要伤害，不要分开。别说狠话，让自己后悔；别留遗憾，让心灵愧对。
我们这一生，谁也不能推断哪一天，就是生命的最后一天；谁也不能预测哪一面，就是今生的最后一面。所以说：有空常把父母看看；有空多和朋友聚聚；有空多与爱人聊聊；有空多把孩子陪陪。
来世，我们再也不是谁的谁了，今生若相遇，好好珍惜吧；今生若相爱，好好守护吧；今生若相知，好好在一起吧！

尽力做好一件事，实乃人生之首务

发表于 2018-06-14 | 分类于 人生哲理

1、年少时我们有足够多的理由去认认真真地喜欢另一个人，而长大后我们有同样多的理由去认认真真辜负另一个人。别沮丧，生活就像心电图，一帆风顺就证明你挂了。
2、做事有始有终值得开始的事就值得完成。聪明人做事总是有始有终。只有一条路不能选择，那就是放弃。最后的措手不及是因为当初游刃有余的自己。现实很近又很冷，梦想很远却很温暖。
3、松驰的琴弦，永远奏不出时代的强音。堕落；人生不在乎掠取多少，而在于追求过程的完美与卓越！磨练，使人难以忍受，使人步履维艰，但它能使强者站得更挺，走得更稳，产生更强的斗志。
4、敢于向黑暗宣战的人，心里必须充满光明。梦想注定是孤独的旅程，路上少不了质疑和嘲笑，但那又怎样，哪怕遍体鳞伤也要勇往直前。东西，让你羡慕，却不能拥有；有些错过，让你留恋，却终生遗憾。
5、这世界上没有绝对的感同身受能帮你的也只有你自己。只要站起来的次数比倒下去的次数多，那就是成功。活得失败的原因就是：讨厌自己明明不甘平凡，却又不好好努力。

6、答应自己，不要去在乎一些么必要的人和事。失因为贪，说真的，老实人很少上当。人生应该树立目标，否则你的精力会白白浪费。山涧的泉水经过一路曲折，才唱出一支美妙的歌。
7、点燃了的火炬不是为了火炬本身，就像我们的美德应该超过自己照亮别人。贝壳虽然死了，却把它的美丽留给了整个世界。尽力做好一件事，实乃人生之首务。
8、应该让别人的生活因为有了你的生存而更加美好。快乐是一种香水，无法倒在别人身上，而自己却不沾上一些。我们的报酬取决于我们所做出的贡献。
9、经典哲理，给予的最需要的方面不在物质财富范围内，它存在于人性特有的领域。无所不能的人实在是一无所能，无所不专的专家实在是一无所专。
10、对骄傲的人不要谦虚，对谦虚的人不要骄傲。比我差的人还没放弃，比我好的人仍在努力，我就更没资格说我无能为力！你配不上自己的野心，也辜负了曾经历的苦难。

你用了多少心，你的成就就会有多少

发表于 2018-06-14 | 分类于 人生哲理

1、当你手中抓住一件东西不放时，你只能拥有一件东西，如果你肯放手，你就有机会选择更多。历史，只有人名真的；小说，只有人名是假的。要随波逐浪，不可随波逐流。
2、你球衣前面的名字，比后面的名字更重要。永远不要埋怨你已经发生的事情，要么就改变它，要么就安静的接受它。养成每天写点什么的习惯，哪怕是记录，哪怕只言片语。
3、据我测算，还可以退五十步，但生活只有五步。不较真，“真”就不出来，还是较吧。我们都是虫，可我是萤火虫。有大快乐的人，必有大哀痛；有大成功的人，必有大孤独。
4、生活中多数人渴望安稳，希望平平静静走完一生，不愿起起伏伏，波波折折，坚信幸福需要安稳，愉快需要平静。做人还是简单点好，爱就爱，恨就恨，不掩饰，不做作。
5、如果你坚信石头会开花，那么开花的不仅仅是石头。当你一门心思撞南墙的时候，南墙就消失了。过所爱的生活，爱所过的生活，快乐的生活，才能生活快乐，快乐的工作，才有快乐人生，生活的理想其实就是理想的生活！

6、前方无绝路，希望在转角。脚是大地上飞翔的翅膀。苦难是化了装的幸福。只要比竞争对手活得长，你就赢了。除非你是三栖动物，否则，总有一个空间不属于你。
7、自己丰富才感知世界丰富，自己善良才感知社会美好，自己坦荡才感受生活喜悦，自己成功才感悟生命壮观！难过时，吃一粒糖，告诉自己生活是甜的！
8、经典哲理，世界上只有想不通的人，没有走不通的路。成功便是站起比倒下多一次。不要放弃，你要配的上自己的野心，也不要辜负了所受的苦难。
9、决定今天的不是今天，而是昨天对人生的态度；决定明天的不是明天，而是今天对事业的作为。我们的今天由过去决定，我们的明天由今天决定！要把昨天当作回忆，今天继续努力，明天就随他去。
10、决定工作效率和工作质量的，首先是工作态度，其次才是能力。你用了多少心，你的成就就会有多少。凡事全力以赴，尽心尽力，才能有所建树。
