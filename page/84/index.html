每一发奋发奋的背后，必有加倍的赏赐

发表于 2018-06-15 | 分类于 早安心语正能量

01、胜利果实，永远挂在树梢上，你可要努力往上跳，才能摘到啊！莫找借口失败，只找理由成功。不为失败找理由，要为成功找方法！
02、成功就是简单的事情不断地重复做。我很努力的去挽回，却发现事与愿违。每一发奋发奋的背后，必有加倍的赏赐。
03、我们都是阳光，无需凭借谁的光。今天做的一切挣扎都是在为明天积蓄力量，所以别放弃。放心吧，生活从不会让强大的人受委屈。
04、为自己找借口的人，永远不会进步。不依靠，不寻找，爱自己，又是新开始！如果你不努力争取你想要的，那你永远都不会拥有它！
05、坚持你的梦想，永不放弃，也许在别人看来它是那么的渺小，但只有我们自己知道它对于我们自己有着怎样的意义。

06、人生，需要我们有足够的勇气，来面对自己的生活，来面对周边的人和物，只有你能够直面这一切时，你才会发现，什么是勇敢。
07、所谓成长，就是逼着你一个人，踉踉跄跄的受伤，跌跌撞撞的坚强。面对艰难困苦，懦弱者被磨去棱角；勇敢者将意志品质磨砺得更为坚强。
08、人生如路，须在荒凉中走出繁华的风景。平凡朴实的梦想，我们用那唯一的坚持信念去支撑那梦想。不要抱怨自己所处的环境，如果改变不了环境，那么就改变自己的心态。
09、早安心语，怀抱一种信仰，只要忠实于自己最好的一面，世界将与你同在。生命是一条艰险的狭谷，只有勇敢的人才能通过。
10、坚韧是成功的一大要素，只要在门上敲得够久够大声，终会把人唤醒的。你的父母正在为你打拼，这就是你今天需要坚强的理由。
11、过去的习惯，决定今天的你，所以，过去的懒惰，决定你今天的一败涂地。你把事情做成了，放个屁人家都觉得很有道理，你失败了，说得再有道理人家也觉得是个屁。

幸福就是用生活最苦，酿造人生的甜酒

发表于 2018-06-15 | 分类于 早安心语正能量

01、你不懂我的时候，就算我把我心掏给你看，你也看不懂。
02、当你在高处的时候，你的朋友知道你是谁，当你坠一落的时候，你才知道你的朋友是谁。
03、不管你干什么，都会有两种结果：一种是笑话，一种是神话。如果你半途而废，只能成为朋友中的笑话；但如果你成功了，你就变成她们眼中的神话。社会就是现实！要么不做，要么做好。
04、只有经过选择之后存留下来的事物，才是我们所真正需要的，那就是美好和快乐。
05、幸福是一种感觉。不依赖慑人的权势，不依赖过人的财富，不依赖超人的才华。依赖的是一颗平常心：常怀一颗笑对人生冷暖的平常心，就有圆融丰满的喜悦常相伴随。幸福似穿鞋，松紧自明；幸福如喝水，冷暖自知。如果说快乐是生理的，那么幸福是精神的。幸福就是用生活最苦，酿造人生的甜酒。
06、早安心语，青春是一场大雨，即使感冒了，还盼望回头再淋它一次。

07、小时侯，幸福是一件东西，拥有就幸福；长大后，幸福是一个目标，达到就幸福；成熟后，发现幸福原来是一种心态，领悟就幸福。
08、生活，就是一种永恒沉重的努力，人生最后悔的事，莫过于轻易的放弃了不该放弃的事；人生最珍贵的东西不是得不到的东西，而是眼前所拥有的东西；人生最美好的东西，不是已失去的东西，而是现在所把握的东西。
09、你虽然在平凡的人群中不起眼，可是要相信每个人都是会闪耀着自己独到的光辉灿烂。
10、使我们视而不见的光亮，对于我们及时黑暗。当我们清醒时，曙光才会破晓。来日方长，太阳是颗启明星！
11、戴三百块的表和三万块的表，时间是一样的；喝三十块的酒和三千块的酒，呕吐是一样的；住三十平米的房和三百平米的房，孤独是一样的。我们需要的是存在感，而存在感，不关爱马仕和兰博基尼的事。总有一天你会明白，你内心真正的快乐，是物质世界永远都给予不了你的。
12、人生就像是向日葵，永远向着太阳，只有夜晚才低下头，这个时候才是自己！

本本分分的做人，求得是无愧于心！

发表于 2018-06-15 | 分类于 早安心语正能量

1、事不强求，求来求去是空；人不强留，留来留去成仇。该来的自会出现，要走的无法挽留。
2、人生没有绝对的公平，付出未必有回报，强留未必能幸福。你在乎的越多，失去的就越多，只有看淡，顺其自然，才能减少痛苦的滋生。
3、友情，不强求，真心实意，就好好相处；虚情假意，就别再联系。真正的朋友，患难与共；虚伪的朋友，不会长久。
4、爱情，不强求，若爱，就不会欺骗；不爱，就直接说穿。爱你的人，再远也会对你眷恋；不爱你的人，再近也不会给你陪伴。
5、财富，不强求，尽自己努力去干，凭自己良心去赚。多多少少，够用就好，金山银山，不如心安。

6、早安心语，幸福，不强求，属于你的永远都在。只要心知足，不贪图，只要看得淡，想得开，幸福就在你身边。
7、命中注定的事，很难改变；上天安排好的，别去挑战。是你的，就是你的，谁也抢不走，争不过。不是你的，不管你怎么争，怎么求，都不会长久拥有。
8、三千繁华，弹指刹那，似水流年，转瞬即逝。很多事，不必弄清楚，糊里糊涂就是最好的答案。很多人，不必去强留，微笑放手就是最好的祝福。
9、人生，不强求，一切顺其自然。不管是感情还是事情，不管是财富还是幸福，一切都会有最好的安排。踏踏实实的做事，要的是坚持到底；本本分分的做人，求得是无愧于心！

经历过了成功，使我懂得了要更加自信

发表于 2018-06-15 | 分类于 早安心语正能量

01、岁月如歌，在悄无声息的流逝中，生活，永远会告诉你很多人生哲理，而随着时间的推移，你会知道的越来越多，感悟的越来越多，明白的越来越多。经历过了成功，使我懂得了要更加自信。经历过了失败，使我懂得了要更加清醒。
02、每个人都有自己的活法，我们没必要去羡慕别人的生活。有的人表面风光，暗地里却不知流了多少眼泪；有的人看似生活窘迫，实际上人家可能过得潇洒快活。幸福没有标准答案，快乐也不止一条道路。自己喜欢的日子，就是最美的日子；适合自己的活法，就是最好的活法。
03、工作是一场赛跑，力量虽重要，但也要思考，速度虽重要，也要了解跑道，耐力虽重要，还要讲究技巧，做到面面具到，才可能捧得鲜花赢得欢笑！
04、怀揣未来的梦想，在每一个平凡而不平淡的日子里，才会笑得更加灿烂。只要不放弃，就没有什么能让自己退缩；只要够坚强，就没有什么能把自己打垮！痛苦是磨砺意志的磨刀石，它能把你的意志之剑磨得更加锋利；痛苦是人生智慧的催熟剂，它能让你的智慧之果更早地成熟。
05、如果，感到此时的自己很辛苦，那告诉自己：容易走的都是下坡路。坚持住，因为你正在走上坡路，走过去，你就一定会有进步。如果，你正在埋怨命运不眷顾，开导自己：命，是失败者的借口；运，是成功者的谦词。命运从来都是掌握在自己手中，埋怨只是一种懦弱的表现！
06、就让我们扬起奋斗的风帆，吹响奋斗的号角，奏响奋斗的乐章，向前吧！去改写自己的命运，去创造自己的奇迹吧！因为，人生需要不停地奋斗。你好，早安！

07、自暴自弃便是命运的奴隶，自强不息是生命的天使；我不想用别人的汗水浇灌自己的心灵，我愿意用别人的棉袄，来温暖自己的躯体。我只想堂堂正正的做人，我只愿光明磊落做事，该记得的我不会遗忘，该遗忘的我不会存放。
08、我们要做聪明的人，做有智慧的人有慈爱又肯诚实对人对己的勇者。就算天大的事情来了，也不逃避它，心平气和地为自己争取最合理的解决之道，不可以做一个弱者，凡是不顺心便跌倒的人是要被社会淘汰的，做一个有弹性的人，当是我们一生追求的目标。
09、一粒尘埃，在空气中凝结，最后生成磅礴的风雨；一粒沙石，在蚌体内打磨，最后结成昂贵的珍珠。有时候，渺小的开始，可以成就雄伟而宏大的事业；有时候，平凡的开始，可以走出崇高而伟大的人生。
10、早安心语，没有哪一种胭脂能涂抹时间，没有哪一件服装能掩饰灵魂，没有哪一套古籍能装潢空虚。
11、人生路上，每个人都会遇到这样或那样的问题，这正是考验我们内心的日寸刻。我们能否战胜困难，其实是在考验我们能否战胜自己。只要心中有阳光，对美好依然向往，内心就会变得强大，奇迹就会发生。
12、要想知道梨子的味道，就要自己去尝一尝。人生就是一个不断尝试的过程，不跨出勇敢的第一步，一切美好的梦想都只不过是虚无缥缈的幻想。有的事情只有自己去做了，才能真正明白其中的道理，靠想象是没有用的。
