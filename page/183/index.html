笑容可以瞒过别人，可心痛却骗不了自己

发表于 2018-06-15 | 分类于 伤感说说

1、或许太重感情的人，日子终究不会好过，信任，依赖，念旧，分分钟把你虐的万劫不复。最痛苦的一种再见是从未说出口，但心里却清楚，一切都已结束。
2、明明喜欢的是你，却要装做无关痛痒的开着你跟别人的玩笑。爱是一生磨难，不爱是一生遗憾。
3、我已用尽全部可以打扰你的借口，但你还是没回头。我错把结局当成了开始，还以为将会是一个美丽的故事。
4、他把刀插进你心脏，说了一句我爱你，你却捂着伤口忍着痛挤出一个微笑对他说：再来一刀我还想听句我爱你。
5、我拿青春来等你，换来的只是“别闹”二字。有时候一直觉得不在意，但是当一切变成了回忆，才发现，竟无法承受这种失落的感觉。

6、女人爱一个男人，往往要比男人爱一个女人的代价高很多，有时候这个代价就是一生。笑容可以瞒过别人，可心痛却骗不了自己。
7、爱一个不爱你的人，就像在机场等一艘船。提着酒瓶摇摇晃晃，笑着摔倒哭着推开所有人的手，我要怎么告诉你我过得很好。
8、我怕我没有足够的勇气一直等你。故事那头她还在等，说书人却合扇说从头。如果忽远忽近的洒脱，是你要的自由，那我宁愿回到一个人的生活。
9、风吹乱了她的头发，裙子上沾了些酒渍，她说她能爱上烟酒，能爱上深夜，能爱上鲜血与铁锈，却唯独再无法爱上一个人。
10、伤感说说，只怕慌忙中抓住谁的手又狼狈跌倒。怕人询问，咽泪装欢。我们总是在最美的时间遇到了不应该遇到的人而爱得好深。

喝醉就去睡，活的再孤独也无所谓

发表于 2018-06-15 | 分类于 伤感说说

1、这世上，能真正对你好的人，遇过一个，就少一个。大张旗鼓的离开都是试探，真正的离开没有告别，悄声无息。
2、抽烟的姿势越来越熟练了，爱人也越来越随便了，从当时的三杯倒到现在的喝不醉，那是堕落，不是成长。
3、累了就喝醉，喝醉就去睡，活的再孤独也无所谓。再喜欢也不要旧情复燃，旧情复燃的结果就是重蹈覆辙。
4、当你在乎的人开始逗别人笑的时候你就真的输了。对一个人的爱累到什么程度才会心甘情愿的去放弃。
5、我吸烟吸到恶心躲在厕所吐满脸都是泪怎么也没见有人从我手中夺走烟抱着我说心疼。那一年，我们觉得世界很小，如今，各奔东西后才知，一别就是一世。

6、失去比得不到可怕很多很多，因为它多个过程叫曾经拥有。心疼每个不开心却依旧在笑的孩子。
7、社会很现实，很少有人去怜悯你的不知所措和情绪低落。喜欢上一个不属于自己的人真难受，连吃个醋都名不正言不顺。
8、伤感说说，很多关系，不用风吹，自然就散了。在感情上我们都不是好演员，我藏不住喜欢，你演不出热情。
9、明明喜欢的是你，却要装做无关痛痒的开着你跟别人的玩笑。就算你知道他满嘴谎话，不还是将错就错的陪他演完这场戏。
10、他即使千般不好万般辜负毕竟是我爱过的人。你最终还是选择了她，却忘了一直有个我。真心总是无人喜，偏偏套路撩人心。

有些人，只是友情客串，不会倾情出演

发表于 2018-06-15 | 分类于 伤感说说

时间在变，物是人非让人感叹；人心善变，人走茶凉让人心寒。遇事了，才知谁给你温暖；困难了，才知谁袖手旁观。有些人，只是友情客串，不会倾情出演。
有些人，只会挑拨离间，不会排忧解难。唯有时间让你看清：谁是虚伪的脸，谁是真心的伴。
有些事，看透了不必揭穿，只适合，睁一只眼闭一只眼。有些人，看清了不必翻脸，只适合，看轻一点看淡一点。不多言，不代表心里没数，不多语，不代表心里没谱，有时候装糊涂，才是最好的自我保护。
人生，因果循环，环环相扣。你真诚，才能收获他人的坦诚；你尊重，才能得到他人的尊敬；你珍惜，才能体会缘分的真谛。
人心，因交换而温暖，因冷漠而陌生；因真挚而信任，因虚伪而离散。无论爱情还友情：不去付出，也会越走越远；不懂感恩，也难撑一片天。
人心善变，勿愿，勿怨。今日非昨，勿悔，勿念。世事变迁，随心，随缘。信与不信，一念之间；懂与不懂，理解之间。
伤感说说，爱与不爱，珍惜之间；人间三千事，淡然一笑间。时间在变，人也在变，做自己，不必解释！

一颗心伤了，就凉了，一个人伤了，就走了，有些离开是因为绝望，有些沉默是因为心凉。沉默，不是懦弱，沉默，是一个人无言的反抗，争辩，是因为在意，哭泣，是因为挽留，而沉默，却是因为不在乎。
当一个人不在乎你了，才会对你的言行不关注，对你的付出无反应，不管你做得多好，说得多真，她都会淡定，不再像从前那样，轻易动怒，瞬间感动。
夫妻之间，不怕争吵，最怕无言，朋友之间，不怕调侃，最怕冷落，若是真的在乎一个人，怎么会面对面无话谈，伤人的永远不是谩骂，不是争辩，而是无尽的沉默。
有多少感情，从无话不说到无话可说，有多少朋友，从娓娓而谈到相对无言，人和人之间，沉默才是最可怕的，因为不重要，所以才看轻，因为不在乎，所以不沟通。
别把沉默给了最爱的人，也别让最在乎自己的人沉默。是不是非得经历过，受伤了，你才会懂得：沉默，是毫不在乎的预兆，沉默，是事不关己的冷漠。
别以为沉默，就是一种顺从，有的沉默，是因为不在乎，我们的情绪波动和身边的人有关，我们的喜怒哀愁和在乎的人有关，而那些对我们毫无影响的人，永远都是陌生人。

3、过错是暂时的遗憾，而错过则是永远的遗憾！
4、环境不会改变，解决之道在于改变自己。
5、勇气是控制恐惧心理，而不是心里毫无恐惧。

7、在实现理想的路途中，必须排除一切干扰，特别是要看清那些美丽的诱惑。
