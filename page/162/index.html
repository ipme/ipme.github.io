做人要学会，走自己的路，看自己的景

发表于 2018-06-15 | 分类于 励志的话

人的一生须练就两项本领：一是说话让人结缘，二是做事让人感动。恶语伤人心，良言利于行。行事之恶，莫大于苛刻；心术之恶，莫大于阴险；言语之恶，莫大于造诬。伤人以言，甚于刀剑。
修炼口德，就是修炼自己的气场，一身正气才能好运多多。口德好才能运势好，运势好才能少走弯路，多些成就。
不知而说，是不聪明；知而不说，是不忠实。君子言简而实，小人言杂而虚。赠人以言，重于珠玉；伤人以言，甚于刀剑。语言切勿刺人骨髓，戏谑切勿中人心病。
与善人言，暖如布帛。伤人之言，痛如刀割。刀疮易好，恶言难消。口是伤人斧，言是割舌刀。闭口深藏舌，安身处处好。得人善言，如获金珠宝玉。见人善言，美于诗赋文章。
励志的话，言不可轻说，若说话更改，不如不说；言不可轻诺，若应诺更改，不如不诺。所以一言半诺，俱宜谨慎为要。

有道德、信义、智谋者，必不多言；惟小人、狂人、妄人者，必会多言。言不中理，不如不言，一言不中，千言无用。口舌祸之门，灭身之斧也。
为人处事之道要有，真诚善良之心别丢。信你的人，别骗；骗你的人，别信。爱你的人，别伤；不爱你的人，别想。有爱无爱，安然对待。谁好谁坏，心里明白。谁是谁非，少理是非。谁真谁假，时间证明。
世界上就没有脾气特别好的人。那些脾气差的人无非是有火直接冲你发，而那些你觉得脾气好的人，其实是不忍心冲你发火而已。永远不要把一个人对你的好，当成理所当然。这个世上，最珍贵的情，便是在乎你的心。
与人相处，永远也不要高估你在别人心中的地位，也许你什么都不是，多你一个也不多，少你一个也不少。要学会，走自己的路，看自己的景。自己永远是自己的主角，不要总在别人的戏剧里跑龙套。人生不过如此，且行且珍惜。

喜我者，我不舍辜负，厌我者，咱形容陌路！

发表于 2018-06-15 | 分类于 励志的话

人活一世，会遇见不同的人，会碰上不同的事。遇事时，有人为你拔刀相助，有人对你落井下石；风光时，有人为你捧场喝彩，有人对你暗地使坏。
人活一辈子，仇不可记。能宽容的就宽容，容人显大度；能放下的就放下，放下显气度；能忘记的就忘记，忘记是智慧。
老话说得好：冤冤相报何时了，得饶人处且饶人。耿耿于怀，只会让心里不舒服；常常怨恨，只会让生活不快乐。
人活一辈子，恩不可忘。贫困无助时，肯借钱给你的人；落后于人时，肯拉你一把的人；你一无所有，能够跟着你的女人；你容颜老去，能够守着你的男人。
励志的话，这些都是生命中，不可多得的人，难能可贵的情！有一颗感恩的心，才能让别人靠近，才能让别人相信。

这一辈子：要想活得舒心，仇不可记。越记越烦心，越记越怨恨。不如赶紧忘记的妙！要想活得安心，恩不可忘。忘了良心不安，忘了自己愧疚。不如趁早回报的好！
做人，要的就是真模样。待人真诚，不算计，对人真心，不作假。想什么说什么，直言不讳，做什么不遮掩，坦坦荡荡。
做事，要的就是真性情。不偷奸耍滑，欺骗谁，不偷工减料，隐瞒谁。做错了的事，能知错就改，做不到的事，不滥竽充数。
昧心钱我从不去赚，骗人事我从来不揽，人再难，也以人品为先，心再苦，也不稔恶藏奸。行得正坐得端睡得安稳，拿得起放得下自在人心。
人善人欺天不欺，心善心好天有报！被欺负没啥，泪流过就好，被嘲笑无碍，心安定就行。喜我者，我不舍辜负，厌我者，咱形容陌路！

突破心理障碍，才能超越自己

发表于 2018-06-15 | 分类于 励志格言

1、人生就像一个动物园，当你以为你在看别人耍猴的时候，却不知自己也是猴子中的一员！从绝望中寻找希望，人生终将辉煌。
2、你不能左右天气，但可以改变心情。你不能改变容貌，但可以掌握自己。你不能预见明天，但可以珍惜今天。尝试去把别人拍过来的砖砌成结实的地基，生活就不会那么辛苦了。
3、励志格言，行动不一定带来快乐，而无行动则决无快乐。你可以一无所有，但绝不能一无是处。突破心理障碍，才能超越自己。
4、人的缺点就像花园里的杂草，如果不及时清理，很快就会占领整座花园。滴水穿石不是靠力，而是因为不舍昼夜。相信自己能力的人，任何事情都能够做到。
5、把事情办好的秘密就是行动。成功之路就是有条理思考之后的行动！行动！行动！你可以看不惯一些东西，但是你应该学会接受，如果你没法改变那一切的话。

6、但愿每次回忆，对生活都不感到负疚。人生路上有阻挡你梦想的砖墙，那是有原因的。这些砖墙让我们来证明我们究竟有多么想要得到我们所需要的
7、世上并没有用来鼓励工作努力的赏赐，所有的赏赐都只是被用来奖励工作成果的。生命的完整，在于宽恕、容忍、等待与爱，如果没有这一切，你拥有了所有，也是虚无。
8、当我们眺望远方的时候，近处的风景便看不清了。如果你希望成功，以恒心为良友，以经验为参谋，以小心为兄弟，以希望为哨兵。
9、思考是一件最辛苦的工作，这可能是为什么很少人愿意思考的原因。成功呈概率分布，关键是你能不能坚持到成功开始呈现的那一刻。
10、自己打败自己是最可悲的失败，自己战胜自己是最可贵的胜利。为别人鼓掌的人也是在给自己的生命加油。拥有梦想只是一种智力，实现梦想才是一种能力。

努力不不一定成功，不努力一定不成功

发表于 2018-06-15 | 分类于 励志格言

1、人生最大的错误，是用健康换取身外之物，人生最大的悲哀，是用生命换取个人的烦恼，人生最大的浪费，是用生命解决自己制造的麻烦！
2、人生最大的哀痛，是子欲孝而亲不在！人生最大的悲剧，是家未富而人先亡，人生最大的可怜，是弥留之际才明白自己是应该做什么的！
3、当心灵趋于平静时，精神便是永恒！把欲望降到最低点，把理性升华到最高点，你会感受到：平安是福，清心是禄，寡欲是寿！
4、努力不不一定成功，不努力一定不成功。永远不抱怨，一切靠自己。人生最大的改变就是去做自己害怕的事情。如果我们想要更多的玫瑰花，就必须种植更多的玫瑰树。
5、伟人之所以伟大，是因为他与别人共处逆境时，别人失去了信心，他却下决心实现自己的目标。世上没有绝望的处境，只有对处境绝望的人。

6、当你手中抓住一件东西不放时，你只能拥有这件东西，如果你肯放手，你就有机会选择别的。人的心若死执自己的观念，不肯放下，那么他的智慧也只能达到某种程度而已。
7、永远不要嘲笑你的教师无知或者单调，因为有一天当你发现你用瞌睡来嘲弄教师实际上很愚蠢时，你在社会上已经碰了很多钉子了。
8、不要幻想自己成为招聘会上的宠儿，当你明白百分之七十的初中生都可以上大学时，你应该明白人们不会去崇拜一个群体的大多数的。
9、励志格言，一个人可以失败许多次，但是只要他没有开始责怪别人，他还不是一个失败者。人生就像弈棋，一步失误，全盘皆输。
10、在必要时候需要弯一弯，转一转，因为太坚强容易折断，我们需要更多的柔软，才能战胜挫折。自以为拥有财富的人，其实是被财富所拥有。
