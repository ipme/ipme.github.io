人生苦短，若虚度年华，则短暂的人生就太长了

发表于 2018-06-15 | 分类于 珍惜时间

1、黄金时代在我们面前而不在我们背后。人生苦短，若虚度年华，则短暂的人生就太长了。只要我们能善用时间，就永远不愁时间不够用。
2、不管饕餮的时间怎样吞噬着一切，我们要在这一息尚存的时候，努力博取我们的声誉，使时间的镰刀不能伤害我们。盛年不再来，一日难再晨，及时当勉励，岁月不待人。
3、不要老叹息过去，它是不再回来的；要明智地改善现在。要以不忧不惧的坚决意志投入扑朔迷离的未来。集腋成裘，聚沙成塔。几秒钟虽然不长，却构成永恒长河中的伟大时代。
4、不要为已消尽之年华叹息，必须正视匆匆溜走的时光。当许多人在一条路上徘徊不前时，他们不得不让开一条大路，让那珍惜时间的人赶到他们的前面去。

5、珍惜时间，敢于浪费哪怕一个钟头时间的人，说明他还不懂得珍惜生命的全部价值。即将来临的一天，比过去的一年更为悠长。荒废时间等于荒废生命。
6、时间，每天得到的都是小时，可是一天的时间给勤勉的人带来智慧与力量，给懒散的人。以此态度求学，则真理可明，以此态度作事，则功业可就。
7、逆水行舟用力撑，一篙松劲退千寻。古云“此日足可惜”，吾辈更应惜秒阴。你热爱生命吗？那么别浪费时间，因为时间是构成生命的材料。

人生天地之间，若白驹过隙，忽然而已

发表于 2018-06-15 | 分类于 珍惜时间

1、一个人越知道时间的价值，越倍觉失时的痛苦呀！逝者如斯夫，不舍昼夜。人生天地之间，若白驹过隙，忽然而已。天可补，海可填，南山可移。日月既往，不可复追。
2、尊重生命尊重他人也尊重自己的生命，是生命进程中的伴随物，也是心理健康的一个条件。懂得生命真谛的人，可以使短促的生命延长。完成工作的方法是爱惜每一分钟。
3、不要以感伤的眼光去看过去，因为过去再也不会回来了，最聪明的办法，就是好好对付你的现在，现在正握在你的手里，你要以堂堂正正的大丈夫气概去迎接如梦如幻的未来。

4、使一个人的有限的生命，更加有效，也即等于延长了人的生命。应当仔细地观察，为的是理解；应当努力地理解，为的是行动。合理安排时间，就等于节约时间。
5、珍惜时间，凡事都要脚踏实地地去工作，不驰于空想，不骛于虚声，惟以求真的态度作踏实的功夫。以此态度求学，则真理可明，以此态度做事，则功业可就。
6、过于求速是做事的最大危险之一。每一点滴的进展都是缓慢而艰巨的，一个人一次只能着手解决一项有限的目标。人寿几何？逝如朝霜。时无重至，华不再阳。

你热爱生命吗？那么别浪费时间！

发表于 2018-06-15 | 分类于 珍惜时间

1、时间就象海绵里的水一样，只要你愿挤，总还是有的。钉子是敲进往的，时间是挤出来的。大豆不挤出油，时间不挤白会溜。善于利用时间的人，永远找得到充裕的时间。
2、用“分”来计算时间的人，比用“时来计算时间的人，时间多五十九倍。时间是由分秒积成的，善于利用零星时间的人，才会做出更大的成绩来。
3、利用寸阴是任何种类的战斗中博得胜利的秘诀。勤勉的人，每周七个全天；懒惰的人，每周七个早晨。起早外出的跛子追不上。辛勤的蜜蜂永远没有时间的悲哀。

4、时间就是生命。时间就像海绵里的水只要愿挤总还是有的。一年之计在于春，一日之计在于晨。在所有的批评家中，最伟大最正确最天才的是时间。
5、世界上最快而又最慢，最长而又最短，最平凡而又最珍贵，最容易被忽视而又最令人后悔的就是时间。人生有两出悲剧：一是万念俱灰，另一是踌躇满志。
6、珍惜时间，必须记住我们学习的时间是有限的。时间有限，不只是由于人生短促，更由于人事纷繁。我们应该力求把我们所有的时间用去做最有益的事情。

把活着的每一天看作生命的最后一天

发表于 2018-06-15 | 分类于 珍惜时间

1、盛年不重来，一日难再晨。及时宜自勉，岁月不待人。一年之计在于春，你看关于学习的名言警句。一日之计在于晨。欢娱不惜时光逝。时间比理性创造出更多的皈依者。
2、“年”教给我们很多“日”不懂的东西。时间是审查一切罪犯的最老练的法官。时间能使隐躲的事物显露，也能使灿烂夺目的东西黯然无光。时间伟大的作者，她能写出未来的结局。
3、与时间抗争者面对的是一个刀枪不进的敌手。时间是最好的医生。时间能缓解极度的悲痛。时间会使钢铁生锈。时间是最伟大公正的裁判。时间能揭露万事。

4、天波易谢，寸暑难留。年难留，时易损。时间是无声的脚步，不会由于我们有很多事情需要处理而稍停片刻。时间是一条金河，莫让它轻轻地在你的指尖溜过。
5、珍惜时间，光阴潮汐不等人。光阴有脚当珍惜，书田无税应勤耕。时间最不偏私，给任何人都是二十四小时；时间也最偏私，给任何人都不是二十四小时。
6、时间待人是同等的，而时间在每个人手里的价值却不同。谁对时间越吝啬，时间对谁就越慷慨。勤奋的人是时间的主人，懒惰的人是时间的奴隶。
