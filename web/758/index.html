回顾最好的昨天，迎接最美好的明天

发表于 2018-06-14 | 分类于 人生哲理

1、累么？累就对了，舒服是留给死人的！付出了不一定有回报，但不付出永远没有回报。泼在你身上的冷水，你应该烧开了泼回去。
2、用平常心来生活，用惭愧心来待人，心来处事，用菩提心契佛心。别荒废了最能拼的年纪，让自己整天无所事事。自己那么差又不努力，还有什么资格唉声叹气。
3、为了自己想要的未来，无论现在有多难熬，都必须得满怀信心的坚持下去。死亡不是失去生命，而是走出了时间。经验是由痛苦中粹取出来的。
4、学会宽恕就是学会顺从自己的心，“恕”字拆开就是“如心”。就算全世界都说我漂亮，但你却说我不漂亮，那么我就是不漂亮。
5、第一个青春是上帝给的；第二个青春是靠自己努力的。每个人都在自己的世界里做着只有自己才能阐释的梦。头脑是日用品，而不是装饰品。

6、有理想在的地方，地狱就是天堂。有希望在的地方，痛苦也成欢乐。每个人都有潜在的能量，只是很容易：被习惯所掩盖，被时间所迷离，被惰性所消磨。
7、只有承担起旅途风雨，才能最终守得住彩虹满天。有时可能别人不在乎你，但你不能不在乎自己。路靠自己走，就算再坎坷，也要自己过。
8、失败并不意味你浪费了时间和生命，失败表明你有理由重新开始。觉得自己做得到和做不到，只在一念之间。
9、经典哲理，跌倒，撞墙，一败涂地，都不用害怕，年轻叫你勇敢。做最好的今天，回顾最好的昨天，迎接最美好的明天。
10、无论你觉得自己多么的不幸，永远有人比你更加不幸。宁愿跑起来被拌倒无数次，也不愿规规矩矩走一辈子。就算跌倒也要豪迈的笑。
