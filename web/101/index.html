相知在急难，独处亦何益

发表于 2018-06-14 | 分类于 友情的短句子

1、友谊永远是一个甜柔的责任，从来不是一种机会。——纪伯伦
2、人与人之间，只有真诚相待，才是真正的朋友。谁要是算计朋友，等于自己欺骗自己。——哈吉·阿布巴卡·伊芒
3、友谊往往是由一种两个人比一个人更容易实现的共同利益结成的，只有在相互满足时这种关系才是纯洁的。——斯特林堡
4、友谊、活跃和青春的歌声会减轻我们的痛苦。——空茨凯维支
5、友谊是慷慨、荣誉的最贤慧的母亲，是感激和仁慈的姐妹，是憎恨和贪婪的死敌；它时时刻刻都准备舍己为人，而且完全出于自愿不用他人恳求。——薄伽丘

6、保持友谊的最好办法就是任何事情也不假手于他，同时也不借钱给他。——保罗
7、相知在急难，独处亦何益。——李白
8、交友投分，切磨箴视。——周兴嗣
9、势力之交，古人羞之。——刘义庆
10、善人同处，则日闻嘉训；恶人从游，则日生邪情。——范晔
